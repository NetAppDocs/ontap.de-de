= ONTAP Befehle zum Aktivieren und Deaktivieren von Storage Failover
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
Die folgenden Befehle dienen zum Aktivieren und Deaktivieren der Storage-Failover-Funktionalität.

|===


| Wenn Sie möchten... | Verwenden Sie diesen Befehl... 


| Takeover aktivieren | `storage failover modify -enabled true -node _nodename_` 


| Takeover deaktivieren | `storage failover modify -enabled false -node _nodename_` 
|===

NOTE: Die Speicherausfallsicherung sollte nur dann deaktiviert werden, wenn dies im Rahmen einer Wartungsmaßnahme erforderlich ist.

.Verwandte Informationen
