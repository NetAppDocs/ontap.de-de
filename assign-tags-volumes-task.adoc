= Tags Volumes zuweisen
:hardbreaks:
:toclevels: 1
:allow-uri-read: 
:toclevels: 1
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
Ab ONTAP 9.14.1 kann mit System Manager Volumes Tags zugewiesen werden, um Objekte als zu einer Kategorie gehörend zu identifizieren, beispielsweise Projekten oder Kostenstellen.

.Über diese Aufgabe
Sie können einem Volume ein Tag zuweisen. Zuerst muss das Tag definiert und hinzugefügt werden. Anschließend kann das Tag auch bearbeitet oder gelöscht werden.

Tags können beim Erstellen eines Volumes hinzugefügt werden oder auch später.

Ein Tag wird definiert, indem ein Schlüssel angegeben und ihm ein Wert im Format "`key:value`" zugeordnet wird. Beispielsweise "`dept:engineering`" oder "`location:san-jose`".

Folgendes sollte beim Erstellen von Tags beachtet werden:

* Schlüssel müssen mindestens ein Zeichen lang sein und dürfen nicht null sein. Werte können null sein.
* Ein Schlüssel kann mit mehreren Werten verknüpft werden, indem die Werte durch ein Komma getrennt werden, zum Beispiel "`location:san-jose,toronto`"
* Tags können für mehrere Ressourcen verwendet werden.
* Die Schlüssel müssen mit einem Kleinbuchstaben beginnen.
* Tags, die Volumes zugewiesen sind, werden gelöscht, wenn das Volume gelöscht wird.
* Tags werden nicht wiederhergestellt, wenn ein Volume aus der Wiederherstellungswarteschlange wiederhergestellt wird.
* Die Tags bleiben erhalten, wenn das Volume verschoben oder geklont wird.
* Tags, die Speicher-VMs in einer Disaster-Recovery-Beziehung zugewiesen sind, werden auf dem Volume am Partnerstandort repliziert.


.Schritte
Um Tags zu verwalten, sind die folgenden Schritte erforderlich:

. Im System Manager auf *Volumes* klicken und dann das Volume auswählen, dem ein Tag hinzugefügt werden soll.
+
Die Tags sind im Abschnitt *Tags* aufgelistet.

. *Tags verwalten* ermöglicht das Ändern vorhandener Tags oder das Hinzufügen neuer Tags.
+
Sie können die Tags hinzufügen, bearbeiten oder löschen.

+
[cols="25,75"]
|===


| Um diese Aktion durchzuführen... | Führen Sie diese Schritte aus... 


 a| 
Tag hinzufügen
 a| 
.. *Tag hinzufügen* auswählen.
.. Einen Schlüssel und dessen Wert oder Werte angeben (mehrere Werte durch Kommas trennen).
.. Auf *Speichern* klicken.




 a| 
Einen Tag bearbeiten
 a| 
.. Den Inhalt in den Feldern *Schlüssel* und *Werte (optional)* anpassen.
.. Auf *Speichern* klicken.




 a| 
Tag löschen
 a| 
.. Klicken Sie image:../media/icon_trash_can_white_bg.gif["Löschsymbol"] neben das Tag, das Sie löschen möchten.


|===
