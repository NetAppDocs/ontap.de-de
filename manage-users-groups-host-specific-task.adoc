= Hostspezifische Benutzer und Gruppen mit ONTAP System Manager verwalten
:allow-uri-read: 


[role="lead"]
Ab ONTAP 9.10.1 ist es möglich, mit System Manager Benutzer und Gruppen zu verwalten, die spezifisch für einen UNIX oder Windows Host sind.

Die folgenden Verfahren können durchgeführt werden:

|===


| Windows | UNIX 


 a| 
* <<Windows-Benutzer und Gruppen anzeigen>>
* <<add-edit-delete-Windows>>
* <<manage-windows-users>>

 a| 
* <<UNIX-Benutzer und Gruppen anzeigen>>
* <<add-edit-delete-UNIX>>
* <<manage-unix-users>>


|===


[[view-windows-users-and-groups]]
== Windows-Benutzer und Gruppen anzeigen

Im System Manager ist eine Liste der Windows-Benutzer und -Gruppen einsehbar.

.Schritte
. Im System Manager auf *Storage > Storage-VMs* klicken.
. Die Storage-VM auswählen und dann die Registerkarte *Einstellungen* wählen.
. Zum Bereich *Host Users and Groups* scrollen.
+
Im Abschnitt *Windows* wird eine Zusammenfassung der Anzahl der Benutzer in jeder Gruppe angezeigt, die der ausgewählten Storage-VM zugeordnet ist.

. Klicken Sie auf image:icon_arrow.gif["Pfeilsymbol"] im Abschnitt *Windows*.
. Klicken Sie auf die Registerkarte *Gruppen* und dann auf image:icon_dropdown_arrow.gif["Dropdown-Symbol"] neben einem Gruppennamen, um Details zu dieser Gruppe anzuzeigen.
. Die Benutzer einer Gruppe werden angezeigt, wenn die Gruppe ausgewählt und anschließend auf die Registerkarte *Benutzer* geklickt wird.




[[add-edit-or-delete-a-windows-group]]
[[add-edit-delete-Windows]]
== Eine Windows-Gruppe hinzufügen, bearbeiten oder löschen

Im System Manager können Windows-Gruppen durch Hinzufügen, Bearbeiten oder Löschen verwaltet werden.

.Schritte
. Im System Manager wird die Liste der Windows-Gruppen angezeigt. Weitere Informationen unter <<Windows-Benutzer und Gruppen anzeigen>>.
. Auf der Registerkarte *Gruppen* lassen sich Gruppen mit den folgenden Aufgaben verwalten:


|===


| Um diese Aktion durchzuführen... | Führen Sie diese Schritte aus... 


 a| 
Gruppe hinzufügen
 a| 
. Klicken Sie image:icon_add.gif["Symbol hinzufügen"].
. Geben Sie die Gruppeninformationen ein.
. Berechtigungen angeben.
. Gruppenmitglieder angeben (lokale Benutzer, Domänenbenutzer oder Domänengruppen hinzufügen).




 a| 
Eine Gruppe bearbeiten
 a| 
. Neben dem Gruppennamen auf image:icon_kabob.gif["Symbol für Menüoptionen"] klicken und anschließend auf *Bearbeiten* klicken.
. Die Gruppeninformationen werden geändert.




 a| 
Gruppe löschen
 a| 
. Aktivieren Sie das Kontrollkästchen neben der Gruppe oder den Gruppen, die gelöscht werden sollen.
. Klicken Sie image:icon_delete_with_can_white_bg.gif["Löschsymbol"].
+
*Hinweis:* Sie können auch eine einzelne Gruppe löschen, indem Sie auf image:icon_kabob.gif["Symbol für Menüoptionen"] neben dem Gruppennamen klicken und dann auf *Löschen* klicken.



|===


[[manage-windows-users]]
== Windows-Benutzerverwaltung

Im System Manager können Sie Windows-Benutzer verwalten, indem Sie sie hinzufügen, bearbeiten, löschen, aktivieren oder deaktivieren. Sie können auch das Kennwort eines Windows-Benutzers ändern.

.Schritte
. Im System Manager wird die Liste der Benutzer für die Gruppe angezeigt. Weitere Informationen unter <<Windows-Benutzer und Gruppen anzeigen>>.
. Auf der Registerkarte *Benutzer* lassen sich Benutzer mit den folgenden Aufgaben verwalten:


|===


| Um diese Aktion durchzuführen... | Führen Sie diese Schritte aus... 


 a| 
Einen Benutzer hinzufügen
 a| 
. Klicken Sie image:icon_add.gif["Symbol hinzufügen"].
. Benutzerinformationen eingeben.




 a| 
Einen Benutzer bearbeiten
 a| 
. Neben dem Benutzernamen auf image:icon_kabob.gif["Symbol für Menüoptionen"] klicken, dann auf *Bearbeiten* klicken.
. Die Benutzerinformationen werden geändert.




 a| 
Einen Benutzer löschen
 a| 
. Aktivieren Sie das Kontrollkästchen neben dem oder den Benutzern, die gelöscht werden sollen.
. Klicken Sie image:icon_delete_with_can_white_bg.gif["Löschsymbol"].
+
*Hinweis:* Ein einzelner Benutzer kann auch gelöscht werden, indem Sie auf image:icon_kabob.gif["Symbol für Menüoptionen"] neben dem Benutzernamen und anschließend auf *Löschen* klicken.





 a| 
Benutzerpasswort ändern
 a| 
. Neben dem Benutzernamen auf image:icon_kabob.gif["Symbol für Menüoptionen"] klicken, dann *Passwort ändern* auswählen.
. Das neue Passwort eingeben und bestätigen.




 a| 
Einen Benutzer aktivieren
 a| 
. Aktivieren Sie das Kontrollkästchen neben jedem deaktivierten Benutzer, den Sie aktivieren möchten.
. Klicken Sie image:icon-enable-with-symbol.gif["Symbol aktivieren"].




 a| 
Einen Benutzer deaktivieren
 a| 
. Aktivierte Benutzer, die deaktiviert werden sollen, werden durch Ankreuzen des Kästchens neben dem jeweiligen Namen ausgewählt.
. Klicken Sie image:icon-disable-with-symbol.gif["Symbol deaktivieren"].


|===

== Eine UNIX Gruppe hinzufügen, bearbeiten oder löschen

Im System Manager ist die Verwaltung von UNIX-Gruppen durch Hinzufügen, Bearbeiten oder Löschen möglich.

.Schritte
. Im System Manager wird die Liste der UNIX-Gruppen angezeigt. Weitere Informationen unter <<UNIX-Benutzer und Gruppen anzeigen>>.
. Auf der Registerkarte *Gruppen* lassen sich Gruppen mit den folgenden Aufgaben verwalten:


|===


| Um diese Aktion durchzuführen... | Führen Sie diese Schritte aus... 


 a| 
Gruppe hinzufügen
 a| 
. Klicken Sie image:icon_add.gif["Symbol hinzufügen"].
. Geben Sie die Gruppeninformationen ein.
. (Optional) Zugehörige Benutzer angeben.




 a| 
Eine Gruppe bearbeiten
 a| 
. Die Gruppe auswählen.
. Klicken Sie image:icon_edit.gif["Bearbeiten-Symbol"].
. Die Gruppeninformationen werden geändert.
. (Optional) Benutzer hinzufügen oder entfernen.




 a| 
Gruppe löschen
 a| 
. Die Gruppe oder die Gruppen, die gelöscht werden sollen, auswählen.
. Klicken Sie image:icon_delete_with_can_white_bg.gif["Löschsymbol"].
