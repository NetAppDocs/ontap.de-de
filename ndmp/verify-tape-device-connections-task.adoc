= Verbindungen von ONTAP NDMP Tape-Geräten überprüfen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Es ist sicherzustellen, dass alle Laufwerke und Medienwechsler in ONTAP als Geräte sichtbar sind.

.Schritte
. Informationen zu allen Laufwerken und Medienwechslern lassen sich mit dem  `storage tape show` Befehl anzeigen.
+
[listing]
----
cluster1::> storage tape show

Node: cluster1-01
Device ID               Device Type     Description                     Status
----------------------  --------------  ------------------------------  --------
sw4:10.11               tape drive      HP LTO-3                        normal
0b.125L1                media changer   HP MSL G3 Series                normal
0d.4                    tape drive      IBM LTO 5 ULT3580               normal
0d.4L1                  media changer   IBM 3573-TL                     normal
...
----
. Wenn ein Tape-Laufwerk nicht angezeigt wird, sollte das Problem überprüft werden.
. Falls kein Medienwechsler angezeigt wird, können Informationen über Medienwechsler mit dem  `storage tape show-media-changer` Befehl angezeigt und das Problem anschließend analysiert werden.
+
[listing]
----
cluster1::> storage tape show-media-changer

Media Changer: sw4:10.11L1
  Description: PX70-TL
         WWNN: 2:00a:000e11:10b919
         WWPN: 2:00b:000e11:10b919
Serial Number: 00FRU7800000_LL1

       Errors: -

Paths:
Node                      Initiator  Alias    Device State              Status
------------------------  ---------  -------  ------------------------  --------
cluster1-01               2b         mc0      in-use                    normal
...
----
