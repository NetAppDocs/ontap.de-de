= Zusammenfassung des Installationsablaufs von ONTAP Mediator
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Die Installation von ONTAP Mediator umfasst die Vorbereitung der Installation, die Bereitstellung des Zugriffs auf die Repositories, das Herunterladen des Installationspakets, die Überprüfung der Codesignatur, die Installation des ONTAP Mediator Pakets und die Durchführung von Konfigurationsaufgaben nach der Installation.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-1.png["Eins"]link:index.html["Vorbereitung der Installation oder des Upgrades von ONTAP Mediator"]
[role="quick-margin-para"]
Um ONTAP Mediator zu installieren oder zu aktualisieren, müssen Sie sicherstellen, dass alle Voraussetzungen erfüllt sind.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-2.png["Zwei"]link:enable-access-repos-task.html["Repository-Zugriff bereitstellen"]
[role="quick-margin-para"]
Der Zugriff auf die Repositories sollte aktiviert werden, damit ONTAP Mediator während des Installationsprozesses auf die benötigten Pakete zugreifen kann.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-3.png["Drei"]link:download-install-pkg-task.html["Das Installationspaket für ONTAP Mediator herunterladen"]
[role="quick-margin-para"]
Das ONTAP Mediator Installationspaket steht auf der ONTAP Mediator Downloadseite zum Herunterladen bereit.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-4.png["Vier"]link:verify-code-signature-task.html["Die Codesignatur des ONTAP Mediator Installationspakets überprüfen"]
[role="quick-margin-para"]
NetApp empfiehlt, die ONTAP Mediator-Code-Signatur vor der Installation des ONTAP Mediator Installationspakets zu überprüfen.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-5.png["Fünf"]link:install-mediator-pkg-task.html["Das ONTAP Mediator Installationspaket installieren"]
[role="quick-margin-para"]
Um ONTAP Mediator zu installieren oder zu aktualisieren, müssen Sie das Installationspaket herunterladen und das Installationsprogramm auf dem Host ausführen.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-6.png["Sechs"]link:verify-install-task.html["Die Installation des ONTAP Mediators überprüfen"]
[role="quick-margin-para"]
Nach der Installation von ONTAP Mediator sollte überprüft werden, ob dieser erfolgreich ausgeführt wird.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-7.png["Sieben"]link:upgrade-host-os-mediator-task.html["Optional können Sie das Host-Betriebssystem aktualisieren und ONTAP Mediator neu installieren oder aktualisieren."]
[role="quick-margin-para"]
Wenn auf Ihrer ONTAP Mediator-Version SCST installiert ist und Sie Ihr Host-Betriebssystem aktualisieren müssen, deinstallieren und installieren Sie ONTAP Mediator im Rahmen der Aktualisierung neu. Sie können die aktuelle Version von ONTAP Mediator oder eine neue Version installieren, sofern eine verfügbar ist. Sie können diesen Schritt überspringen, wenn Sie Ihr Betriebssystem nicht aktualisieren müssen oder wenn das SCST-Paket nicht installiert ist, weil Sie ONTAP Mediator mit Unterstützung für HTTPS und nicht für iSCSI installiert haben.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-8.png["Acht"]link:post-install-config-concept.html["Konfigurationsaufgaben nach der Installation ausführen"]
[role="quick-margin-para"]
Nachdem ONTAP Mediator installiert und in Betrieb genommen wurde, sind zusätzliche Konfigurationsaufgaben erforderlich, um die Funktionen von ONTAP Mediator zu nutzen.
