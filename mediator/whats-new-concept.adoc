:allow-uri-read:

[role="lead"]
Mit jeder neuen Version von ONTAP Mediator werden neue Verbesserungen bereitgestellt. Neuerungen im Überblick.



[[enhancements]]
== Verbesserungen

Informationen zur SCST-Version sind unter <<SCST Support-Matrix>> zu finden.

[cols="25,75"]

1.12.1
 a| 
* Unterstützung für RHEL:
+
** Kompatibel: 10.1 und 9.7.
** Empfohlen: 10.2, 10.0, 9.8, 9.6, 9.4 und 8.10.


* Unterstützung für Rocky Linux 10.2, 9.8 und 8.10.
* Unterstützung für Oracle Linux 10.1 und 9.8 mit sowohl rhck- als auch uek-Kerneln.
* Ab ONTAP Mediator 1.12.1 und ONTAP 9.19.1 kann eine einzelne, mit HTTPS konfigurierte ONTAP Mediator Instanz die folgenden Konfigurationen gleichzeitig überwachen:
+
** Maximal zehn Vier-Knoten- oder fünf Acht-Knoten MetroCluster IP-Konfigurationen bei Installation auf einem Host mit 8-Kern-CPU und mindestens 16GB RAM. Siehe link:https://docs.netapp.com/us-en/ontap-metrocluster/install-ip/concept_mediator_requirements.html#mediator-multiple-mccip["ONTAP Mediator zur Überwachung mehrerer MetroCluster IP Konfigurationen verwenden"^].
** Eine MetroCluster IP Konfiguration mit vier Knoten und eine SnapMirror active sync Peer-Beziehung, wenn sie auf einem Host mit einer 2-Kern-CPU installiert ist. Siehe link:../snapmirror-active-sync/mediator-install-task.html#ontap-mediator["Den ONTAP Mediator für SnapMirror Active Sync konfigurieren"].


* Enthält ein neues Tool zur Reparatur einer beschädigten SCST-Installation. Nach der Installation von ONTAP Mediator befindet sich das Tool unter  `/usr/local/bin/mediator_repair_scst`. Das Tool kann nach einem Betriebssystem-Upgrade verwendet werden, wenn das aktualisierte Betriebssystem einen SCST-Patch benötigt, der im Standard-SCST-Paket nicht verfügbar ist.

HTTPS bietet erhöhte Sicherheit durch die Verschlüsselung ganzer Sitzungen mit SSL/TLS. Sie können wählen, ONTAP Mediator zu installieren, der ausschließlich das HTTPS-Protokoll verwendet. Das SCST-Paket wird bei dieser Installationsmethode nicht installiert. link:index.html#https-iscsi-installation["Erfahren Sie mehr über die HTTPS- und iSCSI-Installationsoptionen für ONTAP Mediator"].

* „Ja“ bedeutet, dass das Betriebssystem für die Installation von ONTAP Mediator empfohlen wird und vollständig kompatibel sowie unterstützt ist.
* „Nein“ bedeutet, dass das Betriebssystem und der ONTAP Mediator nicht kompatibel sind.
* „Kompatibel“ bedeutet, dass Red Hat diese RHEL-Versionen nicht mehr unterstützt, ONTAP Mediator aber weiterhin darauf installiert werden kann.
* ONTAP Mediator 1.6 bietet Unterstützung für Rocky Linux 9 und 8.
* ONTAP Mediator 1.5 war die letzte unterstützte Version für Betriebssysteme der RHEL 7.x Reihe.
* CentOS 8 wurde aufgrund der Neustrukturierung für alle Releases entfernt. CentOS Stream wurde als ungeeignetes Zielbetriebssystem für den Produktiveinsatz eingestuft. Kein Support ist geplant.
