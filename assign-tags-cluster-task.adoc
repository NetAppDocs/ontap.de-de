= Tags einem Cluster mit ONTAP System Manager zuweisen
:hardbreaks:
:toclevels: 1
:allow-uri-read: 
:toclevels: 1
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
Ab ONTAP 9.14.1 kann mit dem System Manager einem Cluster ein Tag zugewiesen werden, um Objekte als zu einer Kategorie wie Projekten oder Kostenstellen zugehörig zu kennzeichnen.

.Über diese Aufgabe
Sie können einem Cluster ein Tag zuweisen. Zuerst muss das Tag definiert und hinzugefügt werden. Anschließend kann das Tag auch bearbeitet oder gelöscht werden.

Tags können beim Erstellen eines Clusters oder auch später hinzugefügt werden.

Ein Tag wird definiert, indem ein Schlüssel angegeben und ihm ein Wert im Format "`key:value`" zugeordnet wird. Beispielsweise "`dept:engineering`" oder "`location:san-jose`".

Folgendes sollte beim Erstellen von Tags beachtet werden:

* Schlüssel müssen mindestens ein Zeichen lang sein und dürfen nicht null sein. Werte können null sein.
* Ein Schlüssel kann mit mehreren Werten verknüpft werden, indem die Werte durch ein Komma getrennt werden, zum Beispiel "`location:san-jose,toronto`"
* Tags können für mehrere Ressourcen verwendet werden.
* Die Schlüssel müssen mit einem Kleinbuchstaben beginnen.


.Schritte
Zur Verwaltung von Tags sind die folgenden Schritte erforderlich:

. Im System Manager auf *Cluster* klicken, um die Übersichtsseite anzuzeigen.
+
Die Tags sind im Abschnitt *Tags* aufgelistet.

. *Tags verwalten* ermöglicht das Ändern vorhandener Tags oder das Hinzufügen neuer Tags.
+
Sie können die Tags hinzufügen, bearbeiten oder löschen.

+
[cols="25,75"]
|===


| Um diese Aktion durchzuführen... | Führen Sie diese Schritte aus... 


 a| 
Tag hinzufügen
 a| 
.. *Tag hinzufügen* auswählen.
.. Einen Schlüssel und dessen Wert oder Werte angeben (mehrere Werte durch Kommas trennen).
.. Auf *Speichern* klicken.




 a| 
Einen Tag bearbeiten
 a| 
.. Den Inhalt in den Feldern *Schlüssel* und *Werte (optional)* anpassen.
.. Auf *Speichern* klicken.




 a| 
Tag löschen
 a| 
.. Klicken Sie image:../media/icon_trash_can_white_bg.gif["Löschsymbol"] neben das Tag, das Sie löschen möchten.


|===
