= Quoten zur Begrenzung der Ressourcennutzung mit ONTAP System Manager festlegen
:hardbreaks:
:toclevels: 1
:allow-uri-read: 
:toclevels: 1
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
Quoten können hinzugefügt werden, um die Menge an Speicherplatz zu begrenzen, die das Quotenziel verwenden kann.

Es kann ein endgültiges Limit und ein flexibles Limit für ein Kontingent festgelegt werden.

Feste Quoten setzen ein endgültiges Limit für die Systemressourcen, jede Operation, die dieses Limit überschreiten würde, schlägt fehl. Weiche Kontingente senden eine Warnmeldung, wenn die Ressourcennutzung einen bestimmten Wert erreicht, sie beeinträchtigen jedoch nicht den Dateizugriff, sodass entsprechende Maßnahmen ergriffen werden können, bevor das Kontingent überschritten wird.

.Schritte
. *Speicher > Kontingente* auswählen.
. Klicken Sie auf *Add*.
