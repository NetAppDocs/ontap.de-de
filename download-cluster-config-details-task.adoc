= Details einer Clusterkonfiguration herunterladen
:hardbreaks:
:toclevels: 1
:allow-uri-read: 
:toclevels: 1
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
Ab ONTAP 9.11.1 ist das Herunterladen der Details der Clusterkonfiguration möglich.

.Schritte
. Im System Manager zu *Cluster > Übersicht* wechseln.
. Klicken Sie auf image:icon-more-kebab-blue-bg.gif["Symbol „Mehr Optionen“"], um eine Liste der Aktionen anzuzeigen.
. *Konfiguration herunterladen* auswählen
. Die HA-Paare auswählen.
. Klicken Sie auf *Download*.
