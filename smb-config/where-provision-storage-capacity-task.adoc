= Informationen zur Bereitstellung von Speicherkapazität für ONTAP SMB
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Bevor Sie ein neues SMB Volume oder einen neuen Qtree erstellen, müssen Sie entscheiden, ob Sie es in einer neuen oder einer bestehenden SVM platzieren und wie umfangreich die Konfiguration der SVM sein soll. Diese Entscheidung bestimmt den Workflow.

.Auswahlmöglichkeiten
* Wenn ein Volume oder ein Qtree auf einer neuen SVM oder auf einer vorhandenen SVM bereitgestellt werden soll, die SMB aktiviert, aber nicht konfiguriert hat, sind die Schritte in „`Konfigurieren des SMB-Zugriffs auf eine SVM`“ und „`Hinzufügen von Speicherkapazität zu einer SMB-fähigen SVM`“ abzuschließen.
+
xref:configure-access-svm-task.adoc[Konfiguration des SMB-Zugriffs auf eine SVM]

+
xref:configure-client-access-shared-storage-concept.adoc[Konfigurieren des SMB Clientzugriffs auf gemeinsam genutzten Speicher]

+
Sie könnten sich dafür entscheiden, eine neue SVM zu erstellen, wenn eine der folgenden Bedingungen zutrifft:

+
** Sie aktivieren SMB zum ersten Mal auf einem Cluster.
** Sie haben bereits SVMs in einem Cluster, in dem Sie die SMB-Unterstützung nicht aktivieren möchten.
** Sie haben eine oder mehrere SMB-fähige SVMs in einem Cluster und möchten eine der folgenden Verbindungen herstellen:
+
*** Zu einer anderen Active Directory Gesamtstruktur oder Arbeitsgruppe.
*** Für einen SMB-Server in einem isolierten Namespace (Mandantenfähigkeit-Szenario). Diese Option ist auch zu wählen, um Speicher auf einer vorhandenen SVM bereitzustellen, auf der SMB aktiviert, aber nicht konfiguriert ist. Dies kann der Fall sein, wenn die SVM für SAN-Zugriff erstellt wurde oder wenn bei der Erstellung der SVM keine Protokolle aktiviert waren.




+
Nachdem SMB auf der SVM aktiviert wurde, kann ein Volume oder Qtree bereitgestellt werden.

* Wenn ein Volume oder ein Qtree auf einer vorhandenen SVM bereitgestellt werden soll, die vollständig für den SMB-Zugriff konfiguriert ist, sind die Schritte in „Hinzufügen von Speicherkapazität zu einer SMB-fähigen SVM“ abzuschließen.
+
xref:configure-client-access-shared-storage-concept.adoc[Konfigurieren des SMB Clientzugriffs auf gemeinsam genutzten Speicher]
