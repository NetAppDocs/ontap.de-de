= ONTAP Zeitdienste für SMB Server konfigurieren
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Bevor ein SMB-Server in einem Active Domain Controller erstellt wird, muss sichergestellt werden, dass die Clusterzeit und die Zeit auf den Domänencontrollern der Domäne, zu der der SMB-Server gehören wird, innerhalb von fünf Minuten übereinstimmen.

.Über diese Aufgabe
Die Cluster-NTP-Dienste sind so zu konfigurieren, dass für die Synchronisierung dieselben NTP-Server wie von der Active Directory-Domäne verwendet werden.

Ab ONTAP 9.5 ist die Einrichtung Ihres NTP-Servers mit symmetrischer Authentifizierung möglich.

.Schritte
. Zeitdienste werden mithilfe des  `cluster time-service ntp server create`Befehls konfiguriert.
+
** Um Zeitdienste ohne symmetrische Authentifizierung zu konfigurieren, ist folgender Befehl einzugeben: `cluster time-service ntp server create -server server_ip_address`
** Um Zeitdienste mit symmetrischer Authentifizierung zu konfigurieren, den folgenden Befehl eingeben: `cluster time-service ntp server create -server server_ip_address -key-id key_id`
`cluster time-service ntp server create -server 10.10.10.1` `cluster time-service ntp server create -server 10.10.10.2`


. Es kann überprüft werden, ob die Zeitdienste korrekt eingerichtet sind, indem der  `cluster time-service ntp server show` Befehl verwendet wird.
+
`cluster time-service ntp server show`

+
[listing]
----

Server                         Version
------------------------------ -------
10.10.10.1                     auto
10.10.10.2                     auto
----
