= ONTAP SMB Konfigurationsworkflow
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Die Konfiguration von SMB umfasst die Beurteilung der physischen Speicher- und Netzwerkanforderungen sowie die Auswahl eines Workflows, der auf Ihr Ziel zugeschnitten ist: die Konfiguration des SMB-Zugriffs auf eine neue oder bestehende SVM oder das Hinzufügen eines Volumes oder Qtree zu einer bestehenden SVM, die bereits vollständig für den SMB-Zugriff konfiguriert ist.
