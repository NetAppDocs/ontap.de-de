= Informationen zur SMB-Serverkonfiguration in ONTAP Arbeitsgruppen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Die Einrichtung eines SMB-Servers als Mitglied einer Arbeitsgruppe besteht aus der Erstellung des SMB-Servers und anschließend der Erstellung lokaler Benutzer und Gruppen.

Sie können einen SMB-Server in einer Arbeitsgruppe konfigurieren, wenn die Microsoft Active Directory-Domäneninfrastruktur nicht verfügbar ist.

Ein SMB-Server im Arbeitsgruppenmodus unterstützt nur die NTLM-Authentifizierung und unterstützt keine Kerberos-Authentifizierung.
