[[fcoe-initiators]]
== FCoE Initiatoren

FCoE-Initiatoren können in Host-Computern mit sowohl FCoE- als auch herkömmlichen FC-Targets in Speicherkontrollern verwendet werden. Der Host-FCoE-Initiator muss mit einem FCoE-DCB-Switch (Data Center Bridging) verbunden sein, eine direkte Verbindung zu einem Target wird nicht unterstützt.

[[fcoe-targets]]
== FCoE Ziele

Sie können FCoE-Zielports mit 4-Gb, 8-Gb oder 16-Gb FC-Ports auf dem Storage Controller kombinieren, unabhängig davon, ob die FC-Ports Add-in-Zieladapter oder Onboard-Ports sind. Sowohl FCoE- als auch FC-Zieladapter können im selben Storage Controller vorhanden sein.

[NOTE]
====
