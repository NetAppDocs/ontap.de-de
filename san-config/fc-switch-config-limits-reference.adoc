= Konfigurationsgrenzen für FC-Switches, die mit ONTAP Systemen verwendet werden
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Fibre Channel Switches haben maximale Konfigurationsgrenzen, einschließlich der Anzahl der unterstützten Anmeldungen pro Port, Portgruppe, Blade und Switch. Die Switch-Hersteller dokumentieren ihre unterstützten Grenzwerte.

Jede FC logische Schnittstelle (LIF) meldet sich an einem FC-Switch-Port an. Die Gesamtzahl der Anmeldungen von einem einzelnen Target auf dem Node entspricht der Anzahl der LIFs plus einer Anmeldung für den zugrunde liegenden physischen Port. Die Konfigurationsgrenzen des Switch-Herstellers für Anmeldungen oder andere Konfigurationswerte dürfen nicht überschritten werden. Dies gilt auch für die auf der Host-Seite verwendeten Initiatoren in virtualisierten Umgebungen mit aktiviertem NPIV. Die Konfigurationsgrenzen des Switch-Herstellers für Anmeldungen dürfen weder für das Target noch für die in der Lösung verwendeten Initiatoren überschritten werden.
