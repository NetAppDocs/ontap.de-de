= Empfohlene FC-Zielportkonfiguration und Geschwindigkeiten für ONTAP Systeme
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
FC-Zielports können für das FC-NVMe-Protokoll genauso konfiguriert und verwendet werden wie für das FC-Protokoll. Die Unterstützung für das FC-NVMe-Protokoll variiert je nach Plattform und ONTAP Version. Mit NetApp Hardware Universe lässt sich die Unterstützung überprüfen.

Für optimale Leistung und höchste Verfügbarkeit wird die empfohlene Zielportkonfiguration gemäß  https://hwu.netapp.com["NetApp Hardware Universe"^] für Ihre spezifische Plattform empfohlen.

[[fc-target-port-supported-speeds]]
== Unterstützte Geschwindigkeiten des FC-Zielports

FC-Zielports können mit unterschiedlichen Geschwindigkeiten konfiguriert werden. Alle von einem Host verwendeten Zielports sollten auf dieselbe Geschwindigkeit eingestellt sein. Die Geschwindigkeit des Zielports sollte der Geschwindigkeit des Geräts entsprechen, mit dem er verbunden ist. Die automatische Aushandlung der Portgeschwindigkeit sollte nicht verwendet werden. Ein Port, der auf automatische Aushandlung eingestellt ist, kann nach einer Übernahme/Rückgabe oder einer anderen Unterbrechung länger für die Wiederherstellung der Verbindung benötigen.

Sie können die Onboard-Ports und Erweiterungsadapter so konfigurieren, dass sie mit den folgenden Geschwindigkeiten arbeiten. Jeder Controller- und Erweiterungsadapter-Port kann individuell für unterschiedliche Geschwindigkeiten konfiguriert werden.
