= Maximale Anzahl an unterstützten FC- und FCoE-Hops in ONTAP
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Die Hop-Anzahl bezeichnet die Anzahl der Switches auf dem Pfad zwischen Initiator (Host) und Ziel (Speichersystem). Die maximal unterstützte FC-Hop-Anzahl zwischen Host und Speichersystem variiert je nach Switch-Lieferant.

In der Dokumentation von Cisco Systems wird dieser Wert auch als _Durchmesser des SAN-Fabrics_ bezeichnet.

Bei FCoE können FCoE Switches mit FC Switches verbunden werden. Für durchgängige FCoE-Verbindungen muss auf den FCoE Switches eine Firmware-Version ausgeführt werden, die Ethernet Inter-Switch-Links (ISLs) unterstützt.

[cols="2*"]
|===
| Switch-Lieferant | Unterstützte Hop-Anzahl 


 a| 
Brocade
 a| 
* 7 für FC
* 5 für FCoE




 a| 
Cisco
 a| 
* 7 für FC
* Bis zu 3 der Switches können FCoE Switches sein.


|===
