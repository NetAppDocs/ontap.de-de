= ONTAP Lizenzen verwalten
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Mit System Manager oder der ONTAP CLI lassen sich die auf Ihrem System installierten Lizenzen anzeigen und verwalten, einschließlich der Anzeige der Lizenz-Seriennummer, der Überprüfung des Lizenzstatus und der Entfernung einer Lizenz.



[[view-details-about-a-license]]
== Details zu einer Lizenz

.Schritte
Wie Details zu einer Lizenz angezeigt werden, hängt davon ab, welche Version von ONTAP verwendet wird und ob der System Manager oder die ONTAP CLI genutzt wird.

[role="tabbed-block"]
====
.System Manager – ONTAP 9.8 und höher
--
. Details zu einer bestimmten Funktionslizenz sind unter *Cluster > Einstellungen* einsehbar.
. Unter *Lizenzen* image:icon_arrow.gif["Pfeilsymbol"] auswählen.
. *Funktionen* auswählen.
. Suchen Sie die lizenzierte Funktion, die angezeigt werden soll, und wählen Sie image:icon_dropdown_arrow.gif["Dropdown-Symbol"] aus, um die Lizenzdetails anzuzeigen.


--
.System Manager – ONTAP 9.7 und früher
--
. *Konfiguration > Cluster > Lizenzen* auswählen.
. Im Fenster *Lizenzen* steht die entsprechende Aktion zur Auswahl:
. Die Registerkarte *Details* auswählen.


--
.CLI
--
. Details zu einer installierten Lizenz:
+
[source, cli]
----
system license show
----


--
====


[[delete-a-license]]
== Lizenz löschen

[role="tabbed-block"]
====
.System Manager – ONTAP 9.8 und höher
--
. Zum Löschen einer Lizenz *Cluster > Einstellungen* auswählen.
. Unter *Lizenzen* image:icon_arrow.gif["Pfeilsymbol"] auswählen.
. *Funktionen* auswählen.
. Wählen Sie die lizenzierte Funktion aus, die Sie löschen möchten, und *Legacy-Schlüssel löschen*.


--
.System Manager – ONTAP 9.7 und früher
--
. *Konfiguration > Cluster > Lizenzen* auswählen.
. Im Fenster *Lizenzen* steht die entsprechende Aktion zur Auswahl:
+
|===
| Wenn Sie möchten... | Folgendes ist zu tun... 


 a| 
Ein bestimmtes Lizenzpaket auf einem Knoten oder eine clusterweite Lizenz löschen
 a| 
Die Registerkarte *Details* auswählen.



 a| 
Ein bestimmtes Lizenzpaket auf allen Knoten im Cluster löschen
 a| 
Auf die Registerkarte *Pakete* wechseln.

|===
. Wählen Sie das Softwarelizenzpaket aus, das Sie löschen möchten, und klicken Sie dann auf *Löschen*.
+
Sie können jeweils nur ein Lizenzpaket löschen.

. Das Bestätigungs Kästchen auswählen und anschließend auf *Löschen* klicken.


--
.CLI
--
. Lizenz löschen:
+
[source, cli]
----
system license delete
----
+
Das folgende Beispiel löscht eine Lizenz mit dem Namen CIFS und der Seriennummer 1-81-0000000000000000000123456 aus dem Cluster:

+
[listing]
----
cluster1::> system license delete -serial-number 1-81-0000000000000000000123456 -package CIFS
----
+
Das folgende Beispiel löscht aus dem Cluster alle Lizenzen unter dem installierten Lizenzpaket Core Bundle für die Seriennummer 123456789:

+
[listing]
----
cluster1::> system license delete { -serial-number 123456789 -installed-license "Core Bundle" }
----


--
