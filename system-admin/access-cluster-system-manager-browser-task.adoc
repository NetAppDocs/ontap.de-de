= Mit System Manager auf einen ONTAP Cluster zugreifen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Wenn Sie für den Zugriff auf und die Verwaltung eines Clusters lieber eine grafische Benutzeroberfläche anstelle der Befehlszeilenschnittstelle (CLI) verwenden, ist dies mit System Manager möglich, das als Webdienst in ONTAP enthalten, standardmäßig aktiviert und über einen Browser zugänglich ist.

[NOTE]
====
Ab ONTAP 9.12.1 ist der System Manager vollständig in die NetApp Console integriert.

Mit der Console ist die Verwaltung Ihrer hybriden Multicloud-Infrastruktur über eine einzige Steuerungsebene möglich, wobei das vertraute System Manager Dashboard erhalten bleibt.

Siehe link:../concepts/sysmgr-integration-console-concept.html["System Manager Integration mit NetApp Console"].

====
.Über diese Aufgabe
Sie können eine Cluster-Management-Netzwerkschnittstelle (LIF) oder eine Node-Management-Netzwerkschnittstelle (LIF) verwenden, um auf System Manager zuzugreifen. Für unterbrechungsfreien Zugriff auf System Manager sollte eine Cluster-Management-Netzwerkschnittstelle (LIF) verwendet werden.

.Bevor Sie beginnen
* Sie benötigen ein Cluster-Benutzerkonto, das mit der Rolle "`admin`" und den Anwendungstypen "`http`" und "`console`" konfiguriert ist.
* Cookies und Websitedaten müssen im Browser aktiviert sein.


.Schritte
. Der Webbrowser wird auf die IP-Adresse der Netzwerkschnittstelle für das Clustermanagement ausgerichtet:
+
** Wenn Sie IPv4 verwenden: `*https://__cluster-mgmt-LIF__*`
** Wenn Sie IPv6 verwenden: `*https://[_cluster-mgmt-LIF_]*`
+

NOTE: Für den Browserzugriff auf System Manager wird ausschließlich HTTPS unterstützt.



+
Wenn der Cluster ein selbstsigniertes digitales Zertifikat verwendet, zeigt der Browser möglicherweise eine Warnung an, dass das Zertifikat nicht vertrauenswürdig ist. Es besteht die Möglichkeit, entweder das Risiko zu akzeptieren, um den Zugriff fortzusetzen, oder ein von einer Zertifizierungsstelle (CA) signiertes digitales Zertifikat auf dem Cluster für die Server-Authentifizierung zu installieren.

. *Optional:* Wenn ein Zugriffsbanner über die CLI konfiguriert wurde, erscheint eine Meldung im *Warnungs*-Dialogfeld, und die gewünschte Option kann ausgewählt werden, um fortzufahren.
+
Diese Option wird auf Systemen, auf denen die Security Assertion Markup Language (SAML) Authentifizierung aktiviert ist, nicht unterstützt.

+
** Wenn Sie nicht fortfahren möchten, auf *Abbrechen* klicken und den Browser schließen.
** Wenn Sie fortfahren möchten, klicken Sie auf *OK*, um zur System Manager-Anmeldeseite zu gelangen.


. Melden Sie sich mit Ihren Cluster-Administrator-Anmeldeinformationen beim System Manager an.
+

NOTE: Ab ONTAP 9.11.1 kann beim Anmelden am System Manager das Gebietsschema festgelegt werden. Das Gebietsschema legt bestimmte Lokalisierungseinstellungen fest, wie Sprache, Währung, Zeit- und Datumsformat sowie ähnliche Einstellungen. Für ONTAP 9.10.1 und frühere Versionen wird das Gebietsschema für den System Manager vom Browser erkannt. Um das Gebietsschema für den System Manager zu ändern, muss das Gebietsschema des Browsers geändert werden.

. *Optional*: Ab ONTAP 9.12.1 kann eine Präferenz für das Erscheinungsbild von System Manager angegeben werden:
+
.. In der oberen rechten Ecke des System Manager auf image:icon-user-blue-bg.png["Benutzerkontosymbol"] klicken, um die Benutzeroptionen zu verwalten.
.. Den Schalter für das *Systemdesign* entsprechend der eigenen Präferenz positionieren:
+
|===


| Position umschalten | Einstellung für das Erscheinungsbild 


 a| 
image:icon-light-theme-sun.png["Symbol für helles Design"] (links)
 a| 
Helles Design (heller Hintergrund mit dunklem Text)



 a| 
OS (Mitte)
 a| 
Standardmäßig wird die Designeinstellung verwendet, die für die Anwendungen des Betriebssystems festgelegt wurde (in der Regel die Designeinstellung des Browsers, mit dem auf System Manager zugegriffen wird).



 a| 
image:icon-dark-theme-moon.png["Symbol für dunkles Design"] (Rechts)
 a| 
Dunkles Design (dunkler Hintergrund mit hellem Text)

|===
