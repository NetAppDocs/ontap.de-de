= Befehle zur Verwaltung von Audit-Einstellungen für Managementaktivitäten
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie verwenden die  `security audit`Befehle, um zu verwalten, welche Verwaltungsaktivitäten in der  `audit.log`Datei protokolliert werden. Sie verwenden die  `cluster log-forwarding`Befehle, um Ziele für das Weiterleiten des Revisionsprotokolls zu verwalten.

|===
| Wenn Sie möchten... | Verwenden Sie diesen Befehl... 


 a| 
Geben Sie an, dass GET-Anfragen von der ONTAP CLI oder den APIs zusätzlich zu den standardmäßigen SET-Anfragen im Revisionsprotokoll (der `audit.log` Datei) aufgezeichnet werden sollen.
 a| 
`security audit modify`



 a| 
Die Einstellungen des Revisionsprotokolls anzeigen
 a| 
`security audit show`



 a| 
Anzeige der aus mehreren Knoten im Cluster zusammengeführten Revisionsprotokolleinträge
 a| 
`security audit log show`



 a| 
Ein Weiterleitungsziel für das Revisionsprotokoll und Sicherheitsmaßnahmen für dessen Übertragung angeben
 a| 
`cluster log-forwarding create`



 a| 
Ein Ziel für das Revisionsprotokoll ändern
 a| 
`cluster log-forwarding modify`



 a| 
Ein Ziel für das Revisionsprotokoll löschen
 a| 
`cluster log-forwarding delete`



 a| 
Die konfigurierten Ziele für das Revisionsprotokoll anzeigen
 a| 
`cluster log-forwarding show`
