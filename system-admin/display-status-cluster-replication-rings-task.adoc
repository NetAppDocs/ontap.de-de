= Den Replikationsstatus der ONTAP Clusterringe anzeigen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können den Status der Replikationsringe des Clusters einsehen, um clusterweite Probleme zu diagnostizieren. Wenn Ihr Cluster Probleme aufweist, kann das Support-Personal Sie bitten, diese Aufgabe durchzuführen, um die Fehlerbehebung zu unterstützen.

.Schritte
. Der Status der Cluster-Replikationsringe lässt sich mit dem  `cluster ring show` Befehl auf der Ebene mit erweiterten Berechtigungen anzeigen.


.Beispiel für die Anzeige des Cluster Ring-Replikationsstatus
Das folgende Beispiel zeigt den Status des VLDB-Replikationsrings auf einem Knoten mit dem Namen node0:

[listing]
----
cluster1::> set -privilege advanced
Warning: These advanced commands are potentially dangerous; use them only when directed to do so by support personnel.
Do you wish to continue? (y or n): y

cluster1::*> cluster ring show -node node0 -unitname vldb
          Node: node0
     Unit Name: vldb
        Status: master
         Epoch: 5
   Master Node: node0
    Local Node: node0
      DB Epoch: 5
DB Transaction: 56
 Number Online: 4
      RDB UUID: e492d2c1-fc50-11e1-bae3-123478563412
----
Weitere Informationen zu  `cluster ring show` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/cluster-ring-show.html["ONTAP-Befehlsreferenz"^].
