= Beschreibungen von ONTAP AutoSupport-Nachrichten abrufen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Die Beschreibungen der AutoSupport-Meldungen, die Sie erhalten, sind über den ONTAP Syslog Translator verfügbar.

.Schritte
. Gehe zu link:https://mysupport.netapp.com/site/bugs-online/syslog-translator["Syslog Übersetzer"^].
. Im Feld **Release** die Version von ONTAP eingeben, die Sie verwenden. Im Feld **Search String** „callhome“ eingeben. *Translate* auswählen.
. Der Syslog Translator listet alphabetisch alle Ereignisse auf, die der von Ihnen eingegebenen Meldungszeichenfolge entsprechen.
