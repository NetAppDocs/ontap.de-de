= Umbenennen eines ONTAP Knotens
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können den Namen eines Knotens nach Bedarf ändern.

.Schritte
. Um einen Node umzubenennen, wird der  `system node rename` Befehl verwendet.
+
Der  `-newname`Parameter gibt den neuen Namen für den Node an. Weitere Informationen zu  `system node rename` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/system-node-rename.html["ONTAP-Befehlsreferenz"^].

+
Wenn Sie mehrere Knoten im Cluster umbenennen möchten, muss der Befehl für jeden Knoten einzeln ausgeführt werden.

+
[NOTE]
====
Der Knotenname darf nicht "`all`" lauten, da "`all`" ein vom System reservierter Name ist.

====


.Beispiel für das Umbenennen eines Knotens
Der folgende Befehl benennt den Knoten "`node1`" in "`node1a`" um:

[listing]
----
cluster1::> system node rename -node node1 -newname node1a
----
