= Webbrowser-Zugriff auf ONTAP System Manager aktivieren oder deaktivieren
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können den Zugriff eines Webbrowsers auf den System Manager aktivieren oder deaktivieren. Außerdem besteht die Möglichkeit, das System Manager Protokoll anzuzeigen.

Sie können den Zugriff eines Webbrowsers auf System Manager mithilfe von  `vserver services web modify -name sysmgr -vserver <cluster_name> -enabled` [`true`|`false`] steuern.

Die System Manager Protokollierung wird in den  `/mroot/etc/log/mlog/sysmgr.log` Dateien des Knotens aufgezeichnet, der zum Zeitpunkt des Zugriffs auf den System Manager die Cluster Management-LIF hostet. Die Protokolldateien können mit einem Browser angezeigt werden. Das System Manager Protokoll ist auch in AutoSupport Nachrichten enthalten.
