= Der ONTAP Baseboard Management Controller ermöglicht die Remoteverwaltung eines Knotens.
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Auf bestimmten Hardwareplattformen wird die Software angepasst, um einen neuen Onboard-Controller im Baseboard Management Controller (BMC) zu unterstützen. Der BMC verfügt über Befehlszeilenbefehle (CLI), mit denen das Gerät remote verwaltet werden kann.

Der BMC funktioniert ähnlich wie der Service Processor (SP) und verwendet viele der gleichen Befehle. Der BMC ermöglicht Folgendes:

* Die BMC-Netzwerkeinstellungen konfigurieren.
* Aus der Ferne kann auf einen Knoten zugegriffen und Knotenverwaltungsaufgaben wie Diagnose, Herunterfahren, Stromzyklus oder Neustart des Knotens durchgeführt werden.


Es bestehen einige Unterschiede zwischen SP und BMC:

* Das BMC steuert die Umgebungsüberwachung von Stromversorgungselementen, Kühlkomponenten, Temperatursensoren, Spannungssensoren und Stromsensoren vollständig. Das BMC meldet Sensordaten über IPMI an ONTAP.
* Einige der Befehle für Hochverfügbarkeit (HA) und Storage unterscheiden sich.
* Das BMC sendet keine AutoSupport-Nachrichten.


Automatische Firmware-Updates sind auch verfügbar, wenn ONTAP mit den folgenden Voraussetzungen betrieben wird:

* Die BMC-Firmware-Revision 1.15 oder höher muss installiert sein.
+
[NOTE]
====
Für ein Upgrade der BMC Firmware von Version 1.12 auf 1.15 oder höher ist ein manuelles Update erforderlich.

====
* BMC startet nach Abschluss eines Firmware-Updates automatisch neu.
+
[NOTE]
====
Knotenoperationen werden während eines BMC Neustarts nicht beeinträchtigt.

====
