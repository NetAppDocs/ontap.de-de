= Informationen zur Struktur von ONTAP AutoSupport Nachrichten, die per E-Mail gesendet werden
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Wenn eine AutoSupport-Nachricht per E-Mail versendet wird, hat die Nachricht einen Standardbetreff, einen kurzen Nachrichtentext und einen großen Anhang im 7z-Dateiformat, der die Daten enthält.

[NOTE]
====
Wenn AutoSupport so konfiguriert ist, dass private Daten ausgeblendet werden, werden bestimmte Informationen, wie beispielsweise der Hostname, im Header, Betreff, Nachrichtentext und in den Anhängen weggelassen oder maskiert.

====


[[subject]]
== Betreff

Die Betreffzeile der vom AutoSupport-Mechanismus versendeten Nachrichten enthält einen Text-String, der den Grund für die Benachrichtigung angibt. Das Format der Betreffzeile ist wie folgt:

HA-Gruppenbenachrichtigung von _System_Name_ (_Message_) _Schweregrad_

* _System_Name_ ist je nach AutoSupport Konfiguration entweder der Hostname oder die System-ID.




[[body]]
== Körper

Der Text der AutoSupport-Nachricht enthält die folgenden Informationen:

* Datum und Zeitstempel der Nachricht
* Version von ONTAP auf dem Knoten, der die Nachricht generiert hat
* System-ID, Seriennummer und Hostname des Node, der die Nachricht generiert hat
* AutoSupport Sequenznummer
* System-ID und Hostname des HA-Partnerknotens




[[attached-files]]
== Angehängte Dateien

Die wichtigsten Informationen in einer AutoSupport-Nachricht sind in Dateien enthalten, die in einer 7z-Datei mit dem Namen  `body.7z` komprimiert und der Nachricht angehängt werden.

Die im Anhang enthaltenen Dateien sind spezifisch für den Typ der AutoSupport-Nachricht.
