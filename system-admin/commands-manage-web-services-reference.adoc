= ONTAP Befehle zur Verwaltung von Webdiensten
:allow-uri-read: 
:imagesdir: ../media/


[role="lead"]
Sie verwenden die `vserver services web` Befehle, um die Verfügbarkeit von Webdiensten für den Cluster oder eine Storage Virtual Machine (SVM) zu verwalten. Sie verwenden die  `vserver services web access`Befehle, um den Zugriff einer Rolle auf einen Webdienst zu steuern.

|===
| Wenn Sie möchten... | Verwenden Sie diesen Befehl... 


 a| 
Ein Webdienst für den Cluster oder eine SVM kann konfiguriert werden:

* Einen Webservice aktivieren oder deaktivieren
* Geben Sie an, ob für den Zugriff auf einen Webdienst ausschließlich HTTPS verwendet werden kann.

 a| 
`vserver services web modify`



 a| 
Anzeige der Konfiguration und Verfügbarkeit von Webdiensten für den Cluster oder eine SVM
 a| 
`vserver services web show`



 a| 
Eine Rolle für den Zugriff auf einen Webdienst im Cluster oder einer SVM autorisieren
 a| 
`vserver services web access create`



 a| 
Die Rollen, die zum Zugriff auf Webdienste im Cluster oder anSVM berechtigt sind, werden angezeigt.
 a| 
`vserver services web access show`



 a| 
Verhindern, dass eine Rolle auf einen Webdienst im Cluster oder einer SVM zugreift
 a| 
`vserver services web access delete`

|===
