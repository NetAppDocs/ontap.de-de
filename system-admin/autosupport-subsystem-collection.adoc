= ONTAP AutoSupport-Subsysteme anzeigen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Jedes Subsystem liefert grundlegende Informationen und Informationen zur Fehlerbehebung, die AutoSupport für seine Meldungen verwendet. Jedes Subsystem ist außerdem mit Auslöseereignissen verknüpft, sodass AutoSupport von den Subsystemen nur die für das jeweilige Auslöseereignis relevanten Informationen erfasst.

AutoSupport sammelt kontextbezogene Inhalte.

.Schritte
. Informationen zu Subsystemen und Auslöseereignissen:
+
[source, console]
----
system node autosupport trigger show
----
