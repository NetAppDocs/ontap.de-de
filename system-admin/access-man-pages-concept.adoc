= Zugriff auf die Manpages der ONTAP CLI
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ONTAP manuell (man) Seiten erklären, wie ONTAP CLI-Befehle verwendet werden. Diese Seiten sind auf der Befehlszeile verfügbar und werden auch in releasespezifischen _Befehlsreferenzen_ veröffentlicht.

An der ONTAP Befehlszeile kann mit dem  `man <command_name>` Befehl die Handbuchseite des angegebenen Befehls angezeigt werden. Wird kein Befehlsname angegeben, erscheint das Inhaltsverzeichnis der Handbuchseiten. Mit dem  `man man` Befehl lassen sich Informationen zum  `man` Befehl selbst anzeigen. Eine Handbuchseite kann durch Eingabe von  `*q*` verlassen werden.

Weitere Informationen zu den in Ihrer Version verfügbaren ONTAP-Befehlen auf Administrator- und Expertenebene finden Sie im link:https://docs.netapp.com/us-en/ontap-cli/["ONTAP-Befehlsreferenz"^].
