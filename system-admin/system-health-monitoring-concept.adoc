= Informationen zur Systemzustandsüberwachung von ONTAP
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Systemzustandsmonitore überwachen proaktiv bestimmte kritische Zustände in Ihrem Cluster und lösen Warnmeldungen aus, wenn ein Fehler oder Risiko erkannt wird. Wenn aktive Warnmeldungen vorliegen, meldet der Systemzustandsstatus einen beeinträchtigten Status für den Cluster. Die Warnmeldungen enthalten die Informationen, die Sie benötigen, um auf den beeinträchtigten Systemzustand zu reagieren.

Bei einem beeinträchtigten Status können Sie Details zum Problem einsehen, einschließlich der wahrscheinlichen Ursache und empfohlener Wiederherstellungsmaßnahmen. Nachdem das Problem behoben wurde, kehrt der Systemzustand automatisch zu OK zurück.

Der Systemzustand spiegelt mehrere separate Zustandsüberwachungen wider. Ein beeinträchtigter Zustand in einer einzelnen Zustandsüberwachung führt zu einem beeinträchtigten Gesamtzustand des Systems.

Detaillierte Informationen dazu, wie ONTAP Cluster Switches für die Systemzustandsüberwachung in Ihrem Cluster unterstützt, sind im _Hardware Universe_ verfügbar.

https://hwu.netapp.com/SWITCH/INDEX["Unterstützte Switches im Hardware Universe"^]

Detaillierte Informationen zu den Ursachen von Cluster Switch Health Monitor (CSHM) AutoSupport-Meldungen sowie zu den erforderlichen Maßnahmen zur Behebung dieser Warnungen sind im Knowledgebase-Artikel verfügbar.

https://kb.netapp.com/Advice_and_Troubleshooting/Data_Storage_Software/ONTAP_OS/AutoSupport_Message%3A_Health_Monitor_Process_CSHM["AutoSupport Meldung: Health Monitor Prozess CSHM"]
