= Informationen zu ONTAP AutoSupport Größen- und Zeitbudgets
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
AutoSupport sammelt Informationen, organisiert nach Subsystemen, und legt für jedes Subsystem ein Größen- und Zeitbudget für die Inhalte fest. Mit wachsender Größe der Speichersysteme bieten AutoSupport Budgets Kontrolle über die AutoSupport Nutzlast, was wiederum eine skalierbare Bereitstellung von AutoSupport Daten ermöglicht.

AutoSupport stellt die Datenerfassung ein und kürzt den AutoSupport-Inhalt, wenn der Inhalt des Subsystems dessen Größen- oder Zeitbudget überschreitet. Kann der Inhalt nicht einfach gekürzt werden (zum Beispiel Binärdateien), lässt AutoSupport den Inhalt weg.

Die Standardgrößen und Zeitbudgets sollten nur auf Anfrage des NetApp Support geändert werden. Die Standardgrößen und Zeitbudgets der Subsysteme können auch mit dem  `autosupport manifest show` Befehl überprüft werden.
