= Informationen zur Clusterverwaltung mit der ONTAP CLI
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ONTAP-Systeme lassen sich über die Befehlszeilenschnittstelle (CLI) verwalten. Die ONTAP Managementoberflächen können genutzt, auf den Cluster zugegriffen, Knoten verwaltet und vieles mehr durchgeführt werden.

Diese Vorgehensweise sollte unter folgenden Umständen angewendet werden:

* Der Funktionsumfang der ONTAP Administratorfähigkeiten soll nachvollzogen werden.
* Sie möchten die Befehlszeilenschnittstelle (CLI) verwenden, nicht System Manager oder ein automatisiertes Skripting-Tool.


.Verwandte Informationen
Einzelheiten zur Syntax und Verwendung der Befehlszeilenschnittstelle sind in der link:../concepts/manual-pages.html["ONTAP-Befehlsreferenz"] Dokumentation zu finden.
