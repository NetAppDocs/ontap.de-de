= Die Berechtigungsstufen für ONTAP CLI-Befehle
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ONTAP Befehle und Parameter sind auf drei Berechtigungsstufen definiert: _admin_, _advanced_ und _diagnostic_. Die Berechtigungsstufen spiegeln die für die Ausführung der Aufgaben erforderlichen Kenntnisse wider.

* *admin*
+
Die meisten Befehle und Parameter sind auf dieser Ebene verfügbar. Sie werden für gängige oder routinemäßige Aufgaben verwendet.

* *fortschrittlich*
+
Befehle und Parameter auf dieser Ebene werden selten verwendet, erfordern fortgeschrittene Kenntnisse und können bei unsachgemäßer Verwendung Probleme verursachen.

+
Erweiterte Befehle oder Parameter werden nur auf Anraten des Supportpersonals verwendet.

* *Diagnostik*
+
Diagnosebefehle und -parameter können Störungen verursachen. Sie werden ausschließlich von Supportmitarbeitern zur Diagnose und Behebung von Problemen verwendet.
