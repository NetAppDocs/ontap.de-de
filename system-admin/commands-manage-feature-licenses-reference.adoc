= Befehle zur Lizenzverwaltung in ONTAP
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können die ONTAP CLI `system license`-Befehle verwenden, um Funktionslizenzen für den Cluster zu verwalten. Sie verwenden die `system feature-usage`-Befehle, um die Funktionsnutzung zu überwachen.

Weitere Informationen zu den in diesem Thema beschriebenen Befehlen finden Sie in der link:https://docs.netapp.com/us-en/ontap-cli/["ONTAP-Befehlsreferenz"^].

Die folgende Tabelle enthält einige der gängigen CLI-Befehle zur Lizenzverwaltung sowie Links zu den Manpages der Befehle für weitere Informationen.

[cols="2,4"]
|===
| Wenn Sie möchten... | Verwenden Sie diesen Befehl... 


 a| 
Alle Pakete, die Lizenzen benötigen, sowie deren aktuellen Lizenzstatus werden angezeigt, einschließlich der folgenden:

* Der Paketname
* Das lizenzierte Verfahren
* Das Ablaufdatum, falls zutreffend

 a| 
link:https://docs.netapp.com/us-en/ontap-cli/system-license-show-status.html["system license show-status"^]



 a| 
Abgelaufene oder ungenutzte Lizenzen anzeigen oder entfernen
 a| 
link:https://docs.netapp.com/us-en/ontap-cli/system-license-clean-up.html["System License Clean-up"^]



 a| 
Zusammenfassung der Funktionsnutzung im Cluster auf Knotenbasis anzeigen
 a| 
https://docs.netapp.com/us-en/ontap-cli/system-feature-usage-show-summary.html["system feature-usage show-summary"]



 a| 
Anzeige des Nutzungsstatus von Funktionen im Cluster auf Knoten- und Wochenbasis
 a| 
https://docs.netapp.com/us-en/ontap-cli/system-feature-usage-show-history.html["system feature-usage show-history"]



 a| 
Status des Lizenzberechtigungsrisikos für jedes Lizenzpaket anzeigen
 a| 
https://docs.netapp.com/us-en/ontap-cli/system-license-entitlement-risk-show.html["system license entitlement-risk show"]

|===
