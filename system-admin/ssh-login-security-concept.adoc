= ONTAP SSH-Anmeldesicherheit
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Ab ONTAP 9.5 sind Informationen über frühere Anmeldungen, erfolglose Anmeldeversuche und Änderungen Ihrer Berechtigungen seit Ihrer letzten erfolgreichen Anmeldung verfügbar.

Sicherheitsrelevante Informationen werden angezeigt, wenn Sie sich erfolgreich als SSH-Administrator anmelden. Es erfolgt eine Benachrichtigung über die folgenden Bedingungen:

* Das letzte Mal, als Ihr Kontoname angemeldet war.
* Die Anzahl der erfolglosen Anmeldeversuche seit der letzten erfolgreichen Anmeldung.
* Ob sich die Rolle seit dem letzten Login geändert hat (zum Beispiel, wenn sich die Rolle des Administratorkontos von "admin" zu "backup" geändert hat)
* Ob die Berechtigungen zum Hinzufügen, Ändern oder Löschen der Rolle seit der letzten Anmeldung geändert wurden.


[NOTE]
====
Wenn einige der angezeigten Informationen verdächtig erscheinen, sollte umgehend die Sicherheitsabteilung kontaktiert werden.

====
Um diese Informationen beim Anmelden zu erhalten, müssen folgende Voraussetzungen erfüllt sein:

* Ihr SSH-Benutzerkonto muss in ONTAP bereitgestellt sein.
* Ihr SSH-Sicherheitslogin muss erstellt werden.
* Ihr Anmeldeversuch muss erfolgreich sein.




[[restrictions-and-other-considerations-for-ssh-login-security]]
== Einschränkungen und weitere Überlegungen zur Sicherheit bei der SSH-Anmeldung

Für die Sicherheitsinformationen zur SSH-Anmeldung gelten die folgenden Einschränkungen und Überlegungen:

* Die Informationen sind nur für SSH-basierte Anmeldungen verfügbar.
* Bei gruppenbasierten Administratorkonten, wie LDAP/NIS- und AD-Konten, können Benutzer die SSH-Anmeldeinformationen anzeigen, wenn die Gruppe, der sie angehören, als Administratorkonto in ONTAP bereitgestellt ist.
+
Warnmeldungen über Änderungen an der Rolle des Benutzerkontos können diesen Benutzern jedoch nicht angezeigt werden. Außerdem können Benutzer, die einer Active Directory Gruppe angehören, die als Administratorkonto in ONTAP bereitgestellt wurde, die Anzahl der fehlgeschlagenen Anmeldeversuche seit ihrer letzten Anmeldung nicht einsehen.

* Die für einen Benutzer gespeicherten Informationen werden gelöscht, wenn das Benutzerkonto aus ONTAP gelöscht wird.
* Die Informationen werden für Verbindungen zu anderen Anwendungen als SSH nicht angezeigt.




[[examples-of-ssh-login-security-information]]
== Beispiele für SSH-Anmeldesicherheitsinformationen

Die folgenden Beispiele veranschaulichen die Art von Informationen, die nach dem Login angezeigt werden.

* Diese Meldung wird nach jedem erfolgreichen Login angezeigt:
+
[listing]
----

Last Login : 7/19/2018 06:11:32
----
* Diese Meldungen werden angezeigt, wenn seit der letzten erfolgreichen Anmeldung erfolglose Anmeldeversuche stattgefunden haben:
+
[listing]
----

Last Login : 4/12/2018 08:21:26
Unsuccessful login attempts since last login - 5
----
* Diese Meldungen werden angezeigt, wenn es fehlgeschlagene Anmeldeversuche gab und Ihre Berechtigungen seit der letzten erfolgreichen Anmeldung geändert wurden:
+
[listing]
----

Last Login : 8/22/2018 20:08:21
Unsuccessful login attempts since last login - 3
Your privileges have changed since last login
----
