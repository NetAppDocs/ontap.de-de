= ONTAP Clusterattribute ändern
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Die Attribute eines Clusters, wie den Clusternamen, den Standort und die Kontaktinformationen, können bei Bedarf angepasst werden.

.Über diese Aufgabe
Die UUID eines Clusters kann nicht geändert werden, da sie bei der Erstellung des Clusters festgelegt wird.

.Schritte
. Zum Ändern von Clusterattributen wird der  `cluster identity modify` Befehl verwendet.
+
Der  `-name`Parameter gibt den Namen des Clusters an. Weitere Informationen zu  `cluster identity modify`und den Regeln für die Angabe des Clusternamens sind in der link:https://docs.netapp.com/us-en/ontap-cli/cluster-identity-modify.html["ONTAP-Befehlsreferenz"^]zu finden.

+
Der  `-location`Parameter gibt den Speicherort für den Cluster an.

+
Der  `-contact` Parameter gibt die Kontaktinformationen wie Name oder E-Mail-Adresse an.



.Beispiel für die Umbenennung eines Clusters
Der folgende Befehl benennt den aktuellen Cluster ("`cluster1`") in "`cluster2`" um:

[listing]
----
cluster1::> cluster identity modify -name cluster2
----
