= Installieren von NetApp-Lizenzen in ONTAP
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Es besteht die Möglichkeit, NetApp Lizenzdateien (NLFs) und Lizenzschlüssel mithilfe von System Manager zu installieren, was die bevorzugte Methode für die Installation von NLFs ist, oder die ONTAP CLI zur Installation von Lizenzschlüsseln zu verwenden. In ONTAP 9.10.1 und höher werden Funktionen mit einer NetApp Lizenzdatei aktiviert, während in Versionen vor ONTAP 9.10.1 ONTAP Funktionen mit Lizenzschlüsseln aktiviert werden.

.Schritte
Falls Sie bereits link:../system-admin/download-nlf-task.html["heruntergeladene NetApp Lizenzdateien"] oder Lizenzschlüssel besitzen, können Sie mithilfe des System Manager oder der ONTAP CLI NLFs und 28-stellige Lizenzschlüssel installieren.

[role="tabbed-block"]
====
.System Manager – ONTAP 9.8 und höher
--
. *Cluster > Einstellungen* auswählen.
. Unter *Lizenzen* image:icon_arrow.gif["Pfeilsymbol"] auswählen.
. Wählen Sie *Durchsuchen*. Wählen Sie die heruntergeladene NetApp Lizenzdatei aus.
. Wenn Lizenzschlüssel hinzugefügt werden sollen, *28-stellige Lizenzschlüssel verwenden* auswählen und die Schlüssel eingeben.


--
.System Manager – ONTAP 9.7 und früher
--
. *Konfiguration > Cluster > Lizenzen* auswählen.
. Unter *Lizenzen* image:icon_arrow.gif["Pfeilsymbol"] auswählen.
. Im Fenster *Pakete* auf *Hinzufügen* klicken.
. Im Dialogfeld *Lizenzpakete hinzufügen* auf *Dateien auswählen* klicken, um die heruntergeladene NetApp Lizenzdatei auszuwählen, und anschließend auf *Hinzufügen* klicken, um die Datei in den Cluster hochzuladen.


--
.CLI
--
. Einen oder mehrere Lizenzschlüssel hinzufügen:
+
[source, cli]
----
system license add
----
+
Das folgende Beispiel installiert Lizenzen vom lokalen Knoten "/mroot/etc/lic_file", falls die Datei an diesem Speicherort existiert:

+
[listing]
----
cluster1::> system license add -use-license-file true
----
+
Das folgende Beispiel fügt dem Cluster eine Liste von Lizenzen mit den Schlüsseln AAAAAAAAAAAAAAAAAAAAAAAAAAAA und BBBBBBBBBBBBBBBBBBBBBBBBBBBB hinzu:

+
[listing]
----
cluster1::> system license add -license-code AAAAAAAAAAAAAAAAAAAAAAAAAAAA, BBBBBBBBBBBBBBBBBBBBBBBBBBBB
----
+
Weitere Informationen zum Link:https://docs.netapp.com/us-en/ontap-cli/system-license-add.html[`system license add`^]-Befehl finden sich in der ONTAP-Befehlsreferenz.
