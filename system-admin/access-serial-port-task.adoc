= Zugriff auf ein ONTAP Cluster über den seriellen Port
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Der Cluster kann direkt über eine Konsole erreicht werden, die an den seriellen Port eines Node angeschlossen ist.

.Schritte
. An der Konsole die Eingabetaste drücken.
+
Das System antwortet mit der Anmeldeaufforderung.

. Am Anmelde-Prompt eine der folgenden Optionen auswählen:
+
|===
| Um auf den Cluster mit ... zuzugreifen | Geben Sie den folgenden Kontonamen ein... 


 a| 
Das Standard-Clusterkonto
 a| 
`*admin*`



 a| 
Ein alternatives administratives Benutzerkonto
 a| 
`_username_`

|===
+
Das System antwortet mit der Aufforderung zur Passworteingabe.

. Das Passwort für das Admin- oder Administrationsbenutzerkonto eingeben und anschließend die Eingabetaste drücken.
