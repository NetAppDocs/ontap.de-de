= Einen ONTAP Knoten remote mit dem SP/BMC verwalten
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Ein Knoten kann mithilfe eines integrierten Controllers, dem sogenannten Service Processor (SP) oder Baseboard Management Controller (BMC), remote verwaltet werden. Dieser Remote-Management-Controller ist in allen aktuellen Plattformmodellen enthalten. Der Controller bleibt unabhängig vom Betriebszustand des Knotens funktionsfähig.

Eine vollständige Übersicht über die Unterstützung von Plattform SP und BMC ist im link:https://mysupport.netapp.com/site/info/sp-bmc["Support-Matrix"^] auf der NetApp Support-Website zu finden.
