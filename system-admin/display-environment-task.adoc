= ONTAP Umweltinformationen anzeigen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sensoren helfen dabei, die Umgebungskomponenten Ihres Systems zu überwachen. Zu den Informationen, die zu Umweltsensoren angezeigt werden können, zählen Typ, Name, Status, Wert und Schwellenwertwarnungen.

.Schritt
. Informationen über Umweltsensoren werden mit dem  `system node environment sensors show` Befehl angezeigt.
