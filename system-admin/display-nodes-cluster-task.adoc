= Details auf Knotenebene in einem ONTAP Cluster
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können Knotennamen, den Status der Knoten und deren Eignung zur Teilnahme am Cluster einsehen. Auf der erweiterten Berechtigungsstufe wird außerdem angezeigt, ob ein Knoten über Epsilon verfügt.

.Schritte
. Informationen über die Knoten in einem Cluster sind mit dem  `cluster show` Befehl verfügbar.
+
Soll in der Ausgabe angezeigt werden, ob ein Knoten Epsilon enthält, ist der Befehl auf dem erweiterten Berechtigungsniveau auszuführen.

+
Weitere Informationen zu  `cluster show` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/cluster-show.html["ONTAP-Befehlsreferenz"^].



.Beispiele für die Anzeige der Knoten in einem Cluster
Das folgende Beispiel zeigt Informationen über alle Knoten in einem vier Knoten Cluster:

[listing]
----

cluster1::> cluster show
Node                  Health  Eligibility
--------------------- ------- ------------
node1                 true    true
node2                 true    true
node3                 true    true
node4                 true    true
----
Das folgende Beispiel zeigt detaillierte Informationen über den Knoten mit dem Namen "`node1`" auf der erweiterten Berechtigungsstufe:

[listing]
----

cluster1::> set -privilege advanced
Warning: These advanced commands are potentially dangerous; use them only when directed to do so by support personnel.
Do you want to continue? {y|n}: y

cluster1::*> cluster show -node node1

       Node: node1
  Node UUID: a67f9f34-9d8f-11da-b484-000423b6f094
    Epsilon: false
Eligibility: true
     Health: true
----
