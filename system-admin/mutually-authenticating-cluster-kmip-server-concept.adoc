= Übersicht zur gegenseitigen Authentifizierung des ONTAP Clusters und eines KMIP Servers
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Die gegenseitige Authentifizierung des Clusters und eines externen Schlüsselmanagers wie eines Key Management Interoperability Protocol (KMIP) Servers ermöglicht es dem Schlüsselmanager, mit dem Cluster über KMIP via SSL zu kommunizieren. Dies erfolgt, wenn eine Anwendung oder bestimmte Funktionalität (zum Beispiel die Storage Encryption Funktionalität) sichere Schlüssel benötigt, um sicheren Datenzugriff bereitzustellen.
