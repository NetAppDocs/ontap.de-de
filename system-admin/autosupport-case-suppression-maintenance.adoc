= Unterdrückung der ONTAP AutoSupport Fall-Erstellung während geplanter Wartungsfenster
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
AutoSupport Fallunterdrückung ermöglicht das Verhindern der Erstellung unnötiger Fälle durch AutoSupport Nachrichten, die während geplanter Wartungsfenster ausgelöst werden.

.Schritte
. Manuelles Auslösen einer AutoSupport-Nachricht mit der Text-String `MAINT=xh`, wobei  `x` die Dauer des Wartungsfensters in Stunden angibt. <node> ist durch den Namen des Knotens zu ersetzen, von dem die AutoSupport-Nachricht gesendet werden soll:
+
[source, console]
----
system node autosupport invoke -node <node> -message MAINT=xh
----
