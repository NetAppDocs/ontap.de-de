= ONTAP Cluster-Details auf Clusterebene anzeigen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Es werden die eindeutige Kennung (UUID), der Name, die Seriennummer, der Standort und die Kontaktinformationen eines Clusters angezeigt.

.Schritte
. Die Attribute eines Clusters werden mit dem  `cluster identity show` Befehl angezeigt.


.Beispiel für die Anzeige von Clusterattributen
Das folgende Beispiel zeigt den Namen, die Seriennummer, den Standort und die Kontaktinformationen eines Clusters.

[listing]
----
cluster1::> cluster identity show

          Cluster UUID: 1cd8a442-86d1-11e0-ae1c-123478563412
          Cluster Name: cluster1
 Cluster Serial Number: 1-80-123456
      Cluster Location: Sunnyvale
       Cluster Contact: jsmith@example.com
----
Weitere Informationen zu  `cluster identity show` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/cluster-identity-show.html["ONTAP-Befehlsreferenz"^].
