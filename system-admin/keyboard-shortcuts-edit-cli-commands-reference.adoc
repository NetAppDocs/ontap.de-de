= ONTAP Tastenkombinationen zum Bearbeiten von CLI-Befehlen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Der aktuell in der Befehlszeile angezeigte Befehl ist der aktive Befehl. Mithilfe von Tastenkombinationen kann der aktive Befehl schnell bearbeitet werden. Diese Tastenkombinationen ähneln denen der UNIX tcsh Shell und des Emacs Editors.

Die folgende Tabelle listet die Tastenkombinationen zum Bearbeiten von CLI-Befehlen auf.  `Ctrl-` gibt an, dass die Strg-Taste gedrückt gehalten wird, während das nachfolgende Zeichen eingegeben wird.  `Esc-` gibt an, dass die Esc-Taste gedrückt und wieder losgelassen wird und anschließend das nachfolgende Zeichen eingegeben wird.

[cols="4a,4a"]
|===
| Wenn Sie möchten... | Verwenden Sie eine dieser Tastenkombinationen... 


 a| 
Den Cursor um ein Zeichen zurückbewegen
 a| 
* Strg+B
* Zurück-Pfeil




 a| 
Den Cursor um ein Zeichen vorwärts bewegen
 a| 
* Strg-F
* Vorwärtspfeil




 a| 
Den Cursor um ein Wort zurück bewegen
 a| 
Esc-B



 a| 
Den Cursor um ein Wort vorwärts bewegen
 a| 
Esc-F



 a| 
Den Cursor an den Anfang der Zeile bewegen
 a| 
Strg+A



 a| 
Den Cursor an das Ende der Zeile bewegen.
 a| 
Strg-E



 a| 
Entfernen Sie den Inhalt der Befehlszeile vom Zeilenanfang bis zur Cursorposition und speichern Sie ihn im Ausschneidepuffer. Der Ausschneidepuffer fungiert als temporärer Speicher, ähnlich wie die sogenannte Zwischenablage in manchen Programmen.
 a| 
Strg-U



 a| 
Entfernen Sie den Inhalt der Befehlszeile vom Cursor bis zum Zeilenende und speichern Sie ihn im Ausschneidepuffer
 a| 
Strg-K



 a| 
Entfernen Sie den Inhalt der Befehlszeile vom Cursor bis zum Ende des folgenden Wortes und speichern Sie ihn im Ausschneidepuffer
 a| 
Esc-D



 a| 
Das Wort vor dem Cursor wird entfernt und im Ausschneidepuffer gespeichert.
 a| 
Strg-W



 a| 
Den Inhalt des Ausschneidepuffers abrufen und an der Cursorposition in die Befehlszeile einfügen
 a| 
Strg-Y



 a| 
Das Zeichen vor dem Cursor löschen
 a| 
* Strg-H
* Rücktaste




 a| 
Das Zeichen an der Cursorposition löschen
 a| 
Strg-D



 a| 
Leitung freimachen
 a| 
Strg+C



 a| 
Bildschirm löschen
 a| 
Strg-L



 a| 
Der aktuelle Inhalt der Befehlszeile wird durch den vorherigen Eintrag in der Verlaufsliste ersetzt.

Bei jeder Wiederholung des Tastaturkürzels bewegt sich der Verlaufscursor zum vorherigen Eintrag.
 a| 
* Strg+P
* Esc-P
* Pfeil nach oben




 a| 
Ersetzt den aktuellen Inhalt der Befehlszeile durch den nächsten Eintrag in der Verlaufsliste. Mit jeder Wiederholung des Tastenkürzels bewegt sich der Verlaufscursor zum nächsten Eintrag.
 a| 
* Strg-N
* Esc-N
* Pfeil nach unten




 a| 
Einen teilweise eingegebenen Befehl erweitern oder gültige Eingaben ab der aktuellen Bearbeitungsposition auflisten
 a| 
* Tab
* Strg-I




 a| 
Kontextsensitive Hilfe anzeigen
 a| 
?



 a| 
Die spezielle Zuordnung für das Fragezeichen  `?`-Zeichen aufheben. Um beispielsweise ein Fragezeichen als Argument in einer Befehlszeile einzugeben, Esc drücken und dann das  `?`-Zeichen eingeben.
 a| 
Esc-?



 a| 
TTY-Ausgabe starten
 a| 
Strg-Q



 a| 
TTY-Ausgabe beenden
 a| 
Strg-S

|===
