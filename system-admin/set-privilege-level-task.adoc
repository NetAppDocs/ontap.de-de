= Die Berechtigungsstufe in der ONTAP CLI festlegen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können die Berechtigungsstufe in der CLI mit dem  `set` Befehl festlegen. Änderungen an den Einstellungen der Berechtigungsstufe gelten nur für die aktuelle Sitzung. Sie sind nicht sitzungsübergreifend persistent.

.Schritte
. Um die Berechtigungsstufe in der CLI festzulegen, den  `set` Befehl mit dem  `-privilege` Parameter verwenden.


.Beispiel für die Festlegung der Berechtigungsstufe
Im folgenden Beispiel wird die Berechtigungsstufe zunächst auf „Erweitert“ und anschließend auf „Admin“ gesetzt:

[listing]
----
cluster1::> set -privilege advanced
Warning: These advanced commands are potentially dangerous; use them only when directed to do so by NetApp personnel.
Do you wish to continue? (y or n): y
cluster1::*> set -privilege admin
----
Weitere Informationen zu  `set` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/set.html["ONTAP-Befehlsreferenz"^].
