= Informationen, die im ONTAP AutoSupport Manifest enthalten sind
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Das AutoSupport Manifest bietet Ihnen eine detaillierte Übersicht der für jede AutoSupport Nachricht gesammelten Dateien. Das AutoSupport Manifest enthält außerdem Informationen zu Sammlungsfehlern, wenn AutoSupport die benötigten Dateien nicht sammeln kann.

Das AutoSupport Manifest enthält die folgenden Informationen:

* Sequenznummer der AutoSupport-Nachricht
* Welche Dateien AutoSupport sind in der AutoSupport-Nachricht enthalten?
* Größe der einzelnen Dateien in Bytes
* Status der AutoSupport Manifest-Sammlung
* Fehlerbeschreibung, falls AutoSupport das Sammeln einer oder mehrerer Dateien fehlgeschlagen ist


Das AutoSupport-Manifest kann mit dem  `system node autosupport manifest show` Befehl angezeigt werden.

Das AutoSupport-Manifest ist in jeder AutoSupport-Nachricht enthalten und wird im XML-Format präsentiert, was bedeutet, dass entweder ein generischer XML-Viewer zum Lesen verwendet werden kann oder die Anzeige über das Digital Advisor Portal erfolgt.
