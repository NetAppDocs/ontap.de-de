= Unterstützte CLI-Befehle für den ONTAP Baseboard Management Controller
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können sich über SSH beim BMC anmelden. Die folgenden Befehle werden von der BMC Befehlszeile unterstützt.

|===
| Befehl | Funktion 


 a| 
System
 a| 
Eine Liste aller Befehle wird angezeigt.



 a| 
Systemkonsole
 a| 
Mit der Systemkonsole verbinden.  `Ctrl+D` zum Beenden der Sitzung verwenden.



 a| 
Systemkern
 a| 
Systemkern sichern und zurücksetzen.



 a| 
System Power Cycle
 a| 
Das System wird ausgeschaltet und anschließend wieder eingeschaltet.



 a| 
System ausschalten
 a| 
Das System ausschalten.



 a| 
System einschalten
 a| 
Das System einschalten.



 a| 
Systemstromstatus
 a| 
Systemstromstatus drucken.



 a| 
System zurücksetzen
 a| 
Das System zurücksetzen.



 a| 
Systemprotokoll
 a| 
Systemkonsolenprotokolle drucken



 a| 
system fru show [id]
 a| 
Alle oder ausgewählte Field Replacable Unit (FRU) Informationen ausgeben.

|===
