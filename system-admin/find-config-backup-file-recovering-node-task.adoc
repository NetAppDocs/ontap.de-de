= Die ONTAP Knotenkonfigurationssicherungsdatei zum Wiederherstellen eines Knotens finden.
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Eine Konfigurationssicherungsdatei, die sich an einer Remote-URL oder auf einem Knoten im Cluster befindet, wird verwendet, um eine Knotenkonfiguration wiederherzustellen.

.Über diese Aufgabe
Sie können entweder eine Cluster- oder eine Knoten-Konfigurationssicherungsdatei verwenden, um eine Knoten-Konfiguration wiederherzustellen.

.Schritt
. Die Konfigurationssicherungsdatei muss dem Knoten zur Verfügung stehen, für den die Konfiguration wiederhergestellt werden soll.
+
|===
| Wenn sich die Konfigurationssicherungsdatei befindet... | Dann... 


 a| 
Unter einer entfernten URL
 a| 
Der  `system configuration backup download`-Befehl auf der erweiterten Berechtigungsebene kann verwendet werden, um ihn auf den wiederherzustellenden Node herunterzuladen.



 a| 
Auf einem Knoten im Cluster
 a| 
.. Mit dem  `system configuration backup show` Befehl auf der Ebene erweiterter Berechtigungen lässt sich die Liste der im Cluster verfügbaren Konfigurationssicherungsdateien anzeigen, die die Konfiguration des wiederherzustellenden Knotens enthalten.
.. Falls die von Ihnen angegebene Konfigurationssicherungsdatei auf dem wiederherzustellenden Knoten nicht vorhanden ist, kann der  `system configuration backup copy` Befehl verwendet werden, um sie auf den wiederherzustellenden Knoten zu kopieren.


|===
+
Wenn Sie den Cluster zuvor neu erstellt haben, sollten Sie eine Konfigurationssicherungsdatei wählen, die nach der Neuerstellung des Clusters erstellt wurde. Falls Sie eine Konfigurationssicherungsdatei verwenden müssen, die vor der Neuerstellung des Clusters erstellt wurde, muss der Cluster nach der Wiederherstellung des Knotens erneut erstellt werden.
