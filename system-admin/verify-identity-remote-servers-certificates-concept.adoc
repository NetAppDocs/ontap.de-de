= Informationen zur Überprüfung der Identität von Remote-Servern mithilfe von Zertifikaten in ONTAP
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ONTAP unterstützt Sicherheitszertifikatsfunktionen zur Überprüfung der Identität von Remote-Servern.

ONTAP Software ermöglicht sichere Verbindungen mithilfe dieser digitalen Zertifikatsfunktionen und Protokolle:

* Das Online Certificate Status Protocol (OCSP) überprüft den Status von digitalen Zertifikatsanforderungen von ONTAP Services über SSL- und Transport Layer Security (TLS)-Verbindungen. Diese Funktion ist standardmäßig deaktiviert.
* Ein Standardsatz vertrauenswürdiger Stammzertifikate ist in ONTAP Software enthalten.
* Key Management Interoperability Protocol (KMIP)-Zertifikate ermöglichen die gegenseitige Authentifizierung eines Clusters und eines KMIP-Servers.
