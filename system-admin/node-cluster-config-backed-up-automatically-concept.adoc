= Informationen zur Planung von Backups der ONTAP Cluster- und Knotenkonfigurationsdateien
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Drei separate Zeitpläne erstellen automatisch Sicherungsdateien für die Cluster- und Knotenkonfiguration und replizieren sie zwischen den Knoten im Cluster.

Die Konfigurationssicherungsdateien werden automatisch nach folgenden Zeitplänen erstellt:

* Alle 8 Stunden
* Täglich
* Wöchentlich


Zu jedem dieser Zeitpunkte wird auf jedem fehlerfreien Knoten im Cluster eine Sicherungsdatei der Knotenkonfiguration erstellt. Alle diese Sicherungsdateien der Knotenkonfiguration werden anschließend zusammen mit der replizierten Clusterkonfiguration in einer einzigen Clusterkonfigurations-Sicherungsdatei zusammengefasst und auf einem oder mehreren Knoten im Cluster gespeichert.
