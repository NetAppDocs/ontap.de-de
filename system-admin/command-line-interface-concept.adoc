= Informationen zur ONTAP Befehlszeile
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Die ONTAP Befehlszeilenschnittstelle (CLI) bietet eine befehlsbasierte Ansicht der Managementoberfläche. Befehle werden an der Eingabeaufforderung des Speichersystems eingegeben, und die Befehlsergebnisse erscheinen als Text.

Die CLI-Befehlszeile wird als  `cluster_name::>` dargestellt.

Wenn die Berechtigungsstufe (d. h. der  `-privilege`Parameter des  `set`Befehls) auf  `advanced` gesetzt ist, enthält die Eingabeaufforderung ein Sternchen (*), zum Beispiel:

`cluster_name::*>`

Weitere Informationen zu  `set` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/set.html["ONTAP-Befehlsreferenz"^].
