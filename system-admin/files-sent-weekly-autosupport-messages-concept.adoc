= Dateien in wöchentlichen AutoSupport-Nachrichten
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Wöchentliche AutoSupport-Meldungen enthalten zusätzliche Konfigurations- und Statusdaten, die hilfreich sind, um Änderungen in Ihrem System im Zeitverlauf nachzuvollziehen.

Die folgenden Informationen sind in den wöchentlichen AutoSupport-Meldungen enthalten:

* Grundlegende Informationen zu jedem Teilsystem
* Inhalte der ausgewählten `/mroot/etc` Verzeichnisdateien
* Protokolldateien
* Ausgabe von Befehlen, die Systeminformationen liefern
* Zusätzliche Informationen, einschließlich Informationen zur replizierten Datenbank (RDB), Servicestatistiken und mehr
