= Kapazitätsauslastung der Systemvolumes in einem ONTAP Cluster anzeigen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Systemvolumes sind FlexVol Volumes, die spezielle Metadaten enthalten, beispielsweise Metadaten für die Überwachungsprotokolle von Dateidiensten. Diese Volumes sind im Cluster sichtbar, sodass die Speichernutzung im Cluster vollständig nachvollzogen werden kann.

Systemvolumes gehören dem Cluster-Management-Server (auch Admin-SVM genannt) und werden automatisch erstellt, wenn die Überwachung der Dateidienste aktiviert ist.

Systemvolumes können mit dem  `volume show` Befehl angezeigt werden, die meisten anderen Volume-Operationen sind jedoch nicht zulässig. Beispielsweise kann ein Systemvolume nicht mit dem  `volume modify` Befehl geändert werden.

Dieses Beispiel zeigt vier Systemvolumes auf der Admin-SVM, die automatisch erstellt wurden, als die Dateidienstüberwachung für eine Daten-SVM im Cluster aktiviert wurde:

[listing]
----
cluster1::> volume show -vserver cluster1
Vserver   Volume       Aggregate    State      Type       Size  Available Used%
--------- ------------ ------------ ---------- ---- ---------- ---------- -----
cluster1  MDV_aud_1d0131843d4811e296fc123478563412
                       aggr0        online     RW          2GB     1.90GB    5%
cluster1  MDV_aud_8be27f813d7311e296fc123478563412
                       root_vs0     online     RW          2GB     1.90GB    5%
cluster1  MDV_aud_9dc4ad503d7311e296fc123478563412
                       aggr1        online     RW          2GB     1.90GB    5%
cluster1  MDV_aud_a4b887ac3d7311e296fc123478563412
                       aggr2        online     RW          2GB     1.90GB    5%
4 entries were displayed.
----
