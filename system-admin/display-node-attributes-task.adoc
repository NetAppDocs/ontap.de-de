= Die Attribute der Knoten in einem ONTAP Cluster anzeigen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können die Attribute eines oder mehrerer Knoten im Cluster anzeigen, zum Beispiel Name, Besitzer, Standort, Modellnummer, Seriennummer, Betriebsdauer des Knotens, Gesundheitszustand und Berechtigung zur Teilnahme an einem Cluster.

.Schritte
. Um die Attribute eines bestimmten Knotens oder aller Knoten in einem Cluster anzuzeigen, den  `system node show` Befehl verwenden.


.Beispiel für die Anzeige von Informationen über einen Knoten
Das folgende Beispiel zeigt detaillierte Informationen zu node1:

[listing]
----
cluster1::> system node show -node node1
                          Node: node1
                         Owner: Eng IT
                      Location: Lab 5
                         Model: model_number
                 Serial Number: 12345678
                     Asset Tag: -
                        Uptime: 23 days 04:42
               NVRAM System ID: 118051205
                     System ID: 0118051205
                        Vendor: NetApp
                        Health: true
                   Eligibility: true
       Differentiated Services: false
           All-Flash Optimized: true
            Capacity Optimized: false
                 QLC Optimized: false
    All-Flash Select Optimized: false
 SAS2/SAS3 Mixed Stack Support: none
----
