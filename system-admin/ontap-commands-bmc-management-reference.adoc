= ONTAP Befehle für BMC Management
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Diese ONTAP Befehle sind auf dem Baseboard Management Controller (BMC) verfügbar.

Der BMC verwendet einige der gleichen Befehle wie der Service Processor (SP). Die folgenden SP-Befehle werden auf dem BMC unterstützt.

|===
| Wenn Sie möchten... | Verwenden Sie diesen Befehl 


 a| 
BMC-Informationen anzeigen
 a| 
`*system service-processor show*`



 a| 
BMC Netzwerkkonfiguration anzeigen/ändern
 a| 
`*system service-processor network show/modify*`



 a| 
Den BMC zurücksetzen
 a| 
`*system service-processor reboot-sp*`



 a| 
Details des aktuell installierten BMC-Firmware-Images anzeigen/ändern
 a| 
`*system service-processor image show/modify*`



 a| 
BMC Firmware aktualisieren
 a| 
`*system service-processor image update*`



 a| 
Status des neuesten BMC Firmware-Updates anzeigen
 a| 
`*system service-processor image update-progress show*`



 a| 
Die automatische Netzwerkkonfiguration für den BMC aktivieren, um eine IPv4- oder IPv6-Adresse im angegebenen Subnetz zu verwenden.
 a| 
`*system service-processor network auto-configuration enable*`



 a| 
Die automatische Netzwerkkonfiguration für eine IPv4- oder IPv6-Adresse im für den BMC angegebenen Subnetz deaktivieren
 a| 
`*system service-processor network auto-configuration disable*`



 a| 
Anzeige der automatischen BMC-Netzwerkkonfiguration
 a| 
`*system service-processor network auto-configuration show*`

|===
Für Befehle, die von der BMC Firmware nicht unterstützt werden, wird die folgende Fehlermeldung zurückgegeben.

[listing]
----
::> Error: Command not supported on this platform.
----
