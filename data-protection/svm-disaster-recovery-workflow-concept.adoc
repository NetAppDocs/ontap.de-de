= ONTAP SnapMirror SVM-Workflow zur Notfallwiederherstellung
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Um sich von einem Ausfall zu erholen und Daten vom Ziel-SVM bereitzustellen, muss das Ziel-SVM aktiviert werden. Die Aktivierung des Ziel-SVM umfasst das Stoppen geplanter SnapMirror-Transfers, das Abbrechen laufender SnapMirror-Transfers, das Aufheben der Replikationsbeziehung, das Stoppen des Quell-SVM und das Starten des Ziel-SVM.
