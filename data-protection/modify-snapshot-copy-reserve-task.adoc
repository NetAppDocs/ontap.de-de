= Die ONTAP Snapshot Reserve anpassen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Es kann sinnvoll sein, eine größere Snapshot-Reserve zu konfigurieren, um zu verhindern, dass Snapshots Speicherplatz belegen, der für das aktive Dateisystem reserviert ist. Die Snapshot-Reserve kann verringert werden, wenn nicht mehr so viel Speicherplatz für Snapshots benötigt wird.

.Schritt
. Die Snapshot-Reserve ändern:
+
`volume modify -vserver _SVM_ -volume _volume_ -percent-snapshot-space _snap_reserve_`

+
Weitere Informationen zu  `volume modify` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/volume-modify.html["ONTAP-Befehlsreferenz"^].

+
Im folgenden Beispiel wird die Snapshot-Reserve für  `vol1` auf 10 Prozent festgelegt:

+
[listing]
----
cluster1::> volume modify -vserver vs0 -volume vol1 -percent-snapshot-space 10
----
