= Ein ONTAP SnapMirror Zielvolume erstellen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Der  `volume create`-Befehl kann auf dem Zielsystem verwendet werden, um ein Zielvolume zu erstellen. Das Zielvolume sollte genauso groß oder größer als das Quellvolume sein. Weitere Informationen zu  `volume create` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/volume-create.html["ONTAP-Befehlsreferenz"^].

.Schritt
. Zielvolume erstellen:
+
`volume create -vserver _SVM_ -volume volume -aggregate _aggregate_ -type DP -size _size_`

+
Das folgende Beispiel erstellt ein 2-GB-Zielvolume mit dem Namen  `volA_dst`:

+
[listing]
----
cluster_dst::> volume create -vserver SVM_backup -volume volA_dst -aggregate node01_aggr -type DP -size 2GB
----
