= ONTAP Snapshot-Speicherverbrauch überwachen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können die Speichernutzung von Snapshots mithilfe des  `df` Befehls überwachen. Der Befehl zeigt den freien Speicherplatz im aktiven Dateisystem und die Snapshot-Reserve an.

.Schritt
. Anzeige des Snapshot-Speicherverbrauchs: `df`
+
Das folgende Beispiel zeigt den Festplattenverbrauch von Snapshots:

+
[listing]
----
cluster1::> df
Filesystem          kbytes  used    avail  capacity
/vol/vol0/          3000000 3000000 0       100%
/vol/vol0/.snapshot 1000000 500000  500000   50%
----


Weitere Informationen zu den in diesem Verfahren beschriebenen Befehlen befinden sich in der link:https://docs.netapp.com/us-en/ontap-cli/["ONTAP-Befehlsreferenz"^].
