= Lokale Ebenen zur Verwendung für ONTAP SnapMirror SVM DR-Beziehungen angeben
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Nachdem eine SVM für die Notfallwiederherstellung erstellt wurde, kann mit der  `aggr-list` Option des  `vserver modify` Befehls eingeschränkt werden, welche lokalen Tiers zum Hosten der SVM DR Ziel-Volumes verwendet werden.

.Schritte
. Erstellen einer Ziel-SVM:
+
`vserver create -vserver _SVM_ -subtype dp-destination`

. Die aggr-list der Disaster-Recovery-SVM wird angepasst, um die lokalen Tiers einzuschränken, die zum Hosten des Volumes der Disaster-Recovery-SVM verwendet werden:
+
`cluster_dest::> vserver modify -vserver _SVM_ -aggr-list <comma-separated-list>`
