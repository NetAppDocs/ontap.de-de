= Verfügbare ONTAP Snapshot-Reserve auf einem Volume prüfen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Es kann sinnvoll sein zu prüfen, wie viel Snapshot-Reserve auf einem Volume verfügbar ist, indem der  `snapshot-reserve-available` Parameter mit dem  `volume show` Befehl verwendet wird. Weitere Informationen zu  `volume show` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/volume-show.html["ONTAP-Befehlsreferenz"^].

.Schritt
. Die auf einem Volume verfügbare Snapshot-Reserve prüfen:
+
`vol show -vserver _SVM_ -volume _volume_ -fields snapshot-reserve-available`

+
Das folgende Beispiel zeigt die verfügbare Snapshot-Reserve für `vol1`:

+
[listing]
----
cluster1::> vol show -vserver vs0 -volume vol1 -fields snapshot-reserve-available

vserver volume snapshot-reserve-available
------- ------ --------------------------
vs0     vol1   4.84GB
----
