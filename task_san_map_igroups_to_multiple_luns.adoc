= Igroups mehreren LUNs zuordnen
:hardbreaks:
:toclevels: 1
:allow-uri-read: 
:toclevels: 1
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
Ab ONTAP 9.9.1 ist es möglich, Initiatorgruppen gleichzeitig zwei oder mehr LUNs zuzuordnen.

. Im System Manager auf *Speicher > LUNs* klicken.
. Wählen Sie die LUNs aus, die Sie zuordnen möchten.
. *Mehr* auswählen und anschließend *Initiatorgruppen zuordnen* wählen.



NOTE: Die ausgewählten igroups werden den ausgewählten LUNs hinzugefügt. Die bereits vorhandenen Zuordnungen werden nicht überschrieben.
