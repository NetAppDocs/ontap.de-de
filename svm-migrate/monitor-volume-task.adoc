= Überwachung der ONTAP Volume Migration
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Zusätzlich zur Überwachung der gesamten SVM-Migration mit dem  `vserver migrate show` Befehl kann auch der Migrationsstatus der in der SVM enthaltenen Volumes überwacht werden.

.Schritte
. Im Ziel-Cluster den Status der Volume-Migration prüfen:
+
[source, cli]
----
vserver migrate show-volume
----
