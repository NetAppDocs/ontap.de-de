= Monitor ONTAP Volume-Migration
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Zusätzlich zur Überwachung der gesamten SVM-Migration mit dem `vserver migrate show` Befehl können Sie den Migrationsstatus der Volumes überwachen, die in der SVM enthalten sind.

.Schritte
. Überprüfen Sie auf dem Zielcluster den Status der Volume-Migration:
+
[source, cli]
----
vserver migrate show-volume
----
