= Quell-ONTAP SVM nach der Clientumstellung manuell entfernen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Wenn die SVM-Migration mit deaktivierter Quellbereinigung durchgeführt wurde, kann die Quell-SVM nach Abschluss der Clientumstellung manuell entfernt werden.

.Schritte
. Überprüfen, ob der Status für die Quellbereinigung bereit ist:
+
[source, cli]
----
vserver migrate show
----
. Quell-Cluster bereinigen:
+
[source, cli]
----
vserver migrate source-cleanup -vserver <SVM_name>
----
