= Entfernen Sie die Quell- ONTAP -SVM manuell nach der Clientumstellung.
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Wenn Sie die SVM-Migration bei deaktivierter Quellbereinigung durchgeführt haben, können Sie die Quell-SVM nach Abschluss der Client-Umstellung manuell entfernen.

.Schritte
. Prüfen Sie, ob der Status für die Quellcodebereinigung bereit ist:
+
[source, cli]
----
vserver migrate show
----
. Reinigen der Quelle:
+
[source, cli]
----
vserver migrate source-cleanup -vserver <SVM_name>
----
