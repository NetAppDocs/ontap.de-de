= Manuelle Umstellung der Clients nach der Migration einer ONTAP SVM
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Standardmäßig wird die Client-Umstellung auf den Ziel-Cluster automatisch durchgeführt, sobald die SVM-Migration den Status „ready-for-cutover“ erreicht hat. Wenn die automatische Client-Umstellung deaktiviert wird, ist die Client-Umstellung manuell durchzuführen.

.Schritte
. Manuelle Clientumstellung:
+
[source, cli]
----
vserver migrate cutover -vserver <SVM_name>
----
. Status des Umstellungsvorgangs prüfen:
+
[source, cli]
----
vserver migrate show
----
