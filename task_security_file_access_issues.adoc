= Diagnose und Behebung von ONTAP Dateizugriffsproblemen mit System Manager
:hardbreaks:
:toclevels: 1
:allow-uri-read: 
:toclevels: 1
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
Ab ONTAP 9.8 lassen sich Bedenken beim Dateizugriff nachverfolgen und anzeigen.

.Schritte
. Im System Manager unter *Storage > Storage-VMs* auswählen.
. Die Storage-VM auswählen, auf der eine Ablaufverfolgung durchgeführt werden soll.
. Klicken Sie auf image:icon_kabob.gif["Symbol für Menüoptionen"]*Mehr*.
. *Dateizugriff nachverfolgen* auswählen.
. Den Benutzernamen und die Client-IP-Adresse angeben, anschließend auf *Start Tracing* klicken.
+
Die Ergebnisse der Ablaufverfolgung werden in einer Tabelle angezeigt. Die Spalte *Gründe* gibt den Grund an, warum auf eine Datei nicht zugegriffen werden konnte.

. In der linken Spalte der Ergebnistabelle auf image:icon_dropdown_arrow.gif["Dropdown-Symbol"] klicken, um die Dateizugriffsberechtigungen anzuzeigen.
