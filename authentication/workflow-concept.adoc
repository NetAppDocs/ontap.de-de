= ONTAP Administrator-Authentifizierung und RBAC-Workflow
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können die Authentifizierung für lokale Administratorkonten oder Remote-Administratorkonten aktivieren. Die Kontoinformationen eines lokalen Kontos befinden sich auf dem Speichersystem und die Kontoinformationen eines Remote-Kontos befinden sich an einem anderen Ort. Jedem Konto kann eine vordefinierte Rolle oder eine benutzerdefinierte Rolle zugewiesen werden.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-1.png["Eins"]Vollständiges Konfigurationsarbeitsblatt
[role="quick-margin-para"]
Vor dem Erstellen von Anmeldekonten und dem Einrichten der rollenbasierten Zugriffssteuerung (RBAC) sollten Informationen zu jedem Element in der link:config-worksheets-reference.html["Konfigurationsarbeitsblätter"] gesammelt werden.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-2.png["Zwei"]Es wird festgestellt, ob das Administratorkonto lokal oder remote ist.
[role="quick-margin-list"]
* *Bei lokaler Installation:* link:enable-password-account-access-task.html["Passwort"], link:enable-ssh-public-key-accounts-task.html["SSH"], link:mfa-overview.html["SSH MFA"] oder link:enable-ssl-certificate-accounts-task.html["SSL"] Zugriff aktivieren.
* *Bei Fernzugriff:* Die Art des Fernzugriffs bestimmen. Abhängig vom Zugriffstyp link:grant-access-active-directory-users-groups-task.html["Active Directory Zugriff aktivieren"], link:grant-access-nis-ldap-user-accounts-task.html["LDAP- oder NIS-Zugriff ermöglichen"] oder link:../system-admin/configure-saml-authentication-task.html["SAML-Authentifizierung (nur für Admin SVM) konfigurieren"].


.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-3.png["Drei"]Rollenbasierte Zugriffssteuerung einrichten
[role="quick-margin-para"]
Die einem Administrator zugewiesene Rolle bestimmt, auf welche Befehle der Administrator Zugriff hat. Die Rolle wird bei der Erstellung des Administratorkontos zugewiesen und kann link:modify-role-assigned-administrator-task.html["modifiziert"] später geändert werden. Vordefinierte Rollen können für link:predefined-roles-cluster-administrators-concept.html["Cluster"] und link:predefined-roles-svm-administrators-concept.html["SVM"] Administratoren verwendet werden oder link:define-custom-roles-task.html["Benutzerdefinierte Rollen festlegen"] nach Bedarf angepasst werden.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-4.png["Vier"]Administratorkonten verwalten
[role="quick-margin-para"]
Je nachdem, wie Sie den Kontozugriff aktiviert haben, kann es erforderlich sein, einen link:manage-public-key-authentication-concept.html["öffentlicher Schlüssel mit einem lokalen Konto"] zuzuordnen, link:manage-ssh-public-keys-and-certificates.html["öffentliche Schlüssel und X.509-Zertifikate"] zu verwalten, link:configure-cisco-duo-mfa-task.html["Cisco Duo 2FA für SSH Logins"] zu konfigurieren, einen link:install-server-certificate-cluster-svm-ssl-server-task.html["CA-signiertes digitales Serverzertifikat"] zu installieren oder link:enable-ad-users-groups-access-cluster-svm-task.html["Active Directory"] sowie link:enable-nis-ldap-users-access-cluster-task.html["LDAP oder NIS"] Zugriff zu konfigurieren. Jede dieser Aufgaben kann vor oder nach der Aktivierung des Kontozugriffs erfolgen.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-5.png["Fünf"]Zusätzliche Sicherheitsfunktionen konfigurieren
[role="quick-margin-list"]
* link:../multi-admin-verify/index.html["Multi-Admin-Verifizierung verwalten"]wenn sichergestellt werden soll, dass bestimmte Vorgänge die Genehmigung von festgelegten Administratoren erfordern.
* link:dynamic-authorization-overview.html["Dynamische Autorisierung verwalten"]wenn zusätzliche Autorisierungsprüfungen dynamisch basierend auf dem Vertrauensniveau eines Benutzers angewendet werden sollen.
* link:configure-jit-elevation-task.html["Just-in-Time (JIT)-Berechtigungserweiterung konfigurieren"]wenn Benutzern vorübergehend erhöhte Berechtigungen zum Ausführen bestimmter Aufgaben gewährt werden sollen.
