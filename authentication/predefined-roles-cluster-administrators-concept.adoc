= Vordefinierte Rollen für ONTAP Clusteradministratoren
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Die vordefinierten Rollen für Clusteradministratoren sollten die meisten Ihrer Anforderungen abdecken. Bei Bedarf können benutzerdefinierte Rollen erstellt werden. Standardmäßig wird einem Clusteradministrator die vordefinierte  `admin` Rolle zugewiesen.

NOTE: Die `autosupport` Rolle ist dem vordefinierten `autosupport` Konto zugewiesen, das von AutoSupport OnDemand verwendet wird. ONTAP verhindert, dass das `autosupport` Konto geändert oder gelöscht wird. ONTAP verhindert außerdem, dass die `autosupport` Rolle anderen Benutzerkonten zugewiesen wird.
