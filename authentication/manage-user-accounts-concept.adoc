= Informationen zur Verwaltung von ONTAP Administratoraccounts
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Je nachdem, wie Sie den Kontozugriff aktiviert haben, kann es erforderlich sein, einen öffentlichen Schlüssel mit einem lokalen Konto zu verknüpfen, ein von einer Zertifizierungsstelle signiertes Server-Digitalzertifikat zu installieren oder den Zugriff auf AD, LDAP oder NIS zu konfigurieren. Alle diese Aufgaben können vor oder nach der Aktivierung des Kontozugriffs ausgeführt werden.
