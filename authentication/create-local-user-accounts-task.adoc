= Informationen zur Aktivierung des lokalen ONTAP Kontozugriffs
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Bei einem lokalen Konto befinden sich die Kontoinformationen, der öffentliche Schlüssel oder das Sicherheitszertifikat auf dem Speichersystem. Der  `security login create` Befehl kann verwendet werden, um lokalen Konten den Zugriff auf eine Admin- oder Daten-SVM zu ermöglichen.

.Verwandte Informationen
* link:https://docs.netapp.com/us-en/ontap-cli/security-login-create.html["security login create"^]
