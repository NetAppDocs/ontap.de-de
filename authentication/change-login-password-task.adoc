= Ändern eines ONTAP Administratorpassworts
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie sollten Ihr anfängliches Passwort unmittelbar nach der ersten Anmeldung im System ändern. Wenn Sie SVM-Administrator sind, können Sie den  `security login password` Befehl verwenden, um Ihr eigenes Passwort zu ändern. Wenn Sie Cluster-Administrator sind, können Sie den  `security login password` Befehl verwenden, um das Passwort eines beliebigen Administrators zu ändern.

.Über diese Aufgabe
Das neue Passwort muss folgende Regeln erfüllen:

* Es darf den Benutzernamen nicht enthalten
* Es muss mindestens acht Zeichen lang sein
* Es muss mindestens einen Buchstaben und eine Zahl enthalten
* Es kann nicht dasselbe sein wie die letzten sechs Passwörter



NOTE: Der  `security login role config modify`Befehl kann verwendet werden, um die Passwortregeln für Konten zu ändern, die einer bestimmten Rolle zugeordnet sind.

.Bevor Sie beginnen
* Sie müssen Cluster- oder SVM-Administrator sein, um Ihr eigenes Passwort zu ändern.
* Sie müssen Cluster-Administrator sein, um das Passwort eines anderen Administrators zu ändern.


.Schritt
. Ändern eines Administratorpassworts: `security login password -vserver _svm_name_ -username _user_name_`
+
Der folgende Befehl ändert das Kennwort des Administrators `admin1` für die SVM``vs1.example.com``. Es erscheint eine Aufforderung, das aktuelle Kennwort einzugeben, danach das neue Kennwort einzugeben und erneut zu bestätigen.

+
[listing]
----
vs1.example.com::>security login password -vserver engData -username admin1
Please enter your current password:
Please enter a new password:
Please enter it again:
----
