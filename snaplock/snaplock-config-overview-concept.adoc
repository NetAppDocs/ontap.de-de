= Informationen zur Konfiguration von ONTAP SnapLock
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Bevor Sie SnapLock verwenden, ist es erforderlich, SnapLock zu konfigurieren, indem verschiedene Aufgaben ausgeführt werden, wie zum Beispiel link:../system-admin/install-license-task.html["die SnapLock-Lizenz installieren"] für jeden Knoten, der ein Aggregat mit einem SnapLock Volume hostet, die Initialisierung des link:../snaplock/initialize-complianceclock-task.html["Compliance Clock"], das Erstellen eines SnapLock Aggregats für Cluster, auf denen ONTAP Versionen vor ONTAP 9.10.1 ausgeführt werden, link:../snaplock/create-snaplock-volume-task.html["Ein SnapLock Volume erstellen und einbinden"] und weitere Aufgaben.
