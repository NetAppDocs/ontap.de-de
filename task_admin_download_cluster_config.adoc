= Eine Clusterkonfiguration mit ONTAP System Manager herunterladen
:hardbreaks:
:toclevels: 1
:allow-uri-read: 
:toclevels: 1
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
Ab ONTAP 9.11.1 kann mit dem System Manager eine Auswahl an Konfigurationsdetails zum Cluster und seinen Nodes heruntergeladen werden. Diese Informationen können für die Bestandsverwaltung, den Hardwareaustausch und Lebenszyklusaktivitäten genutzt werden. Diese Informationen sind besonders hilfreich für Standorte, die keine AutoSupport (ASUP) Daten senden.

Zu den Clusterkonfigurationsdetails gehören der Clustername, die Cluster ONTAP Version, die Cluster-Management-LIF, das Volume und die Anzahl der LIF.

Zu den Konfigurationsdetails des Knotens gehören der Node-Name, die Systemseriennummer, die System-ID, das Systemmodell, die ONTAP Version, MetroCluster-Informationen, SP/BMC-Netzwerkinformationen und Informationen zur Verschlüsselungskonfiguration.

.Schritte
. *Cluster > Übersicht* auswählen.
. Klicken Sie auf image:icon-more-kebab-blue-bg.gif["Symbol „Mehr Optionen“"], um das Dropdown-Menü anzuzeigen.
. *Download configuration* auswählen.
. Wählen Sie die HA Paare aus, dann auf *Herunterladen* klicken.
+
Die Konfiguration wird als Excel-Tabelle heruntergeladen.

+
** Das erste Blatt enthält Clusterdetails.
** Die anderen Blätter enthalten Knotendetails.
