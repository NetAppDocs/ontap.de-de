= Entfernen Sie die zweiten S3-Zugriffsschlüssel, bevor Sie von ONTAP 9.19.1 zurückkehren.
:allow-uri-read: 


[role="lead"]
Wenn Sie auf ONTAP 9.19.1 aktualisiert und einem Benutzer einen zweiten S3-Zugriffsschlüssel hinzugefügt haben, müssen diese Schlüssel entfernt werden, bevor Sie zu einer früheren ONTAP-Version zurückkehren können.

.Schritte
. link:../s3-config/regenerate-access-keys.html#show-s3-access-keys["S3-Zugriffsschlüssel anzeigen"] um festzustellen, ob für S3-Benutzer zweite Zugriffsschlüssel hinzugefügt wurden.
. link:../s3-config/regenerate-access-keys.html#remove-an-s3-access-key["Entfernen Sie die zweiten Zugriffsschlüssel"] für alle S3-Nutzer.
