= Unterstützte ONTAP Revert-Pfade
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können Ihre ONTAP Software nur direkt auf eine Version zurücksetzen, die eine Version älter ist als Ihre aktuelle ONTAP Version. Wenn Sie beispielsweise Version 9.15.1 verwenden, ist ein direktes Zurücksetzen auf 9.13.1 nicht möglich. Zuerst muss auf 9.14.1 zurückgesetzt werden, danach erfolgt ein separates Zurücksetzen von 9.14.1 auf 9.13.1.

Die Rückkehr zu ONTAP 9.4 oder einer früheren Version wird nicht unterstützt. Ein Zurücksetzen auf nicht unterstützte ONTAP Versionen sollte vermieden werden.

Sie können den  `system image show` Befehl verwenden, um die auf jedem Knoten ausgeführte ONTAP-Version zu ermitteln.

Die folgenden unterstützten Wiederherstellungspfade beziehen sich nur auf lokale ONTAP Versionen. Informationen zur Wiederherstellung von ONTAP in der Cloud sind unter  https://docs.netapp.com/us-en/cloud-manager-cloud-volumes-ontap/task-updating-ontap-cloud.html#reverting-or-downgrading["Zurücksetzen oder Herabstufen von Cloud Volumes ONTAP"^] zu finden.


NOTE: link:https://docs.netapp.com/us-en/ontap-afx/administer/revert-ontap.html["AFX Storage-Systeme"^]Das Zurücksetzen der ONTAP Software wird nicht unterstützt.
