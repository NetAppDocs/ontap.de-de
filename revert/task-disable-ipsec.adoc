= IPsec in MetroCluster Konfigurationen vor dem Zurücksetzen von ONTAP 9.12.1 deaktivieren
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Bevor eine MetroCluster Konfiguration von ONTAP 9.12.1 zurückgesetzt wird, muss IPsec deaktiviert werden.

Vor dem Zurücksetzen wird geprüft, ob innerhalb der MetroCluster Konfiguration IPsec-Konfigurationen vorhanden sind. Alle vorhandenen IPsec-Konfigurationen müssen entfernt und IPsec muss deaktiviert werden, bevor mit dem Zurücksetzen fortgefahren werden kann. Das Zurücksetzen von ONTAP ist blockiert, wenn IPsec aktiviert ist, selbst wenn keine Benutzerrichtlinien konfiguriert wurden.
