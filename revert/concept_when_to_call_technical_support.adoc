= Wird technischer Support benötigt, um einen ONTAP Cluster zurückzusetzen?
:allow-uri-read: 


[role="lead"]
Sie sollten sich in den folgenden Situationen an den technischen Support wenden, bevor Sie versuchen, einen ONTAP Cluster zurückzusetzen:

* Eine Produktionsumgebung
+
Ein Produktionscluster sollte nicht ohne Unterstützung des technischen Supports zurückgesetzt werden.

* Sie haben Volumes in ONTAP 9.5 oder einer späteren Version erstellt und müssen auf eine frühere Version zurückkehren.
+
Volumes, die adaptive Komprimierung verwenden, müssen vor dem Zurücksetzen dekomprimiert werden.



Sie können neue oder Testcluster ohne Unterstützung zurücksetzen. Wenn beim Versuch, einen Cluster eigenständig zurückzusetzen, eines der folgenden Probleme auftritt, sollte der technische Support kontaktiert werden:

* Der Rücksetzvorgang schlägt fehl oder kann nicht abgeschlossen werden.
* Der Rücksetzvorgang wird abgeschlossen, aber der Cluster ist in einer Produktionsumgebung nicht verwendbar.
* Der Rücksetzvorgang wird abgeschlossen und der Cluster geht in die Produktion, aber Sie sind mit seinem Verhalten nicht zufrieden.
