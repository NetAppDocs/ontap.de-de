= CORS-Konfiguration vor dem Zurücksetzen von ONTAP 9.16.1 entfernen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Wenn Sie Cross-Origin Resource Sharing (CORS) für den Zugriff auf ONTAP S3 Buckets verwenden, muss dies entfernt werden, bevor Sie von ONTAP 9.16.1 zurückkehren.

Weitere Informationen zu link:../s3-config/cors-integration.html#upgrading-and-reverting["Zurücksetzen von ONTAP Clustern mit CORS"].
