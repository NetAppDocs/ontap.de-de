= Automatische Umschaltung für MetroCluster Konfigurationen nach einer ONTAP Rücksetzung aktivieren
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Nach dem Zurücksetzen einer ONTAP MetroCluster Konfiguration muss die automatische ungeplante Umschaltung aktiviert werden, damit die MetroCluster Konfiguration voll funktionsfähig ist.

.Schritte
. Automatische ungeplante Umschaltung aktivieren:
+
[source, cli]
----
metrocluster modify -auto-switchover-failure-domain auso-on-cluster-disaster
----
. Die MetroCluster Konfiguration validieren:
+
[source, cli]
----
metrocluster check run
----
