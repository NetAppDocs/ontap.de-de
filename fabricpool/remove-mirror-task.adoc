= Einen ONTAP FabricPool Spiegel entfernen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Ein FabricPool Spiegel kann entfernt werden, wenn keine Replikation eines Objektspeichers mehr erforderlich ist.

.Bevor Sie beginnen
Der primäre Objektspeicher muss betriebsbereit sein, andernfalls schlägt der Befehl fehl.

.Schritt
. Einen Objektspeicher-Spiegel in einem FabricPool mit dem  `storage aggregate object-store unmirror -aggregate` Befehl entfernen.
+
[listing]
----
cluster1::> storage aggregate object-store unmirror -aggregate aggr1
----


.Verwandte Informationen
* link:https://docs.netapp.com/us-en/ontap-cli/storage-aggregate-object-store-unmirror.html["storage aggregate Objektspeicher Spiegelung aufheben"^]
