= Einstieg in ONTAP FabricPool
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Die Konfiguration von FabricPool hilft dabei, zu steuern, auf welcher Speicherebene (lokale Performance-Ebene oder Cloud-Ebene) Daten gespeichert werden sollten, abhängig davon, ob häufig auf die Daten zugegriffen wird.

Die für die FabricPool Konfiguration erforderliche Vorbereitung hängt vom Objektspeicher ab, der als Cloud-Tier verwendet wird.
