= Google Cloud Storage als ONTAP FabricPool Cloud-Tier einrichten
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Wenn Sie ONTAP 9.6 oder höher verwenden, kann Google Cloud Storage als Cloud-Ebene für FabricPool eingerichtet werden.



[[additional-considerations-for-using-google-cloud-storage-with-fabricpool]]
== Weitere Überlegungen zur Verwendung von Google Cloud Storage mit FabricPool

* Ein link:install-license-aws-azure-ibm-task.html["FabricPool Lizenz"] ist erforderlich, wenn von lokalen Clustern zu Google Cloud Storage getiert wird.
* Es wird empfohlen, dass der LIF, den ONTAP zur Verbindung mit dem Google Cloud Storage Objektspeicher verwendet, an einem 10 Gbit/s Port angeschlossen ist.
* Auf AFF und FAS Systemen sowie ONTAP Select unterstützt FabricPool die folgenden Google Cloud Objektspeicherklassen:
+
** Google Cloud Multi-Regional
** Google Cloud Regional
** Google Cloud Nearline
** Google Cloud Coldline
+
https://cloud.google.com/storage/docs/storage-classes["Google Cloud: Speicherklassen"^]





.Schritte
. Die Konfigurationsinformationen für Google Cloud Storage werden mit dem  `storage aggregate object-store config create` Befehl und dem  `-provider-type` `GoogleCloud` Parameter angegeben.
+
** Der  `storage aggregate object-store config create` Befehl schlägt fehl, wenn ONTAP mit den angegebenen Informationen nicht auf Google Cloud Storage zugreifen kann.
** Der  `-access-key`Parameter wird verwendet, um den Zugriffsschlüssel für die Autorisierung von Anfragen an den Google Cloud Storage Objektspeicher anzugeben.
** Wenn das Passwort für Google Cloud Storage geändert wird, ist das entsprechende in ONTAP gespeicherte Passwort umgehend zu aktualisieren.
+
Dadurch kann ONTAP ohne Unterbrechung auf die Daten in Google Cloud Storage zugreifen.



+
[listing]
----
storage aggregate object-store config create my_gcp_store_1 -provider-type GoogleCloud  -container-name my-gcp-bucket1 -access-key GOOGAUZZUV2USCFGHGQ511I8
----
. Die Konfigurationsinformationen von Google Cloud Storage lassen sich mit dem  `storage aggregate object-store config show` Befehl anzeigen und überprüfen.
+
Der  `storage aggregate object-store config modify`Befehl ermöglicht die Änderung der Google Cloud Storage-Konfigurationsinformationen für FabricPool.
