= Informationen zu ONTAP FabricPool Spiegeln
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Um sicherzustellen, dass Daten in Datenspeichern im Katastrophenfall zugänglich sind und um den Austausch eines Datenspeichers zu ermöglichen, kann eine FabricPool Spiegelung konfiguriert werden, indem ein zweiter Datenspeicher hinzugefügt wird, sodass Daten synchron auf zwei Datenspeicher gestuft werden. Ein zweiter Datenspeicher kann zu neuen oder bestehenden FabricPool Konfigurationen hinzugefügt, der Spiegelungsstatus überwacht, FabricPool Spiegelungsdetails angezeigt, eine Spiegelung hochgestuft und eine Spiegelung entfernt werden. ONTAP 9.7 oder höher muss ausgeführt werden.
