= Details zum ONTAP FabricPool Spiegel anzeigen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Es lassen sich Details zu einem FabricPool Spiegel anzeigen, um zu sehen, welche Objektspeicher in der Konfiguration enthalten sind und ob der Objektspeicher Spiegel mit dem primären Objektspeicher synchron ist.

.Schritt
. Informationen über einen FabricPool Spiegel werden mit dem  `storage aggregate object-store show` Befehl angezeigt.
+
Dieses Beispiel zeigt die Details zu den primären und gespiegelten Objektspeichern in einem FabricPool Spiegel.

+
[listing]
----
cluster1::> storage aggregate object-store show
----
+
[listing]
----
Aggregate      Object Store Name Availability    Mirror Type
-------------- ----------------- -------------   ----------
aggr1          my-store-1        available       primary
               my-store-2        available       mirror
----
+
Dieses Beispiel zeigt Details zum FabricPool Spiegel an, einschließlich der Information, ob der Spiegel aufgrund einer Resync-Operation beeinträchtigt ist.

+
[listing]
----
cluster1::> storage aggregate object-store show -fields mirror-type,is-mirror-degraded
----
+
[listing]
----
aggregate      object-store-name mirror-type     is-mirror-degraded
-------------- ----------------- ------------- ------------------
aggr1          my-store-1        primary           -
               my-store-2        mirror          false
----


.Verwandte Informationen
