= Alibaba Cloud Object Storage als ONTAP FabricPool Cloud-Tier einrichten
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Wenn Sie ONTAP 9.6 oder höher verwenden, kann Alibaba Cloud Object Storage als Cloud-Tier für FabricPool eingerichtet werden.

.Überlegungen zur Verwendung von Alibaba Cloud Object Storage mit FabricPool
* A link:install-license-aws-azure-ibm-task.html["FabricPool Lizenz"] ist erforderlich, wenn von lokalen ONTAP Clustern zu Alibaba Cloud Object Storage migriert wird.
* Auf AFF und FAS Systemen sowie ONTAP Select unterstützt FabricPool die folgenden Alibaba Object Storage Service Klassen:
+
** Alibaba Object Storage Service Standard
** Alibaba Object Storage Service Infrequent Access
+
https://www.alibabacloud.com/help/doc-detail/51374.htm["Alibaba Cloud: Einführung in die Speicherklassen"^]





Ihr NetApp Vertriebsmitarbeiter kann Informationen zu nicht aufgeführten Speicherklassen bereitstellen.

.Schritte
. Die Konfigurationsinformationen für Alibaba Cloud Object Storage werden mit dem  `storage aggregate object-store config create` Befehl und dem  `-provider-type` `AliCloud` Parameter angegeben.
+
** Der  `storage aggregate object-store config create` Befehl schlägt fehl, wenn ONTAP mit den angegebenen Informationen nicht auf Alibaba Cloud Object Storage zugreifen kann.
** Der  `-access-key`Parameter wird verwendet, um den Zugriffsschlüssel für die Autorisierung von Anfragen an den Alibaba Cloud Object Storage Objektspeicher anzugeben.
** Wenn das Passwort für Alibaba Cloud Object Storage geändert wird, ist das entsprechende in ONTAP gespeicherte Passwort umgehend zu aktualisieren.
+
Dadurch kann ONTAP ohne Unterbrechung auf die Daten im Alibaba Cloud Object Storage zugreifen.



+
[listing]
----
storage aggregate object-store config create my_ali_oss_store_1
-provider-type AliCloud -server oss-us-east-1.aliyuncs.com
-container-name my-ali-oss-bucket -access-key DXJRXHPXHYXA9X31X3JX
----
. Die Konfigurationsinformationen von Alibaba Cloud Object Storage lassen sich mit dem  `storage aggregate object-store config show` Befehl anzeigen und überprüfen.
+
Der  `storage aggregate object-store config modify`Befehl ermöglicht die Änderung der Alibaba Cloud Object Storage-Konfigurationsinformationen für FabricPool.
