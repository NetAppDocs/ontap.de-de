= Installieren Sie eine FabricPool-Lizenz auf einem ONTAP-Cluster
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
NetApp Cloud Tiering ist ab dem 26. April 2026 nicht mehr zum Kauf oder zur Lizenzverlängerung verfügbar. Wenn Sie NetApp Cloud Tiering verwenden, empfiehlt NetApp, dass Sie mit Ihrem NetApp Ansprechpartner zusammenarbeiten, um Ihre bestehenden Tiering-Lizenzen auf ONTAP FabricPool Lizenzen umzustellen.

Bestehende Kunden können NetApp Cloud Tiering weiterhin nutzen und Support dafür erhalten, bis ihr Abonnement oder Lizenzvertrag abläuft. Nach Ablauf des Abonnements haben Sie keinen Zugriff mehr auf NetApp Cloud Tiering-Funktionen oder Support.

Wenn Sie derzeit Cloud-Tiering-Lizenzierung nutzen, sind Sie erst betroffen, wenn Ihre Cloud-Tiering-Lizenz abläuft oder ihre maximale Kapazität erreicht. Kontaktieren Sie NetApp, wenn Sie Ihre Lizenz aktualisieren müssen oder auch früher, um sicherzustellen, dass Ihre Fähigkeit, Daten in die Cloud zu verschieben, nicht unterbrochen wird.

link:https://docs.netapp.com/us-en/data-services-cloud-tiering/task-licensing-cloud-tiering.html["Erfahren Sie mehr über NetApp Cloud-Tiering-Lizenzänderungen"^].

Sie können eine FabricPool-Lizenz mit dem System Manager herunterladen und aktivieren.

Bei der FabricPool Lizenz handelt es sich um eine Cluster-weite Lizenz. Es enthält ein berechtigtes Nutzungslimit, das Sie für Objekt-Storage erwerben, der mit FabricPool im Cluster verknüpft ist. Die Verwendung im Cluster darf die Kapazität des berechtigten Nutzungslimits nicht überschreiten. Wenn Sie die Nutzungsbeschränkung der Lizenz erhöhen müssen, sollten Sie sich an Ihren Vertriebsmitarbeiter wenden.

FabricPool-Lizenzen sind im unbefristeten oder langfristigen Format von 1 oder 3 Jahren erhältlich.

Eine zeitbasierte FabricPool-Lizenz mit 10 TB kostenlosem Speicherplatz ist für Erstbestellungen von FabricPool für bestehende Clusterkonfigurationen verfügbar. Kostenloser Speicherplatz ist bei unbefristeten Lizenzen nicht verfügbar. Eine Lizenz ist nicht erforderlich, wenn Sie NetApp StorageGRID oder ONTAP S3 für die Cloud-Ebene verwenden. Cloud Volumes ONTAP benötigt unabhängig vom verwendeten Anbieter keine FabricPool-Lizenz.

.Bevor Sie beginnen
Sie benötigen Ihre Cluster-UUID, um Ihre NetApp License File (NLF) von der NetApp Support Site abzurufen.

.Schritte
Diese Aufgabe wird nur unterstützt, indem die Lizenzdatei mithilfe von System Manager auf das Cluster hochgeladen wird.

. Laden Sie die NetApp-Lizenzdatei (NLF) für die FabricPool-Lizenz von der herunterlink:https://mysupport.netapp.com/site/global/dashboard["NetApp Support-Website"^].
. Führen Sie die folgenden Aktionen mit System Manager durch, um die FabricPool Lizenz auf das Cluster hochzuladen:
+
.. Klicken Sie im Bereich *Cluster > Einstellungen* auf der *Lizenzen*-Karte auf image:icon_arrow.gif["Pfeilsymbol"].
.. Klicken Sie auf der Seite *Lizenz* auf image:icon_add.gif["Symbol hinzufügen"].
.. Klicken Sie im Dialogfeld *Lizenz hinzufügen* auf *Durchsuchen*, um die heruntergeladene Lizenzdatei auszuwählen, und klicken Sie dann auf *Hinzufügen*, um die Datei auf den Cluster hochzuladen.
