= Installieren einer FabricPool-Lizenz auf einem ONTAP Cluster
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
NetApp Cloud Tiering ist ab dem 26. April 2026 nicht mehr zum Kauf oder zur Lizenzverlängerung verfügbar. Wenn Sie NetApp Cloud Tiering verwenden, empfiehlt NetApp, dass Sie mit Ihrem NetApp Ansprechpartner zusammenarbeiten, um Ihre bestehenden Tiering-Lizenzen auf ONTAP FabricPool Lizenzen umzustellen.

Bestehende Kunden können NetApp Cloud Tiering weiterhin nutzen und Support dafür erhalten, bis ihr Abonnement oder Lizenzvertrag abläuft. Nach Ablauf des Abonnements haben Sie keinen Zugriff mehr auf NetApp Cloud Tiering-Funktionen oder Support.

Wenn Sie derzeit Cloud-Tiering-Lizenzierung nutzen, sind Sie erst betroffen, wenn Ihre Cloud-Tiering-Lizenz abläuft oder ihre maximale Kapazität erreicht. Kontaktieren Sie NetApp, wenn Sie Ihre Lizenz aktualisieren müssen oder auch früher, um sicherzustellen, dass Ihre Fähigkeit, Daten in die Cloud zu verschieben, nicht unterbrochen wird.

link:https://docs.netapp.com/us-en/data-services-cloud-tiering/task-licensing-cloud-tiering.html["Erfahren Sie mehr über NetApp Cloud-Tiering-Lizenzänderungen"^].

Sie können eine FabricPool-Lizenz mit dem System Manager herunterladen und aktivieren.

Die FabricPool-Lizenz ist eine clusterweite Lizenz. Sie umfasst ein berechtigtes Nutzungslimit, das für den Objektspeicher erworben wird, der im Cluster mit FabricPool verknüpft ist. Die Nutzung im gesamten Cluster darf die Kapazität des berechtigten Nutzungslimits nicht überschreiten. Für eine Erhöhung des Nutzungslimits der Lizenz ist der jeweilige Vertriebsmitarbeiter zu kontaktieren.

FabricPool Lizenzen sind als unbefristete oder befristete Lizenzen mit einer Laufzeit von 1 oder 3 Jahren erhältlich.

Eine Lizenz ist nicht erforderlich, wenn Sie NetApp StorageGRID oder ONTAP S3 für die Cloud-Ebene verwenden. Cloud Volumes ONTAP benötigt unabhängig vom verwendeten Anbieter keine FabricPool-Lizenz.

.Bevor Sie beginnen
Sie benötigen Ihre Cluster-UUID, um Ihre NetApp License File (NLF) von der NetApp Support Site abzurufen.

.Schritte
Diese Aufgabe wird nur unterstützt, wenn die Lizenzdatei mithilfe von System Manager in den Cluster hochgeladen wird.

. Laden Sie die NetApp License File (NLF) für die FabricPool Lizenz von der link:https://mysupport.netapp.com/site/global/dashboard["NetApp Support Site"^] herunter.
. Die folgenden Aktionen werden mit dem System Manager durchgeführt, um die FabricPool Lizenz auf das Cluster hochzuladen:
+
.. Im Bereich *Cluster > Einstellungen* auf der Karte *Lizenzen* auf image:icon_arrow.gif["Pfeilsymbol"] klicken.
.. Auf der Seite *Lizenz* auf image:icon_add.gif["Symbol hinzufügen"] klicken.
.. Im Dialogfeld *Lizenz hinzufügen* auf *Durchsuchen* klicken, um die heruntergeladene NLF auszuwählen, und anschließend auf *Hinzufügen* klicken, um die Datei in den Cluster hochzuladen.
