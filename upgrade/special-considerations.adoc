= Vor einem Upgrade auf spezifische ONTAP Konfigurationen prüfen.
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Bei bestimmten Clusterkonfigurationen sind vor einem ONTAP Software-Upgrade spezifische Maßnahmen erforderlich. Wenn beispielsweise eine SAN-Konfiguration vorhanden ist, sollte vor dem Upgrade überprüft werden, dass jeder Host mit der korrekten Anzahl direkter und indirekter Pfade konfiguriert ist.

Anhand der folgenden Tabelle lässt sich feststellen, welche zusätzlichen Schritte gegebenenfalls erforderlich sind.

|===

xref:concept_upgrade_requirements_for_metrocluster_configurations.html[Spezifische Upgrade-Anforderungen für MetroCluster Konfigurationen prüfen]
