= Das ONTAP Software-Image sollte vor einem Upgrade heruntergeladen werden.
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Bevor Sie ONTAP aktualisieren, muss zunächst das Ziel-ONTAP-Software-Image von der NetApp Support-Website heruntergeladen werden. Abhängig von Ihrer ONTAP-Version kann die ONTAP-Software auf einen HTTPS-, HTTP- oder FTP-Server in Ihrem Netzwerk oder in einen lokalen Ordner heruntergeladen werden.

[cols="2"]
|===
| Wenn Sie ausführen... | Das Bild kann an diesem Speicherort heruntergeladen werden... 


| ONTAP 9.6 und höher  a| 
* Ein HTTPS-Server + das CA-Zertifikat des Servers muss auf dem lokalen System installiert sein.
* Ein lokaler Ordner
* Ein HTTP oder FTP Server




| ONTAP 9.4 und höher  a| 
* Ein lokaler Ordner
* Ein HTTP oder FTP Server




| ONTAP 9.0 und höher | Ein HTTP oder FTP Server 
|===
.Über diese Aufgabe
* Wenn ein automatisiertes unterbrechungsfreies Upgrade (ANDU) mit einem link:concept_upgrade_paths.html#types-of-upgrade-paths["direkter Multi-Hop Upgrade-Pfad"] durchgeführt wird, ist es erforderlich, link:download-software-image.html["herunterladen"] das Softwarepaket sowohl für die zwischengeschaltete ONTAP Version als auch für die Ziel-ONTAP Version bereitzustellen, die für das Upgrade benötigt werden. Beispielsweise müssen beim Upgrade von ONTAP 9.8 auf ONTAP 9.13.1 die Softwarepakete für ONTAP 9.12.1 und ONTAP 9.13.1 heruntergeladen werden. Siehe link:concept_upgrade_paths.html#supported-upgrade-paths["Unterstützte Upgrade-Pfade"], um festzustellen, ob für Ihren Upgrade-Pfad das Herunterladen eines zwischengeschalteten Softwarepakets erforderlich ist.
* Wenn ein System mit NetApp Volume Encryption auf ONTAP 9.5 oder höher aktualisiert wird, muss das ONTAP Software-Image für nicht eingeschränkte Länder heruntergeladen werden, das NetApp Volume Encryption enthält.
+
Wenn das ONTAP Software-Image für eingeschränkte Länder zur Aktualisierung eines Systems mit NetApp Volume Encryption verwendet wird, tritt ein Systemabsturz auf und der Zugriff auf die Volumes geht verloren.

* Sie müssen kein separates Softwarepaket für Ihre Firmware herunterladen. Das Firmware-Update für Ihren Cluster ist im ONTAP Software-Upgrade-Paket enthalten und wird auf das Boot-Gerät jedes Knotens kopiert. Die neue Firmware wird anschließend im Rahmen des Upgrade-Prozesses installiert.


.Schritte
. Die gewünschte ONTAP Software befindet sich im link:https://mysupport.netapp.com/site/products/all/details/ontap9/downloads-tab["Software-Downloads"^] Bereich der NetApp Support Site.
+
Für ein ONTAP Select Upgrade ist *ONTAP Select Node Upgrade* auszuwählen.

. Die Software-Image-Datei (zum Beispiel 97_q_image.tgz) wird an den entsprechenden Speicherort kopiert.
+
Je nach Ihrer ONTAP Version ist der Speicherort entweder ein Verzeichnis auf einem HTTP-, HTTPS- oder FTP-Server, von dem das Image an das lokale System bereitgestellt wird, oder ein lokaler Ordner auf dem Speichersystem.
