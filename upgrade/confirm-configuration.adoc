= Bestätigung der Unterstützung der ONTAP-Zielversion für Ihre Hardwarekonfiguration
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Bevor Sie ONTAP aktualisieren, sollte bestätigt werden, dass Ihre Hardwarekonfiguration die Zielversion von ONTAP unterstützt.



[[all-configurations]]
== Alle Konfigurationen

Mit  https://hwu.netapp.com["NetApp Hardware Universe"^] lässt sich bestätigen, dass Ihre Hardwareplattform sowie Ihre Cluster- und Management-Switches in der Zielversion von ONTAP unterstützt werden.

Die Version von ONTAP, auf die Sie ein Upgrade durchführen können, ist möglicherweise durch Ihre Hardwarekonfiguration eingeschränkt. Wenn Ihre Hardware die ONTAP-Softwareversion, auf die Sie ein Upgrade durchführen möchten, nicht unterstützt, müssen zunächst neue Knoten zu Ihrem Cluster hinzugefügt, Ihre Daten migriert, die älteren Knoten entfernt und anschließend Ihre ONTAP-Software aktualisiert werden. Das Verfahren ist zu befolgen, um link:concept_mixed_version_requirements.html#adding-new-nodes-to-an-ontap-cluster["Neue Knoten zu einem ONTAP Cluster hinzufügen"].

Zu den Cluster- und Management-Switches gehören die Cluster-Netzwerk-Switches (NX-OS), die Management-Netzwerk-Switches (IOS) und die Referenz-Konfigurationsdatei (RCF). Wenn Ihre Cluster- und Management-Switches unterstützt werden, aber nicht die für die Zielversion von ONTAP erforderlichen Mindestsoftwareversionen ausführen, sollten die Switches auf unterstützte Softwareversionen aktualisiert werden.

* https://mysupport.netapp.com/site/info/broadcom-cluster-switch["NetApp Downloads: Broadcom Cluster-Switches"^]
* https://mysupport.netapp.com/site/info/cisco-ethernet-switch["NetApp Downloads: Cisco Ethernet-Switches"^]
* https://mysupport.netapp.com/site/info/netapp-cluster-switch["NetApp Downloads: NetApp Cluster-Switches"^]



NOTE: Falls Sie Ihre Switches aktualisieren müssen, NetApp empfiehlt, zunächst das ONTAP-Software-Upgrade durchzuführen und anschließend das Software-Upgrade für Ihre Switches vorzunehmen.



[[metrocluster-configurations]]
== MetroCluster Konfigurationen

Bevor ONTAP aktualisiert wird, kann bei einer MetroCluster Konfiguration das  https://mysupport.netapp.com/matrix["NetApp Interoperabilitätsmatrix-Tool"^] verwendet werden, um zu bestätigen, dass die MetroCluster IP Switches in der Zielversion von ONTAP unterstützt werden.



[[san-configurations]]
== SAN Konfigurationen

Bevor ONTAP aktualisiert wird, kann mit dem  https://mysupport.netapp.com/matrix["NetApp Interoperabilitätsmatrix-Tool"^] überprüft werden, ob die SAN-Konfiguration bei einem für SAN konfigurierten Cluster vollständig unterstützt wird.

Alle SAN-Komponenten, einschließlich der Ziel ONTAP Softwareversion, des Host-Betriebssystems und der Patches, der erforderlichen Host Utilities Software, der Multipathing-Software sowie der Adaptertreiber und Firmware, sollten unterstützt werden.
