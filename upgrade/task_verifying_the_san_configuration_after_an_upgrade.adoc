= Die SAN-Konfiguration nach einem ONTAP Upgrade überprüfen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Nach einem ONTAP Upgrade in einer SAN-Umgebung ist zu überprüfen, ob jeder Initiator, der vor dem Upgrade mit einem LIF verbunden war, erfolgreich wieder mit dem LIF verbunden ist.

. Es sollte sichergestellt werden, dass jeder Initiator mit dem richtigen LIF verbunden ist.
+
Die Liste der Initiatoren sollte mit der Liste verglichen werden, die während der Upgrade-Vorbereitung erstellt wurde. Bei Verwendung von ONTAP 9.11.1 oder höher empfiehlt sich System Manager zur Anzeige des Verbindungsstatus, da dieser eine deutlich übersichtlichere Darstellung als die CLI bietet.

+
[role="tabbed-block"]
====
.System Manager
--
.. Im System Manager auf *Hosts > SAN Initiatorgruppen* klicken.
+
Auf der Seite wird eine Liste der Initiatorgruppen (Initiatorgruppen) angezeigt. Wenn die Liste lang ist, können zusätzliche Seiten der Liste angezeigt werden, indem auf die Seitenzahlen in der unteren rechten Ecke der Seite geklickt wird.

+
Die Spalten zeigen verschiedene Informationen zu den Initiatorgruppen an. Ab Version 9.11.1 wird auch der Verbindungsstatus der Initiatorgruppe angezeigt. Durch das Überfahren der Statusmeldungen mit dem Mauszeiger werden Details angezeigt.



--
.CLI
--
** Liste der iSCSI-Initiatoren:
+
[source, cli]
----
iscsi initiator show -fields igroup,initiator-name,tpgroup
----
** Liste der FC-Initiatoren:
+
[source, cli]
----
fcp initiator show -fields igroup,wwpn,lif
----
