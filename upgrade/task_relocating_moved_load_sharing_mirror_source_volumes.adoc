= Verschobene Load-Sharing Mirror-Quell-Volumes nach einem ONTAP Upgrade neu zuordnen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Nach dem Upgrade von ONTAP ist es erforderlich, die Load-Sharing-Mirror-Quell-Volumes wieder an ihre Speicherorte vor dem Upgrade zu verschieben.

.Schritte
. Den Zielort, an den das Load-Sharing Mirror-Quellvolume verschoben wird, anhand des zuvor erstellten Datensatzes für das Load-Sharing Mirror-Quellvolume bestimmen.
. Das Load-Sharing-Mirror-Quellvolume wird an seinen ursprünglichen Speicherort zurück verschoben:
+
[source, cli]
----
volume move start
----
