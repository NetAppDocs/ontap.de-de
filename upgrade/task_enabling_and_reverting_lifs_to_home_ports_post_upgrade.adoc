= Nach dem ONTAP Upgrade befindet sich jeder LIF an seinem Home-Port.
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Beim Neustart im Rahmen des ONTAP Upgrade-Prozesses werden einige LIFs möglicherweise von ihren Home-Ports auf ihre zugewiesenen Failover-Ports migriert. Nach einem Upgrade ist es erforderlich, alle LIFs, die sich nicht auf ihren Home-Ports befinden, zu aktivieren und auf ihre Home-Ports zurückzusetzen.

.Schritte
. Status aller LIFs anzeigen:
+
[source, cli]
----
network interface show -fields home-port,curr-port
----
+
Wenn *Status Admin* für einen LIF den Wert „down“ oder *Is home* den Wert „false“ hat, mit dem nächsten Schritt fortfahren.

. Die Daten-LIFs aktivieren:
+
[source, cli]
----
network interface modify {-role data} -status-admin up
----
. LIFs auf ihre Home-Ports zurückführen:
+
[source, cli]
----
network interface revert *
----
. Es sollte sichergestellt werden, dass sich alle LIFs in ihren Home-Ports befinden:
+
[source, cli]
----
network interface show
----
+
Dieses Beispiel zeigt, dass sich alle LIFs für SVM vs0 auf ihren Home-Ports befinden.
