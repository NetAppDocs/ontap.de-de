= Vor dem Upgrade von ONTAP vorhandene externe Key-Management-Server-Verbindungen löschen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Bevor ONTAP aktualisiert wird, falls ONTAP 9.2 oder eine frühere Version mit NetApp Storage Encryption (NSE) verwendet wird und ein Upgrade auf ONTAP 9.3 oder höher erfolgt, muss die Befehlszeilenschnittstelle (CLI) verwendet werden, um alle vorhandenen externen Key Management (KMIP) Serververbindungen zu löschen.

.Schritte
. Es sollte sichergestellt werden, dass die NSE-Laufwerke entsperrt, geöffnet und auf die Standard-Hersteller-Sicherheits-ID 0x0 gesetzt sind:
+
[source, cli]
----
storage encryption disk show -disk *
----
+
Weitere Informationen zu  `storage encryption disk show` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/storage-encryption-disk-show.html["ONTAP-Befehlsreferenz"^].

. In den erweiterten Berechtigungsmodus wechseln:
+
[source, cli]
----
set -privilege advanced
----
+
Weitere Informationen zu  `set` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/set.html["ONTAP-Befehlsreferenz"^].

. Die standardmäßige Hersteller-Sicherheits-ID 0x0 dient dazu, den FIPS-Schlüssel den selbstverschlüsselnden Festplatten (SEDs) zuzuweisen:
+
[source, cli]
----
storage encryption disk modify -fips-key-id 0x0 -disk *
----
+
Weitere Informationen zu  `storage encryption disk modify` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/storage-encryption-disk-modify.html["ONTAP-Befehlsreferenz"^].

. Es sollte überprüft werden, ob die Zuweisung des FIPS-Schlüssels zu allen Datenträgern abgeschlossen ist:
+
[source, cli]
----
storage encryption disk show-status
----
+
Weitere Informationen zu  `storage encryption disk show-status` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/storage-encryption-disk-show-status.html["ONTAP-Befehlsreferenz"^].

. Es sollte sichergestellt sein, dass der *Modus* für alle Datenträger auf Daten eingestellt ist.
+
[source, cli]
----
storage encryption disk show
----
+
Weitere Informationen zu  `storage encryption disk show` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/storage-encryption-disk-show.html["ONTAP-Befehlsreferenz"^].

. Anzeigen der konfigurierten KMIP Server:
+
[source, cli]
----
security key-manager keystore show
----
+
Weitere Informationen zu  `security key-manager keystore show` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli//security-key-manager-keystore-show.html["ONTAP-Befehlsreferenz"^].

. Die konfigurierten KMIP-Server löschen:
+
[source, cli]
----
security key-manager delete -address <kmip_ip_address>
----
+
Weitere Informationen zu  `security key-manager delete` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/security-key-manager-key-delete.html["ONTAP-Befehlsreferenz"^].

. Die Konfiguration des externen Schlüsselmanagers löschen:
+
[source, cli]
----
security key-manager external disable
----
+
Weitere Informationen zu  `security key-manager external disable` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli//security-key-manager-external-disable.html["ONTAP-Befehlsreferenz"^].

+

NOTE: Durch diesen Schritt werden die NSE-Zertifikate nicht entfernt.
