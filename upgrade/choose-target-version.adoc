= Eine von NetApp empfohlene ONTAP Zielversion für ein Upgrade auswählen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Wenn mit dem Upgrade Advisor ein Upgrade-Plan für Ihren Cluster erstellt wird, enthält dieser eine empfohlene Ziel-ONTAP-Version für das Upgrade. Die Empfehlung des Upgrade Advisor basiert auf Ihrer aktuellen Konfiguration und Ihrer aktuellen ONTAP Version.

Wenn Sie Upgrade Advisor nicht zur Planung Ihres Upgrades verwenden, sollte die Ziel-ONTAP-Version für das Upgrade anhand von NetApp Empfehlungen oder Ihrer Notwendigkeit, die Mindestversion zu erreichen, um Ihre Leistungsanforderungen zu erfüllen, ausgewählt werden.

* Upgrade auf die neueste verfügbare Version (empfohlen)
+
NetApp empfiehlt, die ONTAP Software auf die neueste Patch-Version der neuesten nummerierten ONTAP Version zu aktualisieren. Falls dies nicht möglich ist, weil die neueste nummerierte Version von den Speichersystemen in Ihrem Cluster nicht unterstützt wird, sollte auf die neueste nummerierte unterstützte Version aktualisiert werden.

* Empfohlene Mindestversion
+
Wenn das Upgrade auf die für Ihren Cluster empfohlene Mindestversion beschränkt werden soll, siehe link:https://kb.netapp.com/Support_Bulletins/Customer_Bulletins/SU2["Empfohlene minimale ONTAP Versionen"^] zur Ermittlung der ONTAP Version, auf die ein Upgrade erfolgen sollte.
