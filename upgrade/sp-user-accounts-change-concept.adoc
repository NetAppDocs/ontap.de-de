= Änderung bei den Benutzerkonten, die nach einem ONTAP Upgrade auf den Service Processor zugreifen können
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Wenn in ONTAP 9.8 oder früher Benutzerkonten erstellt wurden, die mit einer Nicht-Administratorrolle auf den Service Processor (SP) zugreifen können, und ein Upgrade auf ONTAP 9.9.1 oder höher erfolgt, wird jeder Nicht-Administratorwert im  `-role` Parameter in  `admin` geändert.

Weitere Informationen finden sich unter link:../system-admin/accounts-access-sp-concept.html["Konten, die auf den SP zugreifen können"].
