= Das Disk Qualification Package nach einem ONTAP Upgrade aktualisieren
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Nach dem Upgrade Ihrer ONTAP Software sollten das ONTAP Disk Qualification Package (DQP) heruntergeladen und installiert werden. Die DQP wird im Rahmen eines ONTAP Upgrades nicht aktualisiert.

Das DQP enthält die erforderlichen Parameter für die Interaktion von ONTAP mit allen neu qualifizierten Laufwerken. Wenn Ihre Version des DQP keine Informationen für ein neu qualifiziertes Laufwerk enthält, verfügt ONTAP nicht über die Informationen, um das Laufwerk korrekt zu konfigurieren.

Es empfiehlt sich, den DQP vierteljährlich zu aktualisieren. Der DQP sollte auch aus folgenden Gründen aktualisiert werden:

* Immer wenn einem Knoten in Ihrem Cluster ein neuer Laufwerkstyp oder eine neue Laufwerksgröße hinzugefügt wird
+
Wenn Sie beispielsweise bereits 1-TB-Festplatten besitzen und 2-TB-Festplatten hinzufügen, müssen Sie auf das neueste DQP-Update prüfen.

* Immer wenn Sie die Festplatten-Firmware aktualisieren
* Sobald neuere Festplatten-Firmware oder DQP-Dateien verfügbar sind
