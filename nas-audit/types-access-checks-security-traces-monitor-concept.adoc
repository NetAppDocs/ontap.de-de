= Arten von Zugriffsprüfungen, die Sicherheitsüberwachung auf ONTAP SVMs nachverfolgt
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Zugriffsprüfungen für eine Datei oder einen Ordner erfolgen anhand mehrerer Kriterien. Security Traces überwachen Vorgänge auf all diesen Kriterien.

Zu den Arten von Zugriffskontrollen, die von Sicherheitstraces überwacht werden, zählen die folgenden:

* Volume- und Qtree-Sicherheitsstil
* Effektive Sicherheit des Dateisystems, das die Dateien und Ordner enthält, auf denen Operationen angefordert werden
* Benutzerzuordnung
* Berechtigungen auf Freigabeebene
* Berechtigungen auf Exportebene
* Dateiebene-Berechtigungen
* Storage Level Access Guard-Sicherheit
