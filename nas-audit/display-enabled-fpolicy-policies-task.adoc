= Informationen zu aktivierten ONTAP FPolicy Richtlinien
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können Informationen über aktivierte FPolicy Richtlinien anzeigen, um festzustellen, welche FPolicy External Engine verwendet wird, welche Priorität die Richtlinie hat und welcher Storage Virtual Machine (SVM) die FPolicy Richtlinie zugeordnet ist.

.Über diese Aufgabe
Wenn Sie keine Parameter angeben, zeigt der Befehl folgende Informationen an:

* SVM-Name
* Richtlinienname
* Richtlinienpriorität


Mit Befehlsparametern lässt sich die Ausgabe des Befehls nach bestimmten Kriterien filtern.

.Schritt
. Informationen zu aktivierten FPolicy Richtlinien werden mithilfe des entsprechenden Befehls angezeigt:
+
[cols="35,65"]
|===


| Wenn Informationen über aktivierte Richtlinien angezeigt werden sollen... | Geben Sie den Befehl ein... 


 a| 
Auf dem Cluster
 a| 
`vserver fpolicy show-enabled`



 a| 
Auf einer bestimmten SVM
 a| 
`vserver fpolicy show-enabled -vserver vserver_name`



 a| 
Mit dem angegebenen Richtliniennamen
 a| 
`vserver fpolicy show-enabled -policy-name policy_name`



 a| 
Mit der angegebenen Sequenznummer
 a| 
`vserver fpolicy show-enabled -priority integer`

|===


.Beispiel
Das folgende Beispiel zeigt die Informationen zu den aktivierten FPolicy Richtlinien im Cluster:

[listing]
----
cluster1::> vserver fpolicy show-enabled
Vserver                 Policy Name               Priority

----------------------- ------------------------- ----------
vs1.example.com         pol_native                native
vs1.example.com         pol_native2               native
vs1.example.com         pol1                      2
vs1.example.com         pol2                      4
----
