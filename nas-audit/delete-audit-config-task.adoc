= Löschen einer Revisionskonfiguration auf einer ONTAP SVM
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Wenn Sie keine Revisionsdatei- und Verzeichnisereignisse mehr auf der Storage Virtual Machine (SVM) protokollieren und keine Revisionskonfiguration auf der SVM aufrechterhalten möchten, kann die Revisionskonfiguration gelöscht werden.

.Schritte
. Die Revisionskonfiguration deaktivieren:
+
`vserver audit disable -vserver vserver_name`

+
`vserver audit disable -vserver vs1`

. Die Revisionskonfiguration löschen:
+
`vserver audit delete -vserver vserver_name`

+
`vserver audit delete -vserver vs1`
