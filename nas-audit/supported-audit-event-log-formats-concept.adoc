= Informationen zu den unterstützten Formaten für ONTAP Audit-Ereignisprotokolle
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Unterstützte Dateiformate für die konvertierten Audit-Ereignisprotokolle sind  `EVTX` und  `XML` Dateiformate.

Beim Erstellen der Audit-Konfiguration kann der Typ des Dateiformats festgelegt werden. Standardmäßig konvertiert ONTAP die Binärprotokolle in das  `EVTX` Dateiformat.
