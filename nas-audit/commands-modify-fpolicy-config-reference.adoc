= Befehle zum Ändern von FPolicy Konfigurationen in ONTAP
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können FPolicy Konfigurationen anpassen, indem Sie die Elemente ändern, aus denen die Konfiguration besteht. Sie können externe Engines, FPolicy Events, FPolicy Scopes, FPolicy Persistent Stores und FPolicy Policies ändern. FPolicy Policies können außerdem aktiviert oder deaktiviert werden. Wenn die FPolicy Policy deaktiviert wird, wird die Dateiüberwachung für diese Policy eingestellt.

Sie sollten eine FPolicy Richtlinie deaktivieren, bevor deren Konfiguration geändert wird.

[cols="35,65"]
|===


| Wenn Sie Änderungen vornehmen möchten... | Verwenden Sie diesen Befehl... 


 a| 
Externe Engines
 a| 
`vserver fpolicy policy external-engine modify`



 a| 
Veranstaltungen
 a| 
`vserver fpolicy policy event modify`



 a| 
Bereiche
 a| 
`vserver fpolicy policy scope modify`



 a| 
Persistenter Speicher
 a| 
`vserver fpolicy persistent-store modify`



 a| 
Richtlinien
 a| 
`vserver fpolicy policy modify`

|===
Weitere Informationen zu  `vserver fpolicy policy` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/search.html?q=vserver+fpolicy+policy["ONTAP-Befehlsreferenz"^].
