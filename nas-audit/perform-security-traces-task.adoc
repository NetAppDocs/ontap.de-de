= ONTAP Sicherheitsüberprüfungen durchführen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Die Durchführung einer Sicherheitsverfolgung umfasst das Erstellen eines Sicherheitsverfolgungsfilters, das Überprüfen der Filterkriterien, das Generieren von Zugriffsanfragen auf einem SMB- oder NFS-Client, die den Filterkriterien entsprechen, und das Anzeigen der Ergebnisse.

Nachdem Sie die Verwendung eines Sicherheitsfilters zur Erfassung von Protokollinformationen abgeschlossen haben, kann der Filter angepasst und wiederverwendet oder deaktiviert werden, falls er nicht mehr benötigt wird. Nach Ansicht und Analyse der Filterprotokollergebnisse können diese gelöscht werden, wenn sie nicht mehr benötigt werden.
