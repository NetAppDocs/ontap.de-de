= Informationen zu ONTAP FPolicy show-Befehlen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Beim Anzeigen von Informationen zur FPolicy Konfiguration ist es hilfreich zu verstehen, wie die  `show` Befehle funktionieren.

Ein `show` Befehl ohne zusätzliche Parameter zeigt Informationen in zusammengefasster Form an. Zusätzlich verfügt jeder `show` Befehl über dieselben zwei sich gegenseitig ausschließenden optionalen Parameter, `-instance` und `-fields`.

Wenn Sie den  `-instance` Parameter mit einem  `show` Befehl verwenden, zeigt die Befehlsausgabe detaillierte Informationen in Listenform an. In einigen Fällen kann die detaillierte Ausgabe sehr umfangreich sein und mehr Informationen enthalten, als benötigt werden. Mit dem  `-fields` `fieldname[,fieldname...]` Parameter lässt sich die Ausgabe anpassen, sodass nur Informationen zu den von Ihnen angegebenen Feldern angezeigt werden. Welche Felder angegeben werden können, lässt sich durch die Eingabe von  `?` nach dem  `-fields` Parameter ermitteln.

[NOTE]
====
Die Ausgabe eines  `show` Kommandos mit dem  `-fields` Parameter kann weitere relevante und notwendige Felder anzeigen, die mit den angeforderten Feldern in Zusammenhang stehen.

====
Jeder `show` Befehl verfügt über einen oder mehrere optionale Parameter, die die Ausgabe filtern und es ermöglichen, den Umfang der angezeigten Informationen einzugrenzen. Es kann festgestellt werden, welche optionalen Parameter für einen Befehl verfügbar sind, indem `?` nach dem `show` Befehl eingegeben wird.

Der  `show` Befehl unterstützt UNIX-typische Muster und Platzhalter, um mehrere Werte in Befehlsparameterargumenten abzugleichen. Beispielsweise können der Platzhalteroperator (*), der NOT-Operator (!), der OR-Operator (|), der Bereichsoperator (integer...integer), der Kleiner-als-Operator (<), the greater-than operator (>), der Kleiner-gleich-Operator (\<=), and the greater-than or equal to operator (>=) beim Angeben von Werten verwendet werden.

Weitere Informationen zur Verwendung von UNIX-Mustern und Wildcards sind unter xref:../system-admin/command-line-interface-concept.html[Verwendung der ONTAP Befehlszeilenschnittstelle] zu finden.
