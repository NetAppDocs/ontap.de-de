= Audit-Konfiguration auf ONTAP SVMs aktivieren und Audit-Richtlinien für Dateien und Ordner konfigurieren
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Die Implementierung der Überwachung von Datei- und Ordnerzugriffsereignissen ist ein zweistufiger Prozess. Zunächst muss eine Revisionsprotokollierungskonfiguration auf Storage Virtual Machines (SVMs) erstellt und aktiviert werden. Anschließend müssen Revisionsprotokollierungsrichtlinien für die Dateien und Ordner konfiguriert werden, die überwacht werden sollen. Revisionsprotokollierungsrichtlinien können so konfiguriert werden, dass sowohl erfolgreiche als auch fehlgeschlagene Zugriffsversuche überwacht werden.

Sie können sowohl SMB- als auch NFS-Revisionsrichtlinien konfigurieren. SMB- und NFS-Revisionsrichtlinien haben unterschiedliche Konfigurationsanforderungen und Revisionsfunktionen.

Wenn die entsprechenden Revisionsrichtlinien konfiguriert sind, überwacht ONTAP SMB- und NFS-Zugriffsereignisse gemäß den Vorgaben in den Revisionsrichtlinien nur, wenn die SMB- oder NFS-Server ausgeführt werden.
