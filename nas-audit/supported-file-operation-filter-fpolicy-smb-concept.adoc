= Unterstützte Dateioperationen und Filterkombinationen, die ONTAP FPolicy für SMB überwacht
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Bei der Konfiguration Ihres FPolicy-Ereignisses ist zu beachten, dass nur bestimmte Kombinationen von Dateioperationen und Filtern für die Überwachung von SMB-Dateizugriffsvorgängen unterstützt werden.

Die Liste der unterstützten Dateioperationen- und Filterkombinationen für die FPolicy Überwachung von SMB-Dateizugriffsereignissen ist in der folgenden Tabelle aufgeführt:

[cols="30,70"]
|===


| Unterstützte Dateivorgänge | Unterstützte Filter 


 a| 
schließen
 a| 
monitor-ads, offline-bit, close-with-modification, close-without-modification, close-with-read, exclude-directory



 a| 
erstellen
 a| 
monitor-ads, offline-bit



 a| 
Verzeichnis erstellen
 a| 
Aktuell wird für diese Dateioperation kein Filter unterstützt.



 a| 
löschen
 a| 
monitor-ads, offline-bit



 a| 
Verzeichnis löschen
 a| 
Aktuell wird für diese Dateioperation kein Filter unterstützt.



 a|

 a| 
offline-Bit, exclude-dir



 a|

 a| 
monitor-ads, offline-bit, open-with-delete-intent, open-with-write-intent, exclude-dir



 a|

 a| 
monitor-ads, offline-bit, first-read



 a|

 a| 
monitor-ads, offline-bit, first-write, write-with-size-change



 a|

 a| 
monitor-ads, offline-bit



 a|

 a|

|===
Ab ONTAP 9.13.1 können Benutzer Benachrichtigungen über fehlgeschlagene Dateivorgänge aufgrund fehlender Berechtigungen erhalten. Die Liste der unterstützten Kombinationen aus verweigerten Dateizugriffsvorgängen und Filtern für die FPolicy Überwachung von SMB Dateizugriffsereignissen ist in der folgenden Tabelle aufgeführt:
