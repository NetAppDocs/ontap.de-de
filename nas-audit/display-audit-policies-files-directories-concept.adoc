= Informationen zu den auf Dateien und Verzeichnisse angewendeten Revisionsrichtlinien
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Die Anzeige von Informationen über auf Dateien und Verzeichnisse angewendete Revisionsrichtlinien ermöglicht die Überprüfung, ob die entsprechenden Systemzugriffskontrolllisten (SACLs) für bestimmte Dateien und Ordner festgelegt sind.
