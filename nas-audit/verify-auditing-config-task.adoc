= Die ONTAP Revisionsprotokoll-Konfiguration überprüfen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Nach Abschluss der Revisionsprotokollkonfiguration sollte überprüft werden, ob das Revisionsprotokoll ordnungsgemäß konfiguriert und aktiviert ist.

.Schritte
. Überprüfen der Audit-Konfiguration:
+
`vserver audit show -instance -vserver vserver_name`

+
Der folgende Befehl zeigt in Listenform alle Revisionsprotokoll-Konfigurationsinformationen für die Storage Virtual Machine (SVM) vs1 an:

+
`vserver audit show -instance -vserver vs1`

+
[listing]
----

                             Vserver: vs1
                      Auditing state: true
                Log Destination Path: /audit_log
       Categories of Events to Audit: file-ops
                          Log Format: evtx
                 Log File Size Limit: 200MB
        Log Rotation Schedule: Month: -
  Log Rotation Schedule: Day of Week: -
          Log Rotation Schedule: Day: -
         Log Rotation Schedule: Hour: -
       Log Rotation Schedule: Minute: -
                  Rotation Schedules: -
            Log Files Rotation Limit: 0
----
