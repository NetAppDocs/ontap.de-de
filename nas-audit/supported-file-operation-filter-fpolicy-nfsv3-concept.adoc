= Unterstützte Dateioperationen- und Filterkombinationen, die ONTAP FPolicy für NFSv3 überwacht
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Bei der Konfiguration Ihres FPolicy Ereignisses ist zu beachten, dass für die Überwachung von NFSv3-Dateizugriffsvorgängen nur bestimmte Kombinationen von Dateivorgängen und Filtern unterstützt werden.
