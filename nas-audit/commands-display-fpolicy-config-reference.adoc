= Befehle zum Anzeigen von Informationen über FPolicy Konfigurationen in ONTAP
:allow-uri-read: 


[role="lead"]
Die  `fpolicy show` Befehle werden verwendet, um Informationen zur FPolicy Konfiguration anzuzeigen, einschließlich Informationen zu FPolicy External Engines, Ereignissen, Geltungsbereichen und Richtlinien.

[cols="40,60"]
|===


| Wenn Sie Informationen über FPolicy anzeigen möchten... | Verwenden Sie diesen Befehl... 


 a| 
Externe Engines
 a| 
`vserver fpolicy policy external-engine show`



 a| 
Veranstaltungen
 a| 
`vserver fpolicy policy event show`



 a| 
Bereiche
 a| 
`vserver fpolicy policy scope show`



 a| 
Richtlinien
 a| 
`vserver fpolicy policy show`

|===
Weitere Informationen zu  `vserver fpolicy policy` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/search.html?q=vserver+fpolicy+policy["ONTAP-Befehlsreferenz"^].
