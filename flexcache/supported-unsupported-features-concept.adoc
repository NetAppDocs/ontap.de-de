= Unterstützte und nicht unterstützte Funktionen für ONTAP FlexCache Volumes
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Ab ONTAP 9.5 ist die Konfiguration von FlexCache Volumes möglich. FlexVol Volumes werden als Ursprungsvolumes unterstützt und FlexGroup Volumes werden als FlexCache Volumes unterstützt. Ab ONTAP 9.7 werden sowohl FlexVol Volumes als auch FlexGroup Volumes als Ursprungsvolumes unterstützt. Die unterstützten Funktionen und Protokolle für das Ursprungsvolume und das FlexCache Volume unterscheiden sich.


NOTE: Cache Volumes und Ursprungs Volumes können miteinander arbeiten, solange beide auf einer unterstützten Version von ONTAP laufen. Zu beachten ist, dass Funktionen nur dann unterstützt werden, wenn sowohl der Cache als auch der Ursprung mindestens auf der ONTAP Version betrieben werden, in der die Unterstützung eingeführt wurde, oder auf einer späteren ONTAP Version.



[[ontap-version-support-between-flexcache-volumes-and-origin-volumes]]
== ONTAP Versionsunterstützung zwischen FlexCache volumes und Ursprungsvolumes

Die empfohlene ONTAP Version, die zwischen dem Ursprungs-Volume und dem Cache-Volume unterstützt wird, liegt höchstens vier Versionen früher oder vier Versionen später. Wenn der Cache beispielsweise ONTAP 9.14.1 verwendet, ist die früheste Version, die das Ursprungs-Volume verwenden kann, ONTAP 9.10.1.
