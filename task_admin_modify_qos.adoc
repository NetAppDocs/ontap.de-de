= QoS im ONTAP System Manager ändern
:hardbreaks:
:toclevels: 1
:allow-uri-read: 
:toclevels: 1
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
Ab ONTAP 9.8 ist bei der Speicherbereitstellung xref:./performance-admin/guarantee-throughput-qos-task.html[Quality of Service (QoS)] standardmäßig QoS aktiviert. QoS kann während des Bereitstellungsprozesses deaktiviert oder eine benutzerdefinierte QoS-Richtlinie ausgewählt werden. QoS kann auch nach der Speicherbereitstellung angepasst werden.

.Schritte
. Im System Manager *Speicher* und dann *Volumes* auswählen.
. Neben dem Volume, für das die QoS geändert werden soll, image:icon_kabob.gif["Symbol für Menüoptionen"] dann *Bearbeiten* auswählen.
