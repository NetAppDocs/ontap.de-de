= Die RAID-Konfiguration einer lokalen Ebene bearbeiten
:hardbreaks:
:toclevels: 1
:allow-uri-read: 
:toclevels: 1
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
Ab ONTAP 9.11.1 ist es möglich, die RAID-Konfiguration einer lokalen Ebene zu ändern.

.Schritte
. Im System Manager auf *Speicher > Ebenen* klicken.
. Klicken Sie image:icon_kabob.gif["Symbol für Menüoptionen"] neben dem Namen der lokalen Ebene.
. *RAID-Konfiguration bearbeiten* auswählen.
. Die Seite *RAID-Konfiguration bearbeiten* wird angezeigt. Jedes der Felder kann geändert werden:
+
--
.. Wählen Sie einen Datenträgertyp aus.
+
Dieses Feld wird für lokale Tiers mit gemischtem RAID-Typ nicht angezeigt.

.. Einen RAID-Typ auswählen.
.. Die Größe der RAID-Gruppe auswählen.


--



NOTE: Wenn Sie den RAID-Typ ändern, kann die RAID-Gruppengröße nicht geändert werden und umgekehrt. Um beide Felder zu ändern, sollte zunächst ein Feld geändert, die Konfiguration gespeichert und anschließend dieser Bearbeitungsvorgang wiederholt werden, um das andere Feld zu ändern.

.Verwandte Informationen
Siehe link:disks-aggregates/aggregate-creation-workflow-concept.html["Eine lokale Ebene hinzufügen (erstellen)"] für die Erstellung einer lokalen Ebene.
