= pNFS-Befehle, Statistiken und Ereignisprotokolle
:allow-uri-read: 


[role="lead"]
Diese ONTAP CLI-Befehle beziehen sich speziell auf pNFS. Sie können damit konfigurieren, Fehler beheben und Statistiken erfassen.



== Aktivieren Sie NFSv4.1

[listing]
----
nfs modify -vserver SVM -v4.1 enabled
----


== Aktivieren Sie pNFS

[listing]
----
nfs modify -vserver SVM -v4.1-pnfs enabled
----

== NFS-Versionsinformationen für verbundene Clients anzeigen

Sie können NFS-Verbindungen auch mit dem anzeigen `nfs connected-clients show` Befehl. Beachten Sie, dass die angezeigte Liste nur Clients umfasst, die in den letzten 48 Stunden aktiven NFS-Datenverkehr hatten. Im Leerlauf befindliche NFS-Clients (selbst wenn sie noch eingebunden sind) werden möglicherweise erst angezeigt, wenn auf die Einbindung zugegriffen wird. Sie können diese filtern, um nur die zuletzt aufgerufenen Clients anzuzeigen, indem Sie die folgenden Angaben machen: `-idle-time` Besonderheit.

Um beispielsweise Clients mit Aktivität in den letzten 10 Minuten für die pNFS SVM anzuzeigen:

[listing]
----
cluster::*> nfs connected-clients show -vserver PNFS -idle-time <10m>
