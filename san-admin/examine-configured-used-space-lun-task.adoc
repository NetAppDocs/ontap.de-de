= Konfigurierten und belegten Speicherplatz einer LUN prüfen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Die Kenntnis des konfigurierten Speicherplatzes und des tatsächlich für Ihre LUNs verwendeten Speicherplatzes kann dabei helfen, die Menge an Speicherplatz zu bestimmen, die bei der Speicherplatzrückgewinnung zurückgewonnen werden kann, die Menge des reservierten Speicherplatzes, der Daten enthält, sowie die gesamte konfigurierte Größe im Vergleich zur tatsächlich für eine LUN verwendeten Größe.

.Schritt
. Anzeigen des konfigurierten Speicherplatzes im Vergleich zum tatsächlich für eine LUN verwendeten Speicherplatz:
+
`lun show`

+
Das folgende Beispiel zeigt den konfigurierten Speicherplatz im Vergleich zum tatsächlich von den LUNs in der vs3 Storage Virtual Machine (SVM) belegten Speicherplatz:

+
`lun show -vserver vs3 -fields path, size, size-used, space-reserve`

+
[listing]
----
vserver path                    size    space-reserve size-used
------- -----------------       ------- ------------- ---------
vs3     /vol/vol0/lun1          50.01GB disabled      25.00GB
vs3     /vol/vol0/lun1_backup   50.01GB disabled      32.15GB
vs3     /vol/vol0/lun2          75.00GB disabled      0B
vs3     /vol/volspace/lun0      5.00GB  enabled       4.50GB
4 entries were displayed.
----
+
Weitere Informationen zu  `lun show` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/lun-show.html["ONTAP-Befehlsreferenz"^].
