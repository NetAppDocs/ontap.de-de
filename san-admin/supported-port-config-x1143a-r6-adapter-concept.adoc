= Unterstützte Portkonfigurationen für X1143A-R6 Adapter
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Der FC-Zielmodus ist die Standardkonfiguration für die Ports des X1143A-R6 Adapters. Die Ports dieses Adapters können jedoch entweder als 10-Gb-Ethernet- und FCoE-Ports oder als 16-Gb-FC-Ports konfiguriert werden.

Bei Konfiguration für Ethernet und FCoE unterstützen X1143A-R6 Adapter gleichzeitigen NIC- und FCoE-Zieldatenverkehr über denselben 10-GBE-Port. Bei FC-Konfiguration kann jedes Zwei-Port-Paar, das denselben ASIC nutzt, individuell für den FC-Ziel- oder FC-Initiator-Modus konfiguriert werden. Das bedeutet, dass ein einzelner X1143A-R6 Adapter den FC-Zielmodus auf einem Zwei-Port-Paar und den FC-Initiator-Modus auf einem anderen Zwei-Port-Paar unterstützen kann.
