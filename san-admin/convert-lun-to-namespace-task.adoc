= Eine LUN in einen Namensraum konvertieren
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Ab ONTAP 9.11.1 kann die ONTAP CLI verwendet werden, um eine bestehende LUN direkt in einen NVMe Namespace zu konvertieren.

.Bevor Sie beginnen
* Die angegebene LUN sollte keine bestehenden Zuordnungen zu einer Initiatorgruppe haben.
* Die LUN sollte sich nicht in einer für MetroCluster konfigurierten SVM oder in einer SnapMirror active sync Beziehung befinden.
* LUN sollte kein Protokollendpunkt sein oder an einen Protokollendpunkt gebunden sein.
* LUN sollte keinen Präfix- und/oder Suffix-Stream ungleich Null aufweisen.
* Die LUN sollte nicht Teil eines Snapshots sein oder auf der Zielseite der SnapMirror Beziehung als schreibgeschützte LUN vorliegen.


.Schritt
. Eine LUN in einen NVMe-Namensraum konvertieren:
+
[source, cli]
----
vserver nvme namespace convert-from-lun -vserver -lun-path
----
