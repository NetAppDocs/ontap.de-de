= Initiator-WWPNs und iSCSI-Knotennamen für eine Initiatorgruppe angeben
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können die iSCSI-Knotennamen und WWPNs der Initiatoren beim Erstellen einer Initiatorgruppe angeben oder sie später hinzufügen. Wenn Sie die iSCSI-Knotennamen und WWPNs der Initiatoren beim Erstellen der LUN angeben, können diese bei Bedarf später entfernt werden.

In der Host Utilities-Dokumentation finden sich Anweisungen, wie WWPNs ermittelt und die mit einem bestimmten Host verknüpften iSCSI-Node-Namen gefunden werden können. Für Hosts mit ESX-Software kommt die Virtual Storage Console zum Einsatz.
