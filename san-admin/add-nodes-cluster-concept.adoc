= SAN LIF Anforderungen für das Hinzufügen von Knoten zu einem Cluster
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Beim Hinzufügen von Knoten zu einem Cluster sind bestimmte Aspekte zu berücksichtigen.

* Sie müssen gegebenenfalls LIFs auf den neuen Knoten erstellen, bevor Sie LUNs auf diesen neuen Knoten erstellen.
* Diese LIFs sind von den Hosts entsprechend dem Host-Stack und dem Protokoll zu ermitteln.
* Sie müssen LIFs auf den neuen Knoten erstellen, damit LUN- und Volume-Verschiebungen ohne Verwendung des Cluster-Interconnect-Netzwerks möglich sind.
