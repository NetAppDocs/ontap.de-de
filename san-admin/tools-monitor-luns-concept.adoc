= Verfügbare Tools zur effektiven Überwachung Ihrer LUNs
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Tools stehen zur Verfügung, um Ihre LUNs effektiv zu überwachen und Speicherplatzprobleme zu vermeiden.

* Active IQ Unified Manager ist ein kostenloses Tool, das die Verwaltung des gesamten Speichers über alle Cluster in Ihrer Umgebung ermöglicht.
* Der System Manager ist eine grafische Benutzeroberfläche, die in ONTAP integriert ist und die manuelle Verwaltung des Speicherbedarfs auf Clusterebene ermöglicht.
* OnCommand Insight bietet eine einheitliche Sicht auf Ihre Speicherinfrastruktur und ermöglicht die Einrichtung von automatischer Überwachung, Benachrichtigungen und Berichten, wenn der Speicherplatz Ihrer LUNs, Volumes und Aggregate knapp wird.
