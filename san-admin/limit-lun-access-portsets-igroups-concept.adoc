= Möglichkeiten zur Einschränkung des LUN-Zugriffs mit Portsets und Initiatorgruppen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Zusätzlich zur Verwendung von Selective LUN Map (SLM) kann der Zugriff auf Ihre LUNs über Initiatorgruppen und Portsets eingeschränkt werden.

Portsets können in Verbindung mit SLM verwendet werden, um den Zugriff bestimmter Ziele auf bestimmte Initiatoren weiter einzuschränken. Bei Verwendung von SLM mit Portsets sind LUNs auf der Menge der LIFs im Portset auf dem Node, dem die LUN gehört, und auf dem HA-Partner dieses Nodes zugänglich.

Im folgenden Beispiel verfügt host1 über keinen Portsatz. Ohne Portsatz kann host1 auf LUN1 sowohl über LIF1 als auch über LIF2 zugreifen.

image:bsag-c-mode-no-portset.png["Abbildung zur Veranschaulichung des LUN-Zugriffs ohne Portset"]

Sie können den Zugriff auf LUN1 mithilfe eines Portsets einschränken. Im folgenden Beispiel kann Host1 nur über LIF1 auf LUN1 zugreifen. Host1 kann jedoch nicht über LIF2 auf LUN1 zugreifen, da LIF2 nicht in portset1 ist.

image:bsag-c-mode-portset.gif["Abbildung zur Veranschaulichung des LUN-Zugriffs über einen Port"]
