= Adaptereinstellungen anzeigen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können spezifische Befehle verwenden, um Informationen über Ihre FC/UTA-Adapter anzuzeigen.



[[fc-target-adapter]]
== FC Zieladapter

.Schritt
. Mit dem  `network fcp adapter show`Befehl werden Adapterinformationen angezeigt: `network fcp adapter show -instance -node node1 -adapter 0a`
+
Die Ausgabe zeigt Systemkonfigurationsinformationen und Adapterinformationen für jeden verwendeten Steckplatz an.

+
Weitere Informationen zu  `network fcp adapter show` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/network-fcp-adapter-show.html["ONTAP-Befehlsreferenz"^].





[[unified-target-adapter-uta-x1143a-r6]]
== Unified Target Adapter (UTA) X1143A-R6

.Schritte
. Starten Sie Ihren Controller ohne angeschlossene Kabel.
. Mit dem  `system hardware unified-connect show` Befehl lassen sich die Portkonfiguration und die Module anzeigen.
. Die Portinformationen sollten vor der Konfiguration von CNA und Ports eingesehen werden.
