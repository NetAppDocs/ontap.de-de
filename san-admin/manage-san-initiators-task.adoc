== SAN Initiatorgruppen anzeigen und verwalten

Sie können den System Manager verwenden, um eine Liste der Initiatorgruppen (Initiatorgruppen) anzuzeigen. Aus der Liste können zusätzliche Operationen durchgeführt werden.

.Schritte
. Im System Manager auf *Hosts > SAN Initiatorgruppen* klicken.
+
Auf der Seite wird eine Liste der Initiatorgruppen (Initiatorgruppen) angezeigt. Wenn die Liste lang ist, können zusätzliche Seiten der Liste angezeigt werden, indem auf die Seitenzahlen in der unteren rechten Ecke der Seite geklickt wird.

+
Die Spalten zeigen verschiedene Informationen zu den Initiatorgruppen an. Ab Version 9.11.1 wird auch der Verbindungsstatus der Initiatorgruppe angezeigt. Durch das Überfahren der Statusmeldungen mit dem Mauszeiger werden Details angezeigt.

. (Optional): Die folgenden Aufgaben können durch Klicken auf die Symbole in der oberen rechten Ecke der Liste ausgeführt werden:
+
** *Suchen*
** *Laden* Sie die Liste herunter.
** *Anzeigen* oder *Ausblenden* von Spalten in der Liste.
** *Filtern* Sie die Daten in der Liste.


. Folgende Operationen aus der Liste sind möglich:
+
** Klicken Sie auf image:icon_add_blue_bg.png["Symbol hinzufügen"], um eine Initiatorgruppe hinzuzufügen.
** Klicken Sie auf den Namen der Initiatorgruppe, um die *Übersicht*-Seite anzuzeigen, die Details zur Initiatorgruppe enthält.
+
Auf der Seite *Übersicht* können die mit der Initiatorgruppe verknüpften LUNs angezeigt werden, und es besteht die Möglichkeit, Vorgänge zum Erstellen und Zuordnen von LUNs zu initiieren. *Alle SAN-Initiatoren* kann ausgewählt werden, um zur Hauptliste zurückzukehren.

** Fahren Sie mit dem Mauszeiger über die Initiatorgruppe und klicken Sie dann auf image:icon_kabob.gif["Symbol für Menüoptionen"] neben einem Initiatorgruppen-Namen, um die Initiatorgruppe zu bearbeiten oder zu löschen.
** Bewegen Sie den Mauszeiger über den Bereich links neben dem Initiatorgruppenamen und aktivieren Sie das Kästchen. Bei Auswahl von *+Zur Initiatorgruppe hinzufügen* besteht die Möglichkeit, diese Initiatorgruppe einer anderen Initiatorgruppe hinzuzufügen.
** In der Spalte *Speicher-VM* auf den Namen einer Speicher-VM klicken, um Details dazu anzuzeigen.

== SAN Initiatoren anzeigen und verwalten

Im System Manager können Sie eine Liste der Initiatoren anzeigen. Aus der Liste können zusätzliche Operationen durchgeführt werden.

.Schritte
. Im System Manager auf *Hosts > SAN Initiatorgruppen* klicken.
+
Auf der Seite wird eine Liste der Initiatorgruppen (Initiatorgruppen) angezeigt.

. So können die Initiatoren angezeigt werden:
+
** Die Registerkarte *FC Initiators* zeigt eine Liste der FC-Initiatoren an.
** Klicken Sie auf die Registerkarte *iSCSI Initiatoren*, um eine Liste der iSCSI Initiatoren anzuzeigen.
+
Die Spalten enthalten verschiedene Informationen über die Initiatoren.

+
Ab Version 9.11.1 wird auch der Verbindungsstatus des Initiators angezeigt. Durch das Überfahren der Statusmeldungen mit dem Mauszeiger werden Details angezeigt.



. (Optional): Die folgenden Aufgaben können durch Klicken auf die Symbole in der oberen rechten Ecke der Liste ausgeführt werden:
+
** Die Liste kann nach bestimmten Initiatoren durchsucht werden.
** *Laden* Sie die Liste herunter.
** *Anzeigen* oder *Ausblenden* von Spalten in der Liste.
** *Filtern* Sie die Daten in der Liste.
