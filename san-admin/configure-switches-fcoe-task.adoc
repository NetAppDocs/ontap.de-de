= Switches für FCoE konfigurieren
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Die Switches müssen für FCoE konfiguriert sein, bevor der FC-Dienst über die bestehende Ethernet-Infrastruktur ausgeführt werden kann.

.Bevor Sie beginnen
* Ihre SAN Konfiguration muss unterstützt werden.
+
Weitere Informationen zu den unterstützten Konfigurationen sind unter  https://mysupport.netapp.com/matrix["NetApp Interoperabilitätsmatrix-Tool"^] zu finden.

* Auf Ihrem Speichersystem muss ein Unified Target Adapter (UTA) installiert sein.
+
Wenn Sie einen UTA2 verwenden, muss dieser auf  `cna` Modus eingestellt sein.

* Auf Ihrem Host muss ein konvergierter Netzwerkadapter (CNA) installiert sein.


.Schritte
. Die Switch-Dokumentation bietet Anleitungen zur Konfiguration der Switches für FCoE.
. Es sollte sichergestellt sein, dass die DCB-Einstellungen für jeden Knoten im Cluster korrekt konfiguriert sind.
+
[source, cli]
----
run -node node1 -command dcb show
----
+
Die DCB-Einstellungen werden am Switch konfiguriert. Die Switch-Dokumentation bietet Unterstützung, falls die Einstellungen nicht korrekt sind.

. Es sollte überprüft werden, ob die FCoE-Anmeldung funktioniert, wenn der FC-Zielport den Status online  `true` hat.
+
[source, cli]
----
fcp adapter show -fields node,adapter,status,state,speed,fabric-established,physical-protocol
----
+
Wenn der Online-Status des FC-Zielports  `false` ist, empfiehlt sich ein Blick in die Switch-Dokumentation.
