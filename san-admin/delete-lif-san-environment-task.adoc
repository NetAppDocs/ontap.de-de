= Löschen einer LIF in einer SAN-Umgebung
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Bevor Sie eine LIF löschen, sollte sichergestellt sein, dass der mit der LIF verbundene Host über einen anderen Pfad auf die LUNs zugreifen kann.

.Bevor Sie beginnen
Wenn die LIF, die Sie löschen möchten, Mitglied eines Portsets ist, muss die LIF zuerst aus dem Portset entfernt werden, bevor die LIF gelöscht werden kann.

[role="tabbed-block"]
====
.System Manager
--
Eine LIF mit ONTAP System Manager (9.7 und höher) löschen.

.Schritte
. Im System Manager auf *Netzwerk > Übersicht* klicken und anschließend *Netzwerkschnittstellen* auswählen.
. Die Storage-VM auswählen, aus der die LIF gelöscht werden soll.
. Klicken Sie auf image:icon_kabob.gif["Symbol für Menüoptionen"] und wählen Sie *Löschen* aus.


--
.CLI
--
Eine LIF mit der ONTAP CLI löschen.

.Schritte
. Der Name des LIF und der aktuelle Port, der gelöscht werden soll, können überprüft werden:
+
`network interface show –vserver _vserver_name_`

. Die LIF löschen:
+
`network interface delete`

+
`network interface delete -vserver vs1 -lif lif1`

+
Weitere Informationen zu  `network interface delete` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/network-interface-delete.html["ONTAP-Befehlsreferenz"^].

. Überprüfen, ob die LIF gelöscht wurde:
+
`network interface show`

+
`network interface show -vserver vs1`

+
[listing]
----

Vserver Interface  Admin/Oper Address/Mask     Node      Port    Home
------- ---------- ---------- ---------------- --------- ------- ----

----
+
Weitere Informationen zu  `network interface show` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/network-interface-show.html["ONTAP-Befehlsreferenz"^].
