= Einen SAN LIF aus einem Port-Set entfernen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Wenn sich die LIF, die Sie löschen oder verschieben möchten, in einem Portset befindet, muss die LIF aus dem Portset entfernt werden, bevor die LIF gelöscht oder verschoben werden kann.

.Über diese Aufgabe
Schritt 1 der folgenden Vorgehensweise ist nur erforderlich, wenn sich genau eine LIF im Portset befindet. Die letzte LIF in einem Portset kann nicht entfernt werden, wenn das Portset an eine Initiatorgruppe gebunden ist. Andernfalls kann mit Schritt 2 begonnen werden, falls sich mehrere LIFs im Portset befinden.

.Schritte
. Wenn sich nur ein LIF im Port-Set befindet, kann mit dem  `lun igroup unbind` Befehl das Port-Set von der Initiatorgruppe getrennt werden.
+
[NOTE]
====
Wenn eine Initiatorgruppe von einem Portsatz getrennt wird, haben alle Initiatoren in der Initiatorgruppe Zugriff auf alle Ziel-LUNs, die der Initiatorgruppe auf allen Netzwerkschnittstellen zugeordnet sind.

====
+
`cluster1::>lun igroup unbind -vserver vs1 -igroup ig1`

+
Weitere Informationen zu  `lun igroup unbind` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/lun-igroup-unbind.html["ONTAP-Befehlsreferenz"^].

. Mit dem  `lun portset remove` Befehl lässt sich die LIF aus dem Port-Set entfernen.
+
`cluster1::> port set remove -vserver vs1 -portset ps1 -port-name lif1`

+
Weitere Informationen zu  `lun portset remove` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/lun-portset-remove.html["ONTAP-Befehlsreferenz"^].
