= Erfahren Sie mehr über ONTAP LIFs für SAN-Umgebungen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
In Ihrem ONTAP-System befinden sich LUNs und Namensräume in Storage Virtual Machines (SVMs). Netzwerkschnittstellen (LIFs) ermöglichen es Ihren SAN-Clients, eine Verbindung zu einer SVM herzustellen und auf die darin enthaltenen Daten in den LUNs und Namensräumen zuzugreifen.

* Sie sollten für jede Storage Virtual Machine (SVM), die SAN unterstützt, eine Management-LIF konfigurieren.
* Sie können pro Knoten zwei LIFs konfigurieren, eines für jedes Fabric, das mit FC verwendet wird, und für separate Ethernet-Netzwerke für iSCSI.
* Für die Konnektivität wird entweder eine direkte Verbindung oder die Verwendung von Ethernet-Switches unterstützt. Sie müssen LIFs für beide Verbindungsarten konfigurieren.
