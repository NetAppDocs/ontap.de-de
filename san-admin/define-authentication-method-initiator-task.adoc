= Eine Sicherheitsrichtlinienmethode für einen Initiator festlegen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können eine Liste von Initiatoren und deren Authentifizierungsverfahren definieren. Es ist außerdem möglich, das Standard-Authentifizierungsverfahren zu ändern, das für Initiatoren gilt, die kein benutzerdefiniertes Authentifizierungsverfahren haben.

.Über diese Aufgabe
Sie können mithilfe der im Produkt integrierten Sicherheitsrichtlinienalgorithmen eindeutige Passwörter generieren oder die gewünschten Passwörter manuell angeben.

[NOTE]
====
Nicht alle Initiatoren unterstützen hexadezimale CHAP-Geheimpasswörter.

====
.Schritte
. Mit dem  `vserver iscsi security create` Befehl lässt sich eine Sicherheitsrichtlinienmethode für einen Initiator erstellen.
+
`vserver iscsi security create -vserver vs2 -initiator iqn.1991-05.com.microsoft:host1 -auth-type CHAP -user-name bob1 -outbound-user-name bob2`

. Folgen Sie den Anweisungen auf dem Bildschirm, um die Passwörter hinzuzufügen.
+
Erstellt ein Sicherheitsrichtlinienverfahren für den Initiator iqn.1991-05.com.microsoft:host1 mit eingehenden und ausgehenden CHAP-Benutzernamen und -Kennwörtern.
