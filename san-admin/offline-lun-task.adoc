= Eine LUN offline schalten
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Ab ONTAP 9.10.1 kann der System Manager verwendet werden, um LUNs offline zu nehmen. Vor ONTAP 9.10.1 muss die ONTAP CLI verwendet werden, um LUNs offline zu nehmen.

[role="tabbed-block"]
====
.System Manager
--
.Schritte
. Im System Manager auf *Speicher > LUNs* klicken.
. Eine einzelne LUN oder mehrere LUNs offline nehmen


[cols="2"]
|===
| Wenn Sie Folgendes tun möchten… | Tun Sie Folgendes… 


 a| 
Eine einzelne LUN offline nehmen
 a| 
Neben dem LUN-Namen auf image:icon_kabob.gif["Symbol für Menüoptionen"] klicken und *Offline nehmen* auswählen.



 a| 
Mehrere LUNs offline schalten
 a| 
. Wählen Sie die LUNs aus, die offline genommen werden sollen.
. Klicken Sie auf *Mehr* und wählen Sie *Offline nehmen*.


|===
--
.CLI
--
Über die CLI kann immer nur eine LUN gleichzeitig offline genommen werden.

.Schritt
. Die LUN offline schalten:
+
[source, cli]
----
lun offline <lun_name> -vserver <SVM_name>
----
