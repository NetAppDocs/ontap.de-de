= Portsets verwalten
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Zusätzlich zu link:selective-lun-map-concept.html["Selective LUN Map (SLM)"]können Portsets verwendet werden, um weiter einzuschränken, welche LIFs von einem Initiator zum Zugriff auf eine LUN genutzt werden können.

Ab ONTAP 9.10.1 kann im System Manager die den Portsets zugeordneten Netzwerkschnittstellen geändert und Portsets gelöscht werden.



[[change-network-interfaces-associated-with-a-portset]]
== Ändern der einem Portset zugeordneten Netzwerkschnittstellen

. Im System Manager *Netzwerk > Übersicht > Portsets* auswählen.
. Wählen Sie das Portset aus, das Sie bearbeiten möchten, image:icon_kabob.gif["Symbol für Menüoptionen"] dann *Portset bearbeiten* auswählen.




[[delete-a-portset]]
== Löschen eines Portsets

. Im System Manager unter *Network > Overview > Portsets* navigieren.
. Um ein einzelnes Portset zu löschen, wählen Sie das Portset aus, wählen Sie image:icon_kabob.gif["Symbol für Menüoptionen"] und dann *Portsets löschen*.
+
Mehrere Portsets können gelöscht werden, indem die Portsets ausgewählt und auf *Löschen* geklickt wird.
