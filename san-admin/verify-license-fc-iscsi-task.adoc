= Überprüfen Sie Ihre FC- oder iSCSI-Protokolllizenz und fügen Sie sie hinzu.
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Bevor Sie den Blockzugriff für eine Storage Virtual Machine (SVM) mit FC oder iSCSI aktivieren können, benötigen Sie eine Lizenz. Die FC- und iSCSI-Lizenzen sind im Lieferumfang von link:../system-admin/manage-licenses-concept.html#licenses-included-with-ontap-one["ONTAP One"] enthalten.

.Schritte
[role="tabbed-block"]
====
.System Manager
--
Falls ONTAP One nicht vorhanden ist, sollte die FC- oder iSCSI-Lizenz mit ONTAP System Manager (9.7 und höher) überprüft und hinzugefügt werden.

. Im System Manager *Cluster > Einstellungen > Lizenzen* auswählen
. Falls die Lizenz nicht aufgeführt ist, image:icon_add_blue_bg.png["Symbol hinzufügen"] auswählen und den Lizenzschlüssel eingeben.
. Wählen Sie *Add*.


--
.CLI
--
Falls ONTAP One nicht vorhanden ist, sollte die FC- oder iSCSI-Lizenz mit der ONTAP CLI überprüft und hinzugefügt werden.

. Es sollte überprüft werden, ob eine gültige Lizenz für FC oder iSCSI vorhanden ist.
+
[source, cli]
----
system license show
----
+
[listing]
----

Package           Type    Description           Expiration
 ----------------- ------- --------------------- --------------------
 Base              site    Cluster Base License  -
 NFS               site    NFS License           -
 CIFS              site    CIFS License          -
 iSCSI             site    iSCSI License         -
 FCP               site    FCP License           -
----
. Falls Sie keine aktive Lizenz für FC oder iSCSI besitzen, kann Ihr Lizenzcode hinzugefügt werden.
+
[source, cli]
----
license add -license-code <your_license_code>
----
