= Empfohlene MTU Konfigurationen für FCoE Jumbo Frames
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Für Fibre Channel over Ethernet (FCoE) sollten Jumbo-Frames für den Ethernet-Adapter des CNA mit einer MTU von 9000 konfiguriert werden. Jumbo-Frames für den FCoE-Adapter des CNA sollten mit einer MTU von mehr als 1500 konfiguriert werden. Jumbo-Frames sollten nur konfiguriert werden, wenn Initiator, Ziel und alle dazwischenliegenden Switches Jumbo-Frames unterstützen und entsprechend konfiguriert sind.
