= LUNs löschen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können eine LUN aus einer Storage Virtual Machine (SVM) löschen, wenn die LUN nicht mehr benötigt wird.

.Bevor Sie beginnen
Die LUN muss von ihrer Initiatorgruppe getrennt werden, bevor sie gelöscht werden kann.

.Schritte
. Es sollte sichergestellt werden, dass die Anwendung oder der Host die LUN nicht verwendet.
. LUN aus der Initiatorgruppe entfernen:
+
[source, cli]
----
lun mapping delete -vserver <SVM_name> -volume <volume_name> -lun <LUN_name> -igroup <igroup_name>
----
. Die LUN löschen:
+
[source, cli]
----
lun delete -vserver <SVM_name> -volume <volume_name> -lun <LUN_name>
----
. Überprüfen, ob die LUN gelöscht wurde:
+
[source, cli]
----
lun show -vserver <SVM_name>
----
+
[listing]
----
Vserver   Path              State    Mapped  Type     Size
--------- ----------------- -------- ------- -------- ------
vs5       /vol/vol16/lun8   online   mapped  windows  10.00GB
----
