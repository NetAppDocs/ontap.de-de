= Automatisches iSCSI LIF Failover aktivieren oder deaktivieren
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Nach dem Upgrade auf ONTAP 9.11.1 oder höher sollte das automatische LIF-Failover auf allen iSCSI-LIFs, die in ONTAP 9.10.1 oder früher erstellt wurden, manuell aktiviert werden.

Ab ONTAP 9.11.1 kann das automatische LIF-Failover für iSCSI LIFs auf All-flash SAN Array Plattformen aktiviert werden. Wenn ein Storage-Failover auftritt, wird die iSCSI LIF automatisch von ihrem Heimatknoten oder -port auf den HA-Partnerknoten oder -port migriert und nach Abschluss des Failovers wieder zurückmigriert. Oder wenn der Port für die iSCSI LIF fehlerhaft wird, wird die LIF automatisch auf einen funktionsfähigen Port ihres aktuellen Heimatknotens migriert und nach Wiederherstellung des Ports wieder auf ihren ursprünglichen Port zurückmigriert. Dadurch können SAN-Workloads, die auf iSCSI laufen, den I/O-Service nach einem Failover schneller wieder aufnehmen.

In ONTAP 9.11.1 und später ist die automatische LIF-Failover-Funktion für neu erstellte iSCSI-LIFs standardmäßig aktiviert, wenn eine der folgenden Bedingungen zutrifft:

* Auf der SVM sind keine iSCSI LIFs vorhanden.
* Alle iSCSI LIFs auf der SVM sind für automatisches LIF-Failover aktiviert.




[[enable-automatic-iscsi-lif-failover]]
== Automatisches iSCSI LIF-Failover aktivieren

Standardmäßig ist für iSCSI-LIFs, die in ONTAP 9.10.1 und früher erstellt wurden, kein automatisches LIF-Failover aktiviert. Wenn auf der SVM iSCSI-LIFs vorhanden sind, für die kein automatisches LIF-Failover aktiviert ist, werden auch neu erstellte LIFs nicht für automatisches LIF-Failover aktiviert. Wenn das automatische LIF-Failover nicht aktiviert ist und ein Failover-Ereignis auftritt, migrieren Ihre iSCSI-LIFs nicht.

Weitere Informationen zu link:../networking/configure_lifs_cluster_administrators_only_overview.html#lif-failover-and-giveback["LIF-Failover und Giveback"].

.Schritt
. Automatisches Failover für eine iSCSI-LIF aktivieren:
+
[source, cli]
----
network interface modify -vserver <SVM_name> -lif <iscsi_lif> -failover-policy sfo-partner-only -auto-revert true
----
+
Um alle iSCSI LIFs auf der SVM zu aktualisieren, sollte  `-lif*` anstelle von  `lif` verwendet werden.





[[disable-automatic-iscsi-lif-failover]]
== Automatisches iSCSI LIF-Failover deaktivieren

Wenn zuvor das automatische iSCSI LIF-Failover für iSCSI LIFs aktiviert wurde, die in ONTAP 9.10.1 oder früher erstellt wurden, besteht die Möglichkeit, es zu deaktivieren.

.Schritt
. Automatisches Failover für eine iSCSI-LIF deaktivieren:
+
[source, cli]
----
network interface modify -vserver <SVM_name> -lif <iscsi_lif> -failover-policy disabled -auto-revert false
----
+
Um alle iSCSI LIFs auf der SVM zu aktualisieren, sollte  `-lif*` anstelle von  `lif` verwendet werden.
