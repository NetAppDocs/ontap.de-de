= Systemvoraussetzungen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Die Einrichtung von LUNs umfasst das Erstellen einer LUN, das Erstellen einer Initiatorgruppe und das Zuordnen der LUN zur Initiatorgruppe. Ihr System muss bestimmte Voraussetzungen erfüllen, bevor die LUNs eingerichtet werden können.

* Die Interoperabilitätsmatrix muss Ihre SAN-Konfiguration als unterstützt auflisten.
* Ihre SAN-Umgebung muss die SAN-Host- und Controller-Konfigurationsgrenzen erfüllen, die in  https://hwu.netapp.com["NetApp Hardware Universe"^] für Ihre Version der ONTAP Software angegeben sind.
* Es muss eine unterstützte Version von Host Utilities installiert sein.
+
Weitere Informationen enthält die Dokumentation zu Host Utilities.

* Sie benötigen SAN LIFs auf dem Knoten, der die LUN besitzt, und auf dem HA-Partner des Knotens, der die LUN besitzt.


.Verwandte Informationen
* https://mysupport.netapp.com/matrix["NetApp Interoperabilitätsmatrix-Tool"^]
* https://docs.netapp.com/us-en/ontap-sanhost/index.html["ONTAP SAN Hostkonfiguration"]
* https://www.netapp.com/pdf.html?item=/media/19680-tr-4017.pdf["NetApp Technical Report 4017: Best Practices für Fibre Channel SAN"^]
