= Das Netzwerk für optimale Leistung konfigurieren
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Die Leistung von Ethernet Netzwerken variiert stark. Die Leistung des für iSCSI verwendeten Netzwerks lässt sich durch die Auswahl bestimmter Konfigurationswerte maximieren.

.Schritte
. Die Host- und Speicheranschlüsse sind mit demselben Netzwerk verbunden.
+
Am besten wird eine Verbindung zu denselben Switches hergestellt. Routing sollte niemals verwendet werden.

. Die schnellsten verfügbaren Ports sollten ausgewählt und für iSCSI reserviert werden.
+
10 GbE-Ports sind optimal. 1 GbE-Ports sind das Minimum.

. Die Ethernet-Flusssteuerung ist für alle Ports zu deaktivieren.
+
Es sollte link:../networking/networking_reference.html["Netzwerkmanagement"] für die Verwendung der CLI zur Konfiguration der Ethernet-Port-Flusssteuerung angezeigt werden.

. Jumbo Frames aktivieren (typischerweise MTU von 9000).
+
Alle Geräte im Datenpfad, einschließlich Initiatoren, Zielgeräte und Switches, müssen Jumbo-Frames unterstützen. Andernfalls verringert die Aktivierung von Jumbo-Frames die Netzwerkleistung erheblich.
