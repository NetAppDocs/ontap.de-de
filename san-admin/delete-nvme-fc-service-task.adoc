= NVMe-Service aus einer SVM entfernen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Bei Bedarf kann der NVMe-Service von Ihrer Storage Virtual Machine (SVM) entfernt werden.

.Schritte
. Die Berechtigungseinstellung auf „Erweitert“ ändern:
+
`set -privilege advanced`

. Den NVMe-Service auf der SVM beenden:
+
`vserver nvme modify -status -admin down`

. Der NVMe-Dienst wird gelöscht:
+
`vserver nvme delete`
