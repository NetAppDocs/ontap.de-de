= Den NVMe-Service für eine SVM starten
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Bevor Sie das NVMe-Protokoll auf Ihrer Storage Virtual Machine (SVM) verwenden können, muss der NVMe-Dienst auf der SVM gestartet werden.

.Bevor Sie beginnen
NVMe muss als Protokoll auf Ihrem System zugelassen sein.

Folgende NVMe-Protokolle werden unterstützt:

[cols="3*"]
|===


| Protokoll | Ab Version ... | Erlaubt durch... 


| TCP | ONTAP 9.10.1 | Standard 


| FCP | ONTAP 9.4 | Standard 
|===
.Schritte
. Die Berechtigungseinstellung auf „Erweitert“ ändern:
+
`set -privilege advanced`

. Es sollte überprüft werden, ob NVMe als Protokoll zulässig ist:
+
`vserver nvme show`

. Den NVMe-Protokolldienst erstellen:
+
`vserver nvme create`

. Der NVMe-Protokolldienst auf der SVM wird gestartet:
+
`vserver nvme modify -status -admin up`
