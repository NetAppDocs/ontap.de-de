= Einen FC-Service für eine SVM löschen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können einen FC-Service für eine Storage Virtual Machine (SVM) löschen, wenn er nicht mehr benötigt wird.

.Bevor Sie beginnen
Der Administrationsstatus muss „down“ sein, bevor ein FC-Service für eine SVM gelöscht werden kann. Der Administrationsstatus kann mit dem  `vserver fcp modify` Befehl oder dem  `vserver fcp stop` Befehl auf „down“ gesetzt werden.

.Schritte
. Der  `vserver fcp stop`-Befehl kann verwendet werden, um die I/O zur LUN zu stoppen.
+
`vserver fcp stop -vserver vs_1`

. Mit dem  `vserver fcp delete`Befehl lässt sich der Dienst aus der SVM entfernen.
+
`vserver fcp delete -vserver vs_1`

. Verwenden Sie das  `vserver fcp show`, um zu überprüfen, ob der FC-Service von Ihrer SVM entfernt wurde:
+
`vserver fcp show -vserver vs_1`
