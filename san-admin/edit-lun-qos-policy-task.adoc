= LUN QoS-Richtliniengruppe bearbeiten
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Ab ONTAP 9.10.1 kann mit System Manager Quality of Service (QoS)-Richtlinien gleichzeitig mehreren LUNs zugewiesen oder von diesen entfernt werden.


NOTE: Wenn die QoS-Richtlinie auf Volume-Ebene zugewiesen ist, muss sie auf Volume-Ebene geändert werden. Die QoS-Richtlinie kann nur auf LUN-Ebene bearbeitet werden, wenn sie ursprünglich auf LUN-Ebene zugewiesen wurde.

.Schritte
. Im System Manager auf *Speicher > LUNs* klicken.
. Wählen Sie die LUN oder LUNs aus, die bearbeitet werden sollen.
+
Wenn Sie mehrere LUNs gleichzeitig bearbeiten, müssen diese zur selben Storage Virtual Machine (SVM) gehören. Wenn Sie LUNs auswählen, die nicht zur selben SVM gehören, wird die Option zum Bearbeiten der QoS Policy Group nicht angezeigt.

. Klicken Sie auf *Mehr* und wählen Sie *QoS Policy Group bearbeiten*.
