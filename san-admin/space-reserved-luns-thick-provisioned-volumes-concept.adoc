= Konfigurationseinstellungen für speicherplatzreservierte Dateien oder LUNs mit Thick Provisioning-Volumes
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Diese FlexVol volume- und Datei- oder LUN-Konfigurationskombination ermöglicht die Nutzung von Speichereffizienztechnologien und erfordert keine aktive Überwachung Ihres freien Speicherplatzes, da ausreichend Speicherplatz im Voraus zugewiesen wird.

Folgende Einstellungen sind erforderlich, um eine speicherplatzreservierte Datei oder LUN in einem Volume mithilfe von Thick Provisioning zu konfigurieren:

[cols="2*"]
|===
| Volume-Einstellung | Wert 


 a| 
Garantie
 a| 
Volumen



 a| 
Teilreserve
 a| 
100



 a| 
Snapshot-Reserve
 a| 
Beliebig



 a| 
Snapshot Autolöschung
 a| 
Optional



 a| 
Automatisches Wachstum
 a| 
Optional; falls aktiviert, muss der freie Speicherplatz des Aggregats aktiv überwacht werden.

|===
[cols="2*"]
|===
| Datei- oder LUN-Einstellung | Wert 


 a| 
Platzreservierung
 a| 
Aktiviert

|===
