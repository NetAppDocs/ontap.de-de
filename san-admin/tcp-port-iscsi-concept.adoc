= TCP-Port für iSCSI
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Das iSCSI-Protokoll ist in ONTAP so konfiguriert, dass es den TCP-Port 3260 verwendet.

ONTAP unterstützt keine Änderung der Portnummer für iSCSI. Portnummer 3260 ist als Teil der iSCSI-Spezifikation registriert und kann von keiner anderen Anwendung oder keinem anderen Dienst verwendet werden.
