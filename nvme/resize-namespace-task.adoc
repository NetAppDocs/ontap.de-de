= Größe eines Namespace ändern
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Ab ONTAP 9.10.1 kann die Größe eines NVMe Namespace über die ONTAP Befehlszeile (CLI) erhöht oder verringert werden. Mit dem System Manager kann die Größe eines NVMe Namespace erhöht werden.



[[increase-the-size-of-a-namespace]]
== Die Größe eines Namespace vergrößern

[role="tabbed-block"]
====
.System Manager
--
. *Speicher > NVMe Namespaces* auswählen.
. Fahren Sie mit dem Mauszeiger über den Namespace, den Sie vergrößern möchten, klicken Sie auf image:icon_kabob.gif["Symbol für Menüoptionen"] und anschließend auf *Bearbeiten*.
. Unter *CAPACITY* die Größe des Namensraums ändern.


--
.CLI
--
. Geben Sie den folgenden Befehl ein:  `vserver nvme namespace modify -vserver _SVM_name_ -path _path_ –size _new_size_of_namespace_`


--
====


[[decrease-the-size-of-a-namespace]]
== Die Größe eines Namensraums verringern

Die ONTAP CLI muss verwendet werden, um die Größe eines NVMe Namespace zu verringern.

. Die Berechtigungseinstellung auf „Erweitert“ ändern:
+
`set –privilege advanced`

. Die Größe des Namensraums verringern:
+
`vserver nvme namespace modify -vserver _SVM_name_ -path _namespace_path_ -size _new_size_of_namespace_`
