= NVMe-Host-VM-Kennung in ONTAP deaktivieren
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Ab ONTAP 9.14.1 unterstützt ONTAP standardmäßig die Möglichkeit von NVMe/FC-Hosts, virtuelle Maschinen anhand einer eindeutigen Kennung zu identifizieren und für NVMe/FC-Hosts, die Ressourcenauslastung virtueller Maschinen zu überwachen. Dies verbessert die Berichterstellung und Fehlerbehebung auf Host-Seite.

Sie können das Bootargument verwenden, um diese Funktionalität zu deaktivieren. Siehe link:https://kb.netapp.com/on-prem/ontap/da/SAN/SAN-KBs/How_to_disable_NVMe_host_virtual_machine_identifier_in_ONTAP["NetApp Knowledge Base: Wie die NVMe-Host-VM-Kennung in ONTAP deaktiviert wird"^].
