= NVMe Lizenzanforderungen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Ab ONTAP 9.5 ist eine Lizenz zur Unterstützung von NVMe erforderlich. Wenn NVMe in ONTAP 9.4 aktiviert ist, wird nach dem Upgrade auf ONTAP 9.5 eine 90-tägige Kulanzfrist gewährt, um die Lizenz zu erwerben.

Die Lizenz kann mit dem folgenden Befehl aktiviert werden:

`system license add -license-code _NVMe_license_key_`
