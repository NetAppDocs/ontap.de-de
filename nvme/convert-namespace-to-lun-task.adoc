= Einen Namespace in eine LUN konvertieren
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Ab ONTAP 9.11.1 kann die ONTAP CLI verwendet werden, um einen vorhandenen NVMe Namespace direkt in eine LUN zu konvertieren.



[[before-you-start]]
== Bevor Sie beginnen

* Der angegebene NVMe Namespace sollte keine bestehenden Zuordnungen zu einem Subsystem haben.
* Der Namespace sollte weder Teil eines Snapshots sein noch auf der Zielseite der SnapMirror Beziehung als schreibgeschützter Namespace vorliegen.
* Da NVMe-Namespaces nur von bestimmten Plattformen und Netzwerkkarten unterstützt werden, funktioniert diese Funktion nur mit bestimmter Hardware.


.Schritte
. Geben Sie den folgenden Befehl ein, um einen NVMe Namespace in eine LUN zu konvertieren:
+
`lun convert-from-namespace -vserver -namespace-path`

+
Weitere Informationen zu  `lun convert-from-namespace` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/lun-convert-from-namespace.html["ONTAP-Befehlsreferenz"^].
