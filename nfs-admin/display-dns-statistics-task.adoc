= DNS-Statistiken für ONTAP NFS SVMs anzeigen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können DNS-Statistiken für Storage Virtual Machines (SVMs) auf dem Speichersystem anzeigen, um die Leistung zu überwachen und Probleme zu diagnostizieren.

.Schritte
. Mit dem  `statistics catalog object show`Befehl lassen sich die DNS-Objekte identifizieren, von denen Daten angezeigt werden können.
+
`statistics catalog object show -object external_service_op*`

. Die  `statistics start`- und  `statistics stop`-Befehle dienen dazu, eine Datenprobe von einem oder mehreren Objekten zu erfassen.
. Mit dem  `statistics show` Befehl lassen sich die Beispieldaten anzeigen.




[[monitoring-dns-statistics]]
== Überwachung der DNS Statistiken

Die folgenden Beispiele zeigen Leistungsdaten für DNS-Abfragen. Die folgenden Befehle starten die Datenerfassung für eine neue Stichprobe:

[listing]
----
vs1::*> statistics start -object external_service_op -sample-id dns_sample1
vs1::*> statistics start -object external_service_op_error -sample-id dns_sample2
----
Der folgende Befehl zeigt Daten aus der Stichprobe an, indem Zähler angegeben werden, die die Anzahl der gesendeten DNS-Abfragen der Anzahl der empfangenen, fehlgeschlagenen oder abgelaufenen DNS-Abfragen gegenüberstellen:

[listing]
----
vs1::*> statistics show -sample-id dns_sample1 -counter num_requests_sent|num_responses_received|num_successful_responses|num_timeouts|num_request_failures|num_not_found_responses

Object: external_service_op
Instance: vs1:DNS:Query:10.72.219.109
Start-time: 3/8/2016 11:15:21
End-time: 3/8/2016 11:16:52
Elapsed-time: 91s
Scope: vs1

    Counter                                                     Value
    -------------------------------- --------------------------------
    num_not_found_responses                                         0
    num_request_failures                                            0
    num_requests_sent                                               1
    num_responses_received                                          1
    num_successful_responses                                        1
    num_timeouts                                                    0
6 entries were displayed.
----
Der folgende Befehl zeigt Daten aus der Stichprobe an, indem Zähler angegeben werden, die die Anzahl der Male anzeigen, bei denen ein bestimmter Fehler bei einer DNS-Abfrage auf dem jeweiligen Server empfangen wurde:

[listing]
----
vs1::*> statistics show -sample-id dns_sample2 -counter server_ip_address|error_string|count

Object: external_service_op_error
Instance: vs1:DNS:Query:NXDOMAIN:10.72.219.109
Start-time: 3/8/2016 11:23:21
End-time: 3/8/2016 11:24:25
Elapsed-time: 64s
Scope: vs1

    Counter                                                     Value
    -------------------------------- --------------------------------
    count                                                           1
    error_string                                             NXDOMAIN
    server_ip_address                                   10.72.219.109
3 entries were displayed.
----
