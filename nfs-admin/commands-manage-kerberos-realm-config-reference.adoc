= ONTAP Befehle zur Verwaltung von NFS Kerberos Realm-Konfigurationen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Für die Verwaltung von NFS-Kerberos-Realm-Konfigurationen gibt es spezielle ONTAP Befehle.

[cols="35,65"]
|===
| Wenn Sie möchten... | Verwenden Sie diesen Befehl... 


 a| 
Erstellen einer NFS Kerberos Realm-Konfiguration
 a| 
`vserver nfs kerberos realm create`



 a| 
NFS Kerberos Realm-Konfigurationen anzeigen
 a| 
`vserver nfs kerberos realm show`



 a| 
Ändern einer NFS Kerberos Realm-Konfiguration
 a| 
`vserver nfs kerberos realm modify`



 a| 
Löschen einer NFS Kerberos Realm-Konfiguration
 a| 
`vserver nfs kerberos realm delete`

|===
Weitere Informationen zu  `vserver nfs kerberos realm` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/search.html?q=vserver+nfs+kerberos+realm["ONTAP-Befehlsreferenz"^].
