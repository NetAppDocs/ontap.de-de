= ONTAP Befehle zur Verwaltung von NAS Name Service Switch-Einträgen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Namensdienst-Switch-Einträge lassen sich durch Erstellen, Anzeigen, Ändern und Löschen verwalten.

[cols="35,65"]
|===


| Wenn Sie möchten... | Verwenden Sie diesen Befehl... 


 a| 
Einen Name Service Switch-Eintrag erstellen
 a| 
`vserver services name-service ns-switch create`



 a| 
Namensdienst-Switch-Einträge anzeigen
 a| 
`vserver services name-service ns-switch show`



 a| 
Ändern eines Name Service Switch-Eintrags
 a| 
`vserver services name-service ns-switch modify`



 a| 
Löschen eines Name Service Switch-Eintrags
 a| 
`vserver services name-service ns-switch delete`

|===
Weitere Informationen zu  `vserver services name-service ns-switch` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/search.html?q=vserver+services+name-service+ns-switch["ONTAP-Befehlsreferenz"^].
