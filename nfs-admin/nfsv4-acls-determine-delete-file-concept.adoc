= Wie ONTAP NFSv4-ACLs verwendet, um zu bestimmen, ob Dateien gelöscht werden können
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Um festzustellen, ob eine Datei gelöscht werden kann, verwendet ONTAP eine Kombination aus dem DELETE-Bit der Datei und dem DELETE_CHILD-Bit des übergeordneten Verzeichnisses. Weitere Informationen enthält die NFS 4.1 RFC 5661.
