= VMware vStorage über ONTAP NFS aktivieren oder deaktivieren
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können die Unterstützung für VMware vStorage über NFS auf Storage Virtual Machines (SVMs) mithilfe des  `vserver nfs modify` Befehls aktivieren oder deaktivieren.

.Über diese Aufgabe
Standardmäßig ist die Unterstützung für VMware vStorage über NFS deaktiviert.

.Schritte
. Der aktuelle vStorage-Unterstützungsstatus für SVMs wird angezeigt:
+
`vserver nfs show -vserver vserver_name -instance`

. Führen Sie eine der folgenden Aktionen aus:
+
[cols="2*"]
|===
| Wenn Sie möchten... | Führen Sie den folgenden Befehl aus... 


 a| 
VMware vStorage-Unterstützung aktivieren
 a| 
`vserver nfs modify -vserver vserver_name -vstorage enabled`



 a| 
VMware vStorage-Support deaktivieren
 a| 
`vserver nfs modify -vserver vserver_name -vstorage disabled`

|===


.Nachdem Sie fertig sind
Sie müssen das NFS Plug-in für VMware VAAI installieren, bevor Sie diese Funktion nutzen können. Weitere Informationen finden sich unter _Installieren des NetApp NFS Plug-ins für VMware VAAI_.
