= Anzeige der Export-Policy-Netgroup-Warteschlange und des Caches für ONTAP NFS SVMs
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ONTAP verwendet die netgroup-Warteschlange beim Importieren und Auflösen von netgroups und nutzt den netgroup-Cache, um die resultierenden Informationen zu speichern. Bei der Fehlerbehebung von Problemen im Zusammenhang mit Exportrichtlinien-netgroup können die  `vserver export-policy netgroup queue show` und  `vserver export-policy netgroup cache show` Befehle verwendet werden, um den Status der netgroup-Warteschlange und den Inhalt des netgroup-Caches anzuzeigen.

.Schritt
. Führen Sie eine der folgenden Aktionen aus:
+
[cols="20,80"]
|===


| Die Exportrichtlinien-Netzgruppe wird angezeigt... | Geben Sie den Befehl ein... 


 a| 
Warteschlange
 a| 
`vserver export-policy netgroup queue show`



 a| 
Cache
 a| 
`vserver export-policy netgroup cache show -vserver vserver_name`

|===
+
Weitere Informationen zu  `vserver export-policy netgroup` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/search.html?q=vserver+export-policy+netgroup["ONTAP-Befehlsreferenz"^].
