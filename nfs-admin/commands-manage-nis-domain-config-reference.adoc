= ONTAP Befehle zur Verwaltung von NFS NIS-Domänenkonfigurationen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Es gibt spezifische ONTAP Befehle für die Verwaltung von NIS-Domänenkonfigurationen.

[cols="35,65"]
|===


| Wenn Sie möchten... | Verwenden Sie diesen Befehl... 


 a| 
Erstellen einer NIS-Domänenkonfiguration
 a| 
`vserver services name-service nis-domain create`



 a| 
NIS-Domänenkonfigurationen anzeigen
 a| 
`vserver services name-service nis-domain show`



 a| 
Anzeige des Bindungsstatus einer NIS-Domänenkonfiguration
 a| 
`vserver services name-service nis-domain show-bound`



 a| 
NIS Statistiken anzeigen
 a| 
`vserver services name-service nis-domain show-statistics` Verfügbar ab der Berechtigungsstufe „Erweitert“ und höher.



 a| 
NIS-Statistiken löschen
 a| 
`vserver services name-service nis-domain clear-statistics` Verfügbar ab der Berechtigungsstufe „Erweitert“ und höher.



 a| 
Ändern einer NIS Domänenkonfiguration
 a| 
`vserver services name-service nis-domain modify`



 a| 
Löschen einer NIS Domainkonfiguration
 a| 
`vserver services name-service nis-domain delete`



 a| 
Caching für netgroup-by-host-Suchen aktivieren
 a| 
`vserver services name-service nis-domain netgroup-database config modify` Verfügbar ab der Berechtigungsstufe „Erweitert“ und höher.

|===
Weitere Informationen zu  `vserver services name-service nis-domain` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/search.html?q=vserver+services+name-service+nis-domain["ONTAP-Befehlsreferenz"^].
