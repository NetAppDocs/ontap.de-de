= NIS-Statistiken für ONTAP NFS SVMs anzeigen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können NIS-Statistiken für Storage Virtual Machines (SVMs) auf dem Speichersystem anzeigen, um die Leistung zu überwachen und Probleme zu diagnostizieren.

.Schritte
. Mit dem  `statistics catalog object show`Befehl lassen sich die NIS-Objekte identifizieren, von denen Daten angezeigt werden können.
+
`statistics catalog object show -object external_service_op*`

. Die  `statistics start`- und  `statistics stop`-Befehle dienen dazu, eine Datenprobe von einem oder mehreren Objekten zu erfassen.
. Mit dem  `statistics show` Befehl lassen sich die Beispieldaten anzeigen.




[[monitoring-nis-statistics]]
== Überwachung der NIS Statistiken

Die folgenden Beispiele zeigen Leistungsdaten für NIS-Abfragen. Die folgenden Befehle starten die Datenerfassung für eine neue Stichprobe:

[listing]
----
vs1::*> statistics start -object external_service_op -sample-id nis_sample1
vs1::*> statistics start -object external_service_op_error -sample-id nis_sample2
----
Der folgende Befehl zeigt Daten aus der Stichprobe an, indem Zähler angegeben werden, die die Anzahl der gesendeten NIS-Abfragen der Anzahl der empfangenen, fehlgeschlagenen oder abgelaufenen NIS-Abfragen gegenüberstellen:

[listing]
----
vs1::*> statistics show -sample-id nis_sample1 -counter instance|num_requests_sent|num_responses_received|num_successful_responses|num_timeouts|num_request_failures|num_not_found_responses

Object: external_service_op
Instance: vs1:NIS:Query:10.227.13.221
Start-time: 3/8/2016 11:27:39
End-time: 3/8/2016 11:27:56
Elapsed-time: 17s
Scope: vs1

    Counter                                                     Value
    -------------------------------- --------------------------------
    num_not_found_responses                                         0
    num_request_failures                                            1
    num_requests_sent                                               2
    num_responses_received                                          1
    num_successful_responses                                        1
    num_timeouts                                                    0
6 entries were displayed.
----
Der folgende Befehl zeigt Daten aus der Stichprobe an, indem Zähler angegeben werden, die die Anzahl der Male anzeigen, bei denen ein bestimmter Fehler bei einer NIS-Abfrage auf dem jeweiligen Server empfangen wurde:

[listing]
----
vs1::*> statistics show -sample-id nis_sample2 -counter server_ip_address|error_string|count

Object: external_service_op_error
Instance: vs1:NIS:Query:YP_NOTFOUND:10.227.13.221
Start-time: 3/8/2016 11:33:05
End-time: 3/8/2016 11:33:10
Elapsed-time: 5s
Scope: vs1

    Counter                                                     Value
    -------------------------------- --------------------------------
    count                                                           1
    error_string                                          YP_NOTFOUND
    server_ip_address                                   10.227.13.221
3 entries were displayed.
----
