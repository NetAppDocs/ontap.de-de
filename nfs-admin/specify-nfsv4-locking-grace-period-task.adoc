= Die NFSv4-Sperrfrist für ONTAP SVMs angeben
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Um die NFSv4-Sperrfrist festzulegen (d. h. den Zeitraum, in dem Clients versuchen, ihren Sperrstatus von ONTAP während der Serverwiederherstellung zurückzuerhalten), kann die  `-v4-grace-seconds` Option angepasst werden.

.Über diese Aufgabe
Standardmäßig ist diese Option auf  `45` eingestellt.

.Schritte
. Stellen Sie die Berechtigungsstufe auf „Erweitert“ ein:
+
`set -privilege advanced`

. Führen Sie den folgenden Befehl aus:
+
`vserver nfs modify -vserver vserver_name -v4-grace-seconds _number_of_seconds_`

. Zurück zur Administrator-Privilegstufe:
+
`set -privilege admin`
