= ONTAP Befehle zur Verwaltung von Exportrichtlinien
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Für die Verwaltung von Exportrichtlinien gibt es spezielle ONTAP Befehle.

[cols="35,65"]
|===


| Wenn Sie möchten... | Verwenden Sie diesen Befehl... 


 a| 
Informationen zu Exportrichtlinien anzeigen
 a| 
`vserver export-policy show`



 a| 
Umbenennen einer Exportregel
 a| 
`vserver export-policy rename`



 a| 
Exportregel kopieren
 a| 
`vserver export-policy copy`



 a| 
Exportregel löschen
 a| 
`vserver export-policy delete`

|===
Weitere Informationen zu  `vserver export-policy` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/search.html?q=vserver+export-policy["ONTAP-Befehlsreferenz"^].
