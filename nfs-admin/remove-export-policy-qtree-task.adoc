= Entfernen von Exportrichtlinien aus ONTAP NFS-qtrees
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Wenn Sie entscheiden, dass einem Qtree keine bestimmte Exportregel mehr zugewiesen sein soll, kann die Exportregel entfernt werden, indem der Qtree so geändert wird, dass er stattdessen die Exportregel des übergeordneten Volumes übernimmt. Dies ist möglich mit dem  `volume qtree modify` Befehl und dem  `-export-policy` Parameter sowie einer leeren Namenszeichenfolge ("").

.Schritte
. Um eine Exportregel aus einem Qtree zu entfernen, ist folgender Befehl einzugeben:
+
`volume qtree modify -vserver vserver_name -qtree-path /vol/volume_name/qtree_name -export-policy ""`

. Es sollte überprüft werden, ob der Qtree entsprechend angepasst wurde:
+
`volume qtree show -qtree qtree_name -fields export-policy`
