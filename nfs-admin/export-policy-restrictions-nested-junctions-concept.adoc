= Beschränkungen von Exportrichtlinien und verschachtelte Verbindungen für ONTAP NFS FlexVol Volumes
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Wenn Exportregeln so konfiguriert wurden, dass auf einer verschachtelten Verbindung eine weniger restriktive Regel und auf einer übergeordneten Verbindung eine restriktivere Regel gilt, kann der Zugriff auf die Verbindung auf niedrigerer Ebene fehlschlagen.

Es ist darauf zu achten, dass Verbindungspunkte höherer Ebenen weniger restriktive Exportregeln haben als Verbindungspunkte niedrigerer Ebenen.
