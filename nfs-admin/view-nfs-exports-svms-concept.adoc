= Anzeige von Exporten auf NFS-Clients für ONTAP SVMs aktivieren
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
NFS-Clients können den  `showmount -e` Befehl verwenden, um eine Liste der von einem ONTAP NFS-Server verfügbaren Exporte anzuzeigen. Dies kann Benutzern helfen, das Dateisystem zu identifizieren, das sie einbinden möchten.

ONTAP ermöglicht NFS-Clients standardmäßig die Anzeige der Exportliste. In früheren Versionen musste die  `showmount`-Option des  `vserver nfs modify`-Befehls explizit aktiviert werden. Für die Anzeige der Exportliste muss NFSv3 auf der SVM aktiviert sein.

.Beispiel
Der folgende Befehl zeigt das showmount Feature auf der SVM mit dem Namen vs1:

[listing]
----
clusterl : : > vserver nfs show -vserver vs1 -fields showmount
vserver showmount
------- ---------
vs1    enabled
----
Der folgende Befehl, der auf einem NFS-Client ausgeführt wird, zeigt die Liste der Exporte auf einem NFS-Server mit der IP-Adresse 10.63.21.9 an:

[listing]
----
showmount -e 10.63.21.9
Export list for 10.63.21.9:
/unix        (everyone)
/unix/unixl  (everyone)
/unix/unix2  (everyone)
/            (everyone)
----
