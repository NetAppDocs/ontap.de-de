= Statistiken für ONTAP NFS SVMs anzeigen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können NFS-Statistiken für Storage Virtual Machines (SVMs) auf dem Speichersystem anzeigen, um die Leistung zu überwachen und Probleme zu diagnostizieren.

.Schritte
. Mit dem  `statistics catalog object show`Befehl lassen sich die NFS-Objekte identifizieren, von denen Daten angezeigt werden können.
+
`statistics catalog object show -object nfs*`

. Verwenden Sie die  `statistics start` und optionalen  `statistics stop` Befehle, um eine Datenprobe von einem oder mehreren Objekten zu erfassen.
. Mit dem  `statistics show` Befehl lassen sich die Beispieldaten anzeigen.


.Beispiel: Überwachung der NFSv3 Leistung
Das folgende Beispiel zeigt Leistungsdaten für das NFSv3 Protokoll.

Der folgende Befehl startet die Datenerfassung für eine neue Probe:

[listing]
----
vs1::> statistics start -object nfsv3 -sample-id nfs_sample
----
Der folgende Befehl zeigt Daten aus der Stichprobe an, indem Zähler angegeben werden, die die Anzahl erfolgreicher Lese- und Schreibanforderungen im Verhältnis zur Gesamtzahl der Lese- und Schreibanforderungen anzeigen:

[listing]
----

vs1::> statistics show -sample-id nfs_sample -counter read_total|write_total|read_success|write_success

Object: nfsv3
Instance: vs1
Start-time: 2/11/2013 15:38:29
End-time: 2/11/2013 15:38:41
Cluster: cluster1

    Counter                                           Value
    --------------------------- ---------------------------
    read_success                                      40042
    read_total                                        40042
    write_success                                   1492052
    write_total                                     1492052
----
