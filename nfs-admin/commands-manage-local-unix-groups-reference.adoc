= ONTAP Befehle zur Verwaltung von NAS lokalen UNIX-Gruppen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Für die Verwaltung lokaler UNIX-Gruppen gibt es spezielle ONTAP-Befehle.

[cols="35,65"]
|===


| Wenn Sie möchten... | Verwenden Sie diesen Befehl... 


 a| 
Eine lokale UNIX-Gruppe erstellen
 a| 
`vserver services name-service unix-group create`



 a| 
Einen Benutzer zu einer lokalen UNIX Gruppe hinzufügen
 a| 
`vserver services name-service unix-group adduser`



 a| 
Lokale UNIX-Gruppen von einer URI laden
 a| 
`vserver services name-service unix-group load-from-uri`



 a| 
Lokale UNIX-Gruppen anzeigen
 a| 
`vserver services name-service unix-group show`



 a| 
Eine lokale UNIX-Gruppe ändern
 a| 
`vserver services name-service unix-group modify`



 a| 
Einen Benutzer aus einer lokalen UNIX Gruppe löschen
 a| 
`vserver services name-service unix-group deluser`



 a| 
Eine lokale UNIX-Gruppe löschen
 a| 
`vserver services name-service unix-group delete`

|===
Weitere Informationen zu  `vserver services name-service unix-group` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/search.html?q=vserver+services+name-service+unix-group["ONTAP-Befehlsreferenz"^].
