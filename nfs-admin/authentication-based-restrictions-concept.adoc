= Informationen zu authentifizierungsbasierten Einschränkungen für ONTAP NAS SVMs
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Mit authentifizierungsbasierten Einschränkungen lässt sich festlegen, welche Client-Rechner und welche Benutzer eine Verbindung zur Storage Virtual Machine (SVM) herstellen können.

ONTAP unterstützt die Kerberos-Authentifizierung sowohl von UNIX- als auch von Windows-Servern.
