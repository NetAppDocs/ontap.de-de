= Sicherheitsstile auf ONTAP NFS SVM Root-Volumes konfigurieren
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Der Sicherheitsstil des Root-Volumes der Storage Virtual Machine (SVM) wird konfiguriert, um die Art der Berechtigungen für Daten auf dem Root-Volume der SVM festzulegen.

.Schritte
. Der  `vserver create`Befehl mit dem  `-rootvolume-security-style`Parameter dient zur Definition des Sicherheitsstils.
+
Die möglichen Optionen für den Sicherheitsstil des Root-Volumes sind `unix`, `ntfs` oder `mixed`.

. Die Konfiguration, einschließlich des Sicherheitsstils des Root-Volumes der erstellten SVM, anzeigen und überprüfen:
+
`vserver show -vserver _vserver_name_`
