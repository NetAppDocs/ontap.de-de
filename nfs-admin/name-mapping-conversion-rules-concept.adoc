= Namenszuordnungskonvertierungsregeln für ONTAP NAS SVMs
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Ein ONTAP-System verwaltet für jede SVM einen Satz von Konvertierungsregeln. Jede Regel besteht aus zwei Teilen: einem Muster und einer Ersetzung. Die Konvertierung beginnt am Anfang der entsprechenden Liste und führt eine Ersetzung basierend auf der ersten übereinstimmenden Regel durch. Das Muster ist ein regulärer Ausdruck im UNIX-Stil. Die Ersetzung ist eine Zeichenkette mit Escape-Sequenzen, die Teilausdrücke des Musters darstellen, analog zum UNIX `sed` Programm.
