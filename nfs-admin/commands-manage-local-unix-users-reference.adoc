= ONTAP Befehle zur Verwaltung lokaler UNIX-Benutzer für NAS
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Für die Verwaltung lokaler UNIX-Benutzer gibt es spezielle ONTAP-Befehle.

[cols="35,65"]
|===


| Wenn Sie möchten... | Verwenden Sie diesen Befehl... 


 a| 
Einen lokalen UNIX-Benutzer erstellen
 a| 
`vserver services name-service unix-user create`



 a| 
Lokale UNIX-Benutzer von einer URI laden
 a| 
`vserver services name-service unix-user load-from-uri`



 a| 
Lokale UNIX-Benutzer anzeigen
 a| 
`vserver services name-service unix-user show`



 a| 
Einen lokalen UNIX-Benutzer ändern
 a| 
`vserver services name-service unix-user modify`



 a| 
Einen lokalen UNIX-Benutzer löschen
 a| 
`vserver services name-service unix-user delete`

|===
Weitere Informationen zu  `vserver services name-service unix-user` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/search.html?q=vserver+services+name-service+unix-user["ONTAP-Befehlsreferenz"^].
