= ONTAP Befehle zur Verwaltung von NFS Kerberos-Schnittstellenkonfigurationen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Für die Verwaltung von NFS-Kerberos-Schnittstellenkonfigurationen gibt es spezielle ONTAP-Befehle.

[cols="35,65"]
|===


| Wenn Sie möchten... | Verwenden Sie diesen Befehl... 


 a| 
NFS Kerberos auf einer LIF aktivieren
 a| 
`vserver nfs kerberos interface enable`



 a| 
NFS Kerberos-Schnittstellenkonfigurationen
 a| 
`vserver nfs kerberos interface show`



 a| 
Ändern einer NFS Kerberos-Schnittstellenkonfiguration
 a| 
`vserver nfs kerberos interface modify`



 a| 
NFS Kerberos auf einer LIF deaktivieren
 a| 
`vserver nfs kerberos interface disable`

|===
Weitere Informationen zu  `vserver nfs kerberos interface` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/search.html?q=vserver+nfs+kerberos+interface["ONTAP-Befehlsreferenz"^].
