= Informationen zur ONTAP Unterstützung für paralleles NFS
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ONTAP unterstützt paralleles NFS (pNFS). Das pNFS-Protokoll bietet Leistungsverbesserungen, indem es Clients direkten Zugriff auf die Daten einer Reihe von Dateien ermöglicht, die über mehrere Knoten eines Clusters verteilt sind. Dadurch können Clients den optimalen Pfad zu einem Volume ermitteln.
