= ONTAP Befehle zur Verwaltung des NAS Name Service Cache
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können den Namensdienstcache verwalten, indem Sie den Wert für die Gültigkeitsdauer (TTL) ändern. Der TTL-Wert bestimmt, wie lange Namensdienstinformationen im Cache gespeichert bleiben.

[cols="35,65"]
|===


| Wenn Sie den TTL-Wert für ... ändern möchten | Verwenden Sie diesen Befehl... 


 a| 
Unix-Benutzer
 a| 
`vserver services name-service cache unix-user settings`



 a| 
Unix-Gruppen
 a| 
`vserver services name-service cache unix-group settings`



 a| 
Unix Netgroups
 a| 
`vserver services name-service cache netgroups settings`



 a| 
Hosts
 a| 
`vserver services name-service cache hosts settings`



 a| 
Gruppenmitgliedschaft
 a| 
`vserver services name-service cache group-membership settings`

|===
