= Informationen zu den Vorteilen der Aktivierung von NFSv4-ACLs für ONTAP SVMs
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Die Aktivierung von NFSv4 ACLs bietet viele Vorteile.

Die Vorteile der Aktivierung von NFSv4-ACLs umfassen Folgendes:

* Feinere Steuerung des Benutzerzugriffs auf Dateien und Verzeichnisse
* Bessere NFS-Sicherheit
* Verbesserte Interoperabilität mit CIFS
* Aufhebung der NFS-Beschränkung von 16 Gruppen pro Benutzer
