= ONTAP Befehle zur Verwaltung von NFS LDAP-Konfigurationen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Für die Verwaltung von LDAP-Konfigurationen gibt es spezielle ONTAP-Befehle.

[cols="35,65"]
|===


| Wenn Sie möchten... | Verwenden Sie diesen Befehl... 


 a| 
Erstellen einer LDAP-Konfiguration
 a| 
`vserver services name-service ldap create`



 a| 
LDAP-Konfigurationen anzeigen
 a| 
`vserver services name-service ldap show`



 a| 
Ändern einer LDAP-Konfiguration
 a| 
`vserver services name-service ldap modify`



 a| 
Eine LDAP-Konfiguration löschen
 a| 
`vserver services name-service ldap delete`

|===
Weitere Informationen zu  `vserver services name-service ldap` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/search.html?q=vserver+services+name-service+ldap["ONTAP-Befehlsreferenz"^].
