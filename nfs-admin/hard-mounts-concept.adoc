= Informationen zu ONTAP NFS Hard Mounts
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Bei der Fehlersuche bei Mount-Problemen ist sicherzustellen, dass der richtige Mount-Typ verwendet wird. NFS unterstützt zwei Mount-Typen: Soft Mounts und Hard Mounts. Aus Gründen der Zuverlässigkeit empfiehlt sich ausschließlich die Verwendung von Hard Mounts.

Soft-Mounts sollten nicht verwendet werden, insbesondere wenn häufige NFS-Timeouts auftreten können. Durch diese Timeouts können Race Conditions entstehen, die zu Datenbeschädigung führen können.
