= Informationen zu ONTAP Dateizugriff für das NFS-Protokoll
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ONTAP beinhaltet Dateizugriffsfunktionen, die für das NFS-Protokoll verfügbar sind. Ein NFS-Server kann aktiviert und Volumes oder Qtrees können exportiert werden.

Diese Prozedur wird unter folgenden Umständen durchgeführt:

* Sie möchten den Funktionsumfang des ONTAP NFS-Protokolls verstehen.
* Es sollen weniger gängige Konfigurations- und Wartungsaufgaben durchgeführt werden, nicht die grundlegende NFS-Konfiguration.
* Die Befehlszeilenschnittstelle (CLI) soll verwendet werden, nicht der System Manager oder ein automatisiertes Skripttool.
