= ONTAP Befehle zur Verwaltung von Exportregeln
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Für die Verwaltung von Exportregeln gibt es spezielle ONTAP-Befehle.

[cols="35,65"]
|===


| Wenn Sie möchten... | Verwenden Sie diesen Befehl... 


 a| 
Exportregel erstellen
 a| 
`vserver export-policy rule create`



 a| 
Informationen zu Exportregeln
 a| 
`vserver export-policy rule show`



 a| 
Eine Exportregel ändern
 a| 
`vserver export-policy rule modify`



 a| 
Eine Exportregel löschen
 a| 
`vserver export-policy rule delete`

|===
[NOTE]
====
Wenn Sie mehrere identische Exportregeln für verschiedene Clients konfiguriert haben, sollten diese bei der Verwaltung von Exportregeln synchron gehalten werden.

====
Weitere Informationen zu  `vserver export-policy` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/search.html?q=vserver+export-policy["ONTAP-Befehlsreferenz"^].
