= Die ONTAP Benutzer-ID-Domäne für NFSv4 angeben
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Um die Benutzer-ID-Domäne anzugeben, kann die  `-v4-id-domain` Option festgelegt werden.

.Über diese Aufgabe
Standardmäßig verwendet ONTAP die NIS-Domäne für die NFSv4-Benutzer-ID-Zuordnung, sofern eine solche konfiguriert ist. Ist keine NIS-Domäne konfiguriert, wird die DNS-Domäne verwendet. Die Konfiguration der Benutzer-ID-Domäne kann erforderlich sein, beispielsweise wenn mehrere Benutzer-ID-Domänen vorhanden sind. Der Domänenname muss mit der Domänenkonfiguration auf dem Domänencontroller übereinstimmen. Für NFSv3 ist dies nicht erforderlich.

.Schritt
. Führen Sie den folgenden Befehl aus:
+
`vserver nfs modify -vserver vserver_name -v4-id-domain NIS_domain_name`
