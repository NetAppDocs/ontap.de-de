= NFSv4-ACLs für ONTAP SVMs aktivieren oder deaktivieren
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Um NFSv4-ACLs zu aktivieren oder zu deaktivieren, können die Optionen  `-v4.0-acl` und  `-v4.1-acl` geändert werden. Diese Optionen sind standardmäßig deaktiviert.

.Über diese Aufgabe
Die  `-v4.0-acl` oder  `-v4.1-acl` Option steuert das Festlegen und Anzeigen von NFSv4-ACLs; sie steuert nicht die Durchsetzung dieser ACLs für die Zugriffsprüfung.

.Schritt
. Führen Sie eine der folgenden Aktionen aus:
+
[cols="30,70"]
|===


| Wenn Sie möchten... | Dann... 


 a| 
NFSv4.0-ACLs aktivieren
 a| 
Führen Sie den folgenden Befehl aus:

`vserver nfs modify -vserver vserver_name -v4.0-acl enabled`



 a| 
NFSv4.0 ACLs deaktivieren
 a| 
Führen Sie den folgenden Befehl aus:

`vserver nfs modify -vserver vserver_name -v4.0-acl disabled`



 a| 
NFSv4.1-ACLs aktivieren
 a| 
Führen Sie den folgenden Befehl aus:

`vserver nfs modify -vserver vserver_name -v4.1-acl enabled`



 a| 
NFSv4.1 ACLs deaktivieren
 a| 
Führen Sie den folgenden Befehl aus:

`vserver nfs modify -vserver vserver_name -v4.1-acl disabled`

|===
