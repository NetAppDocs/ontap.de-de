= ONTAP Befehle zur Verwaltung von NFS-LDAP-Client-Konfigurationen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Für die Verwaltung von LDAP-Client-Konfigurationen gibt es spezielle ONTAP-Befehle.

[NOTE]
====
SVM-Administratoren können LDAP-Client-Konfigurationen, die von Cluster-Administratoren erstellt wurden, nicht ändern oder löschen.

====
[cols="35,65"]
|===


| Wenn Sie möchten... | Verwenden Sie diesen Befehl... 


 a| 
Erstellen einer LDAP-Client-Konfiguration
 a| 
`vserver services name-service ldap client create`



 a| 
LDAP-Client-Konfigurationen anzeigen
 a| 
`vserver services name-service ldap client show`



 a| 
Ändern einer LDAP-Client-Konfiguration
 a| 
`vserver services name-service ldap client modify`



 a| 
Das BIND-Passwort des LDAP-Clients ändern
 a| 
`vserver services name-service ldap client modify-bind-password`



 a| 
Löschen einer LDAP-Client-Konfiguration
 a| 
`vserver services name-service ldap client delete`

|===
Weitere Informationen zu  `vserver services name-service ldap client` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/search.html?q=vserver+services+name-service+ldap+client["ONTAP-Befehlsreferenz"^].
