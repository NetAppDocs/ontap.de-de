= Informationen zu den Abhängigkeiten bei der Benennung von Dateien und Verzeichnissen in ONTAP NFS und SMB
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Die Namenskonventionen für Dateien und Verzeichnisse hängen sowohl von den Betriebssystemen der Netzwerkclients als auch von den Dateifreigabeprotokollen sowie von den Spracheinstellungen des ONTAP Clusters und der Clients ab.

Das Betriebssystem und die Dateiaustauschprotokolle bestimmen Folgendes:

* Zeichen, die ein Dateiname verwenden darf
* Groß-/Kleinschreibung eines Dateinamens


ONTAP unterstützt Mehrbytezeichen in Datei-, Verzeichnis- und Qtree-Namen, abhängig von der ONTAP Version.
