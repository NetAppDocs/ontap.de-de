= ONTAP Befehle zur Verwaltung lokaler NFS-Netgroups
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Lokale Netzwerkgruppen lassen sich verwalten, indem sie von einem URI geladen, ihr Status auf den Knoten überprüft, angezeigt und gelöscht werden.

[cols="35,65"]
|===


| Wenn Sie möchten... | Verwenden Sie den Befehl... 


 a| 
Netzgruppen aus einer URI laden
 a| 
`vserver services name-service netgroup load`



 a| 
Status der Netgroups auf allen Knoten überprüfen
 a| 
`vserver services name-service netgroup status`

Verfügbar ab der Stufe „Erweitert“ und höher.



 a| 
Lokale Netzgruppen anzeigen
 a| 
`vserver services name-service netgroup file show`



 a| 
Eine lokale Netzgruppe löschen
 a| 
`vserver services name-service netgroup file delete`

|===
Weitere Informationen zu  `vserver services name-service netgroup file` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/search.html?q=vserver+services+name-service+netgroup+file["ONTAP-Befehlsreferenz"^].
