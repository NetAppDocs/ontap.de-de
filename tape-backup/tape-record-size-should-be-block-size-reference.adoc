= Die Größe des Banddatensatzes sollte block_size1 und nicht block_size2 betragen.
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`Tape record size should be block_size1 and not block_size2`

* *Ursache*
+
Der für die lokale Wiederherstellung angegebene Blockierungsfaktor ist falsch.

* *Korrekturmaßnahme*
+
Den Wiederherstellungsauftrag mit  `block_size1` als blockierendem Faktor erneut versuchen.
