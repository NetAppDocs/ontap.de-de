= Fehler beim Abrufen des neuesten Snapshots
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`Failed to get latest snapshot`

* *Ursache*
+
Der neueste Snapshot existiert möglicherweise nicht, da das Volume gerade von SnapMirror initialisiert wird.

* *Korrekturmaßnahme*
+
Nach Abschluss der Initialisierung erneut versuchen.
