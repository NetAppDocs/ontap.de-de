= Abweichung im Backup-Zeitstempel
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`Mismatch in backup time stamp`

* *Ursache*
+
Das beim Bandwechsel eingelegte Band ist nicht Teil des Backup-Satzes.

* *Korrekturmaßnahme*
+
Mit dem  `smtape restore -h`Befehl lassen sich die Header-Informationen eines Bandes überprüfen.
