= Informationen zu ONTAP Tape Seeding
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Tape Seeding ist eine SMTape-Funktionalität, die bei der Initialisierung eines Ziel-FlexVol volume in einer Datensicherungs-Spiegelbeziehung unterstützt.

Tape Seeding ermöglicht die Einrichtung einer Datensicherungs-Spiegelbeziehung zwischen einem Quellsystem und einem Zielsystem über eine Verbindung mit geringer Bandbreite.

Die inkrementelle Spiegelung von Snapshots vom Quellsystem zum Zielsystem ist über eine Verbindung mit geringer Bandbreite möglich. Eine initiale Spiegelung des Basis-Snapshots dauert jedoch über eine Verbindung mit geringer Bandbreite lange. In solchen Fällen kann ein SMTape-Backup des Quell-Volumes auf ein Band durchgeführt werden, und das Band kann verwendet werden, um den initialen Basis-Snapshot auf das Zielsystem zu übertragen. Danach lassen sich inkrementelle SnapMirror-Updates zum Zielsystem über die Verbindung mit geringer Bandbreite einrichten.
