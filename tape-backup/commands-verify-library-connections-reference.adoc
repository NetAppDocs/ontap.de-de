= ONTAP Befehle zur Überprüfung der Tape-Bibliothek-Verbindungen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können Informationen zum Verbindungspfad zwischen einem Speichersystem und einer an das Speichersystem angeschlossenen Tape-Bibliothek anzeigen. Diese Informationen können zur Überprüfung des Verbindungspfads zur Tape-Bibliothek oder zur Fehlerbehebung bei Problemen im Zusammenhang mit den Verbindungspfaden verwendet werden.

Die folgenden Tape-Bibliothek-Details können angezeigt werden, um die Tape-Bibliothek-Verbindungen nach dem Hinzufügen oder Erstellen einer neuen Tape-Bibliothek oder nach der Wiederherstellung eines ausgefallenen Pfads bei Einzelpfad- oder Mehrpfadzugriff auf eine Tape-Bibliothek zu überprüfen. Diese Informationen sind auch bei der Fehlerbehebung von pfadbezogenen Fehlern oder bei einem Zugriffsfehler auf eine Tape-Bibliothek hilfreich.

* Knoten, an dem die Tape-Bibliothek angeschlossen ist
* Geräte-ID
* NDMP Pfad
* Tape-Bibliothek Name
* Zielport- und Initiatorport-IDs
* Single-Path- oder Multipath-Zugriff auf eine Tape-Bibliothek für jeden Ziel- oder FC-Initiatorport
* Details zur Datenintegrität im Zusammenhang mit Pfaden, wie „`Path Errors`“ und „`Path Qual`“
* LUN-Gruppen und LUN-Anzahlen


|===
| Wenn Sie möchten... | Verwenden Sie diesen Befehl... 


 a| 
Informationen zu einer Tape-Bibliothek in einem Cluster
 a| 
`system node hardware tape library show`



 a| 
Pfadinformationen für eine Tape-Bibliothek anzeigen
 a| 
`storage tape library path show`



 a| 
Pfadinformationen für eine Tape-Bibliothek für jeden Initiatorport
 a| 
`storage tape library path show-by-initiator`



 a| 
Verbindungsinformationen zwischen einer Tape-Bibliothek und einem Cluster anzeigen
 a| 
`storage tape library config show`
