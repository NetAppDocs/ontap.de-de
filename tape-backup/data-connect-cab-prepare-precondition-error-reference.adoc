= DATENVERBINDUNG: Fehler bei der Vorbereitung der CAB-Datenverbindungsvoraussetzung
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`DATA CONNECT: CAB data connection prepare precondition error`

* *Ursache*
+
Die NDMP-Datenverbindung schlägt fehl, wenn die Backup-Anwendung die CAB-Erweiterung mit dem NDMP-Server ausgehandelt hat und es eine Diskrepanz im angegebenen NDMP-Datenverbindungsadresstyp zwischen den NDMP_CAB_DATA_CONN_PREPARE- und den NDMP_DATA_CONNECT-Nachrichten gibt.

* *Korrekturmaßnahme*
+
Wenden Sie sich an Ihren Backup-Anwendungsanbieter.
