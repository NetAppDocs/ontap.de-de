= Der Ziel-qtree ist schreibgeschützt
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`Destination qtree is read-only`

* *Ursache*
+
Der Qtree, in den die Wiederherstellung erfolgen soll, ist schreibgeschützt.

* *Korrekturmaßnahme*
+
Es kann versucht werden, die Daten an einem anderen Speicherort wiederherzustellen.
