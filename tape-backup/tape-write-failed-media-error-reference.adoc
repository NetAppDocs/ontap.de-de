= Fehler beim Schreiben auf das Band, neues Band hat einen Medienfehler
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`Tape write failed - new tape encountered media error`

* *Ursache*
+
Das für das Backup verwendete Band ist beschädigt.

* *Korrekturmaßnahme*
+
Das Band sollte ausgetauscht und das Backup erneut versucht werden.
