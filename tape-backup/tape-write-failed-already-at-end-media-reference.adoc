= Fehler beim Schreiben auf das Band, das neue Band befindet sich bereits am Ende des Mediums.
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`Tape write failed - new tape is already at the end of media`

* *Ursache*
+
Auf dem Band ist nicht genügend Speicherplatz vorhanden, um das Backup abzuschließen.

* *Korrekturmaßnahme*
+
Das Band sollte ausgetauscht und das Backup erneut versucht werden.
