= Bandaliase für ein Tape-Laufwerk oder einen Medienwechsler bei ONTAP Tape-Backup entfernen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Aliase können mit dem  `storage tape alias clear` Befehl entfernt werden, wenn persistente Aliase für ein Tape-Laufwerk oder einen Medium-Changer nicht mehr erforderlich sind.

.Schritte
. Ein Alias kann von einem Tape-Laufwerk oder Medienwechsler mit dem  `storage tape alias clear` Befehl entfernt werden.
+
Weitere Informationen zu  `storage tape alias clear` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/storage-tape-alias-clear.html["ONTAP-Befehlsreferenz"^].

+
Der folgende Befehl entfernt die Aliase aller Tape-Laufwerke, indem der Umfang der Alias-Löschoperation auf  `tape` festgelegt wird:

+
[listing]
----
cluster-01::>storage tape alias clear -node cluster-01 -clear-scope tape
----


.Nachdem Sie fertig sind
Wenn eine Tape-Backup- oder Wiederherstellungsoperation mit NDMP durchgeführt wird, muss nach dem Entfernen eines Alias von einem Tape-Laufwerk oder Medienwechsler dem Tape-Laufwerk oder Medienwechsler ein neuer Aliasname zugewiesen werden, um weiterhin Zugriff auf das Tape-Gerät zu haben.
