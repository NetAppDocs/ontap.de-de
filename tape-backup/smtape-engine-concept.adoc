= Informationen zur ONTAP SMTape Engine für FlexVol Volumes
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
SMTape ist eine Disaster Recovery-Lösung von ONTAP, die Datenblöcke auf Tape sichert. Mit SMTape lassen sich Volume-Backups auf Tape durchführen. Ein Backup auf Qtree- oder Subtree-Ebene ist jedoch nicht möglich. SMTape unterstützt Baseline-, Differenzial- und inkrementelle Backups. Für SMTape ist keine Lizenz erforderlich.

Sie können eine SMTape-Sicherung und -Wiederherstellung mithilfe einer NDMP-kompatiblen Backup-Anwendung durchführen. SMTape kann für Backup- und Wiederherstellungsvorgänge nur im NDMP-Modus mit Storage Virtual Machine (SVM)-Bereich verwendet werden.

[NOTE]
====
Der Reversionsprozess wird nicht unterstützt, wenn eine SMTape Tape-Backup- oder Wiederherstellungssitzung läuft. Es ist erforderlich, zu warten, bis die Sitzung abgeschlossen ist, oder die NDMP-Sitzung abzubrechen.

====
Mit SMTape können Sie 255 Snapshots sichern. Für nachfolgende Baseline-, inkrementelle oder differentielle Backups müssen ältere, gesicherte Snapshots gelöscht werden.

Vor der Durchführung einer Basiswiederherstellung muss das Volume, auf das die Daten wiederhergestellt werden, vom Typ  `DP` sein und sich im eingeschränkten Zustand befinden. Nach einer erfolgreichen Wiederherstellung ist dieses Volume automatisch online. Nachfolgende inkrementelle oder differentielle Wiederherstellungen können auf diesem Volume in der Reihenfolge durchgeführt werden, in der die Backups erstellt wurden.
