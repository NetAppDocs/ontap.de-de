= Snapshot konnte nicht erstellt werden
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* *Nachricht*
+
`Failed to create snapshot`

* *Ursache*
+
Das Volume enthält bereits die maximale Anzahl an Snapshot Kopien.

* *Korrekturmaßnahmen*
+
Löschen Sie einige Snapshot Kopien, und versuchen Sie es dann erneut.
