= Volume offline
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`Volume offline`

* *Ursache*
+
Das zu sichernde Volume ist offline.

* *Korrekturmaßnahme*
+
Das Volume online schalten und die Datensicherung erneut versuchen.
