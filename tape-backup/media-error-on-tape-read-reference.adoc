= Medienfehler beim Lesen des Bandes
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`Media error on tape read`

* *Ursache*
+
Das Band, von dem die Daten wiederhergestellt werden, ist beschädigt und enthält möglicherweise nicht die vollständigen Backup-Daten.

* *Korrekturmaßnahme*
+
Wenn Sie sicher sind, dass sich das vollständige Backup auf dem Band befindet, kann der Wiederherstellungsvorgang erneut versucht werden. Befindet sich das vollständige Backup nicht auf dem Band, ist der Wiederherstellungsvorgang nicht möglich.
