= Bänder außerhalb der Reihenfolge
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`Tapes out of order`

* *Ursache*
+
Das erste Band der Bandsequenz für den Wiederherstellungsvorgang besitzt keinen Image-Header.

* *Korrekturmaßnahme*
+
Das Band mit dem Image-Header einlegen und den Auftrag erneut versuchen.
