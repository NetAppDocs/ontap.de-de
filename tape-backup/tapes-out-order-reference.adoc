= Bänder sind nicht in Ordnung
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* *Nachricht*
+
`Tapes out of order`

* *Ursache*
+
Das erste Band der Bandsequenz für den Wiederherstellungsvorgang besitzt nicht den Bildkopf.

* *Korrekturmaßnahmen*
+
Legen Sie das Band mit der Bildkopfzeile ein, und versuchen Sie den Job erneut.
