= Unterstützte Anzahl gleichzeitiger ONTAP Tape-Geräte
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ONTAP unterstützt maximal 64 gleichzeitige Tape-Laufwerk-Verbindungen, 16 Medienwechsler und 16 Bridge- oder Router-Geräte pro Speichersystem (pro Knoten) in beliebiger Kombination von Fibre Channel-, SCSI- oder SAS-Anschlüssen.

Tape-Laufwerke oder Medienwechsler können Geräte in physischen oder virtuellen Tape-Bibliotheken oder Standalone-Geräte sein.

[NOTE]
====
Obwohl ein Speichersystem 64 Tape-Laufwerk-Verbindungen erkennen kann, hängt die maximale Anzahl gleichzeitig durchführbarer Sicherungs- und Wiederherstellungssitzungen von den Skalierbarkeitsgrenzen der Sicherungs-Engine ab.
