= Transfer fehlgeschlagen (ARL-initiierter Abbruch)
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`Transfer failed (ARL initiated abort)`

* *Ursache*
+
Während ein SMTape-Vorgang läuft, wird der SMTape-Vorgang abgebrochen, wenn eine Aggregat-Relokation eingeleitet wird.

* *Korrekturmaßnahme*
+
Der SMTape-Vorgang erfolgt, nachdem der Vorgang zur Umlagerung der Aggregate abgeschlossen ist.
