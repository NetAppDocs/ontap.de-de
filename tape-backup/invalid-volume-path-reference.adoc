= Ungültiger Volume-Pfad
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* *Nachricht*
+
`Invalid volume path`

* *Ursache*
+
Das angegebene Volume für den Backup- oder Wiederherstellungsvorgang wurde nicht gefunden.

* *Korrekturmaßnahmen*
+
Wiederholen Sie den Job mit einem gültigen Volume-Pfad und einem Volume-Namen.
