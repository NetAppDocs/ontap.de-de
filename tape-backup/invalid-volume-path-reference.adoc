= Ungültiger Volume-Pfad
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`Invalid volume path`

* *Ursache*
+
Das für die Sicherungs- oder Wiederherstellungsoperation angegebene Volume wurde nicht gefunden.

* *Korrekturmaßnahme*
+
Den Auftrag mit einem gültigen Volume-Pfad und Volume-Namen erneut ausführen.
