= Fehler: Anzeige fehlgeschlagen: Passwort für Benutzer '<username>' konnte nicht abgerufen werden
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`Error: show failed: Cannot get password for user '<username>'`

* *Ursache*
+
Unvollständige Benutzerkontokonfiguration für NDMP

* *Korrekturmaßnahme*
+
Es ist sicherzustellen, dass das Benutzerkonto mit der SSH-Zugriffsmethode verknüpft ist und das Authentifizierungsverfahren das Benutzerpasswort ist.
