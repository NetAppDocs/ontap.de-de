= NDMPD Session Session_ID nicht aktiv
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* *Nachricht*
+
`ndmpd session session_ID not active`

* *Ursache*
+
Die NDMP-Sitzung ist möglicherweise nicht vorhanden.

* *Korrekturmaßnahmen*
+
Verwenden Sie die `ndmpd status` Befehl zum Anzeigen der aktiven NDMP-Sitzungen.
