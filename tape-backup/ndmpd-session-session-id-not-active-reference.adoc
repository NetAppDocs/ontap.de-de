= ndmpd Sitzung session_ID nicht aktiv
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`ndmpd session session_ID not active`

* *Ursache*
+
Die NDMP-Sitzung existiert möglicherweise nicht.

* *Korrekturmaßnahme*
+
Mit dem  `ndmpd status`Befehl lassen sich die aktiven NDMP-Sitzungen anzeigen.
