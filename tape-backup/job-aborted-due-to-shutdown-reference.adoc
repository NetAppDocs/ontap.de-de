= Auftrag aufgrund von Systemabschaltung abgebrochen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`Job aborted due to shutdown`

* *Ursache*
+
Das Speichersystem wird neu gestartet.

* *Korrekturmaßnahme*
+
Der Auftrag kann nach einem Neustart des Speichersystems erneut ausgeführt werden.
