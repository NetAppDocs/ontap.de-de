= Ressourcenbeschränkung: Kein Thread verfügbar
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`Resource limitation: no available thread`

* *Ursache*
+
Die maximale Anzahl aktiver lokaler Tape-E/A-Threads ist derzeit belegt. Es sind maximal 16 aktive lokale Tape-Laufwerke möglich.

* *Korrekturmaßnahme*
+
Es sollte abgewartet werden, bis einige Bandaufträge abgeschlossen sind, bevor ein neuer Sicherungs- oder Wiederherstellungsauftrag gestartet wird.
