= Volume nicht eingeschränkt
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`Volume not restricted`

* *Ursache*
+
Das Zielvolume, auf das die Daten wiederhergestellt werden, ist nicht eingeschränkt.

* *Korrekturmaßnahme*
+
Das Volume einschränken und den Wiederherstellungsvorgang erneut versuchen.
