= Socket-Lesevorgang: EOF empfangen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`Read Socket received EOF`

* *Ursache*
+
Der Versuch, über eine NDMP-Dreiwegeverbindung mit einem entfernten Tape zu kommunizieren, hat das Dateiende erreicht. Möglicherweise wird eine Dreiwege-Wiederherstellung aus einem Backup-Image mit größerer Blockgröße versucht.

* *Korrekturmaßnahme*
+
Die korrekte Blockgröße angeben und den Wiederherstellungsvorgang erneut versuchen.
