= Neues Band konnte nicht geladen werden
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`Failed to load new tape`

* *Ursache*
+
Fehler im Tape-Laufwerk oder Medium.

* *Korrekturmaßnahme*
+
Das Tape austauschen und den Vorgang erneut versuchen.
