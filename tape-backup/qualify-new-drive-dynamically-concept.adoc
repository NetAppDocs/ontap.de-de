= Wie das ONTAP Speichersystem ein Tape-Laufwerk dynamisch qualifiziert
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Das Speichersystem qualifiziert ein Tape-Laufwerk dynamisch, indem es dessen Hersteller-ID und Produkt-ID mit den Informationen in der Tape-Qualifizierungstabelle abgleicht.

Wenn ein Tape-Laufwerk an das Speichersystem angeschlossen wird, sucht das System nach einer Übereinstimmung der Hersteller-ID und Produkt-ID zwischen den während der Banderkennung erhaltenen Informationen und den Informationen in der internen Tape-Qualifizierungstabelle. Wird eine Übereinstimmung gefunden, kennzeichnet das Speichersystem das Tape-Laufwerk als qualifiziert und kann auf das Tape-Laufwerk zugreifen. Kann das Speichersystem keine Übereinstimmung finden, bleibt das Tape-Laufwerk im nicht qualifizierten Zustand und wird nicht verwendet.
