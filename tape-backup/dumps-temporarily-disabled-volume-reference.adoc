= Dumps auf dem Volume vorübergehend deaktiviert, erneut versuchen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`Dumps temporarily disabled on volume, try again`

* *Ursache*
+
Ein NDMP-Dump-Backup wird auf einem SnapMirror Zielvolume versucht, das Teil einer  `snapmirror break` oder einer  `snapmirror resync` Operation ist.

* *Korrekturmaßnahme*
+
Warten Sie, bis die  `snapmirror break` oder  `snapmirror resync` Operation abgeschlossen ist und anschließend die Dump-Operation durchgeführt werden kann.

+
[NOTE]
====
Immer wenn sich der Status eines SnapMirror Zielvolumes von lesen/schreiben auf lesen/allein oder von lesen/allein auf lesen/schreiben ändert, ist eine Basissicherung erforderlich.

====
