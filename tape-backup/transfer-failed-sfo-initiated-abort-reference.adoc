= Transfer fehlgeschlagen (SFO-initiierter Abbruch)
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`Transfer failed (SFO initiated abort)`

* *Ursache*
+
Der SMTape Vorgang wird aufgrund eines Storage-Failover (Übernahme und Rückgabe) abgebrochen.

* *Korrekturmaßnahme*
+
Der SMTape-Vorgang wird nach Abschluss des Storage-Failover (Takeover und Giveback) durchgeführt.
