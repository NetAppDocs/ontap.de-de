= Bandlesefehler
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`Tape read error`

* *Ursache*
+
Das Tape-Laufwerk ist beschädigt oder das Band enthält nicht das vollständige Backup.

* *Korrekturmaßnahme*
+
Ist das Tape-Laufwerk beschädigt, kann ein anderes Tape-Laufwerk verwendet werden. Enthält das Band nicht die vollständige Sicherung, ist eine Wiederherstellung der Daten nicht möglich.
