= ONTAP Befehle zur Verwaltung von Tape-Laufwerken, Medienwechslern und Tape-Laufwerk-Operationen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Es gibt Befehle zum Anzeigen von Informationen über Tape-Laufwerke und Medienwechsler in einem Cluster, zum Online- und Offline-Schalten eines Tape-Laufwerks, zum Ändern der Tape-Laufwerk-Kassettenposition, zum Festlegen und Löschen des Tape-Laufwerk-Aliasnamens sowie zum Zurücksetzen eines Tape-Laufwerks. Auch Tape-Laufwerk-Statistiken können angezeigt und zurückgesetzt werden.

|===
| Wenn Sie möchten... | Verwenden Sie diesen Befehl... 


 a| 
Ein Tape-Laufwerk online schalten
 a| 
`storage tape online`



 a| 
Aliasnamen für Tape-Laufwerk oder Medienwechsler löschen
 a| 
`storage tape alias clear`



 a| 
Aktivieren oder Deaktivieren einer Tape-Trace-Operation für ein Tape-Laufwerk
 a| 
`storage tape trace`



 a| 
Die Position der Tape-Laufwerk-Kassette ändern
 a| 
`storage tape position`



 a| 
Tape-Laufwerk zurücksetzen
 a| 
`storage tape reset`

[NOTE]
====
Dieser Befehl ist nur auf der Stufe „Erweitert“ verfügbar.

====


 a| 
Einen Aliasnamen für das Tape-Laufwerk oder den Medienwechsler festlegen
 a| 
`storage tape alias set`



 a| 
Ein Tape-Laufwerk offline schalten
 a| 
`storage tape offline`



 a| 
Informationen zu allen Tape-Laufwerken und Medienwechslern
 a| 
`storage tape show`



 a| 
Informationen zu den an den Cluster angeschlossenen Tape-Laufwerken
 a| 
* `storage tape show-tape-drive`
* `system node hardware tape drive show`




 a| 
Informationen zu Medienwechslern, die an den Cluster angeschlossen sind
 a| 
`storage tape show-media-changer`



 a| 
Fehlerinformationen zu den an den Cluster angeschlossenen Tape-Laufwerken anzeigen
 a| 
`storage tape show-errors`



 a| 
Alle qualifizierten und unterstützten ONTAP Tape-Laufwerke, die an jeden Node im Cluster angeschlossen sind, werden angezeigt.
 a| 
`storage tape show-supported-status`



 a| 
Aliase aller Tape-Laufwerke und Medienwechsler anzeigen, die an jeden Node im Cluster angeschlossen sind
 a| 
`storage tape alias show`



 a| 
Die Statistiken eines Tape-Laufwerks auf Null zurücksetzen
 a| 
`storage stats tape zero` `tape_name`

Dieser Befehl ist in der Nodeshell zu verwenden.



 a| 
Von ONTAP unterstützte Tape-Laufwerke anzeigen
 a| 
`storage show tape supported [-v]`

Dieser Befehl ist in der Nodeshell zu verwenden. Mit der  `-v` Option lassen sich weitere Details zu jedem Tape-Laufwerk anzeigen.



 a| 
Tape-Gerät Statistiken bieten Einblick in die Tape-Performance und das Nutzungsmuster.
 a| 
`storage stats tape` `tape_name`

Dieser Befehl ist in der Nodeshell zu verwenden.
