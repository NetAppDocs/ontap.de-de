= Medienfehler beim Schreiben auf Band
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* *Nachricht*
+
`Media error on tape write`

* *Ursache*
+
Das für das Backup verwendete Band ist beschädigt.

* *Korrekturmaßnahmen*
+
Ersetzen Sie das Band, und versuchen Sie es erneut.
