= NDMPD ungültige Versionsnummer: Version_Nummer ``
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`ndmpd invalid version number: version_number`

* *Ursache*
+
Die angegebene NDMP-Version wird vom Speichersystem nicht unterstützt.

* *Korrekturmaßnahmen*
+
Angabe der NDMP-Version 4.
