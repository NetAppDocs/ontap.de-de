= ndmpd ungültige Versionsnummer: version_number ``
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`ndmpd invalid version number: version_number`

* *Ursache*
+
Die angegebene NDMP-Version wird vom Storage-System nicht unterstützt.

* *Korrekturmaßnahme*
+
NDMP-Version 4 angeben.
