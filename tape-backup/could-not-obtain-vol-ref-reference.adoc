= Die Volumenreferenz für das Volume „volume_name“ konnte nicht ermittelt werden.
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`Could not obtain vol ref for Volume vol_name`

* *Ursache*
+
Die Volumenreferenz konnte nicht ermittelt werden, da das Volumen möglicherweise von anderen Operationen verwendet wird.

* *Korrekturmaßnahme*
+
Wiederholen Sie den Vorgang zu einem späteren Zeitpunkt.
