= NFS-Labels werden nicht erkannt
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`Error: Aborting: dump encountered NFS security labels in the file system`

* *Ursache*
+
NFS-Sicherheitsbezeichnungen werden ab ONTAP 9.9.1 unterstützt, wenn NFSv4.2 aktiviert ist. Allerdings werden NFS-Sicherheitsbezeichnungen derzeit von der Dump-Engine nicht erkannt. Wenn NFS-Sicherheitsbezeichnungen in Dateien, Verzeichnissen oder speziellen Dateien in einem beliebigen Dump-Format vorhanden sind, schlägt der Dump fehl.

* *Korrekturmaßnahme*
+
Es ist zu überprüfen, dass keine Dateien oder Verzeichnisse NFS-Sicherheitsbezeichnungen aufweisen.
