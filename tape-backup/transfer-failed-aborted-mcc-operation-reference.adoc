= Übertragung fehlgeschlagen (Abbruch aufgrund eines MetroCluster Vorgangs)
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`Transfer failed (Aborted due to MetroCluster operation)`

* *Ursache*
+
Der SMTape Vorgang wird aufgrund eines Switchover oder Switchback Vorgangs abgebrochen.

* *Korrekturmaßnahme*
+
Der SMTape-Vorgang erfolgt nach Abschluss des Switchover- oder Switchback-Vorgangs.
