= Aktivieren oder Deaktivieren der Protokollierung für ONTAP Tape-Backup und Restore-Vorgänge
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können die Ereignisprotokollierung ein- oder ausschalten.

.Schritte
. Zum Aktivieren oder Deaktivieren der Ereignisprotokollierung den folgenden Befehl in der Clustershell eingeben:
+
`*options _-option_name_ backup.log.enable _-option-value_ {on | off}*`

+
`on`Schaltet die Ereignisprotokollierung ein.

+
`off`schaltet die Ereignisprotokollierung aus.

+
[NOTE]
====
Die Ereignisprotokollierung ist standardmäßig aktiviert.

====
