= Fehler beim Kürzen des Quell-Inodes <inode number>…​
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`Truncation failed for src inode <inode number>. Error <error number>. Skipping inode.`

* *Ursache*
+
Der Inode einer Datei wird gelöscht, wenn die Datei wiederhergestellt wird.

* *Korrekturmaßnahme*
+
Es sollte abgewartet werden, bis der Wiederherstellungsvorgang auf einem Volume abgeschlossen ist, bevor dieses Volume verwendet wird.
