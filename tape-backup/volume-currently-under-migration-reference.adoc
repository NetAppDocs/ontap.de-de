= Das Volume befindet sich derzeit in der Migration
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`Volume is currently under migration`

* *Ursache*
+
Volumemigration und SMTape Backup können nicht gleichzeitig ausgeführt werden.

* *Korrekturmaßnahme*
+
Der Sicherungsauftrag kann nach Abschluss der Volume-Migration erneut ausgeführt werden.
