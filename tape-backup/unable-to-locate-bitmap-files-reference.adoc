= Bitmap-Dateien konnten nicht gefunden werden
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`Unable to locate bitmap files`

* *Ursache*
+
Die für die Datensicherung benötigten Bitmap-Dateien wurden möglicherweise gelöscht. In diesem Fall kann die Datensicherung nicht neu gestartet werden.

* *Korrekturmaßnahme*
+
Die Datensicherung wird erneut durchgeführt.
