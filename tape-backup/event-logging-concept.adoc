= Überwachung von ONTAP Tape-Backup und Wiederherstellungsvorgängen für FlexVol Volumes
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können die Ereignisprotokolldateien einsehen, um die Tape-Backup und Wiederherstellungsvorgänge zu überwachen. ONTAP protokolliert automatisch wichtige Backup- und Wiederherstellungsereignisse sowie deren Zeitpunkt in einer Protokolldatei namens `backup` im  `/etc/log/`Verzeichnis des Controllers. Standardmäßig ist die Protokollierung auf  `on` eingestellt.

Eventprotokolldateien können aus folgenden Gründen angezeigt werden:

* Überprüfung, ob das nächtliche Backup erfolgreich war
* Erfassung von Statistiken zu Sicherungsvorgängen
* Zur Verwendung der Informationen in früheren Ereignisprotokolldateien zur Diagnose von Problemen bei Sicherungs- und Wiederherstellungsvorgängen


Die Ereignisprotokolldateien werden wöchentlich rotiert. Die  `/etc/log/backup` Datei wird in  `/etc/log/backup.0` umbenannt, die  `/etc/log/backup.0` Datei wird in  `/etc/log/backup.1` umbenannt und so weiter. Das System speichert die Protokolldateien bis zu sechs Wochen; daher können bis zu sieben Meldungsdateien (`/etc/log/backup.[0-5]` und die aktuelle  `/etc/log/backup` Datei vorhanden sein.
