= Bandreservierung vorweggenommen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`Tape reservation preempted`

* *Ursache*
+
Das Tape-Laufwerk wird von einem anderen Vorgang verwendet oder das Band wurde vorzeitig geschlossen.

* *Korrekturmaßnahme*
+
Es sollte sichergestellt sein, dass das Tape-Laufwerk nicht von einem anderen Vorgang verwendet wird und dass die DMA-Anwendung den Auftrag nicht abgebrochen hat; anschließend kann ein erneuter Versuch erfolgen.
