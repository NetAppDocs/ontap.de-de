= Maximale Anzahl zulässiger Dumps oder Wiederherstellungen (maximales Sitzungslimit) in Bearbeitung
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`Maximum number of allowed dumps or restores _(maximum session limit)_ in progress`

* *Ursache*
+
Die maximale Anzahl an Sicherungs- oder Wiederherstellungsaufträgen wird bereits ausgeführt.

* *Korrekturmaßnahme*
+
Der Vorgang kann erneut versucht werden, nachdem einige der aktuell laufenden Aufträge abgeschlossen wurden.
