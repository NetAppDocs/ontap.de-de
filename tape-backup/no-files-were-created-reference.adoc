= Es wurden keine Dateien erstellt
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`No files were created`

* *Ursache*
+
Es wurde versucht, ein Verzeichnis-DAR durchzuführen, ohne die erweiterte DAR-Funktionalität zu aktivieren.

* *Korrekturmaßnahme*
+
Die erweiterte DAR-Funktionalität aktivieren und den DAR erneut versuchen.
