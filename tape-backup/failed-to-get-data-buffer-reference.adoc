= Fehler beim Abrufen des Datenpuffers
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`Failed to get data buffer`

* *Ursache*
+
Das Speichersystem hatte keine Puffer mehr.

* *Korrekturmaßnahme*
+
Warten Sie, bis einige Speichersystemoperationen abgeschlossen sind, und versuchen Sie dann den Auftrag erneut.
