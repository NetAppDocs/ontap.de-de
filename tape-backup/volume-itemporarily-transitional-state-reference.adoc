= Das Volumen befindet sich vorübergehend in einem Übergangszustand
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`Volume is temporarily in a transitional state`

* *Ursache*
+
Das zu sichernde Volume befindet sich vorübergehend im nicht eingebundenen Zustand.

* *Korrekturmaßnahme*
+
Nach einer kurzen Wartezeit kann die Datensicherung erneut durchgeführt werden.
