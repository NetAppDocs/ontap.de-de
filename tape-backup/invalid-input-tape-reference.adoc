= Ungültiges Eingabeband
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`Invalid input tape`

* *Ursache*
+
Die Signatur des Sicherungsabbilds im Bandkopf ist ungültig. Das Band enthält beschädigte Daten oder kein gültiges Sicherungsabbild.

* *Korrekturmaßnahme*
+
Der Wiederherstellungsvorgang kann mit einem gültigen Sicherungsabbild erneut ausgeführt werden.
