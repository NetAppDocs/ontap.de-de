= Der für den Dump benötigte Snapshot konnte nicht gesperrt werden.
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`Unable to lock a snapshot needed by dump`

* *Ursache*
+
Der für die Sicherung angegebene Snapshot ist nicht verfügbar.

* *Korrekturmaßnahme*
+
Die Sicherung kann mit einem anderen Snapshot erneut versucht werden.

+
Mit dem  `snap list` Befehl lässt sich die Liste der verfügbaren Snapshots anzeigen.

+
Weitere Informationen zu  `snap list` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/search.html?q=snap+list["ONTAP-Befehlsreferenz"^].
