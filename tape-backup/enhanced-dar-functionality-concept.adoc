= Informationen zu den erweiterten DAR-Funktionen für ONTAP NDMP
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können die erweiterte Funktion zur direkten Zugriffswiederherstellung (DAR) für Verzeichnis DAR sowie für DAR von Dateien und NT-Streams verwenden. Die erweiterte DAR-Funktionalität ist standardmäßig aktiviert.

Die Aktivierung der erweiterten DAR-Funktionalität kann die Backup-Performance beeinträchtigen, da eine Offset-Zuordnung erstellt und auf Tape geschrieben werden muss. Die erweiterte DAR-Funktionalität kann sowohl im Node-basierten als auch im Storage Virtual Machine (SVM)-basierten NDMP-Modus aktiviert oder deaktiviert werden.
