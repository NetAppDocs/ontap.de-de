= Schreibvorgang auf das Band fehlgeschlagen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`Tape write failed`

* *Ursache*
+
Das für das Backup verwendete Band ist beschädigt.

* *Korrekturmaßnahme*
+
Das Band sollte ausgetauscht und der Backup-Job erneut versucht werden.
