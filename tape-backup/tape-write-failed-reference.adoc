= Bandschreibfehler
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`Tape write failed`

* *Ursache*
+
Das für das Backup verwendete Band ist beschädigt.

* *Korrekturmaßnahmen*
+
Ersetzen Sie das Band, und versuchen Sie es erneut.
