= Zugrundeliegendes Aggregat unter Migration
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`Underlying aggregate under migration`

* *Ursache*
+
Wenn ein SMTape-Vorgang auf einem Aggregat gestartet wird, das sich in Migration befindet (Storage-Failover oder Aggregat-Relocation), schlägt der SMTape-Vorgang fehl.

* *Korrekturmaßnahme*
+
Der SMTape-Vorgang wird nach Abschluss der Aggregatmigration durchgeführt.
