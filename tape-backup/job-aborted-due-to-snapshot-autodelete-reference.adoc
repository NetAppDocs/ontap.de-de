= Auftrag aufgrund automatischer Snapshot-Autolöschung abgebrochen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`Job aborted due to snapshot autodelete`

* *Ursache*
+
Das Volume verfügt nicht über genügend Speicherplatz und hat die automatische Löschung von Snapshots ausgelöst.

* *Korrekturmaßnahme*
+
Speicherplatz im Volume freigeben und den Auftrag erneut ausführen.
