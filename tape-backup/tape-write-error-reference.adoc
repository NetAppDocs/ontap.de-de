= Bandschreibfehler
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`Tape write error - The previous tape had less than the required minimum capacity, size MB, for this tape operation, The operation should be restarted from the beginning`

* *Ursache*
+
Die Bandkapazität ist unzureichend, um die Sicherungsdaten aufzunehmen.

* *Korrekturmaßnahme*
+
Bänder mit größerer Kapazität verwenden und den Sicherungsvorgang erneut ausführen.
