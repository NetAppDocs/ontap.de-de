= Die Wiederherstellung der Datei <file name> ist fehlgeschlagen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`Restore of the file file name failed`

* *Ursache*
+
Wenn eine DAR (Direct Access Recovery) einer Datei durchgeführt wird, deren Dateiname mit dem einer LUN auf dem Zielvolume identisch ist, schlägt die DAR fehl.

* *Korrekturmaßnahme*
+
Erneuter DAR-Versuch der Datei.
