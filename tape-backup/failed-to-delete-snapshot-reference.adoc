= Fehler beim Löschen des Snapshots
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`Failed to delete snapshot`

* *Ursache*
+
Der automatische Snapshot konnte nicht gelöscht werden, da er von anderen Vorgängen verwendet wird.

* *Korrekturmaßnahme*
+
Mit dem  `snap` Befehl lässt sich der Status des Snapshots bestimmen. Wenn der Snapshot nicht benötigt wird, kann er manuell gelöscht werden.
