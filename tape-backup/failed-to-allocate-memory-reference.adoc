= Speicherzuweisung fehlgeschlagen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`Failed to allocate memory`

* *Ursache*
+
Dem System ist der Arbeitsspeicher ausgegangen.

* *Korrekturmaßnahme*
+
Der Auftrag kann zu einem späteren Zeitpunkt ausgeführt werden, wenn das System weniger ausgelastet ist.
