= Das Band wird derzeit von anderen Vorgängen verwendet.
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`Tape is currently in use by other operations`

* *Ursache*
+
Das Tape-Laufwerk wird von einem anderen Auftrag verwendet.

* *Korrekturmaßnahme*
+
Die Datensicherung kann erneut versucht werden, nachdem der aktuell aktive Auftrag abgeschlossen ist.
