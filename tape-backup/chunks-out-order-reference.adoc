= Stücke in falscher Reihenfolge
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`Chunks out of order`

* *Ursache*
+
Die Sicherungsbänder werden nicht in der richtigen Reihenfolge wiederhergestellt.

* *Korrekturmaßnahme*
+
Der Wiederherstellungsvorgang sollte erneut versucht und die Bänder in der richtigen Reihenfolge geladen werden.
