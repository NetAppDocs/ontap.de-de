= Fehler beim Lesen des Backup-Images
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`Failed to read backup image`

* *Ursache*
+
Das Band ist beschädigt.

* *Korrekturmaßnahmen*
+
Wenn das Band beschädigt ist, können Sie den Wiederherstellungsvorgang nicht ausführen.
