= Snapshot konnte nicht gesperrt werden
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`Failed to lock snapshot`

* *Ursache*
+
Der Snapshot ist entweder in Verwendung oder wurde gelöscht.

* *Korrekturmaßnahme*
+
Wenn der Snapshot von einem anderen Vorgang verwendet wird, sollte abgewartet werden, bis dieser abgeschlossen ist, und anschließend kann die Sicherung erneut versucht werden. Wenn der Snapshot gelöscht wurde, ist eine Sicherung nicht möglich.
