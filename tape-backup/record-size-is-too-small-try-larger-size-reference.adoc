= Die Tape-Aufzeichnungsgröße ist zu klein. Eine größere Größe sollte verwendet werden.
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`Tape record size is too small. Try a larger size.`

* *Ursache*
+
Der für den Wiederherstellungsvorgang festgelegte Blockierungsfaktor ist kleiner als der Blockierungsfaktor, der während des Backups verwendet wurde.

* *Korrekturmaßnahme*
+
Es wird derselbe Blockierungsfaktor verwendet, der während des Backups angegeben wurde.
