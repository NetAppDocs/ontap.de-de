= Bereits am Ende des Bands
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`Already at the end of tape`

* *Ursache*
+
Das Band enthält keine Daten oder muss zurückgespult werden.

* *Korrekturmaßnahme*
+
Wenn das Band keine Daten enthält, das Band mit der Sicherung verwenden und den Wiederherstellungsvorgang erneut versuchen. Andernfalls das Band zurückspulen und den Wiederherstellungsvorgang erneut versuchen.
