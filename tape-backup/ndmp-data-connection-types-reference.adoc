= NDMP ONTAP Datenverbindungstypen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Im NDMP-Modus mit SVM-Bezug hängen die unterstützten NDMP-Datenverbindungstypen vom LIF-Typ der NDMP-Steuerungsverbindung und dem Status der CAB-Erweiterung ab. Dieser NDMP-Datenverbindungstyp gibt an, ob eine lokale oder eine Drei-Wege-NDMP-Sicherung oder -Wiederherstellung möglich ist.

Sie können eine dreiseitige NDMP-Sicherung oder Wiederherstellung über ein TCP- oder TCP/IPv6-Netzwerk durchführen. Die folgenden Tabellen zeigen die NDMP-Datenverbindungstypen basierend auf dem LIF-Typ der NDMP-Steuerverbindung und dem Status der CAB-Erweiterung.



[[ndmp-data-connection-type-when-cab-extension-is-supported-by-the-backup-application]]
== NDMP-Datenverbindungstyp, wenn die CAB-Erweiterung von der Sicherungsanwendung unterstützt wird

|===
| NDMP-Steuerverbindung LIF-Typ | NDMP-Datenverbindungstyp 


 a| 
Node-Management-LIF
 a| 
LOKAL, TCP, TCP/IPv6



 a| 
Data LIF
 a| 
TCP, TCP/IPv6



 a| 
Cluster-Management LIF
 a| 
LOKAL, TCP, TCP/IPv6



 a| 
Intercluster-LIF
 a| 
LOKAL, TCP, TCP/IPv6

|===
