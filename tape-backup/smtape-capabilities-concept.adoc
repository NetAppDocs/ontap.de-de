= SMTape Funktionen zur Optimierung von ONTAP Tape-Backup und Wiederherstellungsvorgängen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
SMTape-Funktionen wie die Sicherung von Snapshots, inkrementelle und differentielle Backups, die Beibehaltung der Deduplizierungs- und Komprimierungsfunktionen auf wiederhergestellten Volumes sowie Tape-Seeding tragen dazu bei, Tape-Backup und Wiederherstellungsvorgänge zu optimieren.

SMTape bietet folgende Funktionen:

* Bietet eine Lösung zur Notfallwiederherstellung
* Ermöglicht inkrementelle und differentielle Backups
* Sichert Snapshots
* Ermöglicht die Sicherung und Wiederherstellung deduplizierter Volumes und bewahrt die Deduplizierung auf den wiederhergestellten Volumes.
* Sichert komprimierte Volumes und erhält die Komprimierung auf den wiederhergestellten Volumes
* Ermöglicht Tape-Seeding


SMTape unterstützt den Blockierungsfaktor in Vielfachen von 4 KB, im Bereich von 4 KB bis 256 KB.

[NOTE]
====
Daten können nur auf Volumes wiederhergestellt werden, die über maximal zwei aufeinanderfolgende Hauptversionen von ONTAP erstellt wurden.
