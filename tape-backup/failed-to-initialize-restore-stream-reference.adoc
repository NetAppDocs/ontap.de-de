= Fehler beim Initialisieren des Wiederherstellungsstreams
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`Failed to initialize restore stream`

* *Ursache*
+
Diese Fehlermeldung kann aus einem der folgenden Gründe angezeigt werden:

+
** Das Sicherungsabbild ist nicht vom Typ SMTape.
** Der angegebene Bandblockierungsfaktor ist falsch.
** Das Band ist beschädigt oder defekt.
** Für die Wiederherstellung wurde das falsche Band eingelegt.


* *Korrekturmaßnahme*
+
** Wenn das Sicherungsabbild nicht von SMTape stammt, sollte der Vorgang mit einem Band wiederholt werden, das das SMTape-Backup enthält.
** Falls der Blockierungsfaktor falsch ist, sollte der korrekte Blockierungsfaktor angegeben und der Vorgang erneut versucht werden.
** Wenn das Band beschädigt ist, ist die Wiederherstellung nicht möglich.
** Wenn das falsche Band eingelegt ist, sollte der Vorgang mit dem richtigen Band wiederholt werden.
