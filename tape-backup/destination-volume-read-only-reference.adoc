= Das Zielvolume ist schreibgeschützt
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`Destination volume is read-only`

* *Ursache*
+
Der Pfad, zu dem die Wiederherstellungsoperation versucht wird, ist schreibgeschützt.

* *Korrekturmaßnahme*
+
Es kann versucht werden, die Daten an einem anderen Speicherort wiederherzustellen.
