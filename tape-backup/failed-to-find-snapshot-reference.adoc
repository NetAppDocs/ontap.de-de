= Snapshot konnte nicht gefunden werden
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`Failed to find snapshot`

* *Ursache*
+
Der für die Sicherung angegebene Snapshot ist nicht verfügbar.

* *Korrekturmaßnahme*
+
Es wird geprüft, ob der angegebene Snapshot verfügbar ist. Ist dies nicht der Fall, erfolgt ein erneuter Versuch mit dem korrekten Snapshot.
