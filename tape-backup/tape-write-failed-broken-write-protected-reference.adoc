= Schreibvorgang fehlgeschlagen, das neue Band ist beschädigt oder schreibgeschützt.
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`Tape write failed - new tape is broken or write protected`

* *Ursache*
+
Das für die Datensicherung verwendete Band ist beschädigt oder schreibgeschützt.

* *Korrekturmaßnahme*
+
Das Band sollte ausgetauscht und das Backup erneut versucht werden.
