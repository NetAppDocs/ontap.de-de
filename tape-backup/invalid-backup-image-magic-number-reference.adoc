= Ungültige Sicherungsimage-Magic-Nummer
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`Invalid backup image magic number`

* *Ursache*
+
Das Sicherungsabbild ist nicht vom Typ SMTape.

* *Korrekturmaßnahme*
+
Wenn das Sicherungsabbild nicht von SMTape stammt, sollte der Vorgang mit einem Band wiederholt werden, das das SMTape-Backup enthält.
