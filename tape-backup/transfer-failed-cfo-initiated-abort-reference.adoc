= Transfer fehlgeschlagen (CFO-initiierter Abbruch)
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`Transfer failed (CFO initiated abort)`

* *Ursache*
+
Der SMTape Vorgang wird aufgrund eines Storage-Failover (Übernahme und Rückgabe) Vorgangs eines CFO-Aggregats abgebrochen.

* *Korrekturmaßnahme*
+
Die SMTape-Operation erfolgt, nachdem das Storage-Failover des CFO Aggregats abgeschlossen ist.
