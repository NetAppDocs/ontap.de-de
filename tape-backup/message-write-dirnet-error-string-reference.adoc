= Meldung von Write Dirnet: Fehlerzeichenfolge
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`Message from Write Dirnet: error_string`

* *Ursache*
+
Bei der Backup-Kommunikation zu einem entfernten Bandlaufwerk in einer NDMP-Dreiwegeverbindung ist ein Fehler aufgetreten.

* *Korrekturmaßnahme*
+
Die Netzwerkverbindung zum Remote-Mover sollte überprüft werden.
