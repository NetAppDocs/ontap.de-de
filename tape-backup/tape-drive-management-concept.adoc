= Informationen zur Verwaltung von ONTAP Tape-Laufwerken
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können die Verbindungen der Tape-Bibliothek überprüfen und Informationen zum Tape-Laufwerk anzeigen, bevor ein Tape-Backup oder eine Wiederherstellung durchgeführt wird. Ein nicht qualifiziertes Tape-Laufwerk kann verwendet werden, indem es als qualifiziertes Tape-Laufwerk emuliert wird. Zusätzlich können Tape-Aliase zugewiesen und entfernt sowie vorhandene Aliase angezeigt werden.

Beim Sichern von Daten auf Band werden diese in Banddateien gespeichert. Dateimarken trennen die Banddateien, und die Dateien haben keine Namen. Eine Banddatei wird über ihre Position auf dem Band angegeben. Eine Banddatei wird mit einem Tape-Gerät geschrieben. Beim Lesen der Banddatei muss ein Gerät mit demselben Komprimierungstyp verwendet werden, der auch beim Schreiben dieser Banddatei verwendet wurde.
