= Die Größe der Tape-Aufzeichnung muss im Bereich zwischen 4 KB und 256 KB liegen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`Tape record size must be in the range between 4KB and 256KB`

* *Ursache*
+
Der für den Sicherungs- oder Wiederherstellungsvorgang festgelegte Blockierungsfaktor liegt nicht im zulässigen Bereich.

* *Korrekturmaßnahme*
+
Ein Blockierungsfaktor im Bereich von 4 KB bis 256 KB kann angegeben werden.
