= Informationen zur NDMP-Unterstützung mit ONTAP FlexGroup Volumes
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Ab ONTAP 9.7 wird NDMP auf FlexGroup Volumes unterstützt.

Ab ONTAP 9.7 wird der Befehl ndmpcopy für den Datentransfer zwischen FlexVol und FlexGroup Volumes unterstützt.

Wenn von ONTAP 9.7 auf eine frühere Version zurückgekehrt wird, bleiben die inkrementellen Übertragungsinformationen der vorherigen Übertragungen nicht erhalten, sodass nach der Rückkehr eine Basiskopie durchgeführt werden muss.

Ab ONTAP 9.8 werden die folgenden NDMP-Funktionen auf FlexGroup Volumes unterstützt:

* Die NDMP_SNAP_RECOVER-Nachricht in der Erweiterungsklasse 0x2050 kann zur Wiederherstellung einzelner Dateien in einem FlexGroup Volume verwendet werden.
* Die NDMP-Erweiterung für wiederaufnehmbare Backups (RBE) wird für FlexGroup Volumes unterstützt.
* Die Umgebungsvariablen EXCLUDE und MULTI_SUBTREE_NAMES werden für FlexGroup Volumes unterstützt.
