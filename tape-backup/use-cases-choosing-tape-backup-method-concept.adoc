= Anwendungsfälle für ONTAP SMTape und Dump Backup Engines
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ONTAP unterstützt zwei Backup-Engines: SMTape und dump. Die Anwendungsfälle der SMTape- und dump-Backup-Engines sind zu berücksichtigen, um die geeignete Backup-Engine für Tape-Backup- und Wiederherstellungsvorgänge auszuwählen.

Dump kann in folgenden Fällen verwendet werden:

* Direkte Wiederherstellung (DAR) von Dateien und Verzeichnissen
* Sicherung einer Teilmenge von Unterverzeichnissen oder Dateien in einem bestimmten Pfad
* Bestimmte Dateien und Verzeichnisse während der Datensicherung ausschließen
* Langzeitarchivierung von Backups


SMTape kann in folgenden Fällen verwendet werden:

* Lösung zur Notfallwiederherstellung
* Beibehalten der Deduplizierungseinsparungen und Deduplizierungseinstellungen der gesicherten Daten während eines Wiederherstellungsvorgangs
* Sicherung großer Datenmengen
