= DATA LISTEN: Fehler bei der Vorbereitung der CAB Datenverbindungsvoraussetzung
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`DATA LISTEN: CAB data connection prepare precondition error`

* *Ursache*
+
Der NDMP-Datenempfang schlägt fehl, wenn die Backup-Anwendung die CAB-Erweiterung mit dem NDMP-Server ausgehandelt hat und ein Unterschied im angegebenen NDMP-Datenverbindungsadressentyp zwischen den NDMP_CAB_DATA_CONN_PREPARE- und den NDMP_DATA_LISTEN-Nachrichten besteht.

* *Korrekturmaßnahme*
+
Wenden Sie sich an Ihren Backup-Anwendungsanbieter.
