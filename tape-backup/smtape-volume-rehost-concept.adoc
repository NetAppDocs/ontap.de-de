= Funktionsweise von SMTape bei ONTAP Volume-Rehost-Operationen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
SMTape-Operationen können nicht gestartet werden, wenn ein Volume Rehost-Vorgang auf einem Volume ausgeführt wird. Wenn ein Volume an einem Volume Rehost-Vorgang beteiligt ist, sollten keine SMTape-Sitzungen auf diesem Volume gestartet werden.

Wenn eine Volume Rehost Operation ausgeführt wird, schlägt das SMTape Backup oder die SMTape Wiederherstellung fehl. Wenn ein SMTape Backup oder eine SMTape Wiederherstellung ausgeführt wird, schlagen Volume Rehost Operationen mit einer entsprechenden Fehlermeldung fehl. Diese Bedingung gilt sowohl für NDMP-basierte als auch für CLI-basierte Backup- oder Wiederherstellungsoperationen.
