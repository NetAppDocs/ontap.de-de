= Informationen zu qualifizierten ONTAP Tape-Laufwerken
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie müssen ein qualifiziertes Tape-Laufwerk verwenden, das getestet wurde und auf einem Speichersystem einwandfrei funktioniert. Tape-Aliasing kann verwendet und Tape-Reservierungen aktiviert werden, um sicherzustellen, dass jeweils nur ein Speichersystem auf ein Tape-Laufwerk zugreift.

Ein qualifiziertes Tape-Laufwerk ist ein Tape-Laufwerk, das getestet wurde und auf Speichersystemen ordnungsgemäß funktioniert. Tape-Laufwerke können für bestehende ONTAP Versionen mithilfe der Konfigurationsdatei qualifiziert werden.
