= Ungültige Prüfsumme für Backup-Images
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* *Nachricht*
+
`Invalid backup image checksum`

* *Ursache*
+
Das Band ist beschädigt.

* *Korrekturmaßnahmen*
+
Wenn das Band beschädigt ist, können Sie den Wiederherstellungsvorgang nicht ausführen.
