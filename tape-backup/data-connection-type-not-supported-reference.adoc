= Der Datenverbindungstyp ["NDMP4_ADDR_TCP"|"NDMP4_ADDR_TCP_IPv6"] wird für ["IPv6"|"IPv4"] Steuerverbindungen nicht unterstützt.
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`Data connection type ["NDMP4_ADDR_TCP"|"NDMP4_ADDR_TCP_IPv6"] not supported for ["IPv6"|"IPv4"] control connections`

* *Ursache*
+
Im knotenbezogenen NDMP-Modus muss die hergestellte NDMP-Datenverbindung denselben Netzwerkadresstyp (IPv4 oder IPv6) wie die NDMP-Steuerverbindung aufweisen.

* *Korrekturmaßnahme*
+
Wenden Sie sich an Ihren Backup-Anwendungsanbieter.
