= Wie Tape-Backup und Wiederherstellungsvorgänge während der Notfallwiederherstellung in ONTAP MetroCluster Konfigurationen beeinflusst werden
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Tape-Backup- und Wiederherstellungsvorgänge können während der Notfallwiederherstellung in einer MetroCluster Konfiguration gleichzeitig durchgeführt werden. Es ist wichtig zu verstehen, wie diese Vorgänge während der Notfallwiederherstellung beeinflusst werden.

Wenn Tape-Backup- und Wiederherstellungsvorgänge auf einem Volume einer SVM in einer Disaster-Recovery-Beziehung durchgeführt werden, können nach einem Switchover und Switchback weiterhin inkrementelle Tape-Backup- und Wiederherstellungsvorgänge durchgeführt werden.
