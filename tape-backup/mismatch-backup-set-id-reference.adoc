= Fehler in Backup-Satz-ID
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`Mismatch in backup set ID`

* *Ursache*
+
Das beim Bandwechsel eingelegte Band ist nicht Teil des Backup-Satzes.

* *Korrekturmaßnahme*
+
Das richtige Band muss eingelegt werden, anschließend kann der Auftrag erneut versucht werden.
