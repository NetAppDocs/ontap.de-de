= Diskrepanz bei der Backup-Satz-ID
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Nachricht*
+
`Mismatch in backup set ID`

* *Ursache*
+
Das während einer Bandänderung geladene Band ist nicht Teil des Backup-Satzes.

* *Korrekturmaßnahmen*
+
Legen Sie das richtige Band ein, und versuchen Sie es erneut.
