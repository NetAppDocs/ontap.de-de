= Informationen zum Hinzufügen von Tape-Laufwerken und Tape-Bibliotheken zu ONTAP Speichersystemen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können Tape-Laufwerke und Tape-Bibliotheken dynamisch zum Speichersystem hinzufügen (ohne das Speichersystem offline zu nehmen).

Wenn ein neuer Medienwechsler hinzugefügt wird, erkennt das Speichersystem dessen Vorhandensein und fügt ihn der Konfiguration hinzu. Wenn der Medienwechsler bereits in den Aliasinformationen referenziert ist, werden keine neuen logischen Namen erstellt. Wenn die Bibliothek nicht referenziert ist, erstellt das Speichersystem einen neuen Alias für den Medienwechsler.

In einer Tape-Bibliothek-Konfiguration muss ein Tape-Laufwerk oder ein Medium-Changer auf LUN 0 eines Zielports konfiguriert werden, damit ONTAP alle Medium-Changer und Tape-Laufwerke an diesem Zielport erkennt.
