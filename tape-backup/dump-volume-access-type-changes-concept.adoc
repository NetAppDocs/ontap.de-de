= Funktionsweise von Dump, wenn sich der Zugriffstyp eines ONTAP-Volumes ändert
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Wenn sich der Status eines SnapMirror Zielvolumes oder eines SnapVault sekundären Volumes von lesen/schreiben auf nur lesen oder von nur lesen auf lesen/schreiben ändert, ist eine Baseline-Tape-Backup- oder Wiederherstellungsoperation erforderlich.

SnapMirror Ziel- und SnapVault sekundäre Volumes sind schreibgeschützte Volumes. Bei Tape-Backup- und Restore-Vorgängen auf solchen Volumes ist eine Basissicherung oder ein Restore erforderlich, sobald sich der Status des Volumes von schreibgeschützt zu lesen/schreiben oder von lesen/schreiben zu schreibgeschützt ändert.
