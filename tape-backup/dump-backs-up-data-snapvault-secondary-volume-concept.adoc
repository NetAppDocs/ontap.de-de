= Funktionsweise von Dump auf einem ONTAP SnapVault sekundäres Volume
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Tape-Backup-Operationen können für Daten durchgeführt werden, die auf dem SnapVault sekundären Volume gespiegelt sind. Es kann nur die auf dem SnapVault sekundären Volume gespiegelte Datenmenge auf Band gesichert werden, nicht jedoch die Metadaten der SnapVault Beziehung.

Wenn die Spiegelungsbeziehung zum Datenschutz unterbrochen wird (`snapmirror break` oder wenn eine SnapMirror-Resynchronisierung erfolgt, ist stets ein Baseline-Backup erforderlich.

.Verwandte Informationen
* link:https://docs.netapp.com/us-en/ontap-cli/snapmirror-break.html["snapmirror break"^]
