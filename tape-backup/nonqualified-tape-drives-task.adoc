= Ein nicht qualifiziertes Tape-Laufwerk für ONTAP Tape-Backup verwenden
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können ein nicht qualifiziertes Tape-Laufwerk in einem Speichersystem verwenden, wenn es ein qualifiziertes Tape-Laufwerk emulieren kann. Es wird dann wie ein qualifiziertes Tape-Laufwerk behandelt. Um ein nicht qualifiziertes Tape-Laufwerk zu verwenden, muss zunächst festgestellt werden, ob es eines der qualifizierten Tape-Laufwerke emuliert.

.Über diese Aufgabe
Ein nicht qualifiziertes Tape-Laufwerk ist ein Laufwerk, das an das Speichersystem angeschlossen ist, aber von ONTAP nicht unterstützt oder erkannt wird.

.Schritte
. Die nicht qualifizierten Tape-Laufwerke, die an ein Speichersystem angeschlossen sind, werden mit dem  `storage tape show-supported-status` Befehl angezeigt.
+
Der folgende Befehl zeigt die an das Speichersystem angeschlossenen Tape-Laufwerke sowie den Unterstützungs- und Qualifizierungsstatus jedes einzelnen Tape-Laufwerks an. Nicht qualifizierte Tape-Laufwerke werden ebenfalls aufgeführt.  `tape_drive_vendor_name` ist ein nicht qualifiziertes Tape-Laufwerk, das an das Speichersystem angeschlossen ist, aber von ONTAP nicht unterstützt wird.

+
[listing]
----

cluster1::> storage tape show-supported-status -node Node1

          Node: Node1
                                    Is
          Tape Drive                Supported  Support Status
          --------------------      ---------  --------------
          "tape_drive_vendor_name"  false      Nonqualified tape drive
          Hewlett-Packard C1533A    true       Qualified
          Hewlett-Packard C1553A    true       Qualified
          Hewlett-Packard Ultrium 1 true       Qualified
          Sony SDX-300C             true       Qualified
          Sony SDX-500C             true       Qualified
          StorageTek T9840C         true       Dynamically Qualified
          StorageTek T9840D         true       Dynamically Qualified
          Tandberg LTO-2 HH         true       Dynamically Qualified
----
. Das qualifizierte Tape-Laufwerk emulieren.
