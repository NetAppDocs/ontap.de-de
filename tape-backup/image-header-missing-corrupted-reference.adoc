= Bildkopf fehlt oder ist beschädigt
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* *Nachricht*
+
`Image header missing or corrupted`

* *Ursache*
+
Das Band enthält kein gültiges SMTape Backup.

* *Korrekturmaßnahmen*
+
Versuchen Sie es mit einem Band, das ein gültiges Backup enthält, erneut.
