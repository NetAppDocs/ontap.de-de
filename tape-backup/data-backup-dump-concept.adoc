= Informationen zur ONTAP Dump Engine für FlexVol Volumes
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Dump ist eine Snapshot-basierte Backup und Recovery Lösung von ONTAP, die das Sichern von Dateien und Verzeichnissen aus einem Snapshot auf ein Tape-Gerät sowie das Wiederherstellen der gesicherten Daten auf ein Speichersystem ermöglicht.

Sie können Ihre Dateisystemdaten, wie Verzeichnisse, Dateien und deren zugehörige Sicherheitseinstellungen, mithilfe der Dump-Sicherung auf ein Tape-Gerät sichern. Es ist möglich, ein gesamtes Volume, einen gesamten Qtree oder einen Teilbaum zu sichern, der weder ein gesamtes Volume noch ein gesamter Qtree ist.

Ein Dump-Backup oder eine Wiederherstellung kann mit NDMP-kompatiblen Backup-Anwendungen durchgeführt werden.

Wenn Sie ein Dump-Backup durchführen, können Sie den für das Backup zu verwendenden Snapshot angeben. Wenn Sie keinen Snapshot für das Backup angeben, erstellt die Dump-Engine einen Snapshot für das Backup. Nach Abschluss der Sicherungsoperation löscht die Speicherabbildsicherung diesen Snapshot.

Mit der Dump-Engine sind Level-0-, inkrementelle oder differentielle Backups auf Tape möglich.

[NOTE]
====
Nach dem Downgrade auf eine Version vor Data ONTAP 8.3 muss vor der Durchführung einer inkrementellen Sicherung eine Basissicherung durchgeführt werden.
