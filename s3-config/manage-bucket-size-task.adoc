= Die Größe des ONTAP S3-Buckets kann erhöht oder verringert werden.
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Bei Bedarf kann die Größe eines vorhandenen Buckets vergrößert oder verkleinert werden.

.Schritte
Der System Manager oder die ONTAP CLI kann zur Verwaltung der Bucket-Größe verwendet werden.

[role="tabbed-block"]
====
.System Manager
--
. *Storage > Buckets* auswählen und den Bucket suchen, der geändert werden soll.
. Klicken Sie image:icon_kabob.gif["Symbol für Menüoptionen"] neben den Bucket-Namen und wählen Sie *Bearbeiten*.
. Im Fenster *Bucket bearbeiten* die Kapazität des Buckets ändern.
. *Speichern*.


--
.CLI
--
. Bucket-Kapazität ändern:
+
[source, cli]
----
vserver object-store-server bucket modify -vserver <SVM_name> -bucket <bucket_name> -size {<integer>[KB|MB|GB|TB|PB]}
----
