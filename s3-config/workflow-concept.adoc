= ONTAP S3 Konfigurationsworkflow
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Die Konfiguration von S3 beinhaltet die Beurteilung der physischen Speicher- und Netzwerkanforderungen sowie die Auswahl eines Workflows, der auf Ihr Ziel zugeschnitten ist: die Konfiguration des S3-Zugriffs auf eine neue oder bestehende SVM oder das Hinzufügen eines Buckets und von Benutzern zu einer bestehenden SVM, die bereits vollständig für den S3-Zugriff konfiguriert ist.


NOTE: Die Konfiguration des Network Time Protocol (NTP) ist erforderlich, um die Zeitsynchronisierung zwischen Clustern und Clients sicherzustellen. Für den Clientzugriff ist häufig ein gültiger Zeitstempel mit einer Differenz von mindestens 15 Minuten zwischen dem ONTAP S3 Objektspeicher und dem Client erforderlich. link:../system-admin/manage-cluster-time-concept.html["Informationen zur Konfiguration von NTP"].

Wenn Sie den S3-Zugriff auf eine neue Storage-VM mithilfe des System Manager konfigurieren, werden Sie aufgefordert, Zertifikats- und Netzwerkinformationen einzugeben, und die Storage-VM sowie der S3-Objektspeicherserver werden in einem einzigen Vorgang erstellt.
