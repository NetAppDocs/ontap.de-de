= ONTAP S3 Storage-Servicelevel
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ONTAP beinhaltet vordefinierte Speicherdienste, die entsprechenden minimalen Leistungsfaktoren zugeordnet sind.

Die tatsächlich verfügbaren Speicherdienste in einem Cluster oder einer SVM werden durch die Art des Speichers bestimmt, aus dem ein Aggregat in der SVM besteht.
