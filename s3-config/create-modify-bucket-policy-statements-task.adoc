= Informationen zu ONTAP S3-Bucket- und Objektspeicher-Serverrichtlinien
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Der Zugriff von Benutzern und Gruppen auf S3-Ressourcen wird durch Bucket- und Objektspeicher-Serverrichtlinien gesteuert. Bei einer geringen Anzahl von Benutzern oder Gruppen ist die Zugriffskontrolle auf Bucket-Ebene wahrscheinlich ausreichend, aber bei vielen Benutzern und Gruppen ist es einfacher, den Zugriff auf der Objektspeicher-Server-Ebene zu steuern.
