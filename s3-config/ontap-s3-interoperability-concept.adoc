= ONTAP S3 Interoperabilität
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Der ONTAP S3 Server interagiert normal mit anderen ONTAP Funktionen, außer wie in dieser Tabelle angegeben.

[cols="3*"]
|===
| Merkmalsbereich | Unterstützt | Nicht unterstützt 


 a| 
Cloud Volumes ONTAP
 a| 
* Azure Clients in ONTAP 9.9.1 und späteren Versionen
* AWS Clients in ONTAP 9.11.0 und späteren Versionen
* Google Cloud Clients in ONTAP 9.12.1 und späteren Versionen

 a| 
* Cloud Volumes ONTAP für jeden Client in ONTAP 9.8 und früheren Versionen




 a| 
Datenschutz
 a| 
* Cloud Sync
* Objektsperre; Governance und Compliance (beginnend mit ONTAP 9.14.1)
* link:ontap-s3-supported-actions-reference.html#bucket-operations["Objektversionierung"] (beginnend mit ONTAP 9.11.1)
* Ungespiegelte MetroCluster Aggregates (beginnend mit ONTAP 9.12.1)
* Gespiegelte MetroCluster Aggregate (beginnend mit ONTAP 9.14.1)
* link:../s3-snapmirror/index.html["SnapMirror S3"] (beginnend mit ONTAP 9.10.1)
* SnapMirror (nur NAS-Volumes; ab ONTAP 9.12.1)
* SnapLock (nur NAS-Volumes; ab ONTAP 9.14.1)

 a| 
* Erasure Coding
* NDMP
* SMTape
* SnapMirror (synchron und asynchron)
* SnapMirror Cloud
* SVM Disaster Recovery
* SyncMirror (SyncMirror gespiegelte Aggregate werden in MetroCluster Konfigurationen ab ONTAP 9.14.1 unterstützt. SyncMirror wird außerhalb von MetroCluster Konfigurationen nicht unterstützt.)




 a| 
Verschlüsselung
 a| 
* NetApp Aggregate Encryption (NAE)
* NetApp Volume Encryption (NVE)
* NetApp Storage Encryption (NSE)
* TLS/SSL

 a| 
* SLAG




 a| 
MetroCluster Umgebungen
 a| 
-
 a| 
SnapMirror S3



 a| 
Speichereffizienz
 a| 
* Deduplizierung
* Komprimierung
* Verdichtung

 a| 
* Effizienzgewinne auf Aggregatebene (Mitglieder, die demselben Aggregat angehören, können die volumenübergreifende Deduplizierung nutzen, Mitglieder, die verschiedenen Aggregaten angehören, jedoch nicht)
* Volume-Klon des FlexGroup Volume, das ONTAP S3-Buckets enthält
* FlexClone technology (Volumes, Dateien und LUNs)




 a| 
Quality of Service (QoS)
 a| 
* QoS-Maximum (Obergrenze)
* QoS-Mindestwerte (Untergrenzen)
* Verschachtelte QoS-Richtlinien
+
link:../performance-admin/policy-group-assignment-restrictions.html["Einschränkungen bei der Zuweisung von QoS-Richtliniengruppen"]


 a| 



 a| 
Zusätzliche Funktionen
 a|

* link:../s3-config/create-bucket-lifecycle-rule-task.html["Bucket Lifecycle-Management"] (beginnend mit ONTAP 9.13.1)
* FabricPool Cloud-Tier (link:../s3-config/index.html["nur nativ S3"])
* FabricPool lokale Ebene (link:../s3-multiprotocol/index.html["Nur NAS-Volumes"])
* link:../flexcache/supported-unsupported-features-concept.html#supported-features["FlexCache Volumes"] (beginnend mit ONTAP 9.18.1)

 a| 
* FPolicy
* Qtrees
* Kontingente
* FabricPool Cloud-Tier (nur NAS-Volumes)
* FabricPool lokale Tier (nur natives S3)
