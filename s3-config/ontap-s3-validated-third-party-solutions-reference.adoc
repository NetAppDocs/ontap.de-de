= Validierte Drittanbieterlösungen mit S3 in ONTAP
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
S3 ist ein universeller Standard und dies ist keine umfassende Liste der unterstützten Anwendungen, sondern nur eine Liste von Lösungen, die in Zusammenarbeit mit den jeweiligen Partnern validiert wurden. Falls die gesuchte Lösung nicht aufgeführt ist, kann Ihr NetApp Account Representative kontaktiert werden.



[[third-party-solutions-validated-using-native-s3-buckets]]
== Drittanbieterlösungen, validiert mit nativen S3-Buckets

* Amazon SageMaker
* Apache Hadoop S3A Client
* Apache Kafka
* Apache Spark
* Commvault (V11)
* Confluent Kafka
* NetBackup
* Red Hat Quay
* Rubrik
* Snowflake
* Trino
* Veeam (V12)



NOTE: Diese Lösungen sind speziell für die Verwendung von nativen S3-Buckets in ONTAP validiert. Einige dieser Lösungen, wie beispielsweise solche im Zusammenhang mit Versionierung, Objektsperren und anderen Funktionen, werden bei der Verwendung von link:../s3-multiprotocol/index.html["S3 NAS Buckets (S3 in Multiprotokoll NAS Volumes)"] nicht unterstützt.
