= Hardwarekonfigurationen anzeigen, um Probleme mit ONTAP System Manager zu ermitteln
:hardbreaks:
:toclevels: 1
:allow-uri-read: 
:toclevels: 1
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
Ab ONTAP 9.8 kann mit dem System Manager die Konfiguration der Hardware im Netzwerk angezeigt und der Zustand der Hardwaresysteme und Verkabelungskonfigurationen ermittelt werden.

.Schritte
Um die Hardwarekonfigurationen anzuzeigen, sind die folgenden Schritte erforderlich:

. Im System Manager *Cluster > Hardware* auswählen.
. Fahren Sie mit der Maus über die Komponenten, um Status und weitere Details anzuzeigen.
+
Verschiedene Arten von Informationen sind einsehbar:

+
** <<Informationen zu Controllern>>
** <<Informationen zu Festplattengehäusen>>
** <<Informationen zu Speicherschaltern>>


. Ab ONTAP 9.12.1 können Verkabelungsinformationen im System Manager angezeigt werden. Das Kästchen *Show Cables* kann aktiviert werden, um die Verkabelung anzuzeigen. Wenn der Mauszeiger über ein Kabel gehalten wird, werden die Verbindungsinformationen angezeigt.
+
** <<Informationen zur Verkabelung>>

--
.Ports
--
* Ein Port wird rot hervorgehoben, wenn er nicht aktiv ist.
* Wenn Sie mit der Maus über den Port fahren, werden der Status des Ports und weitere Details angezeigt.
* Konsolenports können nicht angezeigt werden.
+
*Anmerkungen*:

+
** Ab ONTAP 9.11.1 werden SAS-Ports nur dann rot hervorgehoben, wenn sie sich in einem Fehlerzustand befinden oder wenn ein verkabelter Port, der verwendet wird, offline geht. Die Ports erscheinen weiß, wenn sie offline und nicht verkabelt sind.
** Bei ONTAP 9.10.1 und älteren Versionen sind deaktivierte SAS-Ports rot hervorgehoben.




--
.FRUs
--
Informationen zu FRUs erscheinen nur, wenn der Zustand einer FRU nicht optimal ist.

* Defekte Netzteile in Knoten oder Gehäusen.
* In den Knoten wurden hohe Temperaturen festgestellt.
* Defekte Lüfter an den Knoten oder am Gehäuse.


--
.Adapterkarten
--
* Karten mit definierten Teilenummernfeldern werden in den Steckplätzen angezeigt, wenn externe Karten eingelegt wurden.
* Anschlüsse werden auf den Karten angezeigt.
* Für eine unterstützte Karte können Bilder dieser Karte angezeigt werden. Wenn sich die Karte nicht in der Liste der unterstützten Teilenummern befindet, erscheint eine allgemeine Grafik.


--

== Informationen zu Festplattengehäusen

Sie können Folgendes anzeigen:

[role="tabbed-block"]
====
.Festplattenregale
--
* Die Vorder- und Rückansicht kann angezeigt werden.
* Sie können die folgenden Festplattengehäusemodelle anzeigen:
+
[cols="35,65"]
|===


| Wenn Ihr System läuft... | Anschließend kann im System Manager angezeigt werden... 


| ONTAP 9.9.1 und höher | Alle Regale, die _nicht_ als „end of service“ oder „end of availability“ gekennzeichnet sind 


| ONTAP 9.8 | DS4243, DS4486, DS212C, DS2246, DS224C und NS224 
|===


--
.Regalanschlüsse
--
* Sie können den Portstatus einsehen.
* Sie können Informationen zum Remote-Port anzeigen, wenn der Port verbunden ist.


--
.Shelf-FRUs
--
* Informationen zu Netzteilfehlern werden angezeigt.

[[information-about-storage-switches]]
== Informationen zu Speicherschaltern

Sie können Folgendes anzeigen:

[role="tabbed-block"]
====
.Storage Switches
--
* Das Display zeigt Schalter, die als Speicherschalter verwendet werden, um Regale mit Knoten zu verbinden.
* Ab ONTAP 9.9.1 zeigt System Manager Informationen über einen Switch an, der sowohl als Storage Switch als auch als Cluster fungiert und auch zwischen Knoten eines HA-Paares gemeinsam genutzt werden kann.
* Die folgenden Informationen werden angezeigt:
+
** Switch-Name
** IP-Adresse
** Seriennummer
** SNMP Version
** Systemversion


* Sie können die folgenden Storage-Switch-Modelle anzeigen:
+
[cols="35,65"]
|===


| Wenn Ihr System läuft... | Anschließend kann im System Manager angezeigt werden... 


| ONTAP 9.11.1 oder höher | Cisco Nexus 3232C Cisco Nexus 9336C-FX2 NVIDIA SN2100 


| ONTAP 9.10.1 und 9.9.1 | Cisco Nexus 3232C Cisco Nexus 9336C-FX2 


| ONTAP 9.8 | Cisco Nexus 3232C 
|===


--
.Storage Switch Ports
--
* Die folgenden Informationen werden angezeigt:
+
** Identitätsname
** Identitätsindex
** Status
** Fernverbindung
** Weitere Details




--

[[information-about-cabling]]
== Informationen zur Verkabelung

Ab ONTAP 9.12.1 sind die folgenden Verkabelungsinformationen sichtbar:

* *Verkabelung* zwischen Controllern, Switches und Shelves, wenn keine Storage-Bridges verwendet werden
* *Konnektivität*, die die IDs und MAC-Adressen der Ports an beiden Enden des Kabels anzeigt
