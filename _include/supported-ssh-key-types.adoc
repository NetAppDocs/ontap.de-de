Die folgende Tabelle zeigt die für ONTAP SSH-Verbindungen unterstützten Host-Schlüssel-Algorithmen. Diese Schlüsseltypen gelten nicht für die Konfiguration der öffentlichen SSH-Authentifizierung.

[cols="30,30,30"]
|===


| ONTAP Release | Im FIPS-Modus unterstützte Schlüsseltypen | Unterstützte Schlüsseltypen im Nicht-FIPS-Modus 


 a| 
9.11.1 und später
 a| 
ecdsa-sha2-nistp256
 a| 
ecdsa-sha2-nistp256 + rsa-sha2-512 + rsa-sha2-256 + ssh-ed25519 + ssh-dss + ssh-rsa



 a| 
9.10.1 und früher
 a| 
ecdsa-sha2-nistp256 + ssh-ed25519
 a| 
ecdsa-sha2-nistp256 + ssh-ed25519 + ssh-dss + ssh-rsa

|===

NOTE: Die Unterstützung für den Host-Key-Algorithmus ssh-ed25519 wird ab ONTAP 9.11.1 entfernt.

Weitere Informationen finden sich unter link:../networking/configure_network_security_using_federal_information_processing_standards_fips.html["Netzwerksicherheit mithilfe von FIPS konfigurieren"].
