.Über diese Aufgabe
Eine 1:1-Zuordnung von LIFs und NICs erzielt den größten Leistungszuwachs, ist aber für die Aktivierung von Trunking nicht erforderlich. Mindestens zwei installierte NICs können einen Leistungsvorteil bieten, sind jedoch nicht erforderlich.

Alle LIFs in der Trunking-Gruppe sollten derselben Failover-Gruppe angehören. Es ist zu beachten, dass ein Controller-Failover auf demselben Node dazu führen kann, dass die LIFs offline gehen, wenn sie in einer Failover-Gruppe auf demselben Node konfiguriert sind. Wenn die LIFs nicht in einer Failover-Gruppe auf demselben Node konfiguriert sind und ein Failover auf einen anderen Node erfolgt, funktioniert das Trunking nicht mehr.

Die Trunking Failover-Gruppe ist immer dann anzupassen, wenn Verbindungen (und die zugrunde liegenden Netzwerkkarten) zu einer Failover-Gruppe hinzugefügt oder daraus entfernt werden.

.Bevor Sie beginnen
* Die Port-Namen, die den NICs zugeordnet sind, müssen bekannt sein, um eine Failover-Gruppe zu erstellen.
* Die Ports müssen sich alle auf demselben Node befinden.
