= SVM Anwendungsfälle
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Dienstanbieter nutzen SVMs in sicheren mandantenfähigen Umgebungen, um die Daten jedes Mandanten zu isolieren, jedem Mandanten eine eigene Authentifizierung und Administration bereitzustellen und die Kostenverrechnung zu vereinfachen. Es ist möglich, einer SVM mehrere LIFs zuzuweisen, um unterschiedliche Kundenanforderungen zu erfüllen, und QoS kann eingesetzt werden, um zu verhindern, dass die Workloads eines Mandanten die Workloads anderer Mandanten „bullying“.

Administratoren nutzen SVMs für ähnliche Zwecke im Unternehmen. Daten verschiedener Abteilungen können getrennt werden, oder Speichervolumes, auf die Hosts zugreifen, werden in einer SVM und Benutzerfreigabe-Volumes in einer anderen gehalten. Einige Administratoren platzieren iSCSI/FC LUNs und NFS-Datenspeicher in einer SVM und SMB-Freigaben in einer anderen.
