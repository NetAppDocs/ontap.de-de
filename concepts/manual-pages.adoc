= ONTAP-Befehlsreferenz
:hardbreaks:
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Für jede wichtige ONTAP-Version sind die gängigen CLI-Befehle (ONTAP-Handbuchseiten oder Manpages) im link:https://docs.netapp.com/us-en/ontap-cli/["ONTAP Befehlsreferenz"^] gebündelt. Diese Befehlsreferenzen erläutern die Nutzung der CLI-Befehle in jeder ONTAP-Version.

Manpages sind auch an der ONTAP Befehlszeile mit dem  `man` Befehl verfügbar. Weitere Informationen zu  `man` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/man.html["ONTAP-Befehlsreferenz"^].
