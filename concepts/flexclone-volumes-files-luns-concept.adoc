= FlexClone Volumes, Dateien und LUNs
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Die _FlexClone_ Technologie referenziert Snapshot-Metadaten, um beschreibbare, zeitpunktgenaue Kopien eines Volumes zu erstellen. Kopien teilen sich Datenblöcke mit ihren übergeordneten Elementen und verbrauchen keinen Speicherplatz außer dem für Metadaten erforderlichen, bis Änderungen in die Kopie geschrieben werden. FlexClone Dateien und FlexClone LUNs verwenden dieselbe Technologie, außer dass kein zugrunde liegender Snapshot erforderlich ist.

Während das Erstellen herkömmlicher Kopien Minuten oder sogar Stunden dauern kann, ermöglicht die FlexClone Software das nahezu sofortige Kopieren selbst größter Datensätze. Das macht sie ideal für Situationen, in denen mehrere Kopien identischer Datensätze benötigt werden (zum Beispiel bei einer virtuellen Desktop-Bereitstellung) oder temporäre Kopien eines Datensatzes (etwa beim Testen einer Anwendung mit einem Produktionsdatensatz).

Sie können ein vorhandenes FlexClone Volume klonen, ein Volume mit LUN-Klonen klonen oder Spiegel- und Tresordaten klonen. Sie können ein FlexClone Volume von seinem übergeordneten Volume trennen, in diesem Fall erhält die Kopie eigenen Speicherplatz.
