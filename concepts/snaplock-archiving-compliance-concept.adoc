= WORM-Speicher
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
_SnapLock_ ist eine leistungsstarke Compliance-Lösung für Organisationen, die _write once, read many (WORM)_ Speicher verwenden, um kritische Dateien für regulatorische und Governance-Zwecke unverändert aufzubewahren.

Eine einzelne Lizenz berechtigt zur Nutzung von SnapLock im strikten _Compliance mode_, um externe Vorgaben wie SEC Rule 17a-4(f) zu erfüllen, sowie im weniger restriktiven _Enterprise mode_, um interne Vorschriften zum Schutz digitaler Assets einzuhalten. SnapLock verwendet eine manipulationssichere _ComplianceClock_, um zu bestimmen, wann die Aufbewahrungsfrist für eine WORM-Datei abgelaufen ist.

_SnapLock für SnapVault_ kann verwendet werden, um Snapshots auf Sekundärspeichern WORM-geschützt abzulegen. SnapMirror kann verwendet werden, um WORM-Dateien zur Notfallwiederherstellung und für andere Zwecke an einen anderen geografischen Standort zu replizieren.
