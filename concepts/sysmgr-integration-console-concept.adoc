= ONTAP System Manager in die NetApp Console integrieren
:hardbreaks:
:toclevels: 1
:allow-uri-read: 
:toclevels: 1
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
Sie können ONTAP 9.10.1 und spätere Versionen mit System Manager in der NetApp Console verwalten. Diese Integration ermöglicht eine effiziente Überwachung der hybriden Multicloud-Infrastruktur über eine einheitliche Steuerungsebene, wobei die vertraute System Manager Benutzeroberfläche erhalten bleibt.

Die Konsole ermöglicht das Erstellen und Verwalten von Cloud-Speicher (zum Beispiel Cloud Volumes ONTAP), die Nutzung von NetApp Datenservices (zum Beispiel Cloud Backup) sowie die Steuerung vieler lokaler und Edge-Speichergeräte.

Die Konsole bietet zwei Möglichkeiten, Ihre Cluster zu entdecken und zu verwalten:

* Direkte Erkennung für das Management über System Manager (ONTAP 9.12.1 und höher)
* Erkennung über einen Konsolenagenten.


Der Konsolenagent ist eine in Ihrer Umgebung installierte Software. Dies ermöglicht den Zugriff auf Verwaltungsfunktionen über System Manager sowie auf NetApp Cloud-Dienste, die Funktionen wie Datenreplizierung, Backup und Recovery, Datenklassifizierung, Daten-Tiering und mehr bieten.

Weitere Informationen zu link:https://docs.netapp.com/us-en/storage-management-ontap-onprem/index.html["Lokale ONTAP Clusterverwaltung mit der NetApp Console"^].

.Schritte
. Melden Sie sich an der NetApp Console an.
+
.. Wenn ein Console-Login vorhanden ist, kann dieses verwendet werden.
.. Wenn dies das erste Mal ist, wählen Sie „Mit Ihren NetApp Support Site-Anmeldedaten anmelden“ und geben Sie Ihre Anmeldedaten auf der Console-Anmeldeseite ein.


. Ihr lokaler Cluster kann in der Konsole mithilfe eines Console Agent oder durch direkte Erkennung angezeigt werden. Weitere Informationen unter: https://docs.netapp.com/us-en/storage-management-ontap-onprem/task-discovering-ontap.html[discovering on-premises ONTAP clusters^].
+
.. Informationen dazu, wie link:https://docs.netapp.com/us-en/storage-management-ontap-onprem/task-manage-ontap-direct.html["Cluster verwalten, die direkt entdeckt wurden"^].
.. Informationen dazu, wie link:https://docs.netapp.com/us-en/storage-management-ontap-onprem/task-manage-ontap-connector.html["Cluster verwalten, die mit einem Console Agent entdeckt wurden"^].


. Verwaltung des lokalen Clusters mit System Manager. Auf der Seite „Systeme meiner Arbeitsumgebungen“ den Cluster auswählen und in der Liste „Dienste“ System Manager auswählen.
+
.. Weitere Informationen zu link:https://docs.netapp.com/us-en/storage-management-ontap-onprem/task-manage-ontap-direct.html["Verwaltung von Clustern, die direkt entdeckt wurden"^].
.. Informationen dazu, wie link:https://docs.netapp.com/us-en/storage-management-ontap-onprem/task-manage-ontap-connector.html["Cluster verwalten, die mit einem Console Agent entdeckt wurden"^].






[[netapp-console-and-ontap-9-12-1]]
== NetApp Console und ONTAP 9.12.1

Hinweis: Wenn Sie ONTAP 9.12.1 verwenden, erscheint eine Meldung, die dazu auffordert, die NetApp Console auszuprobieren.

+ Wenn der Cluster eine Verbindung zur NetApp Console hat, wird eine Anmeldeaufforderung angezeigt.

. *Weiter zur NetApp Console* auswählen, um dem Link zur Console zu folgen.
+

NOTE: Wenn Ihre Systemeinstellungen externe Netzwerke blockieren, können Sie nicht auf die Konsole zugreifen. Um über die Konsole auf System Manager zuzugreifen, muss sichergestellt sein, dass Ihr System die Adresse „cloudmanager.cloud.netapp.com“ erreichen kann. Andernfalls kann an der Eingabeaufforderung die mit Ihrem ONTAP System installierte Version von System Manager ausgewählt werden.

. Auf der Anmeldeseite der Konsole *Mit Ihren NetApp Support Site Credentials* anmelden und Ihre Anmeldeinformationen eingeben.
+
Wenn Sie die Konsole bereits genutzt haben und über einen Login mit E-Mail-Adresse und Passwort verfügen, ist weiterhin diese Login-Option zu verwenden.

+
https://docs.netapp.com/us-en/console-setup-admin/task-logging-in.html["Weitere Informationen zum Anmelden an der NetApp Console"^].

. Falls Sie dazu aufgefordert werden, einen Namen für Ihr neues NetApp Console Konto eingeben.
+
In den meisten Fällen erstellt die Console automatisch ein Konto für Sie auf Basis der Daten aus Ihrem Cluster.

. Geben Sie die Anmeldeinformationen des Clusteradministrators für den Cluster ein.


.Ergebnis
System Manager wird angezeigt und der Cluster kann nun über die Konsole verwaltet werden.
