= Netzwerkarchitektur im Überblick
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Die Netzwerkarchitektur für eine ONTAP Rechenzentrumsimplementierung besteht typischerweise aus einer Cluster-Verbindung, einem Management-Netzwerk für die Cluster-Administration und einem Datennetzwerk. NICs (Network Interface Cards) stellen physische Ports für Ethernet-Verbindungen bereit. HBAs (Host Bus Adapter) stellen physische Ports für FC-Verbindungen bereit.
