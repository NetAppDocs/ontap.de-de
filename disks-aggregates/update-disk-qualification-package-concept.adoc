= Wann das ONTAP Disk Qualification Package aktualisiert werden sollte
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Das Disk Qualification Package (DQP) bietet volle Unterstützung für neu qualifizierte Laufwerke. Bevor Sie die Firmware eines Laufwerks aktualisieren oder neue Laufwerkstypen oder -größen zu einem Cluster hinzufügen, muss das DQP aktualisiert werden. Eine bewährte Vorgehensweise ist es, das DQP regelmäßig zu aktualisieren, beispielsweise vierteljährlich oder halbjährlich.

Sie müssen das DQP in den folgenden Situationen herunterladen und installieren:

* Immer wenn ein neuer Laufwerkstyp oder eine neue Laufwerksgröße dem Knoten hinzugefügt wird
+
Wenn Sie beispielsweise bereits 1-TB-Festplatten besitzen und 2-TB-Festplatten hinzufügen, müssen Sie auf das neueste DQP-Update prüfen.

* Immer wenn Sie die Festplatten-Firmware aktualisieren
* Sobald neuere Festplatten-Firmware oder DQP-Dateien verfügbar sind
* Immer wenn auf eine neue Version von ONTAP aktualisiert wird.
+
Die DQP wird im Rahmen eines ONTAP Upgrades nicht aktualisiert.
