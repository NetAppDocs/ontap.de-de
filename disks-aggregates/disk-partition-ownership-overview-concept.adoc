= Verwaltung der Besitzverhältnisse von ONTAP Festplatten und Partitionen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können die Besitzverhältnisse von Festplatten und Partitionen verwalten.

Folgende Aufgaben sind möglich:

* *link:display-partition-ownership-task.html["Anzeige des Datenträger- und Partitionsbesitzes"]*
+
Sie können die Datenträgerbesitzverhältnisse einsehen, um festzustellen, welcher Knoten den Speicher kontrolliert. Sie können außerdem auf Systemen mit gemeinsam genutzten Datenträgern die Partitionsbesitzverhältnisse einsehen.

* *link:configure-auto-assignment-disk-ownership-task.html["Einstellungen für die automatische Zuweisung des Festplattenbesitzes ändern"]*
+
Sie können eine vom Standard abweichende Richtlinie für die automatische Zuweisung des Festplattenbesitzes auswählen oder die automatische Zuweisung des Festplattenbesitzes deaktivieren.

* *link:manual-assign-disks-ownership-manage-task.html["Manuelle Zuweisung der Besitzrechte für nicht partitionierte Datenträger"]*
+
Wenn Ihr Cluster nicht für die automatische Zuweisung des Festplattenbesitzes konfiguriert ist, muss der Besitz manuell zugewiesen werden.

* *link:manual-assign-ownership-partitioned-disks-task.html["Manuelle Zuweisung der Besitzrechte partitionierter Festplatten"]*
+
Sie können die Besitzverhältnisse der Container-Disk oder der Partitionen manuell oder mithilfe der automatischen Zuweisung festlegen, genau wie bei nicht partitionierten Disks.

* *link:remove-failed-disk-task.html["Entfernen einer defekten Festplatte"]*
+
Eine Festplatte, die vollständig ausgefallen ist, wird von ONTAP nicht mehr als verwendbare Festplatte betrachtet, und die Festplatte kann sofort vom Gehäuse getrennt werden.

* *link:remove-ownership-disk-task.html["Besitzrechte von einer Festplatte entfernen"]*
+
ONTAP schreibt die Eigentümerinformationen auf die Festplatte. Vor dem Entfernen einer Ersatzfestplatte oder ihres Gehäuses aus einem Knoten empfiehlt es sich, die Eigentümerinformationen zu entfernen, damit sie ordnungsgemäß in einen anderen Knoten integriert werden kann.
