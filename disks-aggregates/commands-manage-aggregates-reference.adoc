= ONTAP Befehle zur Verwaltung lokaler Ebenen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie verwenden den  `storage aggregate` Befehl, um Ihre lokalen Tiers zu verwalten.


NOTE: Vor ONTAP 9.7 verwendet System Manager den Begriff _aggregate_, um eine _lokale Ebene_ zu beschreiben. Unabhängig von der ONTAP Version verwendet die ONTAP CLI den Begriff _aggregate_. Weitere Informationen zu lokalen Ebenen finden sich unter link:../disks-aggregates/index.html["Festplatten und lokale Ebenen"].

|===
| Wenn Sie möchten... | Verwenden Sie diesen Befehl... 


 a| 
Die Größe des Caches für alle lokalen Flash Pool Ebenen anzeigen
 a| 
`storage aggregate show -fields hybrid-cache-size-total -hybrid-cache-size-total >0`



 a| 
Festplatteninformationen und Status für eine lokale Ebene anzeigen
 a| 
`storage aggregate show-status`



 a| 
Ersatzfestplatten nach Knoten anzeigen
 a| 
`storage aggregate show-spare-disks`



 a| 
Die lokalen Root-Tiers im Cluster anzeigen
 a| 
`storage aggregate show -has-mroot true`



 a| 
Grundlegende Informationen und Status für lokale Ebenen
 a| 
`storage aggregate show`



 a| 
Den in lokalen Tiers verwendeten Speichertyp anzeigen
 a| 
`storage aggregate show -fields storage-type`



 a| 
Eine lokale Stufe online bringen
 a| 
`storage aggregate online`



 a| 
Eine lokale Ebene löschen
 a| 
`storage aggregate delete`



 a| 
Eine lokale Ebene in den eingeschränkten Zustand versetzen
 a| 
`storage aggregate restrict`



 a| 
Eine lokale Ebene umbenennen
 a| 
`storage aggregate rename`



 a| 
Eine lokale Ebene offline nehmen
 a| 
`storage aggregate offline`



 a| 
Den RAID-Typ für eine lokale Ebene ändern
 a| 
`storage aggregate modify -raidtype`
