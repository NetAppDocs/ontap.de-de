= Ermitteln, welche Volumes sich auf einer lokalen ONTAP Ebene befinden.
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Möglicherweise muss ermittelt werden, welche Volumes sich auf einer lokalen Ebene befinden, bevor Vorgänge wie das Verschieben oder das Offline-Schalten auf der lokalen Ebene durchgeführt werden.


NOTE: Vor ONTAP 9.7 verwendet System Manager den Begriff _aggregate_, um eine _lokale Ebene_ zu beschreiben. Unabhängig von der ONTAP Version verwendet die ONTAP CLI den Begriff _aggregate_. Weitere Informationen zu lokalen Ebenen finden sich unter link:../disks-aggregates/index.html["Festplatten und lokale Ebenen"].

.Schritte
. Die Volumes, die sich auf einer lokalen Ebene befinden, werden angezeigt, wenn folgender Befehl eingegeben wird:
+
`volume show -aggregate aggregate_name`

+
Alle Volumes, die sich auf der angegebenen lokalen Ebene befinden, werden angezeigt.
