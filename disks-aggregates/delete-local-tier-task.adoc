= Löschen einer ONTAP lokalen Ebene
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Eine lokale Ebene kann gelöscht werden, wenn sich auf der lokalen Ebene keine Volumes befinden.


NOTE: Vor ONTAP 9.7 verwendet System Manager den Begriff _aggregate_, um eine _lokale Ebene_ zu beschreiben. Unabhängig von der ONTAP Version verwendet die ONTAP CLI den Begriff _aggregate_. Weitere Informationen zu lokalen Ebenen finden sich unter link:../disks-aggregates/index.html["Festplatten und lokale Ebenen"].

Der `storage aggregate delete` Befehl löscht eine lokale Speicherebene. Der Befehl schlägt fehl, wenn sich Volumes auf der lokalen Speicherebene befinden. Wenn der lokalen Speicherebene ein Objektspeicher zugeordnet ist, werden zusätzlich zur Löschung der lokalen Speicherebene auch die Objekte im Objektspeicher gelöscht. An der Objektspeicher Konfiguration werden durch diesen Befehl keine Änderungen vorgenommen.

Das folgende Beispiel löscht eine lokale Tier mit dem Namen "`aggr1`":

....
> storage aggregate delete -aggregate aggr1
....
