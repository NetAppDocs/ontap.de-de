= Die Cache-Aufbewahrungsrichtlinie für ONTAP Flash Pool lokale Tiers festlegen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können Volumes in Flash Pool lokalen Tiers Cache-Aufbewahrungsrichtlinien zuweisen. Daten in Volumes mit einer hohen Cache-Aufbewahrungsrichtlinie verbleiben länger im Cache, während Daten in Volumes mit einer niedrigen Cache-Aufbewahrungsrichtlinie schneller entfernt werden. Dadurch wird die Leistung Ihrer kritischen Workloads erhöht, da wichtige Informationen für einen längeren Zeitraum schneller zugänglich sind.

.Bevor Sie beginnen
Es sollte bekannt sein, ob Ihr System Bedingungen aufweist, die verhindern könnten, dass die Cache-Aufbewahrungsrichtlinie einen Einfluss darauf hat, wie lange Ihre Daten im Cache verbleiben.

.Schritte
Die folgenden Schritte werden im erweiterten Berechtigungsmodus der CLI durchgeführt:

. Die Berechtigungseinstellung auf „Erweitert“ ändern:
+
`set -privilege advanced`

. Die Cache-Aufbewahrungsrichtlinie des Volumes überprüfen:
+
Standardmäßig ist die Aufbewahrungsrichtlinie für den Cache "`normal`".

. Cache-Aufbewahrungsrichtlinie festlegen:
+
`volume modify -volume volume_name -vserver vserver_name -caching-policy policy_name`

. Es sollte überprüft werden, ob die Cache-Aufbewahrungsrichtlinie des Volumes auf die ausgewählte Option geändert wurde.
. Die Berechtigungseinstellung wieder auf "admin" setzen:
+
`set -privilege admin`
