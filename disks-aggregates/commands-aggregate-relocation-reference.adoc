= ONTAP Befehle für eine lokale Tier-Verlagerung
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Es gibt spezielle ONTAP Befehle zum Verschieben der lokalen Tier-Zuständigkeit innerhalb eines HA-Paares.


NOTE: Vor ONTAP 9.7 verwendet System Manager den Begriff _aggregate_, um eine _lokale Ebene_ zu beschreiben. Unabhängig von der ONTAP Version verwendet die ONTAP CLI den Begriff _aggregate_. Weitere Informationen zu lokalen Ebenen finden sich unter link:../disks-aggregates/index.html["Festplatten und lokale Ebenen"].

|===


| Wenn Sie möchten... | Verwenden Sie diesen Befehl... 


 a| 
Den lokalen Tier-Umstellungsprozess starten
 a| 
`storage aggregate relocation start`



 a| 
Den lokalen Tier-Umstellungsprozess überwachen
 a| 
`storage aggregate relocation show`

|===
