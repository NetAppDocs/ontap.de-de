= SSDs zu einem ONTAP SSD Storage Pool hinzufügen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Durch das Hinzufügen von SSDs zu einem SSD-Speicherpool erhöhen sich die physische und die nutzbare Größe des Speicherpools sowie die Größe der Zuordnungseinheit. Die größere Zuordnungseinheit wirkt sich auch auf Zuordnungseinheiten aus, die bereits lokalen Speicherebenen zugewiesen wurden.

.Bevor Sie beginnen
Sie müssen festgestellt haben, dass diese Operation das Cache-Limit Ihres HA-Paares nicht überschreiten wird. ONTAP verhindert nicht, dass das Cache-Limit überschritten wird, wenn SSDs zu einem SSD-Speicherpool hinzugefügt werden, und dies kann dazu führen, dass die neu hinzugefügte Speicherkapazität nicht verfügbar ist.

.Über diese Aufgabe
Beim Hinzufügen von SSDs zu einem bestehenden SSD-Speicherpool müssen die SSDs entweder dem einen oder dem anderen Knoten desselben HA-Paares gehören, dem bereits die vorhandenen SSDs im Speicherpool gehören. Es können SSDs hinzugefügt werden, die einem der beiden Knoten des HA-Paares gehören.

Die SSD, die Sie dem Speicherpool hinzufügen, muss die gleiche Größe wie die derzeit im Speicherpool verwendete Festplatte haben.

[role="tabbed-block"]
====
.System Manager
--
Ab ONTAP 9.12.1 ist es möglich, mit System Manager SSDs zu einem SSD-Speicherpool hinzuzufügen.

.Schritte
. Klicken Sie auf *Speicher > Ebenen* und suchen Sie den Abschnitt *Speicherpools*.
. Den Speicherpool suchen, auf image:icon_kabob.gif["Symbol für Menüoptionen"] klicken und *Festplatten hinzufügen* auswählen.
. Wählen Sie den Datenträgertyp und die Anzahl der Datenträger.
. Die geschätzte Cache-Größe prüfen.


--
.CLI
--
.Schritte
. *Optional:* Die aktuelle Größe der Zuordnungseinheit und den verfügbaren Speicherplatz für den Speicherpool anzeigen:
+
`storage pool show -instance _sp_name_`

. Verfügbare SSDs:
+
`storage disk show -container-type spare -type SSD`

. Fügen Sie die SSDs dem Speicherpool hinzu:
+
`storage pool add -storage-pool sp_name -disk-list _disk1,disk2…_`

+
Das System zeigt an, welche lokalen Flash Pool Ebenen durch diesen Vorgang vergrößert werden und um welchen Betrag, und fordert Sie zur Bestätigung des Vorgangs auf.
