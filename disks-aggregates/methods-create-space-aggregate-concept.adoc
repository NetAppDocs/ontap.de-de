= Methoden zur Schaffung von Speicherplatz in einer ONTAP lokalen Ebene
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Wenn der Speicherplatz auf einer lokalen Speicherebene erschöpft ist, können verschiedene Probleme auftreten, von Datenverlust bis hin zur Deaktivierung der Volume-Garantie. Es gibt mehrere Möglichkeiten, mehr Speicherplatz auf einer lokalen Speicherebene zu schaffen.

Alle Methoden haben unterschiedliche Konsequenzen. Vor der Durchführung von Maßnahmen empfiehlt sich das Lesen des entsprechenden Abschnitts in der Dokumentation.

Im Folgenden sind einige gängige Methoden zur Speicherplatzbeschaffung in der lokalen Speicherebene aufgeführt, in der Reihenfolge von den geringsten bis zu den größten Auswirkungen:

* Dem lokalen Tier werden Festplatten hinzugefügt.
* Einige Volumes auf eine andere lokale Ebene mit verfügbarem Speicherplatz verschieben.
* Die Größe der volumengarantierten Volumes in der lokalen Tier verkleinern.
* Nicht benötigte Volume-Snapshots löschen, wenn der Garantietyp des Volumes "`none`" ist.
* Nicht benötigte Volumes löschen.
* Speichersparende Funktionen wie Deduplizierung oder Komprimierung können genutzt werden.
* (Vorübergehend) Funktionen deaktivieren, die eine große Menge an Metadaten verwenden.
