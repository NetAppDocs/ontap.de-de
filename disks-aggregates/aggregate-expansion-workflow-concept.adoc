= Workflow zum Hinzufügen von Kapazität zu einer ONTAP lokalen Ebene
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Um die Kapazität einer lokalen Ebene zu erhöhen, müssen zunächst die lokale Ebene identifiziert werden, zu der hinzugefügt werden soll, der benötigte neue Speicherplatz bestimmt, neue Festplatten installiert, die Festplattenzugehörigkeit zugewiesen und gegebenenfalls eine neue RAID-Gruppe erstellt werden.

Sie können entweder den System Manager oder die ONTAP CLI verwenden, um Kapazität hinzuzufügen.
