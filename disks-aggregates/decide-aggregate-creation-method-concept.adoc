= Entscheiden, welche Methode zur Erstellung lokaler ONTAP-Tiers verwendet werden kann.
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Obwohl ONTAP Best-Practice-Empfehlungen für das automatische Hinzufügen lokaler Ebenen bereitstellt, muss festgestellt werden, ob die empfohlenen Konfigurationen in Ihrer Umgebung unterstützt werden. Falls dies nicht der Fall ist, sind Entscheidungen bezüglich der RAID-Richtlinie und der Festplattenkonfiguration zu treffen und die lokalen Ebenen anschließend manuell zu erstellen.

Wenn eine lokale Ebene automatisch erstellt wird, analysiert ONTAP die verfügbaren Ersatzfestplatten im Cluster und generiert eine Empfehlung, wie Ersatzfestplatten gemäß Best Practices zur Hinzufügung lokaler Ebenen verwendet werden sollten. ONTAP zeigt die empfohlenen Konfigurationen an. Es besteht die Möglichkeit, die Empfehlungen zu übernehmen oder die lokalen Ebenen manuell hinzuzufügen.


NOTE: Vor ONTAP 9.7 verwendet System Manager den Begriff _aggregate_, um eine _lokale Ebene_ zu beschreiben. Unabhängig von der ONTAP Version verwendet die ONTAP CLI den Begriff _aggregate_. Weitere Informationen zu lokalen Ebenen finden sich unter link:../disks-aggregates/index.html["Festplatten und lokale Ebenen"].



[[before-you-can-accept-ontap-recommendations]]
== Bevor Sie ONTAP Empfehlungen annehmen können

Wenn eine oder mehrere der folgenden Festplattenbedingungen vorliegen, müssen diese behoben werden, bevor die Empfehlungen von ONTAP übernommen werden:

* Fehlende Datenträger
* Schwankungen bei der Anzahl der Ersatzfestplatten
* Nicht zugewiesene Datenträger
* Nicht-Null-Ersatzfestplatten
* Festplatten, die Wartungstests unterzogen werden




[[when-you-must-use-the-manual-method]]
== Wann die manuelle Methode verwendet werden muss

In vielen Fällen ist das empfohlene Layout der lokalen Ebene optimal für Ihre Umgebung. Wenn Ihre Umgebung jedoch die folgenden Konfigurationen aufweist, muss die lokale Ebene mit der manuellen Methode erstellt werden.


NOTE: Ab ONTAP 9.11.1 ist es möglich, mit System Manager lokale Ebenen manuell hinzuzufügen.

* Lokale Ebenen mit Array-LUNs von Drittanbietern
* Virtuelle Festplatten mit Cloud Volumes ONTAP oder ONTAP Select
* MetroCluster System
* SyncMirror
* MSATA-Festplatten
* Flash Pool Tiers
* Mehrere Festplattentypen oder -größen sind mit dem Node verbunden.
