= Entfernen einer ausgefallenen ONTAP Festplatte
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Eine vollständig ausgefallene Festplatte wird von ONTAP nicht mehr als nutzbare Festplatte gezählt und kann sofort vom Festplattengehäuse getrennt werden. Eine teilweise ausgefallene Festplatte sollte jedoch so lange angeschlossen bleiben, bis der Rapid RAID Recovery-Prozess abgeschlossen ist.

.Über diese Aufgabe
Wenn Sie eine Festplatte entfernen, weil sie defekt ist oder übermäßig viele Fehlermeldungen ausgibt, sollte die Festplatte weder in diesem noch in einem anderen Speichersystem wiederverwendet werden.

.Schritte
. Verwenden Sie die Befehlszeilenschnittstelle (CLI), um die Festplatten-ID der ausgefallenen Festplatte zu ermitteln:
+
`storage disk show -broken`

+
Wenn die Festplatte nicht in der Liste der ausgefallenen Festplatten erscheint, ist sie möglicherweise teilweise ausgefallen, und eine Rapid RAID Recovery ist im Gange. In diesem Fall wird empfohlen, zu warten, bis die Festplatte in der Liste der ausgefallenen Festplatten aufgeführt ist (was bedeutet, dass der Rapid RAID Recovery Prozess abgeschlossen ist), bevor die Festplatte entfernt wird.

+
Ermitteln Sie die Modellnummer aus der letzten Spalte der Ausgabe.

. Den physischen Standort der Festplatte bestimmen, die entfernt werden soll:
+
`storage disk set-led -action on -disk _disk_name_ 2`

+
Die Fehler-LED auf der Vorderseite der Festplatte leuchtet.

. Die Festplatte ist gemäß den Anweisungen im Hardwarehandbuch für das jeweilige Festplattengehäusemodell aus dem Festplattengehäuse zu entfernen.
