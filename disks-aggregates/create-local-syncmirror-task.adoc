= ONTAP lokale Tiers mit aktiviertem SyncMirror hinzufügen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können SyncMirror aktivieren, wenn Sie manuell eine lokale Ebene erstellen, um die Daten der lokalen Ebene synchron zu spiegeln.

Weitere Informationen zu link:../concepts/mirrored-unmirrored-aggregates-concept.html["gespiegelte und ungespiegelte lokale Tiers"].

.Bevor Sie beginnen
* Der Cluster muss ausschließlich mit internem Speicher initialisiert werden.
* Die Cluster-Einrichtung muss auf beiden Knoten abgeschlossen sein.


.Über diese Aufgabe
Dieses Verfahren erstellt auf jedem Cluster-Node gespiegelte lokale Tiers gleicher Größe, und jedes lokale Tier verfügt über 44 Festplatten.

.Schritte
. Automatische Speicherzuweisung deaktivieren:
+
`storage disk option modify -node * -autoassign off`

. Bestätigen Sie, dass die automatische Zuweisung deaktiviert ist:
+
`storage disk option show`

. Das externe Shelf anschließen.
. Weisen Sie jedem Node die externen Laufwerke zu, wobei Pool 1 angegeben wird:
+
`storage disk assign -disk <disk ID> -owner <node name> -pool 1`

. Die lokale Root-Ebene auf jedem Knoten spiegeln:
+
`storage aggregate mirror -aggregate <node1 root-aggr>`

+
`storage aggregate mirror -aggregate <node2 root-aggr>`

+

NOTE: Die Laufwerke in Pool 1 werden automatisch so partitioniert, dass sie denen in Plex 0 entsprechen.

. Auf Knoten 1 eine gespiegelte Daten lokale Datenebene mit einer Anzahl von 44 Festplatten erstellen. Dadurch werden 22 Partitionen aus Pool 0 und 22 Partitionen aus Pool 1 ausgewählt.
+
`storage aggregate create -node <node1 name> -aggregate <node1 aggr-name> -diskcount 44 -mirror true`

. Auf Knoten 2 eine gespiegelte Daten lokale Datenebene mit einer Anzahl von 44 Festplatten erstellen. Dadurch werden 22 Partitionen aus Pool 0 und 22 Partitionen aus Pool 1 ausgewählt.
+
`storage aggregate create -node <node2 name> -aggregate <node2 aggr-name> -diskcount 44 -mirror true`

. Es wird überprüft, ob lokale Ebenen gleicher Größe erfolgreich erstellt wurden:
+
`storage aggregate show`
