= ONTAP RAID-Schutzstufen für Festplatten
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ONTAP unterstützt drei RAID-Schutzstufen für lokale Speicherebenen. Die RAID-Schutzstufe bestimmt die Anzahl der Paritätsfestplatten, die im Falle eines Festplattenausfalls zur Datenwiederherstellung zur Verfügung stehen.

Mit RAID-Schutz kann ONTAP im Falle eines Festplattenausfalls in einer RAID-Gruppe die ausgefallene Festplatte durch eine Ersatzfestplatte ersetzen und mithilfe von Parität die Daten der ausgefallenen Festplatte rekonstruieren.

* *RAID4*
+
Mit RAID4-Schutz kann ONTAP eine Ersatzfestplatte verwenden, um die Daten einer ausgefallenen Festplatte innerhalb der RAID-Gruppe zu ersetzen und wiederherzustellen.

* *RAID-DP*
+
Mit RAID-DP Schutz kann ONTAP bis zu zwei Ersatzfestplatten verwenden, um die Daten von bis zu zwei gleichzeitig ausgefallenen Festplatten innerhalb der RAID-Gruppe zu ersetzen und wiederherzustellen.

* *RAID-TEC*
+
Mit RAID-TEC Schutz kann ONTAP bis zu drei Ersatzfestplatten verwenden, um die Daten von bis zu drei gleichzeitig ausgefallenen Festplatten innerhalb der RAID-Gruppe zu ersetzen und wiederherzustellen.
