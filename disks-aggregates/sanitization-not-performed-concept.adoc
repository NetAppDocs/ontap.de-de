= Informationen darüber, wann die ONTAP Festplattenbereinigung nicht durchgeführt werden kann
:allow-uri-read: 


[role="lead"]
Unter diesen Umständen kann keine Festplattenbereinigung durchgeführt werden.

* Im Übernahmemodus für Systeme in einem HA-Paar wird dies nicht unterstützt.
* Dies kann nicht auf Festplatten durchgeführt werden, die aufgrund von Lese- oder Schreibproblemen ausgefallen sind.
* Wenn Sie das Zufallsmuster verwenden, kann es nicht auf mehr als 100 Datenträger gleichzeitig durchgeführt werden.
* Es wird auf Array LUNs nicht unterstützt.
