= Laufwerke zu einem ONTAP Node oder Shelf hinzufügen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie fügen einem Node oder Shelf Laufwerke hinzu, um die Anzahl der Hot Spares zu erhöhen oder um dem lokalen Tier zusätzlichen Speicherplatz hinzuzufügen.


NOTE: Vor ONTAP 9.7 verwendet System Manager den Begriff _aggregate_, um eine _lokale Ebene_ zu beschreiben. Unabhängig von der ONTAP Version verwendet die ONTAP CLI den Begriff _aggregate_. Weitere Informationen zu lokalen Ebenen finden sich unter link:../disks-aggregates/index.html["Festplatten und lokale Ebenen"].

.Bevor Sie beginnen
Das Laufwerk, das Sie hinzufügen möchten, muss von Ihrer Plattform unterstützt werden. Dies kann mit dem link:https://hwu.netapp.com/["NetApp Hardware Universe"^] bestätigt werden.

Die Mindestanzahl an HDDs, die Sie in einem einzigen Vorgang hinzufügen sollten, beträgt sechs. Die Mindestanzahl an SSDs, die Sie in einem einzigen Vorgang hinzufügen sollten, beträgt drei. Das Hinzufügen eines Einzellaufwerks kann die Leistung beeinträchtigen.

.Schritte für das NetApp Hardware Universe
. Im Dropdown-Menü **Produkte** die eigene Hardwarekonfiguration auswählen
. Wählen Sie Ihre Plattform aus.
. Die verwendete Version von ONTAP auswählen und anschließend **Ergebnisse anzeigen**.
. Unterhalb der Grafik **Hier klicken, um alternative Ansichten anzuzeigen** auswählen. Die Ansicht auswählen, die der eigenen Konfiguration entspricht.


image:hardware-universe-more-info-graphic.png["Screenshot der Hardware Universe Grafik (hier klicken)"]

.Schritte zur Installation der Laufwerke
. Prüfen Sie das link:https://mysupport.netapp.com/site/["NetApp Support Site"^] auf neuere Firmware-Dateien für Laufwerke und Gehäuse sowie Datenträgerqualifizierungspakete.
+
Falls Ihr Node oder Ihr Shelf nicht über die neuesten Versionen verfügt, sollten diese vor der Installation des neuen Laufwerks aktualisiert werden.

+
Bei neuen Laufwerken, die keine aktuelle Firmware-Version besitzen, wird die Laufwerksfirmware automatisch (unterbrechungsfrei) aktualisiert.

. Sorgen Sie für eine ordnungsgemäße Erdung.
. Die Blende wird vorsichtig von der Vorderseite der Plattform entfernt.
. Der richtige Steckplatz für das neue Laufwerk ist zu identifizieren.
+

NOTE: Die korrekten Steckplätze für das Hinzufügen von Laufwerken variieren je nach Plattformmodell und ONTAP Version. In einigen Fällen ist es erforderlich, Laufwerke der Reihe nach in bestimmte Steckplätze einzusetzen. Beispielsweise werden beim AFF A800 die Laufwerke in bestimmten Abständen eingesetzt, sodass Gruppen leerer Steckplätze entstehen. Beim AFF A220 werden neue Laufwerke in die jeweils nächsten leeren Steckplätze eingesetzt, von außen zur Mitte des Shelfs.

+
Die Schritte in **Bevor Sie beginnen** geben an, welche Steckplätze für Ihre Konfiguration in der link:https://hwu.netapp.com/["NetApp Hardware Universe"^] korrekt sind.

. Das neue Laufwerk einsetzen:
+
.. Bei geöffnetem Nockenhebel das neue Laufwerk mit beiden Händen einsetzen.
.. Drücken, bis das Laufwerk stoppt.
.. Den Nockenhebel so schließen, dass das Laufwerk vollständig in der Mittelebene sitzt und der Hebel hörbar einrastet. Darauf achten, den Nockenhebel langsam zu schließen, damit er korrekt mit der Vorderseite des Laufwerks abschließt.


. Es sollte überprüft werden, ob die Aktivitäts-LED (grün) des Laufwerks leuchtet.
+
Leuchtet die Aktivitäts-LED des Laufwerks dauerhaft, bedeutet dies, dass das Laufwerk mit Strom versorgt ist. Blinkt die Aktivitäts-LED des Laufwerks, bedeutet dies, dass das Laufwerk mit Strom versorgt ist und E/A-Vorgänge stattfinden. Wenn die Firmware des Laufwerks automatisch aktualisiert wird, blinkt die LED.

. Um ein weiteres Laufwerk hinzuzufügen, wiederholen Sie die Schritte 4 bis 6.
+
Die neuen Laufwerke werden erst erkannt, nachdem sie einem Knoten zugewiesen wurden. Die neuen Laufwerke können manuell zugewiesen werden, oder ONTAP weist die neuen Laufwerke automatisch zu, sofern Ihr Knoten die Regeln für die automatische Laufwerkszuweisung erfüllt.

. Nachdem alle neuen Laufwerke erkannt wurden, sollte überprüft werden, ob sie hinzugefügt wurden und die Eigentümerschaft korrekt angegeben ist.


.Schritte zur Bestätigung der Installation
. Die Liste der Datenträger wird angezeigt:
+
`storage aggregate show-spare-disks`

+
Die neuen Laufwerke, die dem richtigen Knoten zugeordnet sind, werden angezeigt.

. **Optional (nur für ONTAP 9.3 und früher),** die neu hinzugefügten Laufwerke auf Null setzen:
+
`storage disk zerospares`

+
Laufwerke, die zuvor in einer ONTAP lokalen Tier verwendet wurden, müssen vor dem Hinzufügen zu einer anderen lokalen Tier genullt werden. In ONTAP 9.3 und früher kann das Nullsetzen je nach Größe der nicht genullten Laufwerke im Node mehrere Stunden dauern. Das Nullsetzen der Laufwerke kann Verzögerungen verhindern, falls die Größe einer lokalen Tier schnell erhöht werden muss. In ONTAP 9.4 oder später ist dies kein Problem, da die Laufwerke dort mit _Fast Zeroing_ innerhalb von Sekunden genullt werden.



.Ergebnisse
Die neuen Laufwerke sind einsatzbereit. Sie können sie einer lokalen Tier hinzufügen, sie auf die Liste der Hot Spares setzen oder sie beim Erstellen einer neuen lokalen Tier hinzufügen.
