= Medienkosten einer lokalen ONTAP-Ebene festlegen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Ab ONTAP 9.11.1 ist es mit dem System Manager möglich, die Medienkosten einer lokalen Ebene festzulegen.

.Schritte
. Im System Manager auf *Speicher > Ebenen* klicken und dann in den gewünschten lokalen Ebenenkacheln auf *Medienkosten festlegen* klicken.
. *Aktive und inaktive Stufen* auswählen, um den Vergleich zu ermöglichen.
. Eine Währungsart und ein Betrag werden eingegeben.


Wenn Sie die Medienkosten eingeben oder ändern, wird die Änderung in allen Medientypen vorgenommen.
