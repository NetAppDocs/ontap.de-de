= ONTAP Befehle zur Anzeige von Informationen über Storage Shelves
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Der  `storage shelf show` Befehl zeigt Konfigurations- und Fehlerinformationen für Ihre Festplattengehäuse an.

|===
| Wenn Sie anzeigen möchten... | Verwenden Sie diesen Befehl... 


 a| 
Allgemeine Informationen zur Regalkonfiguration und zum Hardwarestatus
 a| 
`storage shelf show`



 a| 
Detaillierte Informationen zu einem bestimmten Shelf, einschließlich Stack-ID
 a| 
`storage shelf show -shelf`



 a| 
Ungeklärte, vom Kunden zu behebende Fehler nach Shelf
 a| 
`storage shelf show -errors`



 a| 
Informationen zur Bucht
 a| 
`storage shelf show -bay`



 a| 
Verbindungsinformationen
 a| 
`storage shelf show -connectivity`



 a| 
Informationen zur Kühlung, einschließlich Temperatursensoren und Kühlventilatoren
 a| 
`storage shelf show -cooling`



 a| 
Informationen zu E/A-Modulen
 a| 
`storage shelf show -module`



 a| 
Port-Informationen
 a| 
`storage shelf show -port`



 a| 
Informationen zur Stromversorgung, einschließlich Netzteilen, Stromsensoren und Spannungssensoren
 a| 
`storage shelf show -power`

|===
