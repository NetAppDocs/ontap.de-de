= Wie Warnungen bei niedrigem Ersatzfestplattenbestand bei der Verwaltung Ihrer ONTAP Ersatzfestplatten helfen können
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Standardmäßig werden Warnungen in der Konsole und in den Protokollen ausgegeben, wenn weniger als eine Ersatzfestplatte vorhanden ist, die den Attributen jeder Festplatte in Ihrem Speichersystem entspricht.

Sie können den Schwellenwert für diese Warnmeldungen ändern, damit Ihr System den Best Practices entspricht.

.Über diese Aufgabe
Die RAID-Option "`min_spare_count`" sollte auf "`2`" gesetzt werden, damit stets die empfohlene Mindestanzahl an Ersatzfestplatten vorhanden ist.

.Schritt
. Die Option auf "`2`" setzen:
+
`storage raid-options modify -node _nodename_ -name min_spare_count -value 2`



.Verwandte Informationen
* link:https://docs.netapp.com/us-en/ontap-cli/storage-raid-options-modify.html["storage raid-options modify"^]
