= ONTAP-Laufwerke manuell schnell nullen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Bei Systemen, die neu mit ONTAP 9.4 oder höher installiert wurden, und Systemen, die mit ONTAP 9.4 oder höher neu initialisiert wurden, wird _fast zeroing_ verwendet, um Laufwerke zu nullen.

Mit der _schnellen Nullsetzung_ werden Laufwerke innerhalb von Sekunden genullt. Dies erfolgt automatisch vor der Bereitstellung und verkürzt die Zeit für die Initialisierung des Systems, das Erstellen lokaler Tiers oder das Erweitern lokaler Tiers beim Hinzufügen von Ersatzlaufwerken erheblich.

_Fast zeroing_ wird sowohl auf SSDs als auch auf HDDs unterstützt.


NOTE: Die Schnellnullung wird auf Systemen, die von ONTAP 9.3 oder älter aktualisiert wurden, nicht unterstützt. ONTAP 9.4 oder höher muss neu installiert oder das System reinitialisiert werden. In ONTAP 9.3 und älter werden Laufwerke ebenfalls automatisch von ONTAP genullt, jedoch dauert der Vorgang länger.

Wenn Sie ein Laufwerk manuell nullen müssen, können Sie eine der folgenden Methoden verwenden. In ONTAP 9.4 und höher dauert das manuelle Nullen eines Laufwerks ebenfalls nur wenige Sekunden.

[role="tabbed-block"]
====
.CLI Befehl
--
*Einen CLI-Befehl verwenden, um Laufwerke schnell auf Null zu setzen*

.Über diese Aufgabe
Für die Ausführung dieses Befehls sind Administratorrechte erforderlich.

.Schritte
. Den CLI-Befehl eingeben:
+
[source, cli]
----
storage disk zerospares
----


--
.Optionen im Bootmenü
--
*Im Bootmenü Optionen zum schnellen Nullsetzen von Laufwerken auswählen*

.Über diese Aufgabe
* Die Verbesserung der schnellen Nullstellung wird von Systemen, die von einer Version vor ONTAP 9.4 aktualisiert wurden, nicht unterstützt.


.Schritte
. Im Bootmenü steht eine der folgenden Optionen zur Auswahl:
+
** (4) Konfiguration bereinigen und alle Festplatten initialisieren
** (9a) Alle Festplatten entpartitionieren und deren Eigentümerinformationen entfernen
** (9b) Konfiguration bereinigen und Knoten mit vollständigen Festplatten initialisieren




--
