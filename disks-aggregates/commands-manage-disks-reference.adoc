= ONTAP Befehle zur Verwaltung von Festplatten
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können die  `storage disk` und  `storage aggregate` Befehle verwenden, um Ihre Festplatten zu verwalten.


NOTE: Vor ONTAP 9.7 verwendet System Manager den Begriff _aggregate_, um eine _lokale Ebene_ zu beschreiben. Unabhängig von der ONTAP Version verwendet die ONTAP CLI den Begriff _aggregate_. Weitere Informationen zu lokalen Ebenen finden sich unter link:../disks-aggregates/index.html["Festplatten und lokale Ebenen"].

|===


| Wenn Sie möchten... | Verwenden Sie diesen Befehl... 


 a| 
Eine Liste der Ersatzfestplatten, einschließlich partitionierter Festplatten, nach Besitzer anzeigen
 a| 
`storage aggregate show-spare-disks`



 a| 
RAID-Typ der Festplatte, aktuelle Nutzung und RAID-Gruppe nach lokaler Ebene anzeigen
 a| 
`storage aggregate show-status`



 a| 
Anzeige des RAID-Typs, der aktuellen Nutzung, der lokalen Ebene und der RAID-Gruppe, einschließlich Ersatzfestplatten, für physische Festplatten
 a| 
`storage disk show -raid`



 a| 
Zeigen Sie eine Liste der defekten Festplatten einschließlich des Laufwerksmodells an
 a| 
`storage disk show -broken`



 a| 
Den Namen des Laufwerks vor dem Cluster (Nodescope) für eine Festplatte anzeigen
 a| 
`storage disk show -primary-paths` (fortschrittlich)



 a| 
Die LED für eine bestimmte Festplatte oder ein bestimmtes Shelf leuchtet auf.
 a| 
`storage disk set-led`



 a| 
Den Prüfsummentyp für eine bestimmte Festplatte anzeigen
 a| 
`storage disk show -fields checksum-compatibility`



 a| 
Prüfsummentyp für alle Ersatzfestplatten anzeigen
 a| 
`storage disk show -fields checksum-compatibility -container-type spare`



 a| 
Informationen zu Festplattenkonnektivität und Platzierung
 a| 
`storage disk show -fields disk,primary-port,secondary-name,secondary-port,shelf,bay`



 a| 
Die Namen der Datenträger vor der Clusterung für bestimmte Datenträger werden angezeigt
 a| 
`storage disk show -disk diskname -fields diskpathnames`



 a| 
Die Liste der Datenträger im Wartungszentrum anzeigen
 a| 
`storage disk show -maintenance`



 a| 
SSD-Verschleißlebensdauer anzeigen
 a| 
`storage disk show -ssd-wear`



 a| 
Partitionierung einer gemeinsam genutzten Festplatte aufheben
 a| 
`storage disk unpartition` (verfügbar auf Diagnoseebene)



 a| 
Alle nicht mit Nullen initialisierten Datenträger auf null setzen
 a| 
`storage disk zerospares`



 a| 
Einen laufenden Bereinigungsprozess auf einer oder mehreren angegebenen Festplatten anhalten
 a| 
`system node run -node nodename -command disk sanitize`



 a| 
Informationen zur Speicherverschlüsselung von Festplatten anzeigen
 a| 
`storage encryption disk show`



 a| 
Authentifizierungsschlüssel von allen verknüpften Schlüsselverwaltungsservern abrufen
 a| 
`security key-manager restore`
