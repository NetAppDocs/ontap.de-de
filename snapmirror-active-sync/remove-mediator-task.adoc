= ONTAP Mediator oder ONTAP Cloud Mediator entfernen
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
Wenn eine bestehende ONTAP Mediator oder ONTAP Cloud Mediator Konfiguration aus den ONTAP Clustern entfernt werden soll, ist dies mit dem  `snapmirror mediator remove` Befehl möglich. Es kann jeweils nur ein Typ von Mediator verwendet werden, daher muss eine Instanz entfernt werden, bevor die andere installiert wird.

.Schritte
ONTAP Mediator oder ONTAP Cloud Mediator kann entfernt werden, indem einer der folgenden Schritte abgeschlossen wird.

[role="tabbed-block"]
====
.ONTAP Mediator
--
. ONTAP Mediator entfernen:
+
`snapmirror mediator remove -mediator-address <address> -peer-cluster <peerClusterName>`

+
Beispiel:

+
[listing]
----
snapmirror mediator remove -mediator-address 12.345.678.90 -peer-cluster cluster_xyz
----


--
.ONTAP Cloud Mediator
--
. ONTAP Cloud Mediator entfernen:
+
`snapmirror mediator remove -peer-cluster <peerClusterName> -type cloud`

+
Beispiel:

+
[listing]
----
snapmirror mediator remove -peer-cluster cluster_xyz -type cloud
----


--
====
