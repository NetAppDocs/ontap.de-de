= ONTAP SnapMirror Löschvorgang schlägt fehl, wenn auf dem Zielvolume eine Sperre eingerichtet ist
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
Die folgenden Informationen sind relevant, wenn der SnapMirror Löschvorgang fehlschlägt, weil auf einem der Ziel-Volumes eine Umleitungssperre gesetzt ist.

.Problem:
SnapMirror Löschvorgang schlägt fehl, wenn für eines der Ziel-Volumes eine Umleitungssperre gesetzt ist.

.Lösung
Die folgenden Vorgänge werden ausgeführt, um die Umleitung erneut zu versuchen und die Sperre vom Ziel-Volume zu entfernen.

* SnapMirror Resynchronisierung
* SnapMirror Aktualisierung
