= ONTAP Mediator oder ONTAP Cloud Mediator nicht erreichbar oder Mediator-Quorumstatus ist falsch
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
Die folgenden Informationen gelten, wenn der ONTAP Mediator oder ONTAP Cloud Mediator nicht erreichbar ist oder der Quorumstatus des Mediators auf „false“ steht.

.Problem:
Nach Ausführung des  `snapmirror failover start` Befehls wird in der Ausgabe für den  `snapmirror failover show` Befehl eine Meldung angezeigt, die darauf hinweist, dass entweder der ONTAP Mediator oder ONTAP Cloud Mediator nicht konfiguriert ist.

Siehe link:mediator-install-task.html["Den ONTAP Mediator und die Cluster für SnapMirror active sync konfigurieren"] oder link:cloud-mediator-config-task.html["Der ONTAP Cloud Mediator für SnapMirror Aktiv-Sync wird konfiguriert"].

....
Cluster1::> snapmirror failover show
Source Destination Error
Path Path Type Status start-time end-time Reason
-------- ----------- -------- --------- ---------- ---------- ----------
vs0:/cg/cg vs1:/cg/cg planned failed 10/1/2020 10/1/2020 SnapMirror failover cannot start because the source-side precheck failed. reason: Mediator not configured.
05:50:42 05:50:43
....
.Ursache:
Mediator ist nicht konfiguriert oder es bestehen Probleme mit der Netzwerkverbindung.

.Lösung:
Wenn der ONTAP Mediator nicht konfiguriert ist, muss der ONTAP Mediator konfiguriert werden, bevor eine SnapMirror active sync-Beziehung hergestellt werden kann. Etwaige Netzwerkverbindungsprobleme beheben. Sicherstellen, dass der Mediator verbunden ist und der Quorum-Status sowohl auf der Quell- als auch auf der Zielseite mit dem Befehl snapmirror mediator show auf true steht. Weitere Informationen finden sich unter link:mediator-install-task.html["Den ONTAP Mediator konfigurieren"].

....
cluster::> snapmirror mediator show
Mediator Address Peer Cluster     Connection Status Quorum Status
---------------- ---------------- ----------------- -------------
10.234.10.143    cluster2         connected         true
....
