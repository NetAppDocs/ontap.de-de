= Informationen zur NAS-Verwaltung mit ONTAP System Manager
:hardbreaks:
:toclevels: 1
:allow-uri-read: 
:toclevels: 1
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
Die Themen in diesem Abschnitt erläutern die Konfiguration und Verwaltung von NAS-Umgebungen mit System Manager in ONTAP 9.7 und späteren Versionen.

Wenn Sie den klassischen System Manager verwenden (nur in ONTAP 9.7 und früher verfügbar), finden Sie hierzu die folgenden Themen:

* https://docs.netapp.com/us-en/ontap-system-manager-classic/nfs-config/index.html["NFS Konfigurationsübersicht"^]
* https://docs.netapp.com/us-en/ontap-system-manager-classic/smb-config/index.html["SMB Konfigurationsübersicht"^]


System Manager unterstützt Workflows für:

* Erstkonfiguration der Cluster, die für NAS-Dateidienste vorgesehen sind.
* Zusätzliche Volume-Bereitstellung für sich ändernde Speicheranforderungen.
* Konfiguration und Wartung von branchenüblichen Authentifizierungs- und Sicherheitseinrichtungen.


Mit dem System Manager können Sie NAS-Services auf Komponentenebene verwalten:

* Protokolle: NFS, SMB oder beides (NAS Multiprotokoll)
* Namensdienste – DNS, LDAP und NIS
* Namensdienstschalter
* Kerberos und TLS-Sicherheit
* Exporte und Freigaben
* Qtrees
* Namenszuordnung von Benutzern und Gruppen
