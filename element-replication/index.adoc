= Replikation zwischen NetApp Element Software und ONTAP Übersicht
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Die Geschäftskontinuität auf einem Element System kann sichergestellt werden, indem SnapMirror verwendet wird, um Snapshots eines Element Volumes auf ein ONTAP Ziel zu replizieren. Bei einem Ausfall am Element Standort können Daten vom ONTAP System an Clients bereitgestellt und das Element System nach Wiederherstellung des Dienstes wieder aktiviert werden.

Ab ONTAP 9.4 können Snapshots einer auf einem ONTAP Knoten erstellten LUN auf ein Element System repliziert werden. Eine LUN könnte während eines Ausfalls am Element Standort erstellt worden sein, oder eine LUN wird zur Migration von Daten von ONTAP zu Element Software verwendet.

link:https://docs.netapp.com/us-en/element-software/storage/concept_snapmirror_overview.html["Konfiguration der Replikation von NetApp Element Software und ONTAP"^].
