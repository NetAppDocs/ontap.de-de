= Die maximale Kapazitätsgrenze einer Storage-VM im ONTAP System Manager verwalten
:hardbreaks:
:toclevels: 1
:allow-uri-read: 
:toclevels: 1
:nofooter: 
:icons: font

:imagesdir: ./media/


[role="lead"]
Ab ONTAP 9.13.1 kann mit System Manager eine maximale Kapazitätsgrenze für eine Storage-VM aktiviert und ein Schwellenwert festgelegt werden, der Warnungen auslöst, wenn der belegte Speicher einen bestimmten Prozentsatz der maximalen Kapazität erreicht.



[[enable-a-maximum-capacity-limit-for-a-storage-vm]]
[[enable-max-cap]]
== Eine maximale Kapazitätsgrenze für eine Storage-VM aktivieren

Ab ONTAP 9.13.1 können Sie die maximale Kapazität festlegen, die allen Volumes in einer Storage-VM zugewiesen werden kann. Die maximale Kapazität kann beim Hinzufügen einer Storage-VM oder beim Bearbeiten einer vorhandenen Storage-VM aktiviert werden.

.Schritte
. Wählen Sie *Storage* > *Storage VMs*.
. Einer der folgenden Schritte kann durchgeführt werden:
+
--
** Um eine Storage-VM hinzuzufügen, image:icon_add_blue_bg.gif["Symbol hinzufügen"] auswählen.
** Um eine Storage-VM zu bearbeiten, image:icon_kabob.gif["Symbol für Menüoptionen"] neben dem Namen der Storage-VM auswählen und anschließend auf *Bearbeiten* klicken.


--
. Geben Sie die Einstellungen für die Storage-VM ein oder ändern Sie diese und aktivieren Sie das Kästchen mit der Bezeichnung „Maximale Kapazitätsgrenze aktivieren“.
. Geben Sie die maximale Kapazitätsgröße an.
. Geben Sie den Prozentsatz der maximalen Kapazität an, den Sie als Schwellenwert für die Auslösung von Warnmeldungen verwenden möchten.
. Auf *Speichern* klicken.




[[edit-the-maximum-capacity-limit-of-a-storage-vm]]
[[edit-max-cap-limit-svm]]
== Die maximale Kapazitätsgrenze einer Storage-VM bearbeiten

Ab ONTAP 9.13.1 kann das maximale Kapazitätslimit einer bestehenden Storage-VM bearbeitet werden, wenn das <<enable-max-cap,Das maximale Kapazitätslimit wurde aktiviert.>> bereits besteht.

.Schritte
. Wählen Sie *Storage* > *Storage VMs*.
. Klicken Sie image:icon_kabob.gif["Symbol für Menüoptionen"] neben den Namen der Storage-VM und dann auf *Bearbeiten*.
+
Das Kästchen mit der Bezeichnung „Maximale Kapazitätsgrenze aktivieren“ ist bereits aktiviert.

. Einer der folgenden Schritte kann durchgeführt werden:


[cols="35,65"]
|===


| Aktion | Schritte 


 a| 
Maximale Kapazitätsbegrenzung deaktivieren
 a| 
. Deaktivieren Sie das Kästchen.
. Auf *Speichern* klicken.




 a| 
Die maximale Kapazitätsgrenze anpassen
 a| 
. Geben Sie die neue maximale Kapazitätsgröße an. (Es kann keine Größe angegeben werden, die kleiner ist als der bereits zugewiesene Speicherplatz in der Storage-VM.)
. Geben Sie den neuen Prozentsatz der maximalen Kapazität an, den Sie als Schwellenwert zur Auslösung von Warnmeldungen verwenden möchten.
. Auf *Speichern* klicken.


|===
