= Was unterbrechungsfreier Betrieb für Hyper-V und SQL Server über SMB bedeutet
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Unterbrechungsfreier Betrieb für Hyper-V und SQL Server über SMB bezeichnet die Kombination von Funktionen, die es den Anwendungsservern und den darin enthaltenen virtuellen Maschinen oder Datenbanken ermöglichen, während zahlreicher administrativer Aufgaben online zu bleiben und eine kontinuierliche Verfügbarkeit zu gewährleisten. Dies umfasst sowohl geplante als auch ungeplante Ausfallzeiten der Speicherinfrastruktur.

Folgende unterbrechungsfreie Betriebsabläufe für Anwendungsserver über SMB werden unterstützt:

* Geplante Übernahme und Rückgabe
* Ungeplante Übernahme
* Upgrade
* Geplante Aggregatverlagerung (ARL)
* LIF-Migration und Failover
* Geplante Volumenverschiebung
