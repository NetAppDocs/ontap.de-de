= Anzeige des unterbrechungsfreien Betriebsstatus mithilfe der Systemzustandsüberwachung
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Die  `system health`Befehle ermöglichen die Anzeige von Informationen über die allgemeine Systemgesundheit des Clusters und die Gesundheit des CIFS-NDO-Subsystems, die Reaktion auf Warnmeldungen, die Konfiguration zukünftiger Warnmeldungen sowie die Anzeige von Informationen darüber, wie die Zustandsüberwachung konfiguriert ist.

.Schritte
. Der Gesundheitszustand wird durch die Durchführung geeigneter Maßnahmen überwacht:
+
|===
| Wenn Sie anzeigen möchten... | Geben Sie den Befehl ein... 


 a| 
Der Gesundheitszustand des Systems, der den Gesamtzustand der einzelnen Gesundheitsmonitore widerspiegelt
 a| 
`*system health status show*`



 a| 
Informationen zum Gesundheitszustand des CIFS-NDO Subsystems
 a| 
`*system health subsystem show -subsystem CIFS-NDO -instance*`

|===
. Informationen darüber, wie die CIFS-NDO-Alarmüberwachung konfiguriert ist, werden durch Ausführen der entsprechenden Aktionen angezeigt:
+
|===
| Wenn Informationen über ... angezeigt werden sollen | Geben Sie den Befehl ein... 


 a| 
Die Konfiguration und der Status des Health Monitors für das CIFS-NDO-Subsystem, wie überwachte Knoten, Initialisierungsstatus und Status
 a| 
`*system health config show -subsystem CIFS-NDO*`



 a| 
Die CIFS-NDO-Warnungen, die ein Gesundheitsmonitor potenziell generieren kann
 a| 
`*system health alert definition show -subsystem CIFS-NDO*`



 a| 
CIFS-NDO-Gesundheitsmonitor-Richtlinien, die bestimmen, wann Warnmeldungen ausgelöst werden
 a| 
`*system health policy definition show -monitor node-connect*`

|===
+
[NOTE]
====
Der  `-instance`Parameter zeigt detaillierte Informationen an.

====


.Beispiele
Die folgende Ausgabe zeigt Informationen über den allgemeinen Gesundheitszustand des Clusters und des CIFS-NDO-Subsystems:

[listing]
----
cluster1::> system health status show
Status
---------------
ok

cluster1::> system health subsystem show -instance -subsystem CIFS-NDO

                   Subsystem: CIFS-NDO
                      Health: ok
        Initialization State: initialized
Number of Outstanding Alerts: 0
 Number of Suppressed Alerts: 0
                        Node: node2
  Subsystem Refresh Interval: 5m
----
