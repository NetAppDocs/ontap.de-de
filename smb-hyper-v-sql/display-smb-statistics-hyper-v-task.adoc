= SMB Statistiken in ONTAP anzeigen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Es lassen sich verschiedene SMB Statistiken anzeigen, um die Leistung zu überwachen und Probleme zu diagnostizieren.

.Schritte
. Die  `statistics start` und optionalen  `statistics stop` Befehle dienen dazu, eine Datenprobe zu erfassen.
. Führen Sie eine der folgenden Aktionen aus:
+
|===
| Wenn Sie Statistiken für... | Führen Sie den folgenden Befehl aus... 


 a| 
Alle Versionen von SMB
 a| 
`*statistics show -object cifs*`



 a| 
SMB 1.0
 a| 
`*statistics show -object smb1*`



 a| 
SMB 2.x und SMB 3.0
 a| 
`*statistics show -object smb2*`



 a| 
SMB-Subsystem des Knotens
 a| 
`*statistics show -object nblade_cifs*`

|===


.Verwandte Informationen
