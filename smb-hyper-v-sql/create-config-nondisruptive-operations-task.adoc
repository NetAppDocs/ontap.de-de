= Übersicht über ONTAP Konfigurationen für unterbrechungsfreie Operationen mit Hyper-V und SQL Server über SMB
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Es gibt mehrere ONTAP Konfigurationsschritte, die für die Vorbereitung von Hyper-V und SQL Server Installationen erforderlich sind, damit ein unterbrechungsfreier Betrieb über SMB möglich ist.

Bevor die ONTAP Konfiguration für unterbrechungsfreien Betrieb mit Hyper-V und SQL Server über SMB erstellt wird, müssen die folgenden Aufgaben abgeschlossen sein:

* Zeitdienste müssen auf dem Cluster eingerichtet werden.
* Für die SVM muss das Netzwerk eingerichtet werden.
* Die SVM muss erstellt werden.
* Die Data LIF-Schnittstellen müssen auf der SVM konfiguriert sein.
* DNS muss auf der SVM konfiguriert sein.
* Die gewünschten Namensdienste müssen für die SVM eingerichtet sein.
* Der SMB-Server muss erstellt werden.
