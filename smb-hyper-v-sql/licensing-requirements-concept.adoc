= ONTAP und Lizenzierungsanforderungen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Bestimmte ONTAP- und Lizenzierungsanforderungen sind zu berücksichtigen, wenn SQL Server oder Hyper-V over SMB-Lösungen für unterbrechungsfreien Betrieb auf SVMs erstellt werden.



[[ontap-version-requirements]]
== ONTAP Versionsanforderungen

* Hyper-V über SMB
+
ONTAP unterstützt unterbrechungsfreie Vorgänge über SMB-Freigaben für Hyper-V unter Windows 2012 oder höher.

* SQL Server über SMB
+
ONTAP unterstützt unterbrechungsfreie Vorgänge über SMB-Freigaben für SQL Server 2012 oder höher, die unter Windows 2012 oder höher ausgeführt werden.



Die aktuellsten Informationen zu den unterstützten Versionen von ONTAP, Windows Server und SQL Server für unterbrechungsfreie Vorgänge über SMB-Freigaben sind in der Interoperabilitätsmatrix verfügbar.

https://mysupport.netapp.com/matrix["NetApp Interoperabilitätsmatrix-Tool"^]



[[licensing-requirements]]
== Lizenzanforderungen

Folgende Lizenzen werden benötigt:

* CIFS
* FlexClone (nur für Hyper-V über SMB)
+
Diese Lizenz ist erforderlich, wenn Remote VSS für Backups verwendet wird. Der Schattenkopiedienst verwendet FlexClone, um zeitpunktgenaue Kopien von Dateien zu erstellen, die dann beim Erstellen eines Backups verwendet werden.

+
Eine FlexClone Lizenz ist optional, wenn eine Sicherungsmethode verwendet wird, die Remote VSS nicht nutzt.



Die FlexClone-Lizenz ist in link:../system-admin/manage-licenses-concept.html#licenses-included-with-ontap-one["ONTAP One"] enthalten. Wenn ONTAP One nicht vorhanden ist, empfiehlt sich link:../system-admin/manage-license-task.html["Es wird geprüft, ob die erforderlichen Lizenzen installiert sind."] und, falls erforderlich, link:../system-admin/install-license-task.html["sie installieren"].
