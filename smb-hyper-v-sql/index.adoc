= SMB-Konfiguration für Microsoft Hyper-V und SQL Server Übersicht
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ONTAP Funktionen ermöglichen unterbrechungsfreie Vorgänge für zwei Microsoft Anwendungen über das SMB Protokoll: Microsoft Hyper-V und Microsoft SQL Server.

Diese Verfahren sind anzuwenden, wenn unter den folgenden Umständen SMB unterbrechungsfrei betrieben werden soll:

* Der grundlegende SMB-Protokoll-Dateizugriff wurde konfiguriert.
* Sie möchten SMB 3.0 oder neuere Dateifreigaben, die sich in SVMs befinden, für die Speicherung der folgenden Objekte aktivieren:
+
** Hyper-V Dateien für virtuelle Maschinen
** SQL Server-Systemdatenbanken
