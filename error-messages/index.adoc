= Erfahren Sie mehr über die ONTAP EMS-Konfiguration
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ONTAP 9 kann so konfiguriert werden, dass wichtige EMS (Event Management System)-Ereignisbenachrichtigungen direkt an eine E-Mail-Adresse, einen Syslog-Server, einen Simple Management Network Protocol (SNMP) Traphost oder eine Webhook-Anwendung gesendet werden, sodass Sie unmittelbar über Systemprobleme benachrichtigt werden, die eine zeitnahe Aufmerksamkeit erfordern.

Da wichtige Ereignisbenachrichtigungen nicht standardmäßig aktiviert sind, ist das EMS so zu konfigurieren, dass Benachrichtigungen entweder an eine E-Mail-Adresse, einen Syslog-Server, einen SNMP-Traphost oder eine Webhook-Anwendung gesendet werden.

Versionsspezifische Versionen von link:https://docs.netapp.com/us-en/ontap-ems-9131/["ONTAP 9 EMS Referenz"^] anzeigen.

Wenn Ihre EMS-Ereigniszuordnung veraltete ONTAP-Befehlssätze verwendet (wie event destination, event route), wird empfohlen, die Zuordnung zu aktualisieren. link:../error-messages/convert-ems-routing-to-notifications-task.html["Informationen zur Aktualisierung Ihrer EMS-Zuordnung aus veralteten ONTAP Befehlen"].
