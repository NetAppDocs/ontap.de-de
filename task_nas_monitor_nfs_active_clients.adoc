= Aktive NFS-Clients mit ONTAP System Manager überwachen
:hardbreaks:
:toclevels: 1
:allow-uri-read: 
:toclevels: 1
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
Ab ONTAP 9.8 zeigt der System Manager an, welche NFS-Client-Verbindungen aktiv sind, wenn NFS auf einem Cluster lizenziert ist.

Dies ermöglicht eine schnelle Überprüfung, welche NFS-Clients aktiv mit einer Storage-VM verbunden sind, welche verbunden, aber im Leerlauf sind, und welche getrennt sind.

Für jede NFS-Client-IP-Adresse zeigt die Anzeige *NFS Clients* Folgendes an: * Zeitpunkt des letzten Zugriffs * IP-Adresse der Netzwerkschnittstelle * NFS-Verbindungsversion * Name der Storage-VM

Darüber hinaus wird in der Anzeige *Storage>Volumes* eine Liste der in den letzten 48 Stunden aktiven NFS-Clients angezeigt, und in der Anzeige *Dashboard* ist die Anzahl der NFS-Clients enthalten.

.Schritt
. NFS-Client-Aktivität anzeigen: *Hosts > NFS Clients* auswählen.
