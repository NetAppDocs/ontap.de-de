= Lasten werden ausgeglichen, indem Volumes mit ONTAP System Manager auf eine andere Ebene verschoben werden.
:hardbreaks:
:toclevels: 1
:allow-uri-read: 
:toclevels: 1
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
Ab ONTAP 9.9.1 ist das Verschieben von Volumes basierend auf der Analyse aktiver und inaktiver Datenspeicherung möglich. In ONTAP 9.8 kann auch der System Manager verwendet werden, um ein Volume auf eine andere Tier zu verschieben und so die Last auszugleichen.

Weitere Informationen finden sich unter link:concept_nas_file_system_analytics_overview.html["Übersicht zur Dateisystemanalyse"].

.Schritte
. *Speicher > Volumes* auswählen.
. Wählen Sie das Volume oder die Volumes aus, die Sie verschieben möchten, und klicken Sie dann auf *Verschieben*.
. Wählen Sie eine vorhandene Ebene (Aggregat) aus, zu der Sie das oder die Volumes verschieben möchten.
