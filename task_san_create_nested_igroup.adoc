= Eine verschachtelte Initiatorgruppe erstellen
:hardbreaks:
:toclevels: 1
:allow-uri-read: 
:toclevels: 1
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
Ab ONTAP 9.9.1 kann eine Initiatorgruppe erstellt werden, die aus anderen bereits vorhandenen igroups besteht.

. Im System Manager auf *Host > SAN Initiatorgruppen* klicken und dann auf *Hinzufügen*.
. Geben Sie den *Namen* und die *Beschreibung* der Initiatorgruppe ein.
+
Die Beschreibung dient als Alias der Initiatorgruppe.

. Die *Storage-VM* und das *Host-Betriebssystem* auswählen.
+

NOTE: Der Betriebssystemtyp einer verschachtelten Initiatorgruppe kann nach der Erstellung der Initiatorgruppe nicht mehr geändert werden.

. Unter *Mitglieder der Initiatorgruppe* die Option *Vorhandene Initiatorgruppe* auswählen.
+
Die *Suche* kann verwendet werden, um die Initiatorgruppen zu finden und auszuwählen, die hinzugefügt werden sollen.
