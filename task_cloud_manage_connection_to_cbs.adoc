= Die Verbindung zu Cloud Backup verwalten
:hardbreaks:
:toclevels: 1
:allow-uri-read: 
:toclevels: 1
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
Ab ONTAP 9.9.1 kann mit dem System Manager eine Datensicherung in der Cloud mithilfe von Cloud Backup durchgeführt werden. Die Verbindung zu Cloud Backup lässt sich verwalten, und Details zur Anzahl und Kapazität der mit dem Service gesicherten Volumes werden angezeigt.

.Bevor Sie beginnen
Es empfiehlt sich, ein Konto in der NetApp Console einzurichten. Für das Dienstkonto ist die Rolle „Account Admin“ zu erstellen. (Andere Dienstkontorollen verfügen nicht über die erforderlichen Berechtigungen, um eine Verbindung vom System Manager herzustellen.) Siehe link:task_cloud_backup_data_using_cbs.html["Daten mit Cloud Backup sichern"] für Details.



[[view-the-status-of-the-connection-to-cloud-backup]]
== Status der Verbindung zu Cloud Backup anzeigen

Verschiedene Details zur Verbindung mit Cloud Backup sind einsehbar.

.Schritte
. Zu *Schutz > Übersicht* wechseln.
. Im Abschnitt *Cloud Backup Service* sind die folgenden Details sichtbar:
+
** Status der Verbindung.
** Der Cloud-Anbieter.
** Der Cloud Manager Arbeitsbereich.
** Die Anzahl der gesicherten Volumes.
** Der Cloud-Anbieter verwendete Kapazität.
** Die Cloud Manager Connector ID.






[[modify-the-connection-with-cloud-backup]]
== Die Verbindung mit Cloud Backup anpassen

Sie können die Verbindung zu Cloud Backup ändern.

.Schritte
. Zu *Schutz > Übersicht* wechseln.
. Im Abschnitt *Cloud Backup Service* auf image:icon_kabob.gif["Symbol für Menüoptionen"] klicken.
. Sie können eines der folgenden Änderungsverfahren auswählen:
+
** *Bearbeiten*: Ermöglicht die Änderung der Datensicherungsstrategie und des IPspace.
** *Deaktivieren*: Stoppt alle weiteren Backup-Vorgänge in die Cloud für den Cluster.
** *Verknüpfung aufheben*: Entfernt die Verwaltung von Backups beim Cloud-Anbieter aus dem System Manager. Backups werden jedoch weiterhin erstellt und sie können über die NetApp Console verwaltet werden.
