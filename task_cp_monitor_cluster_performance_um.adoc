= Clusterleistung mit ONTAP Unified Manager überwachen
:hardbreaks:
:toclevels: 1
:allow-uri-read: 
:toclevels: 1
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
Mit Active IQ Unified Manager lässt sich die Verfügbarkeit maximieren und die Kontrolle über Ihre NetApp AFF und FAS Speicherinfrastruktur für verbesserte Skalierbarkeit, Supportfähigkeit, Leistung und Sicherheit aufrechterhalten.

Active IQ Unified Manager überwacht kontinuierlich den Systemzustand und sendet Warnmeldungen, sodass Ihre Organisation IT-Personalressourcen freisetzen kann. Der Speicherstatus kann sofort über ein zentrales Dashboard angezeigt und Probleme durch empfohlene Maßnahmen schnell behoben werden.

Die Datenverwaltung wird vereinfacht, da Sie Daten erkennen, überwachen und Benachrichtigungen erhalten können, um den Speicher proaktiv zu verwalten und Probleme schnell zu beheben. Die Effizienz der Administration wird verbessert, da Sie Petabytes an Daten über ein einziges Dashboard überwachen und Ihre Daten in großem Umfang verwalten können.

Mit Active IQ Unified Manager kann auf schwankende Geschäftsanforderungen reagiert und die Performance mithilfe von Leistungsdaten und fortschrittlichen Analysen optimiert werden. Die Berichtsfunktionen bieten Zugriff auf Standardberichte oder ermöglichen die Erstellung individueller Betriebsberichte, um die spezifischen Anforderungen Ihres Unternehmens zu erfüllen.
