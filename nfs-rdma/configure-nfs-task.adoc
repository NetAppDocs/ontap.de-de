= Die NFS-Konfiguration anpassen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
In den meisten Fällen ist es nicht erforderlich, die Konfiguration der NFS-fähigen Storage-VM für NFS über RDMA zu ändern.

Sollten Sie jedoch Probleme mit Mellanox-Chips und der LIF-Migration haben, sollte die NFSv4-Sperrfrist verlängert werden. Standardmäßig beträgt die Sperrfrist 45 Sekunden. Ab ONTAP 9.10.1 beträgt der Maximalwert der Sperrfrist 180 Sekunden.

.Schritte
. Stellen Sie die Berechtigungsstufe auf „Erweitert“ ein:
+
`set -privilege advanced`

. Führen Sie den folgenden Befehl aus:
+
`vserver nfs modify -vserver _SVM_name_ -v4-grace-seconds _number_of_seconds_`



Weitere Informationen zu dieser Aufgabe finden sich unter xref:../nfs-admin/specify-nfsv4-locking-grace-period-task.adoc[Die NFSv4-Sperrfrist für ONTAP SVMs angeben].
