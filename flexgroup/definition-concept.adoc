= Informationen zu ONTAP FlexGroup Volumes
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Ein FlexGroup Volume ist ein Scale-out-NAS-Container, der hohe Performance sowie automatische Lastverteilung und Skalierbarkeit bietet. Ein FlexGroup Volume enthält mehrere Mitglieds-Volumes (Constituents), die den Datenverkehr automatisch und transparent aufteilen. _Mitglieds-Volumes_ sind die zugrunde liegenden FlexVol Volumes, aus denen ein FlexGroup Volume besteht.

image:fg-overview-flexgroup.gif["FlexGroup Volume dargestellt als eine Gruppe von FlexVol Volumes, die als Mitgliedsvolumes oder Bestandteile bezeichnet werden"]

FlexGroup Volumes bieten folgende Vorteile:

* Hohe Skalierbarkeit
+
Mehrere FlexGroup Volumes können auf einem Cluster bereitgestellt werden, solange die Anzahl der Mitgliedsvolumes die Knoten- oder Clustergrenzen nicht überschreitet.

+
Ab ONTAP 9.12.1P2 beträgt die maximale Kapazität eines einzelnen FlexGroup Volumes 60 PB, mit 400 Milliarden Dateien auf einem 10-Knoten-Cluster, wenn link:../volumes/enable-large-vol-file-support-task.html["Großvolumen-Unterstützung ist aktiviert."]. Ohne Unterstützung für große Volumes beträgt die maximale Kapazität eines einzelnen FlexGroup Volumes 20 PB.



[NOTE]
====
Obwohl die maximale Kapazität eines einzelnen FlexGroup Volumes 60PB beträgt (200 Member Volumes x 300TB = 60PB), wird die beste Leistung erzielt, wenn die genutzte Kapazität der Member Volumes unter 80% bleibt (200 Member Volumes x 240TB = 48PB).

====
* Hohe Leistung
+
FlexGroup Volumes können die Ressourcen des Clusters nutzen, um Workloads mit hohem Durchsatz und geringer Latenz zu bedienen.

* Vereinfachtes Management
+
Ein FlexGroup Volume ist ein einzelner Namespace-Container, der auf ähnliche Weise wie FlexVol Volumes verwaltet werden kann.
