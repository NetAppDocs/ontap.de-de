= Zusammenfassung des Workflows zum Datenschutz für ONTAP FlexGroup Volumes
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Es können SnapMirror Disaster Recovery (DR) Beziehungen für FlexGroup Volumes erstellt werden. Auch das Sichern und Wiederherstellen von FlexGroup Volumes mithilfe der SnapVault Technologie ist möglich, ebenso wie das Erstellen einer einheitlichen Datensicherungsbeziehung, die dasselbe Ziel für Backup und DR verwendet.

.Über diese Aufgabe
Der SnapMirror-Beziehungstyp ist immer  `XDP` für FlexGroup Volumes. Die Art des Datenschutzes, die durch eine SnapMirror Beziehung bereitgestellt wird, wird durch die von Ihnen verwendete Replikationsrichtlinie bestimmt. Entweder kann die Standardrichtlinie oder eine benutzerdefinierte Richtlinie des erforderlichen Typs für die gewünschte Replikationsbeziehung verwendet werden.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-1.png["1"]Cluster und SVMs peeren
[role="quick-margin-para"]
Falls die Cluster und SVMs noch nicht miteinander verbunden sind, die link:../peering/create-cluster-relationship-93-later-task.html["Cluster-Peers"] und die link:../peering/create-intercluster-svm-peer-relationship-93-later-task.html["SVM Peers"] erstellen.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-2.png["2"] Einen Arbeitszeitplan erstellen
[role="quick-margin-para"]
Sie müssen link:../data-protection/create-replication-job-schedule-task.html["einen Arbeitsplan erstellen"] festlegen, wann die SnapMirror-Aktualisierungen erfolgen.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-3.png["3"]Je nach Art des Datenschutzes ist einer dieser Wege zu wählen:
[role="quick-margin-list"]
* *Falls SnapMirror DR:*
+
link:create-snapmirror-relationship-task.html["Eine SnapMirror Beziehung erstellen."]Beim Erstellen der Beziehung kann die Standardrichtlinie `MirrorAllSnapshots` oder eine benutzerdefinierte Richtlinie vom Typ `async-mirror` ausgewählt werden.

* *Falls SnapMirror vault:*
+
link:create-snapvault-relationship-task.html["Eine SnapMirror Vault-Beziehung erstellen."]Beim Erstellen der Beziehung kann die Standardrichtlinie `XDPDefault` oder eine benutzerdefinierte Richtlinie vom Typ `vault` ausgewählt werden.

* *Bei einheitlichem Datenschutz:*
+
link:create-unified-data-protection-relationship-task.html["Eine einheitliche Datenschutzbeziehung erstellen."]Beim Erstellen der Beziehung kann die Standardrichtlinie `MirrorAndVault` oder eine benutzerdefinierte Richtlinie vom Typ `mirror-vault` ausgewählt werden.
