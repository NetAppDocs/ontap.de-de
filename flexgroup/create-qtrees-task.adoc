= Qtrees mit ONTAP FlexGroup Volumes erstellen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Ab ONTAP 9.3 können Sie Qtrees mit FlexGroup Volumes erstellen. Qtrees ermöglichen es Ihnen, Ihre FlexGroup Volumes in kleinere Segmente zu unterteilen, die Sie einzeln verwalten können.

.Über diese Aufgabe
* Wenn das FlexGroup Volume auf dem Quell-Cluster Qtrees in einer SnapMirror Beziehung enthält, muss der Ziel-Cluster ONTAP 9.3 oder höher ausführen (eine Version der ONTAP Software, die Qtrees unterstützt).
* Ab ONTAP 9.5 werden Qtree-Statistiken für FlexGroup Volumes unterstützt.


.Schritte
. Einen Qtree im FlexGroup Volume erstellen:
+
[source, cli]
----
volume qtree create -vserver <vserver_name> -volume <volume_name> -qtree <qtree_name>
----
+
Optional lassen sich der Sicherheitsstil, SMB-Oplocks, UNIX-Berechtigungen und die Exportrichtlinie für den Qtree angeben.

+
[listing]
----
cluster1::> volume qtree create -vserver vs0 -volume fg1 -qtree qtree1 -security-style mixed
----
