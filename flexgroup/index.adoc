= Informationen zur Verwaltung von ONTAP FlexGroup Volumes mit der CLI
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können FlexGroup Volumes einrichten, verwalten und schützen, um Skalierbarkeit und Leistung zu gewährleisten. Ein FlexGroup Volume ist ein Scale-out-Volume, das hohe Leistung sowie automatische Lastverteilung bietet.

Sie können FlexGroup Volumes konfigurieren, wenn Folgendes zutrifft:

* Sie möchten bewährte Verfahren anwenden, nicht jede verfügbare Option untersuchen.
* Sie verfügen über Cluster-Administratorrechte, nicht über SVM-Administratorrechte.



NOTE: Ab ONTAP 9.5 ersetzen FlexGroup Volumes die Infinite Volumes, die in ONTAP 9.5 oder späteren Versionen nicht unterstützt werden.

Siehe link:supported-unsupported-config-concept.html["Unterstützte und nicht unterstützte Konfigurationen für FlexGroup Volumes"] für weitere Informationen.

.Verwandte Informationen
Konzeptuelle Informationen zu FlexVol Volumes sind auf FlexGroup Volumes anwendbar. Informationen zu FlexVol Volumes und ONTAP Technologie sind in der ONTAP Reference Library und in Technical Reports (TRs) verfügbar.
