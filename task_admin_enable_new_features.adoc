= Neue Funktionen lassen sich durch das Hinzufügen von Lizenzschlüsseln mit ONTAP System Manager aktivieren.
:hardbreaks:
:toclevels: 1
:allow-uri-read: 
:toclevels: 1
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
In Versionen vor ONTAP 9.10.1 werden ONTAP-Funktionen mit Lizenzschlüsseln aktiviert, und Funktionen in ONTAP 9.10.1 und später werden mit einer NetApp Lizenzdatei aktiviert. Lizenzschlüssel und NetApp Lizenzdateien können mithilfe des System Manager hinzugefügt werden.

Ab ONTAP 9.10.1 wird der System Manager verwendet, um eine NetApp License File zu installieren und so mehrere lizenzierte Funktionen gleichzeitig zu aktivieren. Die Verwendung einer NetApp License File vereinfacht die Lizenzinstallation, da separate Lizenzschlüssel für einzelne Funktionen nicht mehr hinzugefügt werden müssen. Die NetApp License File wird von der NetApp Support Site heruntergeladen.

Wenn Sie bereits Lizenzschlüssel für bestimmte Funktionen besitzen und auf ONTAP 9.10.1 aktualisieren, können Sie diese Lizenzschlüssel weiterhin verwenden.

.Schritte
. *Cluster > Einstellungen* auswählen.
. Unter *Lizenzen* image:icon_arrow.gif["Pfeilsymbol"] auswählen.
. Wählen Sie *Durchsuchen*. Wählen Sie die heruntergeladene NetApp Lizenzdatei aus.
. Wenn Lizenzschlüssel hinzugefügt werden sollen, *28-stellige Lizenzschlüssel verwenden* auswählen und die Schlüssel eingeben.
