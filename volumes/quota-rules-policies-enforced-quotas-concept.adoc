= Erfahren Sie mehr über ONTAP Kontingente, Kontingentregeln und Kontingentrichtlinien
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Quoten werden in Quotenregeln definiert, die für FlexVol Volumes spezifisch sind. Diese Quotenregeln werden in einer Quotenrichtlinie für eine Storage Virtual Machine (SVM) zusammengefasst und auf jedem Volume der SVM aktiviert.

Eine Quotenregel gilt immer spezifisch für ein Volume. Quotenregeln haben keine Wirkung, bis Quoten auf dem in der Quotenregel definierten Volume aktiviert werden.

Eine Quotenrichtlinie ist eine Sammlung von Quotenregeln für alle Volumes einer SVM. Quotenrichtlinien werden nicht zwischen SVMs geteilt. Eine SVM kann bis zu fünf Quotenrichtlinien haben, was die Möglichkeit bietet, Sicherungskopien von Quotenrichtlinien zu erstellen. Einer SVM ist jeweils nur eine Quotenrichtlinie zugewiesen. Beim Initialisieren oder Ändern von Quoten auf einem Volume werden die Quotenregeln der aktuell der SVM zugewiesenen Quotenrichtlinie aktiviert.

Ein Kontingent ist die tatsächliche Beschränkung, die ONTAP durchsetzt, oder die tatsächliche Nachverfolgung, die ONTAP durchführt. Eine Kontingentregel führt immer zu mindestens einem Kontingent und kann viele weitere abgeleitete Kontingente zur Folge haben. Die vollständige Liste der durchgesetzten Kontingente ist nur in Kontingentberichten sichtbar.

Die Aktivierung ist der Prozess, bei dem ONTAP ausgelöst wird, um verbindliche Quoten aus dem aktuellen Satz von Quotenregeln in der zugewiesenen Quotenrichtlinie zu erstellen. Die Aktivierung erfolgt auf Volume-Basis. Die erste Aktivierung von Quoten auf einem Volume wird als Initialisierung bezeichnet. Nachfolgende Aktivierungen werden je nach Umfang der Änderungen entweder als Reinitialisierung oder als Größenanpassung bezeichnet.
