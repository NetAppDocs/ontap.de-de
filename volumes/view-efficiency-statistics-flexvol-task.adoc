= Effizienzstatistiken eines ONTAP FlexVol Volumes anzeigen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können die Details der auf einem FlexVol volume ausgeführten Effizienzmaßnahmen einsehen. Dies kann zur Bewertung der Effektivität Ihrer administrativen Prozesse oder im Rahmen der Kapazitätsplanung erfolgen.

.Schritte
. Mit dem  `volume efficiency stat` Befehl lassen sich die Statistiken der Effizienzoperationen auf einem FlexVol volume anzeigen.


.Beispiel
Mit dem folgenden Befehl können Sie die Statistiken der Effizienzoperationen auf dem Volume VolA anzeigen:
`volume efficiency stat -vserver vs1 -volume VolA`

[listing]
----
cluster1::> volume efficiency stat -vserver vs1 -volume VolA

               Vserver Name: vs1
                Volume Name: VolA
                Volume Path: /vol/VolA
Inline Compression Attempts: 0
----
Weitere Informationen zu  `volume efficiency stat` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/volume-efficiency-stat.html["ONTAP-Befehlsreferenz"^].
