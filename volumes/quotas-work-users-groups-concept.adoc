= Erfahren Sie, wie ONTAP Kontingente mit Benutzern und Gruppen funktionieren
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können einen Benutzer oder eine Gruppe als Ziel eines Benutzerkontingents festlegen. Bei der Definition eines Benutzerkontingents sind einige Implementierungsunterschiede zu beachten.

Einige der Unterschiede, die Sie beachten sollten, sind folgende:

* Benutzer oder Gruppe
* UNIX oder Windows
* Spezielle Benutzer und Gruppen
* Sind mehrere IDs enthalten?


Es gibt außerdem verschiedene Möglichkeiten, IDs für Benutzer je nach Umgebung festzulegen.
