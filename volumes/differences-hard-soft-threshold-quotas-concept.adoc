= Unterschiede zwischen den harten, weichen und Schwellenwertquoten von ONTAP
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Feste Quoten verhindern Operationen, während weiche Quoten Benachrichtigungen auslösen.

Feste Quoten setzen ein endgültiges Limit für die Systemressourcen, jede Operation, die dieses Limit überschreiten würde, schlägt fehl. Die folgenden Einstellungen erstellen feste Quoten:

* Parameter für das Festplattenlimit
* Parameter „Files Limit“


Weiche Quoten senden eine Warnmeldung, wenn die Ressourcennutzung einen bestimmten Wert erreicht, beeinträchtigen jedoch nicht die Datenzugriffsoperationen, sodass geeignete Maßnahmen ergriffen werden können, bevor die Quote überschritten wird. Die folgenden Einstellungen erstellen weiche Quoten:

* Schwellenwert für den Parameter „Festplattenlimit“
* Soft Disk Limit Parameter
* Parameter „Soft Files Limit“


Schwellenwert- und Soft-Disk-Kontingente ermöglichen Administratoren, mehr als eine Benachrichtigung zu einem Kontingent zu erhalten. Typischerweise wird der Schwellenwert für das Festplattenlimit auf einen Wert gesetzt, der nur geringfügig unter dem Festplattenlimit liegt, sodass der Schwellenwert eine „letzte Warnung“ gibt, bevor Schreibvorgänge fehlschlagen.
