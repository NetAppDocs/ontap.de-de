= Erfahren Sie mehr über die Zuweisung von ONTAP Kontingentrichtlinien
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Eine Quotenrichtlinie ist eine Zusammenfassung der Quotenregeln für alle FlexVol volumes einer SVM. Bei der Zuweisung der Quotenrichtlinien sind bestimmte Aspekte zu beachten.

* Eine SVM verfügt zu jedem Zeitpunkt über eine zugewiesene Quotenrichtlinie. Wenn eine SVM erstellt wird, wird eine leere Quotenrichtlinie erstellt und der SVM zugewiesen. Diese Standardquotenrichtlinie trägt den Namen "default", sofern bei der Erstellung der SVM kein anderer Name angegeben wird.
* Eine SVM kann bis zu fünf Quotenrichtlinien haben. Wenn eine SVM fünf Quotenrichtlinien hat, kann keine neue Quotenrichtlinie für die SVM erstellt werden, bis eine vorhandene Quotenrichtlinie gelöscht wurde.
* Wenn Sie eine Quotenregel erstellen oder Quotenregeln für eine Quotenrichtlinie ändern müssen, können Sie eine der folgenden Vorgehensweisen wählen:
+
** Wenn Sie mit einer Quotenrichtlinie arbeiten, die einer SVM zugewiesen ist, muss die Quotenrichtlinie der SVM nicht zugewiesen werden.
** Wenn Sie mit einer nicht zugewiesenen Quotenrichtlinie arbeiten und diese Quotenrichtlinie anschließend der SVM zuweisen, muss eine Sicherungskopie der Quotenrichtlinie vorhanden sein, auf die bei Bedarf zurückgegriffen werden kann.
+
Beispielsweise kann eine Kopie der zugewiesenen Kontingentrichtlinie erstellt, die Kopie geändert, der SVM zugewiesen und die ursprüngliche Kontingentrichtlinie umbenannt werden.



* Eine Quotenrichtlinie kann auch dann umbenannt werden, wenn sie der SVM zugewiesen ist.
