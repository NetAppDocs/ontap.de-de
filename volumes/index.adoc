= Logische Speicherverwaltung mit der ONTAP CLI
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Mit der ONTAP CLI können Sie FlexVol Volumes erstellen und verwalten, mit der FlexClone Technologie effiziente Kopien von Volumes, Dateien und LUNs erstellen, Qtrees und Quotas anlegen sowie Effizienzfunktionen wie Deduplizierung und Komprimierung verwalten.

Diese Vorgehensweise sollte unter folgenden Umständen angewendet werden:

* Es besteht Interesse am Umfang der ONTAP FlexVol volume-Funktionen und an den Speicher-Effizienzmerkmalen.
* Die Befehlszeilenschnittstelle (CLI) soll verwendet werden, nicht der System Manager oder ein automatisiertes Skripttool.
