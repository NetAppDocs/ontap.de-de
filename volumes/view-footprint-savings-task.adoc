= Zeigen Sie die Einsparungen beim ONTAP-Volume-Footprint mit oder ohne temperaturempfindliche Speichereffizienz an
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Je nach Ihrer ONTAP Version können Sie die Einsparungen beim physischen Speicherplatz für jedes Volume einsehen. Dies kann zur Bewertung der Effektivität Ihrer administrativen Prozesse oder im Rahmen der Kapazitätsplanung erfolgen.

.Über diese Aufgabe
Ab ONTAP 9.11.1 kann mit dem Befehl  `volume show-footprint` die Einsparung beim physischen Speicherplatzbedarf von Volumes mit aktivierter temperaturabhängiger Speichereffizienz (TSSE) angezeigt werden. Ab ONTAP 9.13.1 kann mit demselben Befehl die Einsparung beim physischen Speicherplatzbedarf von Volumes ohne aktivierte TSSE angezeigt werden.

.Schritte
. Anzeige der Einsparungen beim Volume-Footprint:
+
[source, cli]
----
volume show-footprint
----
+
.Beispielausgabe mit aktiviertem TSSE
[listing]
----
      Vserver : vs0
      Volume  : vol_tsse_75_per_compress

      Feature                                         Used       Used%
      --------------------------------             ----------    -----
      Volume Data Footprint                           10.15GB      13%
      Volume Guarantee                                     0B       0%
      Flexible Volume Metadata                        64.25MB       0%
      Delayed Frees                                   235.0MB       0%
      File Operation Metadata                             4KB       0%

      Total Footprint                                 10.45GB      13%

      Footprint Data Reduction                         6.85GB       9%
           Auto Adaptive Compression                   6.85GB       9%
      Effective Total Footprint                        3.59GB       5%
----
+
.Beispielausgabe ohne aktiviertes TSSE
[listing]
----
      Vserver : vs0
      Volume  : vol_file_cg_75_per_compress

      Feature                                         Used       Used%
      --------------------------------             ----------    -----
      Volume Data Footprint                            5.19GB       7%
      Volume Guarantee                                     0B       0%
      Flexible Volume Metadata                        32.12MB       0%
      Delayed Frees                                   90.17MB       0%
      File Operation Metadata                             4KB       0%

      Total Footprint                                  5.31GB       7%

      Footprint Data Reduction                         1.05GB       1%
           Data Compaction                             1.05GB       1%
      Effective Total Footprint                        4.26GB       5%
----
