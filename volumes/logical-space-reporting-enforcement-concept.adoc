= Logische Speicherplatzberichterstattung und -durchsetzung für ONTAP Volumes
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Ab ONTAP 9.4 ist es möglich, den Benutzern den belegten logischen Speicherplatz eines Volumes und den verbleibenden Speicherplatz anzuzeigen. Ab ONTAP 9.5 ist es möglich, die von Benutzern belegte Menge an logischem Speicherplatz zu begrenzen.

Die Meldung und Durchsetzung des logischen Speicherplatzes sind standardmäßig deaktiviert.

Die folgenden Volumentypen unterstützen die Meldung und Durchsetzung von logischem Speicherplatz.

[cols="3*"]
|===
| Volumentyp | Wird die Speicherberichterstattung unterstützt? | Wird die Speicherplatzerzwingung unterstützt? 


 a| 
FlexVol Volumes
 a| 
Ja, beginnend mit ONTAP 9.4
 a| 
Ja, beginnend mit ONTAP 9.5



 a| 
SnapMirror Zielvolumen
 a| 
Ja, beginnend mit ONTAP 9.8
 a| 
Ja, beginnend mit ONTAP 9.13.1



 a| 
FlexGroup Volumes
 a| 
Ja, beginnend mit ONTAP 9.9.1
 a| 
Ja, beginnend mit ONTAP 9.9.1



 a| 
FlexCache Volumes
 a| 
Die Ursprungseinstellung wird am Cache verwendet.
 a| 
Nicht zutreffend

|===
