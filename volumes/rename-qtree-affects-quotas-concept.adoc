= Erfahren Sie, wie ONTAP mit Kontingenten beim Umbenennen eines Qtree umgeht
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Wenn ein Qtree mit ONTAP umbenannt wird, werden die Quota-Regeln für diesen Qtree automatisch aktualisiert. Wenn ein Qtree mit dem CIFS- oder NFS-Client umbenannt wird, müssen die Quota-Regeln für diesen Qtree aktualisiert werden.

[NOTE]
====
Wenn Sie einen Qtree mit Ihrem CIFS- oder NFS-Client umbenennen und die Quota-Regeln für diesen Qtree nicht mit dem neuen Namen aktualisieren, bevor Sie die Quotas neu initialisieren, werden keine Quotas auf den Qtree angewendet und explizite Quotas für den Qtree, einschließlich Baumquotas sowie Benutzer- oder Gruppenquotas für den Qtree, können in abgeleitete Quotas umgewandelt werden.

====
