= ONTAP Volumes mit System Manager verwalten
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Nachdem im System Manager eine Liste der Volumes angezeigt wurde, stehen verschiedene Aktionen zur Verwaltung der Volumes zur Verfügung.

Einige Volume-Typen sind im System Manager nicht verfügbar, darunter die folgenden Volumes:

* Vol0
* Volumes vom Typ DEL und TMP
* FlexGroup-Bestandteile
* Replizierte Volumes in einer MetroCluster Konfiguration


.Schritte
. Im System Manager wählen Sie *Storage > Volumes*.
+
Die Liste der Volumes wird angezeigt.

+
SVM-Administratoren sehen nur eine begrenzte Auswahl an Optionen und dürfen möglicherweise keine Volumes hinzufügen.

. Folgendes ist möglich:


[cols="25,75"]
|===


| Um diese Aufgabe auszuführen... | Diese Maßnahmen können ergriffen werden... 


 a| 
Fügen Sie ein Volume hinzu
 a| 
Wählen Sie image:icon_add_blue_bg.gif["Symbol hinzufügen"]. Siehe link:../task_admin_add_a_volume.html["Fügen Sie ein Volume hinzu"].



 a| 
Mehrere Volumes verwalten
 a| 
Aktivieren Sie die Kontrollkästchen neben den Volumes.

* Wählen Sie image:icon_delete_with_can_white_bg.gif["Löschsymbol"], um die ausgewählten Volumes zu löschen.
* Wählen Sie image:icon_protect.gif["Symbol „Protect“"] aus, um den ausgewählten Volumes eine Datensicherungsstrategie zuzuweisen.
* Wählen Sie image:icon-more-kebab-white-bg.gif["Mehr Symbol"] aus, um eine der folgenden Aktionen für alle ausgewählten Volumes auszuführen:
+
** Kontingent aktivieren
** Offline nehmen
** Bewegen
** Gelöschte Volumes anzeigen






 a| 
Einzelnes Volume verwalten
 a| 
Wählen Sie neben dem Volume image:icon_kabob.gif["Symbol für Menüoptionen"] und anschließend eine der folgenden Aktionen aus:

* Bearbeiten
* Größenänderung (Ab ONTAP 9.10.1 und nur für Online-Volumes und DP FlexVol Volumes)
* Löschen
* Klonen
* Offline nehmen (oder online bringen)
* Kontingent aktivieren (oder Kontingent deaktivieren)
* Exportrichtlinie bearbeiten
* Mountpfad bearbeiten
* Bewegen
* Cloud-Tier Einstellungen bearbeiten
* Schützen




 a| 
Volume umbenennen
 a| 
Ein Volume kann auf der Übersichtsseite umbenannt werden.

Wählen Sie image:icon-edit-pencil-blue-outline.png["Bearbeiten-Symbol"] neben dem Datenträgernamen aus und ändern Sie dann den Namen des Datenträgers.

|===
