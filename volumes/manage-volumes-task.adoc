= Verwalten Sie ONTAP-Volumes mit System Manager
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Nachdem Sie in System Manager eine Liste der Volumes angezeigt haben, können Sie zum Verwalten der Volumes verschiedene Aktionen durchführen.

Einige Volume-Typen sind mit System Manager nicht verfügbar, einschließlich der folgenden Volumes:

* Vol0
* Volumen vom Typ DEL und TMP
* FlexGroup-Komponenten
* Replizierte Volumes in einer MetroCluster-Konfiguration


.Schritte
. Im System Manager wählen Sie *Storage > Volumes*.
+
Die Liste der Volumes wird angezeigt.

+
SVM-Administratoren sehen nur eine begrenzte Auswahl an Optionen und dürfen möglicherweise keine Volumes hinzufügen.

. Sie können Folgendes durchführen:


[cols="25,75"]
|===


| Aufgabe durchführen... | Ergreifen Sie diese Maßnahmen... 


 a| 
Fügen Sie ein Volume hinzu
 a| 
Wählen Sie image:icon_add_blue_bg.gif["Symbol hinzufügen"]. Siehe link:../task_admin_add_a_volume.html["Fügen Sie ein Volume hinzu"].



 a| 
Management mehrerer Volumes
 a| 
Aktivieren Sie die Kontrollkästchen neben den Volumes.

* Wählen Sie image:icon_delete_with_can_white_bg.gif["Symbol Löschen"], um die ausgewählten Volumes zu löschen.
* Wählen Sie image:icon_protect.gif["Symbol schützen"] aus, um den ausgewählten Volumes eine Datensicherungsstrategie zuzuweisen.
* Wählen Sie image:icon-more-kebab-white-bg.gif["Weitere Symbole"] aus, um eine der folgenden Aktionen für alle ausgewählten Volumes auszuführen:
+
** Quoten aktivieren
** Versetzen in den Offline-Modus
** Bewegen
** Gelöschte Volumes Anzeigen






 a| 
Management eines einzelnen Volumes
 a| 
Wählen Sie neben dem Volume image:icon_kabob.gif["Symbol für Menüoptionen"] und anschließend eine der folgenden Aktionen aus:

* Bearbeiten
* Größe der Größe ändern (ab ONTAP 9.10.1, nur für Online-Volumes und DP FlexVol Volumes)
* Löschen
* Klonen
* In Offline-Modus versetzen (oder Online versetzen)
* Quoten aktivieren (oder Quota deaktivieren)
* Exportrichtlinie Bearbeiten
* Mount Path Bearbeiten
* Bewegen
* Einstellungen Für Cloud-Ebene Bearbeiten
* Sichern




 a| 
Benennen Sie ein Volume um
 a| 
Sie können ein Volume über die Übersichtsseite umbenennen.

Wählen Sie image:icon-edit-pencil-blue-outline.png["Symbol bearbeiten"] neben dem Datenträgernamen aus und ändern Sie dann den Namen des Datenträgers.

|===
