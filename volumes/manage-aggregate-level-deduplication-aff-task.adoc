= Verwaltung der Inline-Deduplizierung auf Aggregatebene in AFF Systemen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Die Deduplizierung auf Aggregatebene entfernt doppelte Blöcke über Volumes hinweg, die zum selben Aggregat gehören. Die Deduplizierung auf Aggregatebene kann direkt auf AFF Systemen durchgeführt werden. Diese Funktion ist standardmäßig für alle neu erstellten Volumes und alle aktualisierten Volumes aktiviert, bei denen die Inline-Deduplizierung für Volumes aktiviert ist.

.Über diese Aufgabe
Die Deduplizierungsoperation entfernt doppelte Blöcke, bevor Daten auf die Festplatte geschrieben werden. Nur Volumes mit der  `space guarantee` auf  `none` gesetzten Option können an der Inline-Deduplizierung auf Aggregatebene teilnehmen. Dies ist die Standardeinstellung auf AFF Systemen.

[NOTE]
====
Die Inline-Deduplizierung auf Aggregatebene wird manchmal auch als volumenübergreifende Inline-Deduplizierung bezeichnet.

====
.Schritt
. Inline-Deduplizierung auf Aggregatebene in AFF-Systemen verwalten:
+
[cols="2*"]
|===
| Wenn Sie möchten... | Verwenden Sie diesen Befehl 


 a| 
Inline-Deduplizierung auf Aggregatebene aktivieren
 a| 
`volume efficiency modify -vserver vserver_name -volume vol_name -cross-volume-inline-dedupe true`



 a| 
Inline-Deduplizierung auf Aggregatebene deaktivieren
 a| 
`volume efficiency modify -vserver vserver_name -volume vol_name -cross-volume-inline-dedupe false`



 a| 
Anzeige des Inline-Deduplizierungsstatus auf Aggregatebene
 a| 
`volume efficiency config -volume vol_name`

|===


.Beispiele
Der folgende Befehl zeigt den Status der Inline-Deduplizierung auf Aggregatebene an:

[listing]
----

wfit-8020-03-04::> volume efficiency config -volume choke0_wfit_8020_03_0

Inline Dedupe:                                true
Data Compaction:                              true

----
