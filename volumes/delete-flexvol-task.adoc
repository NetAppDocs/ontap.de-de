= Löschen eines ONTAP FlexVol Volume
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können ein nicht mehr benötigtes FlexVol Volume löschen.

.Bevor Sie beginnen
Es dürfen keine Anwendungen auf die Daten in dem Volume zugreifen, das Sie löschen möchten.

[NOTE]
====
Falls ein Volume versehentlich gelöscht wurde, siehe link:https://kb.netapp.com/Advice_and_Troubleshooting/Data_Storage_Software/ONTAP_OS/How_to_use_the_Volume_Recovery_Queue["NetApp Knowledge Base: Verwendung der Volume-Wiederherstellungswarteschlange"^].

====
.Schritte
. Falls das Volume eingebunden wurde, es aushängen:
+
`volume unmount -vserver vserver_name -volume volume_name`

. Wenn das Volume Teil einer SnapMirror Beziehung ist, kann die Beziehung mit dem  `snapmirror delete` Befehl gelöscht werden.
. Wenn das Volume online ist, wird es offline geschaltet:
+
`volume offline -vserver vserver_name volume_name`

. Das Volume löschen:
+
`volume delete -vserver vserver_name volume_name`





[[result]]
== Ergebnis

Das Volume wird zusammen mit allen zugehörigen Quota-Richtlinien und Qtrees gelöscht.
