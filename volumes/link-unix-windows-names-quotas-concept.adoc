= UNIX- und Windows-Namensverknüpfung für ONTAP Quotas
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
In einer gemischten Umgebung können sich Benutzer entweder als Windows-Benutzer oder als UNIX-Benutzer anmelden. Es ist möglich, Quoten so zu konfigurieren, dass die UNIX-ID und die Windows-ID eines Benutzers als ein und derselbe Benutzer erkannt werden.

Kontingente für Windows-Benutzernamen werden einem UNIX-Benutzernamen zugeordnet oder umgekehrt, wenn beide der folgenden Bedingungen erfüllt sind:

* Der  `user-mapping`Parameter ist in der Quotenregel für den Benutzer auf "on" gesetzt.
* Die Benutzernamen wurden mit den  `vserver name-mapping` Befehlen zugeordnet.


Wenn ein UNIX- und ein Windows-Name einander zugeordnet werden, werden sie bei der Bestimmung der Kontingentnutzung als dieselbe Person behandelt.
