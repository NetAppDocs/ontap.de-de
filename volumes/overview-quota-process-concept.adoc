= Erfahren Sie mehr über ONTAP Kontingente
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Quotas ermöglichen es, den von einem Benutzer, einer Gruppe oder einem Qtree verwendeten Speicherplatz und die Anzahl der Dateien einzuschränken oder zu verfolgen. Quotas werden auf ein bestimmtes FlexVol volume oder einen bestimmten Qtree angewendet.

Quoten können weich oder hart sein. Weiche Quoten veranlassen ONTAP, eine Benachrichtigung zu senden, wenn festgelegte Grenzwerte überschritten werden, und harte Quoten verhindern, dass ein Schreibvorgang erfolgreich ist, wenn festgelegte Grenzwerte überschritten werden.

Wenn ONTAP eine Anfrage von einem Benutzer oder einer Benutzergruppe zum Schreiben auf ein FlexVol volume erhält, prüft es, ob für diesen Benutzer oder diese Benutzergruppe Quoten auf diesem volume aktiviert sind und ermittelt Folgendes:

* Ob das endgültige Limit erreicht wird
+
Falls ja, schlägt der Schreibvorgang fehl, wenn das endgültige Limit erreicht ist und die Benachrichtigung über das endgültige Limit gesendet wird.

* Ob das weiche Limit überschritten wird
+
Wenn ja, ist der Schreibvorgang erfolgreich, wenn das Soft-Limit überschritten wird und die Soft-Quota-Benachrichtigung gesendet wird.

* Ob ein Schreibvorgang das Softlimit nicht überschreitet
+
Wenn ja, ist der Schreibvorgang erfolgreich und es wird keine Benachrichtigung gesendet.
