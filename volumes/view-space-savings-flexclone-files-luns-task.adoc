= ONTAP Speicherplatzeinsparungen mit FlexClone-Dateien und FlexClone-LUNs anzeigen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können den Prozentsatz des durch Block-Sharing auf einem Volume mit FlexClone Dateien und FlexClone LUNs eingesparten Speicherplatzes anzeigen. Dies kann Teil der Kapazitätsplanung sein.

.Schritte
. Um die durch FlexClone Dateien und FlexClone LUNs erzielte Speicherplatzersparnis anzuzeigen, den folgenden Befehl eingeben:
+
`df -s volname`

+
`volname` ist der Name des FlexVol volume.

+
[NOTE]
====
Wenn der  `df -s` Befehl auf einem Deduplizierung-aktivierten FlexVol volume ausgeführt wird, lässt sich der durch Deduplizierung sowie durch FlexClone Dateien und LUNs eingesparte Speicherplatz anzeigen.

====


.Beispiel
Das folgende Beispiel zeigt die Speicherplatzersparnis auf einem FlexClone volume test1:

[listing]
----
systemA> df -s test1

Filesystem         used   saved   %saved Vserver
/vol/test1/        4828    5744     54%  vs1
----
Weitere Informationen zu den in diesem Verfahren beschriebenen Befehlen befinden sich in der link:https://docs.netapp.com/us-en/ontap-cli/["ONTAP-Befehlsreferenz"^].
