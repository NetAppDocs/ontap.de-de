= Führen Sie einen ONTAP-Effizienzvorgang mithilfe der Zeitplanung durch.
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können die Planung von Deduplizierungs- oder Datenkomprimierungsvorgängen auf einem Volume anpassen. Die Konfigurationsoptionen für einen Zeitplan und eine Effizienzrichtlinie schließen sich gegenseitig aus.

.Über diese Aufgabe
Dieser Vorgang wird mit dem  `volume efficiency modify` Befehl durchgeführt. Weitere Informationen zu  `volume efficiency modify` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/volume-efficiency-modify.html["ONTAP-Befehlsreferenz"^].

.Schritte
. Mit dem  `volume efficiency modify` Befehl lässt sich die Planung von Deduplizierungs- oder Datenkomprimierungsvorgängen auf einem Volume anpassen.


.Beispiele
Der folgende Befehl ändert die Planung der Effizienzvorgänge für VolA, sodass diese montags bis freitags um 23 Uhr ausgeführt werden:

`volume efficiency modify -vserver vs1 -volume VolA -schedule mon-fri@23`

.Verwandte Informationen
* link:run-efficiency-operations-depending-new-data-task.html["Effizienzoperationen werden abhängig von der Menge der neu geschriebenen Daten ausgeführt."]
