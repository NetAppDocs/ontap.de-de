= Methoden zum Löschen von ONTAP FlexClone Dateien und FlexClone LUNs
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können mehrere Methoden verwenden, um FlexClone-Dateien und FlexClone-LUNs zu löschen. Das Verständnis der verfügbaren Methoden unterstützt die Planung der Klonverwaltung.

Sie können die folgenden Methoden verwenden, um FlexClone Dateien und FlexClone LUNs zu löschen:

* Ein FlexVol Volume kann so konfiguriert werden, dass Klone mit aktivierter Autodelete-Funktion automatisch gelöscht werden, wenn der freie Speicherplatz in einem FlexVol Volume unter einen bestimmten Schwellenwert sinkt.
* Clients können so konfiguriert werden, dass sie Klone mithilfe des NetApp Manageability SDK löschen.
* Clients können zum Löschen von Klonen mithilfe der NAS- und SAN-Protokolle verwendet werden.
+
Die langsamere Löschmethode ist standardmäßig aktiviert, da diese Methode das NetApp Manageability SDK nicht verwendet. Das System kann jedoch so konfiguriert werden, dass beim Löschen von FlexClone Dateien die schnellere Löschmethode verwendet wird, wenn die  `volume file clone deletion` Befehle verwendet werden.
