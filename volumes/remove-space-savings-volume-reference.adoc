= Platzersparnisse aus einem ONTAP Volume entfernen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können die durch Effizienzoptimierungen auf einem Volume erzielten Speicherplatzeinsparungen rückgängig machen. Es muss jedoch ausreichend Speicherplatz vorhanden sein, um eine Rückgängigmachung zu ermöglichen.

Es stehen verschiedene Informationsquellen zur Verfügung, die bei der Planung und Umsetzung der Entfernung der Platzersparnis helfen.
