= ONTAP-Befehle zur Verwaltung von Kontingentregeln und Kontingentrichtlinien
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Die `volume quota policy rule` Befehle ermöglichen die Konfiguration von Quotenregeln, und die `volume quota policy` Befehle sowie einige `vserver` Befehle ermöglichen die Konfiguration von Quotenrichtlinien. Je nach Anforderung stehen die folgenden Befehle zur Verwaltung von Kontingentregeln und Kontingentrichtlinien zur Verfügung:


NOTE: Die folgenden Befehle können nur auf FlexVol Volumes ausgeführt werden.



[[commands-for-managing-quota-rules]]
== Befehle zur Verwaltung von Quotenregeln

[cols="2*"]
|===
| Wenn Sie möchten... | Verwenden Sie diesen Befehl... 


 a| 
Eine neue Quotenregel erstellen
 a| 
`volume quota policy rule create`



 a| 
Eine bestehende Quotenregel löschen
 a| 
`volume quota policy rule delete`



 a| 
Eine bestehende Quotenregel ändern
 a| 
`volume quota policy rule modify`



 a| 
Informationen zu konfigurierten Kontingentregeln
 a| 
`volume quota policy rule show`

|===


[[commands-for-managing-quota-policies]]
== Befehle zur Verwaltung von Quotenrichtlinien

[cols="2*"]
|===
| Wenn Sie möchten... | Verwenden Sie diesen Befehl... 


 a| 
Eine Quotenrichtlinie und die darin enthaltenen Quotenregeln duplizieren
 a| 
`volume quota policy copy`



 a| 
Eine neue, leere Quotenrichtlinie erstellen
 a| 
`volume quota policy create`



 a| 
Eine bestehende Quotenrichtlinie löschen, die derzeit keiner Storage Virtual Machine (SVM) zugewiesen ist
 a| 
`volume quota policy delete`



 a| 
Eine Quotenrichtlinie umbenennen
 a| 
`volume quota policy rename`



 a| 
Informationen zu Quotenrichtlinien
 a| 
`volume quota policy show`



 a| 
Einer SVM wird eine Quotenrichtlinie zugewiesen.
 a| 
`vserver modify -quota-policy _policy_name_`



 a| 
Der Name der einer SVM zugewiesenen Quotenrichtlinie wird angezeigt.
 a| 
`vserver show`

|===
Weitere Informationen zu den in diesem Verfahren beschriebenen Befehlen befinden sich in der link:https://docs.netapp.com/us-en/ontap-cli/["ONTAP-Befehlsreferenz"^].
