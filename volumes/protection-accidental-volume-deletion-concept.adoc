= Schutz vor versehentlichem Löschen von ONTAP-Volumes
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Das Standardverhalten beim Löschen von Volumes erleichtert die Wiederherstellung versehentlich gelöschter FlexVol volumes.

Eine  `volume delete`Anfrage an ein Volume, das den Typ  `RW` oder  `DP` hat (wie in der  `volume show`Befehlsausgabe zu sehen), führt dazu, dass dieses Volume in einen teilweise gelöschten Zustand versetzt wird. Standardmäßig bleibt es mindestens 12 Stunden in einer Wiederherstellungswarteschlange, bevor es vollständig gelöscht wird.


NOTE: Durch das Löschen der SVM, die das gelöschte Volume enthält, wird die Volume Recovery Queue (VRQ) geleert. Löschen Sie eine SVM nur, wenn Sie sicher sind, dass keine Volumes, die der SVM gehören, wiederhergestellt werden müssen. Volumes in der Volume Recovery Queue können nicht existieren, wenn die zugehörige SVM gelöscht wird.
