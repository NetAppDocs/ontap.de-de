= Visualisierung der ONTAP-Speichereffizienz
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Mit dem  `storage aggregate show-efficiency` Befehl werden Informationen zur Speichereffizienz aller Aggregate im System angezeigt.

Der  `storage aggregate show-efficiency` Befehl verfügt über drei verschiedene Ansichten, die durch Übergabe von Befehlsoptionen aufgerufen werden können.

Weitere Informationen zu  `storage aggregate show-efficiency` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/storage-aggregate-show-efficiency.html["ONTAP-Befehlsreferenz"^].



[[default-view]]
== Standardansicht

Die Standardansicht zeigt das Gesamtverhältnis für jedes der Aggregate an.

`cluster1::> storage aggregate show-efficiency`



[[detailed-view]]
== Detailansicht

Die Detailansicht wird mit der  `-details` Befehlsoption angezeigt. In dieser Ansicht werden die folgenden Informationen angezeigt:

* Gesamtwirkungsgradverhältnis für jedes der Aggregate.
* Gesamtverhältnis ohne Momentaufnahmen.
* Verhältnisaufteilung für die folgenden Effizienztechnologien: Volume Deduplizierung, Volume Komprimierung, Snapshots, Klone, Datenkompaktierung und Aggregate Inline-Deduplizierung.


`cluster1::> storage aggregate show-efficiency -details`



[[advanced-view]]
== Erweiterte Ansicht

Die erweiterte Ansicht ähnelt der Detailansicht und zeigt sowohl logische als auch physische verwendete Details an.

Dieser Befehl muss auf der erweiterten Berechtigungsebene ausgeführt werden. Zur erweiterten Berechtigungsebene wechseln Sie mit dem  `set -privilege advanced` Befehl.

Die Eingabeaufforderung ändert sich zu `cluster::*>`.

`cluster1::> set -privilege advanced`

Die erweiterte Ansicht wird mit der  `-advanced` Befehlsoption angezeigt.

`cluster1::*> storage aggregate show-efficiency -advanced`

Um die Verhältnisse für ein einzelnes Aggregat anzuzeigen, wird der  `-aggregate _aggregate_name_` Befehl aufgerufen. Dieser Befehl kann sowohl auf Administratorebene als auch auf der erweiterten Berechtigungsebene ausgeführt werden.

`cluster1::> storage aggregate show-efficiency -aggregate aggr1`

Weitere Informationen zu  `set -privilege advanced` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/set.html["ONTAP-Befehlsreferenz"^].
