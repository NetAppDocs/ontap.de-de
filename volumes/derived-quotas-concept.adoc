= Erfahren Sie mehr über ONTAP basierte Quoten
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Eine Quote, die aufgrund einer Standardquote und nicht aufgrund einer expliziten Quote (einer Quote mit einem bestimmten Ziel) erzwungen wird, wird als _abgeleitete Quote_ bezeichnet.

Die Anzahl und der Speicherort der abgeleiteten Kontingente hängen vom Kontingenttyp ab:

* Ein Standardbaumkontingent auf einem Volume erzeugt abgeleitete Standardbaumkontingente für jeden Qtree auf dem Volume.
* Ein Standardbenutzer- oder Gruppenkontingent erstellt für jeden Benutzer oder jede Gruppe, der oder die eine Datei auf derselben Ebene (Volume oder Qtree) besitzt, ein abgeleitetes Benutzer- oder Gruppenkontingent.
* Ein Standardbenutzer- oder Gruppenkontingent auf einem Volume erzeugt ein abgeleitetes Standardbenutzer- oder Gruppenkontingent auf jedem Qtree, der ebenfalls über ein Baumkontingent verfügt.


Die Einstellungen abgeleiteter Kontingente, einschließlich Limits und Benutzerzuordnung, entsprechen den Einstellungen der entsprechenden Standardkontingente. Beispielsweise erstellt ein Standard-Baumkontingent mit einem Festplattenlimit von 20 GB auf einem Volume abgeleitete Baumkontingente mit einem Festplattenlimit von 20 GB für die Qtrees im Volume. Handelt es sich bei einem Standardkontingent um ein Tracking-Kontingent (ohne Limits), sind auch die abgeleiteten Kontingente Tracking-Kontingente.

Abgeleitete Kontingente lassen sich durch das Erstellen eines Kontingentberichts anzeigen. Im Bericht wird ein abgeleitetes Benutzer- oder Gruppenkontingent durch einen Kontingentbezeichner gekennzeichnet, der entweder leer ist oder ein Sternchen (*) enthält. Ein abgeleitetes Baumkontingent hingegen verfügt über einen Kontingentbezeichner. Zur Identifizierung eines abgeleiteten Baumkontingents ist auf dem Volume nach einem Standardbaumkontingent mit denselben Grenzwerten zu suchen.

Explizite Quoten interagieren auf folgende Weise mit abgeleiteten Quoten:

* Abgeleitete Quoten werden nicht erstellt, wenn für dasselbe Ziel bereits eine explizite Quote existiert.
* Wenn beim Erstellen einer expliziten Quote für ein Ziel bereits eine abgeleitete Quote existiert, kann die explizite Quote durch Größenänderung aktiviert werden, anstatt eine vollständige Quoteninitialisierung durchführen zu müssen.
