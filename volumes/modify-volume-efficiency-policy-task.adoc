= Ändern einer ONTAP Effizienzrichtlinie
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können eine Effizienzrichtlinie für ein Volume so anpassen, dass Deduplizierung und Datenkomprimierung für eine andere Dauer ausgeführt werden oder der Auftragszeitplan mithilfe des  `volume efficiency policy modify` Befehls geändert wird. Weitere Informationen zu  `volume efficiency policy modify` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/volume-efficiency-policy-modify.html["ONTAP-Befehlsreferenz"^].

.Schritte
. Mit dem  `volume efficiency policy modify` Befehl lässt sich eine Effizienzrichtlinie für ein Volume ändern.


.Beispiele
Der folgende Befehl ändert die Effizienzrichtlinie mit dem Namen policy1 so, dass sie stündlich ausgeführt wird:

`volume efficiency policy modify -vserver vs1 -policy policy1 -schedule hourly`

Der folgende Befehl ändert eine Effizienzrichtlinie namens pol2 auf einen Schwellenwert von 30%:

`volume efficiency policy modify -vserver vs1 -policy pol1 -type threshold -start-threshold-percent 30%`
