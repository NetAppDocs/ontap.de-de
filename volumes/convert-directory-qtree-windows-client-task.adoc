= Verwenden Sie einen Windows-Client, um ein ONTAP Volume-Verzeichnis in einen Qtree zu konvertieren.
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Um ein Verzeichnis mithilfe eines Windows-Clients in einen Qtree umzuwandeln, wird das Verzeichnis umbenannt, ein Qtree auf dem Speichersystem erstellt und der Inhalt des Verzeichnisses in den Qtree verschoben.

.Über diese Aufgabe
Für diese Vorgehensweise muss der Windows Explorer verwendet werden. Die Windows-Befehlszeile oder die DOS-Eingabeaufforderung kann nicht verwendet werden.

.Schritte
. Windows Explorer öffnen.
. Klicken Sie auf die Ordnerdarstellung des Verzeichnisses, das Sie ändern möchten.
+
[NOTE]
====
Das Verzeichnis muss sich im Stammverzeichnis des übergeordneten Volumes befinden.

====
. Im Menü *Datei* die Option *Umbenennen* auswählen, um diesem Verzeichnis einen anderen Namen zu geben.
. Auf dem Speichersystem mit dem  `volume qtree create` Befehl einen neuen Qtree mit dem ursprünglichen Namen des Verzeichnisses erstellen. Weitere Informationen zu  `volume qtree create` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/volume-qtree-create.html["ONTAP-Befehlsreferenz"^].
. Im Windows Explorer den umbenannten Verzeichnisordner öffnen und die darin enthaltenen Dateien auswählen.
. Ziehen Sie diese Dateien in die Ordnerdarstellung des neuen qtree.
+
[NOTE]
====
Je mehr Unterordner der Ordner enthält, den Sie verschieben, desto länger dauert der Verschiebevorgang.

====
. Im Menü *Datei* die Option *Löschen* auswählen, um den umbenannten, nun leeren Verzeichnisordner zu löschen.
