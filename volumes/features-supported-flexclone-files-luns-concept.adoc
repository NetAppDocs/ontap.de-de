= ONTAP-Funktionen, die von FlexClone Dateien und FlexClone LUNs unterstützt werden
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
FlexClone Dateien und FlexClone LUNs funktionieren mit verschiedenen ONTAP Funktionen, wie Deduplizierung, Snapshots, Kontingenten und Volume SnapMirror.

Folgende Funktionen werden von FlexClone Dateien und FlexClone LUNs unterstützt:

* Deduplizierung
* Snapshots
* Zugriffskontrolllisten
* Kontingente
* FlexClone Volumes
* NDMP
* Volume SnapMirror
* Der  `volume move` Befehl
* Platzreservierung
* HA-Konfiguration
