= Eine Effizienzrichtlinie von einem ONTAP Volume entkoppeln
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können eine Effizienzrichtlinie von einem Volume trennen, um weitere zeitplanbasierte Deduplizierungs- und Datenkomprimierungsvorgänge auf dem Volume zu beenden. Nach dem Trennen einer Effizienzrichtlinie muss diese manuell ausgelöst werden.

.Schritt
. Mit dem  `volume efficiency modify` Befehl lässt sich eine Volume Effizienzrichtlinie von einem Volume trennen.


.Beispiel
Der folgende Befehl trennt die Effizienzrichtlinie vom Volume VolA: `volume efficiency modify -vserver vs1 -volume VolA -policy -`

Weitere Informationen zu  `volume efficiency modify` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/volume-efficiency-modify.html["ONTAP-Befehlsreferenz"^].
