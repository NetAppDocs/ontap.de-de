= Erfahren Sie mehr über die Erstellung eines ONTAP FlexClone Volumes aus einer SnapMirror Quelle oder einem Zielvolume
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können ein FlexClone Volume aus dem Quell- oder Zielvolume in einer bestehenden Volume SnapMirror Beziehung erstellen. Wenn Sie eine Version von ONTAP verwenden, die älter als ONTAP 9.12.1 ist, und Beziehungen vom Typ DP nutzen, die die  `DPDefault` Richtlinie verwenden, kann dies dazu führen, dass zukünftige SnapMirror Replikationsvorgänge nicht erfolgreich abgeschlossen werden.

Die Replikation funktioniert möglicherweise nicht, weil beim Erstellen des FlexClone Volumes ein Snapshot gesperrt wird, der von SnapMirror verwendet wird. In diesem Fall stoppt SnapMirror die Replikation auf das Zielvolume, bis das FlexClone Volume zerstört oder vom übergeordneten Volume getrennt wird. Es bestehen zwei Möglichkeiten, dieses Problem zu adressieren:

* Wenn Sie das FlexClone Volume nur vorübergehend benötigen und einen vorübergehenden Stopp der SnapMirror Replikation in Kauf nehmen können, kann das FlexClone Volume erstellt und entweder gelöscht oder, wenn möglich, vom übergeordneten Volume getrennt werden.
+
Die SnapMirror Replikation wird normal fortgesetzt, wenn das FlexClone Volume gelöscht oder von seinem übergeordneten Volume getrennt wird.

* Wenn eine vorübergehende Unterbrechung der SnapMirror Replikation nicht akzeptabel ist, kann ein Snapshot im SnapMirror Quellvolume erstellt werden, und dieser Snapshot kann anschließend zum Erstellen des FlexClone Volumes verwendet werden. (Wenn das FlexClone Volume aus dem Zielvolume erstellt wird, muss gewartet werden, bis dieser Snapshot auf das SnapMirror Zielvolume repliziert wurde.)
+
Diese Methode zum Erstellen eines Snapshots im SnapMirror Quellvolume ermöglicht das Erstellen des Klons, ohne einen Snapshot zu sperren, der von SnapMirror verwendet wird.



Wenn Sie XDP-Beziehungen mit ONTAP 9.12.1 oder höher mit der  `MirrorAllSnapshots` Richtlinie verwenden, sollte dieses Problem nicht auftreten.
