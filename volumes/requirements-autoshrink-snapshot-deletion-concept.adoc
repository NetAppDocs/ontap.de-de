= Voraussetzungen für die Aktivierung von ONTAP Autoshrink und automatischer Snapshot-Löschung
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Die Autoshrink-Funktion kann zusammen mit automatischer Snapshot-Löschung verwendet werden, sofern bestimmte Konfigurationsanforderungen erfüllt sind.

Wenn sowohl die Autoshrink-Funktionalität als auch die automatische Snapshot-Löschung aktiviert werden sollen, muss die Konfiguration die folgenden Anforderungen erfüllen:

* ONTAP muss so konfiguriert sein, dass versucht wird, die Volume-Größe zu erhöhen, bevor Snapshots gelöscht werden (die  `-space-mgmt-try-first` Option muss auf  `volume_grow` gesetzt werden).
* Der Auslöser für die automatische Snapshot-Löschung muss die vollständige Auslastung des Volumes sein (der  `trigger`Parameter muss auf  `volume`gesetzt werden).
