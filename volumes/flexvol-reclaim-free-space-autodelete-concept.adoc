= Erfahren Sie mehr darüber, wie ONTAP FlexVol Volumes mit der automatischen Löschfunktion freien Speicherplatz zurückgewinnen.
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können die Einstellung zur automatischen Löschung eines FlexVol volume aktivieren, um FlexClone Dateien und FlexClone LUNs automatisch zu löschen. Durch das Aktivieren der automatischen Löschung lässt sich eine Zielmenge an freiem Speicherplatz im Volume zurückgewinnen, wenn das Volume nahezu voll ist.

Sie können ein Volume so konfigurieren, dass FlexClone-Dateien und FlexClone-LUNs automatisch gelöscht werden, wenn der freie Speicherplatz im Volume unter einen bestimmten Schwellenwert fällt, und dass das Löschen von Klonen automatisch gestoppt wird, wenn eine Zielmenge an freiem Speicherplatz im Volume wiederhergestellt ist. Obwohl der Schwellenwert, bei dem die automatische Löschung von Klonen beginnt, nicht festgelegt werden kann, lässt sich bestimmen, ob ein Klon zur Löschung geeignet ist, und die Zielmenge an freiem Speicherplatz für ein Volume kann festgelegt werden.

Ein Volume löscht automatisch FlexClone Dateien und FlexClone LUNs, wenn der freie Speicherplatz im Volume unter einen bestimmten Schwellenwert sinkt und wenn _beide_ der folgenden Bedingungen erfüllt sind:

* Die automatische Löschfunktion ist für das Volume aktiviert, das die FlexClone Dateien und FlexClone LUNs enthält.
+
Die automatische Löschfunktion kann für ein FlexVol Volume mithilfe des  `volume snapshot autodelete modify` Befehls aktiviert werden. Der  `-trigger` Parameter muss auf  `volume` oder  `snap_reserve` gesetzt werden, damit ein Volume FlexClone Dateien und FlexClone LUNs automatisch löscht. Weitere Informationen zu  `volume snapshot autodelete modify` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/volume-snapshot-autodelete-modify.html["ONTAP-Befehlsreferenz"^].

* Die automatische Löschfunktion ist für die FlexClone Dateien und FlexClone LUNs aktiviert.
+
Sie können die automatische Löschung für eine FlexClone Datei oder FlexClone LUN mit dem  `file clone create` Befehl mit dem  `-autodelete` Parameter aktivieren. Dadurch ist es möglich, bestimmte FlexClone Dateien und FlexClone LUNs zu erhalten, indem die automatische Löschung für die Klone deaktiviert wird und andere Volume-Einstellungen die Klon-Einstellung nicht überschreiben. Weitere Informationen zu  `file clone create` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/search.html?q=file+clone+create["ONTAP-Befehlsreferenz"^].
