= Ändern Sie den Schwellenwert für die inaktive Datenkomprimierung des ONTAP-Volumes
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Die Häufigkeit, mit der ONTAP einen Kaltdatenscan durchführt, kann durch Anpassen des Kälteschwellenwerts für Volumes mit temperaturabhängiger Speichereffizienz geändert werden.

.Bevor Sie beginnen
Sie müssen Cluster- oder SVM-Administrator sein und die ONTAP CLI auf der Berechtigungsstufe „advanced“ verwenden.

.Über diese Aufgabe
Der Kälteschwellenwert kann zwischen 1 und 60 Tagen liegen. Der Standardwert beträgt 14 Tage.

.Schritte
. Berechtigungsstufe festlegen:
+
`set -privilege advanced`

. Inaktive Datenkomprimierung auf einem Volume ändern:
+
`volume efficiency inactive-data-compression modify -vserver <vserver_name> -volume <volume_name> -threshold-days <integer>`



Weitere Informationen zu  `volume efficiency inactive-data-compression modify` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/volume-efficiency-inactive-data-compression-modify.html#description["ONTAP-Befehlsreferenz"^].
