= Erfahren Sie mehr darüber, wie ONTAP Autoshrink mit dem Löschen von Snapshots interagiert
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Da die Autoshrink-Funktion die Größe eines FlexVol volume verringert, kann sie sich auch darauf auswirken, wann Volume-Snapshots automatisch gelöscht werden.

Die Autoshrink-Funktion interagiert auf folgende Weise mit der automatischen Löschung von Volume-Snapshots:

* Wenn sowohl der  `grow_shrink` automatische Größenanpassung Modus als auch die automatische Snapshot-Löschung aktiviert sind, kann eine Verkleinerung der Volume-Größe eine automatische Snapshot-Löschung auslösen.
+
Dies liegt daran, dass die Snapshot-Reserve auf einem Prozentsatz der Volume-Größe basiert (standardmäßig 5 Prozent), und dieser Prozentsatz nun auf einer kleineren Volume-Größe basiert. Dadurch können Snapshots die Reserve überschreiten und automatisch gelöscht werden.

* Wenn der  `grow_shrink` automatische Größenanpassungsmodus aktiviert ist und ein Snapshot manuell gelöscht wird, kann dies eine automatische Volumenverkleinerung auslösen.
