= Erfahren Sie, wie ONTAP Speicherplatzreservierung mit FlexClone Dateien und FlexClone LUNs funktioniert.
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Bei der Verwendung von FlexClone Dateien und FlexClone LUNs ist es wichtig zu verstehen, wie das Attribut für die Speicherplatzreservierung funktioniert.

Standardmäßig erben FlexClone Dateien und LUNs das Attribut für die Speicherplatzreservierung von der übergeordneten Datei bzw. der übergeordneten LUN. Sie können jedoch FlexClone Dateien und FlexClone LUNs mit deaktivierter Speicherplatzreservierung erstellen, wenn das FlexVol volume nicht genügend Speicherplatz aufweist. Dies ist auch dann möglich, wenn das Attribut im jeweiligen übergeordneten Element aktiviert ist.

Beachten Sie, dass der Klonvorgang fehlschlägt, wenn das FlexVol Volume nicht genügend Speicherplatz enthält, um eine FlexClone Datei oder FlexClone LUN mit der gleichen Speicherplatzreservierung wie die des übergeordneten Volumes zu erstellen.
