= ONTAP-Befehle zur Verwaltung von FlexVol Volumes
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Die ONTAP CLI bietet spezifische Befehle zur Verwaltung von FlexVol Volumes. Je nach Bedarf stehen die folgenden Befehle zur Verwaltung von FlexVol Volumes zur Verfügung:

[cols="2*"]
|===
| Wenn Sie möchten... | Verwenden Sie diesen Befehl... 


 a| 
Ein Volume online schalten
 a| 
`volume online`



 a| 
Die Größe eines Volumens ändern
 a| 
`volume size`



 a| 
Das zugehörige Aggregat eines Volumens ermitteln
 a| 
`volume show`



 a| 
Das zugehörige Aggregat für alle Volumes auf einer Storage Virtual Machine (SVM) ermitteln
 a| 
`volume show -vserver -fields aggregate`



 a| 
Das Format eines Volumes bestimmen
 a| 
`volume show -fields block-type`



 a| 
Ein Volume mithilfe einer Verbindung auf ein anderes Volume einbinden
 a| 
`volume mount`



 a| 
Ein Volume in den eingeschränkten Zustand versetzen
 a| 
`volume restrict`



 a| 
Volume umbenennen
 a| 
`volume rename`



 a| 
Ein Volume offline schalten
 a| 
`volume offline`

|===
Weitere Informationen zu  `volume` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/search.html?q=volume["ONTAP-Befehlsreferenz"^].
