= ONTAP-Befehle zum Konfigurieren des Löschens von FlexClone Dateien
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Wenn Clients FlexClone-Dateien löschen, ohne das NetApp Manageability SDK zu verwenden, können die  `volume file clone deletion` Befehle verwendet werden, um das schnellere Löschen von FlexClone-Dateien aus einem FlexVol Volume zu ermöglichen. Erweiterungen und eine Mindestgröße für FlexClone-Dateien werden verwendet, um ein schnelleres Löschen zu ermöglichen.

Mit den  `volume file clone deletion` Befehlen lässt sich eine Liste unterstützter Erweiterungen und eine Mindestgröße für FlexClone Dateien in einem Volume festlegen. Die schnellere Löschmethode wird nur für FlexClone Dateien angewendet, die die Anforderungen erfüllen. Für FlexClone Dateien, die die Anforderungen nicht erfüllen, wird die langsamere Löschmethode verwendet.

Wenn Clients FlexClone Dateien und FlexClone LUNs von einem Volume mithilfe des NetApp Manageability SDK löschen, gelten die Anforderungen an Erweiterung und Größe nicht, da immer die schnellere Löschmethode verwendet wird.

[cols="2*"]
|===
| Zu... | Verwenden Sie diesen Befehl... 


 a| 
Eine Erweiterung zur Liste der unterstützten Erweiterungen für das Volume hinzufügen
 a| 
`volume file clone deletion add-extension`



 a| 
Die Mindestgröße der FlexClone-Dateien, die mithilfe der schnelleren Löschmethode aus dem Volume gelöscht werden können, ändern
 a| 
`volume file clone deletion modify`



 a| 
Eine Erweiterung aus der Liste der unterstützten Erweiterungen für das Volume entfernen
 a| 
`volume file clone deletion remove-extension`



 a| 
Die unterstützte Liste der Erweiterungen und die Mindestgröße der FlexClone Dateien, die Clients mithilfe der schnelleren Löschmethode aus dem Volume löschen können, sind verfügbar.
 a| 
`volume file clone deletion show`

|===
Weitere Informationen zu den in diesem Verfahren beschriebenen Befehlen befinden sich in der link:https://docs.netapp.com/us-en/ontap-cli/["ONTAP-Befehlsreferenz"^].
