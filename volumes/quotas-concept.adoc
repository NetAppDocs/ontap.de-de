= Vorteile der Verwendung von ONTAP-Kontingenten
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Kontingente ermöglichen die Verwaltung und Überwachung der Ressourcennutzung mit FlexVol volumes.

Die Definition von Quoten bietet mehrere Vorteile. Sie können Standard-, explizite, abgeleitete und Tracking-Quoten verwenden, um die Festplattennutzung optimal zu verwalten.

.Ressourcenverbrauch begrenzen
Es besteht die Möglichkeit, den Speicherplatz oder die Anzahl der von einem Benutzer oder einer Gruppe verwendeten oder in einem qtree enthaltenen Dateien zu begrenzen.

.Ressourcennutzung nachverfolgen
Der Speicherplatzbedarf oder die Anzahl der von einem Benutzer, einer Gruppe oder einem qtree verwendeten Dateien kann ohne Begrenzung erfasst werden.

.Benutzer benachrichtigen
Benachrichtigungen können generiert werden, wenn die Ressourcennutzung bestimmte Schwellenwerte erreicht. Dies warnt Benutzer, wenn ihre Festplatten- oder Dateinutzung zu hoch ist.
