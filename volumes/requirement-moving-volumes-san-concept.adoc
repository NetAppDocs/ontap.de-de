= Anforderungen für das Verschieben von ONTAP SAN-Volumes
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Vor dem Verschieben eines Volumes in einer SAN-Umgebung sind Vorbereitungen erforderlich.

Vor dem Verschieben eines Volumes, das LUNs oder Namespaces enthält, müssen die folgenden Anforderungen erfüllt sein:

* Für Volumes, die eine oder mehrere LUNs enthalten, ist es empfehlenswert, mindestens zwei Pfade pro LUN (LIFs) zu jedem Knoten im Cluster zu haben.
+
Dadurch werden Single Points of Failure eliminiert und das System kann Komponentenausfälle überstehen.

* Für Volumes, die Namespaces enthalten, muss das Cluster mit ONTAP 9.6 oder höher betrieben werden.
+
Das Verschieben von Volumes wird für NVMe Konfigurationen mit ONTAP 9.5 nicht unterstützt.
