= ONTAP Kontrollpunkte und Effizienzvorgänge
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Checkpoints werden intern verwendet, um den Ausführungsprozess einer Effizienzoperation zu protokollieren. Wenn eine Effizienzoperation aus irgendeinem Grund gestoppt wird (wie etwa Systemstillstand, Systemstörung, Neustart oder weil die letzte Effizienzoperation fehlgeschlagen oder gestoppt wurde) und Checkpoint-Daten vorhanden sind, kann die Effizienzoperation vom letzten Checkpoint-Datei aus fortgesetzt werden.

Ein Kontrollpunkt wird erstellt:

* in jeder Phase oder Teilphase der Operation
* wenn Sie den  `sis stop` Befehl ausführen
* wenn die Dauer abläuft


Weitere Informationen zu den in diesem Verfahren beschriebenen Befehlen befinden sich in der link:https://docs.netapp.com/us-en/ontap-cli/["ONTAP-Befehlsreferenz"^].
