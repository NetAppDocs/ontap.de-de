= ONTAP-Befehle zum Aktivieren und Ändern von Kontingenten
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
`volume quota`Befehle ermöglichen es, den Status von Kontingenten zu ändern und die Protokollierung von Kontingentmeldungen zu konfigurieren. Je nach Anforderung stehen die folgenden Befehle zur Verfügung, um Kontingente zu aktivieren und zu ändern:

[cols="2*"]
|===
| Wenn Sie möchten... | Verwenden Sie diesen Befehl... 


 a| 
Kontingente aktivieren (auch _initialisieren_ genannt)
 a| 
`volume quota on`



 a| 
Bestehende Quoten anpassen
 a| 
`volume quota resize`



 a| 
Quoten deaktivieren
 a| 
`volume quota off`



 a| 
Die Protokollierung von Kontingenten ändern, Kontingente aktivieren, deaktivieren oder die Größe vorhandener Kontingente anpassen
 a| 
`volume quota modify`

|===
Weitere Informationen zu den in diesem Verfahren beschriebenen Befehlen befinden sich in der link:https://docs.netapp.com/us-en/ontap-cli/["ONTAP-Befehlsreferenz"^].
