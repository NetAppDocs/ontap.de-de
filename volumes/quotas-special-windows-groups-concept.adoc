= Erfahren Sie, wie ONTAP Kontingente mit speziellen Windows-Gruppen funktionieren
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Es gibt mehrere spezielle Windows-Gruppen, die Quoten anders verarbeiten als andere Windows-Gruppen. Es ist wichtig zu verstehen, wie Quoten für diese speziellen Gruppen angewendet werden.

[NOTE]
====
ONTAP unterstützt keine Gruppenkontingente basierend auf Windows-Gruppen-IDs. Wenn eine Windows-Gruppen-ID als Kontingentziel angegeben wird, gilt das Kontingent als Benutzerkontingent.

====
.Alle
Wenn das Quotenziel die Gruppe „Everyone“ ist, wird eine Datei, deren ACL angibt, dass der Eigentümer „Everyone“ ist, unter der SID für „Everyone“ gezählt.

.BUILTIN\Administratoren
Wenn das Kontingentziel die Gruppe BUILTIN\Administrators ist, wird der Eintrag als Benutzerkontingent betrachtet und dient ausschließlich der Nachverfolgung. Für BUILTIN\Administrators können keine Einschränkungen festgelegt werden. Wenn ein Mitglied von BUILTIN\Administrators eine Datei erstellt, gehört die Datei BUILTIN\Administrators und wird unter der SID von BUILTIN\Administrators (nicht der persönlichen SID des Benutzers) gezählt.
