= Informationen im ONTAP System Manager suchen, filtern und sortieren
:hardbreaks:
:toclevels: 1
:allow-uri-read: 
:toclevels: 1
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
Im System Manager können Sie nach verschiedenen Aktionen, Objekten und Informationsthemen suchen. Sie können auch Tabellendaten nach bestimmten Einträgen durchsuchen.

System Manager bietet zwei Arten der Suche:

* <<Globale Suche>>
+
Wenn Sie im Feld oben auf jeder Seite ein Suchargument eingeben, durchsucht System Manager die gesamte Benutzeroberfläche nach passenden Ergebnissen. Anschließend können Sie die Ergebnisse sortieren und filtern.

+
Ab ONTAP 9.12.1 bietet der System Manager auch Suchergebnisse von der NetApp Support Site, um Links zu relevanten Supportinformationen bereitzustellen.

* <<Tabellenraster-Suche>>
+
Ab ONTAP 9.8 durchsucht der System Manager bei Eingabe eines Sucharguments im Feld am oberen Rand eines Tabellenrasters nur die Spalten und Zeilen dieser Tabelle nach Übereinstimmungen.





[[global-searching]]
== Globale Suche

Am oberen Rand jeder Seite im System Manager steht ein globales Suchfeld zur Verfügung, mit dem verschiedene Objekte und Aktionen in der Benutzeroberfläche durchsucht werden können. Beispielsweise ist eine Suche nach Objekten anhand ihres Namens, nach Seiten in der Navigationsspalte (auf der linken Seite), nach verschiedenen Aktionen wie „Add Volume“ oder „Add License“ sowie nach Links zu externen Hilfethemen möglich. Die Ergebnisse können außerdem gefiltert und sortiert werden.


NOTE: Für bessere Ergebnisse werden Such-, Filter- und Sortiervorgänge eine Minute nach der Protokollierung und fünf Minuten nach dem Erstellen, Ändern oder Löschen eines Objekts durchgeführt.



[[getting-search-results]]
=== Suchergebnisse abrufen

Die Suche unterscheidet nicht zwischen Groß- und Kleinschreibung. Es können verschiedene Textzeichenfolgen eingegeben werden, um die gewünschte Seite, Aktion oder Informationsthemen zu finden. Es werden bis zu 20 Ergebnisse aufgelistet. Wenn mehr Ergebnisse gefunden werden, kann auf *Mehr anzeigen* geklickt werden, um alle Ergebnisse anzuzeigen. Die folgenden Beispiele beschreiben typische Suchvorgänge:

|===


| Suchtyp | Beispiel für Suchzeichenfolge | Beispiel-Suchergebnisse 


| Nach Objektnamen | vol_ | vol_lun_dest auf Storage-VM: svm0 (Volume) /vol/vol…est1/lun auf Storage-VM: svm0 (LUN) svm0:vol_lun_dest1 Rolle: Destination (Relationship) 


| Nach Standort in der Schnittstelle | Volumen | Volume hinzufügen (Aktion) Schutz – Übersicht (Seite) Gelöschtes Volume wiederherstellen (Hilfe) 


| Nach Aktionen | hinzufügen | Volume hinzufügen (Aktion) Netzwerk – Übersicht (Seite) Volumes und LUNs erweitern (Hilfe) 


| Nach Hilfeinhalten | SAN | Storage – Übersicht (Seite) SAN-Übersicht (Hilfe) SAN-Speicher für Datenbanken bereitstellen (Hilfe) 
|===


[[global-search-results-from-netapp-support-site]]
=== Globale Suchergebnisse von der NetApp Support-Website

Ab ONTAP 9.12.1 wird für Benutzer, die bei Active IQ Digital Advisor (auch bekannt als Digital Advisor) registriert sind, im System Manager eine weitere Ergebnisspalte angezeigt, die Links zu NetApp Support Site-Informationen einschließlich System Manager Produktinformationen bereitstellt.

Die Suchergebnisse enthalten folgende Informationen:

* *Titel* der Information, die ein Link zum Dokument im HTML-, PDF-, EPUB- oder einem anderen Format ist.
* *Inhaltstyp*, der angibt, ob es sich um ein Produktdokumentationsthema, einen KnowledgeBase-Artikel oder eine andere Art von Information handelt.
* *Kurzbeschreibung* des Inhalts.
* *Erstellt*-Datum bei der ersten Veröffentlichung.
* *Aktualisiert* Datum, an dem es zuletzt aktualisiert wurde.


Sie können die folgenden Aktionen ausführen:

[cols="35,65"]
|===


| Aktion | Ergebnis 


 a| 
*ONTAP System Manager* auswählen und anschließend Text in das Suchfeld eingeben.
 a| 
Die Suchergebnisse enthalten NetApp Support Site-Informationen zum System Manager.



 a| 
*Alle Produkte* auswählen und anschließend Text in das Suchfeld eingeben.
 a| 
Die Suchergebnisse enthalten NetApp Support Site-Informationen für alle NetApp Produkte, nicht nur für System Manager.



 a| 
Auf ein Suchergebnis klicken.
 a| 
Die Informationen von der NetApp Support Site werden in einem separaten Browserfenster oder Tab angezeigt.



 a| 
*Weitere Ergebnisse anzeigen* auswählen.
 a| 
Wenn mehr als zehn Ergebnisse angezeigt werden, kann nach dem zehnten Ergebnis auf *Weitere Ergebnisse anzeigen* geklickt werden, um weitere Ergebnisse anzuzeigen. Jedes Mal, wenn auf *Weitere Ergebnisse anzeigen* geklickt wird, werden, sofern verfügbar, weitere zehn Ergebnisse angezeigt.



 a| 
Kopieren Sie den Link.
 a| 
Der Link wird in die Zwischenablage kopiert. Der Link kann in eine Datei oder in ein Browserfenster eingefügt werden.



 a| 
Klicken Sie image:icon-pin-blue.png["Stecknadelsymbol"].
 a| 
Das Panel, in dem die Ergebnisse angezeigt werden, ist fixiert, sodass es angezeigt bleibt, wenn Sie in einem anderen Panel arbeiten.



 a| 
Klicken Sie image:icon-x-close.png["Schließen-Symbol"].
 a| 
Das Ergebnis-Panel ist nicht mehr angeheftet und wird geschlossen.

|===


[[filtering-search-results]]
=== Suchergebnisse filtern

Sie können die Ergebnisse mithilfe von Filtern eingrenzen, wie in den folgenden Beispielen gezeigt:

|===


| Filter | Syntax | Beispiel für Suchzeichenfolge 


| Nach Objekttyp | <type>:<objectName> | Volume:vol_2 


| Nach Objektgröße | <type><size-symbol><number><units> | LUNs<500 MB 


| Durch defekte Festplatten | "defekte Festplatte" oder "beeinträchtigte Festplatte" | fehlerhafte Festplatte 


| Nach Netzwerkschnittstelle | <IP address> | 172.22.108.21 
|===


[[sorting-search-results]]
=== Suchergebnisse sortieren

Wenn alle Suchergebnisse angezeigt werden, sind sie alphabetisch sortiert. Die Ergebnisse können sortiert werden, indem Sie auf image:icon_filter.png["Filtersymbol"] klicken und auswählen, wie die Ergebnisse sortiert werden sollen.



[[table-grid-searching]]
== Tabellenraster-Suche

Ab ONTAP 9.8 erscheint immer dann, wenn der System Manager Informationen in einem Tabellen- oder Rasterformat anzeigt, eine Suchschaltfläche am oberen Rand der Tabelle.

Wenn Sie auf *Suchen* klicken, erscheint ein Textfeld, in das Sie ein Suchargument eingeben können. System Manager durchsucht die gesamte Tabelle und zeigt nur die Zeilen an, die Text enthalten, der Ihrem Suchargument entspricht.

Sie können ein Sternchen ( * ) als „Wildcard“-Zeichen als Ersatz für Zeichen verwenden. Beispielsweise kann die Suche nach `vol*` Zeilen liefern, die Folgendes enthalten:

* vol_122_D9
* vol_lun_dest1
* vol2866
* volspec1
* volum_dest_765
* Volumen
* volume_new4
* volume9987
