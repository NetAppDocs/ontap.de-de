= Die Volumenverschlüsselungslizenz auf einem ONTAP Cluster installieren
:allow-uri-read: 


[role="lead"]
Eine VE-Lizenz berechtigt Sie zur Nutzung der Funktion auf allen Knoten im Cluster. Diese Lizenz ist erforderlich, bevor Daten mit NVE verschlüsselt werden können. Sie ist im Lieferumfang von link:../system-admin/manage-licenses-concept.html#licenses-included-with-ontap-one["ONTAP One"] enthalten.

Vor ONTAP One war die VE-Lizenz im Verschlüsselungspaket enthalten. Das Verschlüsselungspaket wird nicht mehr angeboten, ist aber weiterhin gültig. Obwohl es derzeit nicht erforderlich ist, können bestehende Kunden sich dafür entscheiden, link:../system-admin/download-nlf-task.html["Upgrade auf ONTAP One"].

.Bevor Sie beginnen
* Sie müssen Cluster-Administrator sein, um diese Aufgabe ausführen zu können.
* Sie müssen den VE Lizenzschlüssel von Ihrem Vertriebsmitarbeiter erhalten haben oder ONTAP One installiert haben.


.Schritte
. link:../system-admin/manage-license-task.html["Es sollte überprüft werden, ob die VE Lizenz installiert ist."].
+
Der Name des VE Lizenzpakets ist  `VE`.

. Wenn die Lizenz nicht installiert ist, link:../system-admin/install-license-task.html["System Manager oder die ONTAP Befehlszeilenschnittstelle (CLI) kann zur Installation verwendet werden."].
