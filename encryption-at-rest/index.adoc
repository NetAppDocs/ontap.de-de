= Informationen zur ONTAP Daten im Ruhezustand Verschlüsselung
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
NetApp bietet sowohl software- als auch hardwarebasierte Verschlüsselungstechnologien, um sicherzustellen, dass Daten im Ruhezustand nicht gelesen werden können, wenn das Speichermedium umfunktioniert, zurückgegeben, verlegt oder gestohlen wird.

* Die softwarebasierte Verschlüsselung mittels NetApp Volume Encryption (NVE) unterstützt die Datenverschlüsselung jeweils eines Volumens
* Die hardwarebasierte Verschlüsselung mittels NetApp Storage Encryption (NSE) unterstützt die vollständige Festplattenverschlüsselung (FDE) der Daten während des Schreibvorgangs.
