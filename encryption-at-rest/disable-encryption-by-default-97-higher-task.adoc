= ONTAP Verschlüsselung standardmäßig deaktiviert
:allow-uri-read: 


[role="lead"]
Ab ONTAP 9.7 ist die Aggregat- und Volumenverschlüsselung standardmäßig aktiviert, sofern Sie über eine Volumenverschlüsselungslizenz (VE) verfügen und einen integrierten oder externen Schlüsselmanager verwenden. Falls erforderlich, kann die Verschlüsselung standardmäßig für den gesamten Cluster deaktiviert werden.

.Bevor Sie beginnen
Sie müssen Cluster-Administrator sein, um diese Aufgabe ausführen zu können, oder SVM-Administrator, dem der Cluster-Administrator die entsprechende Berechtigung delegiert hat.

.Schritt
. Um die Verschlüsselung für den gesamten Cluster in ONTAP 9.7 oder höher standardmäßig zu deaktivieren, ist folgender Befehl auszuführen:
+
`options -option-name encryption.data_at_rest_encryption.disable_by_default -option-value on`
