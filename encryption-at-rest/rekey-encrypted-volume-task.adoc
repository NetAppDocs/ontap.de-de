= Den Verschlüsselungsschlüssel für ein Volume mit dem Befehl volume encryption rekey start in ONTAP ändern
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Es gilt als bewährte Sicherheitsmaßnahme, den Verschlüsselungsschlüssel eines Volumes regelmäßig zu ändern. Ab ONTAP 9.3 kann der Verschlüsselungsschlüssel mit dem  `volume encryption rekey start` Befehl geändert werden.

.Über diese Aufgabe
Sobald Sie einen Schlüsselerneuerungsvorgang starten, muss dieser abgeschlossen werden. Eine Rückkehr zum alten Schlüssel ist nicht möglich. Wenn während des Vorgangs Leistungsprobleme auftreten, kann der Vorgang mit dem  `volume encryption rekey pause` Befehl angehalten und mit dem  `volume encryption rekey resume` Befehl fortgesetzt werden.

Bis zum Abschluss der Neuverschlüsselung verfügt das Volume über zwei Schlüssel. Neue Schreibvorgänge und die zugehörigen Lesevorgänge verwenden den neuen Schlüssel. Andernfalls verwenden Lesevorgänge den alten Schlüssel.

[NOTE]
====
Es ist nicht möglich, mit  `volume encryption rekey start` ein SnapLock Volume neu zu verschlüsseln.

====
.Schritte
. Ändern eines Verschlüsselungsschlüssels:
+
`volume encryption rekey start -vserver SVM_name -volume volume_name`

+
Der folgende Befehl ändert den Verschlüsselungsschlüssel für  `vol1` auf SVM ``vs1``:

+
[listing]
----
cluster1::> volume encryption rekey start -vserver vs1 -volume vol1
----
. Status des Schlüsselerneuerungsvorgangs überprüfen:
+
`volume encryption rekey show`

+
Weitere Informationen zu  `volume encryption rekey show` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/volume-encryption-rekey-show.html["ONTAP-Befehlsreferenz"^].

+
Der folgende Befehl zeigt den Status des Rekey-Vorgangs an:

+
[listing]
----
cluster1::> volume encryption rekey show

Vserver   Volume   Start Time           Status
-------   ------   ------------------   ---------------------------
vs1       vol1     9/18/2017 17:51:41   Phase 2 of 2 is in progress.
----
. Nach Abschluss der Schlüsselerneuerung kann überprüft werden, ob das Volume für die Verschlüsselung aktiviert ist:
+
`volume show -is-encrypted true`

+
Weitere Informationen zu  `volume show` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/volume-show.html["ONTAP-Befehlsreferenz"^].

+
Der folgende Befehl zeigt die verschlüsselten Volumes auf  `cluster1` an:

+
[listing]
----
cluster1::> volume show -is-encrypted true

Vserver  Volume  Aggregate  State  Type  Size  Available  Used
-------  ------  ---------  -----  ----  -----  --------- ----
vs1      vol1    aggr2     online    RW  200GB    160.0GB  20%
----
