= ONTAP NetApp Volume Encryption-Workflow
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie müssen die Schlüsselverwaltungsdienste konfigurieren, bevor Sie die Volume-Verschlüsselung aktivieren können. Die Verschlüsselung kann auf einem neuen Volume oder auf einem vorhandenen Volume aktiviert werden.

image:nve-workflow.gif["NetApp Volume Encryption-Workflow. Dies zeigt die Schritte zur Aktivierung der Verschlüsselung auf einem Volume."]

link:../encryption-at-rest/install-license-task.html["Die VE-Lizenz muss installiert werden."] und konfigurieren Sie die Schlüsselverwaltungsdienste, bevor Sie Daten mit NVE verschlüsseln können. Vor der Installation der Lizenz sollte link:cluster-version-support-nve-task.html["feststellen, ob Ihre ONTAP Version NVE unterstützt"] beachtet werden.
