= Verschieben eines verschlüsselten Volumes in ONTAP
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können den  `volume move start` Befehl verwenden, um ein verschlüsseltes Volume zu verschieben. Das verschobene Volume kann sich auf demselben Aggregat oder auf einem anderen Aggregat befinden.

.Über diese Aufgabe
Der Vorgang schlägt fehl, wenn der Zielknoten oder das Zielvolume die Volume-Verschlüsselung nicht unterstützt.

Die  `-encrypt-destination`Option für `volume move start` ist bei verschlüsselten Volumes standardmäßig aktiviert. Die Anforderung, explizit anzugeben, dass das Zielvolume nicht verschlüsselt werden soll, stellt sicher, dass die Daten auf dem Volume nicht versehentlich entschlüsselt werden.

.Bevor Sie beginnen
Sie müssen Cluster-Administrator sein, um diese Aufgabe ausführen zu können.

.Schritte
. Ein vorhandenes verschlüsseltes Volume kann verschoben werden, wobei die Daten auf dem Volume verschlüsselt bleiben:
+
`volume move start -vserver _SVM_name_ -volume _volume_name_ -destination-aggregate _aggregate_name_`

+
Weitere Informationen zu  `volume move start` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/volume-move-start.html["ONTAP-Befehlsreferenz"^].

+
Der folgende Befehl verschiebt ein vorhandenes Volume mit dem Namen `vol1` in das Zielaggregat `aggr3` und lässt die Daten auf dem Volume verschlüsselt:

+
[listing]
----
cluster1::> volume move start -vserver vs1 -volume vol1 -destination-aggregate aggr3
----
. Es sollte überprüft werden, ob das Volume für die Verschlüsselung aktiviert ist:
+
`volume show -is-encrypted true`

+
Weitere Informationen zu  `volume show` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/volume-show.html["ONTAP-Befehlsreferenz"^].

+
Der folgende Befehl zeigt die verschlüsselten Volumes auf  `cluster1` an:

+
[listing]
----
cluster1::> volume show -is-encrypted true

Vserver  Volume  Aggregate  State  Type  Size  Available  Used
-------  ------  ---------  -----  ----  -----  --------- ----
vs1      vol1    aggr3     online    RW  200GB    160.0GB  20%
----
