= Informationen zur Verschlüsselung von ONTAP Volume-Daten mit NVE
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Ab ONTAP 9.7 ist die Aggregat- und Volumenverschlüsselung standardmäßig aktiviert, wenn die VE-Lizenz sowie die integrierte oder externe Schlüsselverwaltung vorhanden sind. Für ONTAP 9.6 und frühere Versionen kann die Verschlüsselung auf einem neuen oder einem bestehenden Volume aktiviert werden. Die VE-Lizenz muss installiert und die Schlüsselverwaltung aktiviert sein, bevor die Volumenverschlüsselung aktiviert werden kann. NVE ist FIPS-140-2 Level 1-konform.
