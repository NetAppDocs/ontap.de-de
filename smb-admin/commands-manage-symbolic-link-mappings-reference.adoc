= ONTAP Befehle zur Verwaltung von SMB Symbol-Link-Zuordnungen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Es gibt spezielle ONTAP Befehle zur Verwaltung von Symbol-Link-Zuordnungen.

|===
| Wenn Sie möchten... | Verwenden Sie diesen Befehl... 


 a| 
Erstellen einer Symbol-Link-Zuordnung
 a| 
`vserver cifs symlink create`



 a| 
Informationen zu Symbol-Link-Zuordnungen anzeigen
 a| 
`vserver cifs symlink show`



 a| 
Eine Symbol-Link-Zuordnung ändern
 a| 
`vserver cifs symlink modify`



 a| 
Symbol-Link-Zuordnung löschen
 a| 
`vserver cifs symlink delete`

|===
Weitere Informationen zu  `vserver cifs symlink` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/search.html?q=vserver+cifs+symlink["ONTAP-Befehlsreferenz"^].
