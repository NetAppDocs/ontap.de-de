= Informationen zur Anzeige von ONTAP SMB-Dateisicherheits- und Überwachungsrichtlinien
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Informationen zur Dateisicherheit von Dateien und Verzeichnissen in Volumes auf Storage Virtual Machines (SVMs) können angezeigt werden. Informationen zu Überwachungsrichtlinien auf FlexVol volumes können angezeigt werden. Wenn konfiguriert, können Informationen zu Storage-Level Access Guard und Dynamic Access Control Sicherheitseinstellungen auf FlexVol volumes angezeigt werden.



[[displaying-information-about-file-security]]
== Anzeige von Informationen zur Dateisicherheit

Es lassen sich Informationen über die auf Daten in Volumes und Qtrees (für FlexVol volumes) angewendete Dateisicherheit mit den folgenden Sicherheitsstilen anzeigen:

* NTFS
* UNIX
* Gemischt




[[displaying-information-about-audit-policies]]
== Anzeige von Informationen über Prüfungsrichtlinien

Informationen zu Überwachungsrichtlinien für die Überwachung von Zugriffsereignissen auf FlexVol volumes über die folgenden NAS-Protokolle können angezeigt werden:

* SMB (alle Versionen)
* NFSv4.x




[[displaying-information-about-storage-level-access-guard-slag-security]]
== Anzeige von Informationen zur Storage-Level Access Guard (SLAG) Sicherheit

Die Storage-Level Access Guard-Sicherheit kann auf FlexVol volumes und qtree-Objekte mit den folgenden Sicherheitsstilen angewendet werden:

* NTFS
* Gemischt
* UNIX (wenn auf der SVM, die das Volume enthält, ein CIFS-Server konfiguriert ist)




[[displaying-information-about-dynamic-access-control-dac-security]]
== Anzeigen von Informationen zur Dynamic Access Control (DAC) Sicherheit

Die Dynamic Access Control-Sicherheit kann auf ein Objekt innerhalb eines FlexVol volume mit den folgenden Sicherheitsstilen angewendet werden:

* NTFS
* Gemischt (wenn das Objekt über effektive NTFS-Sicherheit verfügt)
