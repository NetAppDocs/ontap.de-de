= ONTAP Befehle zur Verwaltung von Passwörtern für SMB-Server-Computerkonten
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie müssen die Befehle zum Ändern, Zurücksetzen und Deaktivieren von Passwörtern sowie zum Konfigurieren automatischer Aktualisierungszeitpläne kennen. Es besteht außerdem die Möglichkeit, einen Zeitplan auf dem SMB Server zu konfigurieren, damit dieser automatisch aktualisiert wird.

|===
| Wenn Sie möchten... | Verwenden Sie diesen Befehl... 


 a| 
Das Domänenkontokennwort wird geändert, wenn ONTAP mit AD-Diensten synchronisiert ist.
 a| 
`vserver cifs domain password change`



 a| 
Das Domänenkontokennwort wird zurückgesetzt, wenn ONTAP nicht mit den AD-Diensten synchronisiert ist.
 a| 
`vserver cifs domain password reset`



 a| 
SMB-Server für automatische Änderungen von Computerkontokennwörtern konfigurieren
 a| 
`vserver cifs domain password schedule modify -vserver vserver_name -is-schedule-enabled true`



 a| 
Automatische Änderung von Computerkontokennwörtern auf SMB-Servern deaktivieren
 a| 
`vserver cifs domain password schedule modify -vserver vs1 -is-schedule-enabled false`

|===
Weitere Informationen zu  `vserver cifs domain password` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/search.html?q=vserver+cifs+domain+password["ONTAP-Befehlsreferenz"^].
