= Empfehlungen zu ONTAP SMB BranchCache
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Bevor Sie BranchCache konfigurieren, sollten bestimmte Empfehlungen berücksichtigt werden, wenn entschieden wird, bei welchen SMB-Freigaben das BranchCache-Caching aktiviert werden soll.

Die folgenden Empfehlungen sind bei der Entscheidung zu berücksichtigen, welchen Betriebsmodus Sie verwenden und auf welchen SMB-Freigaben Sie BranchCache aktivieren:

* Die Vorteile von BranchCache sind reduziert, wenn sich die remote zwischengespeicherten Daten häufig ändern.
* BranchCache-Dienste sind vorteilhaft für Freigaben, die Dateiinhalte enthalten, die von mehreren Remote-Office-Clients wiederverwendet werden, oder für Dateiinhalte, auf die ein einzelner Remote-Benutzer wiederholt zugreift.
* Das Aktivieren des Cachings für schreibgeschützte Inhalte wie Daten in Snapshots und SnapMirror-Zielen kann in Betracht gezogen werden.
