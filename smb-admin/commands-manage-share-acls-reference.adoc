= ONTAP Befehle zur Verwaltung von SMB-Freigabezugriffskontrolllisten
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie müssen die Befehle zur Verwaltung von SMB-Zugriffskontrolllisten (ACLs) kennen, einschließlich deren Erstellung, Anzeige, Änderung und Löschung.

|===
| Wenn Sie möchten... | Verwenden Sie diesen Befehl... 


 a| 
Eine neue ACL erstellen
 a| 
`vserver cifs share access-control create`



 a| 
ACLs anzeigen
 a| 
`vserver cifs share access-control show`



 a| 
Eine ACL ändern
 a| 
`vserver cifs share access-control modify`



 a| 
Eine ACL löschen
 a| 
`vserver cifs share access-control delete`

|===
