= Informationen zur Unterstützung von ONTAP SMB-Servern
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können SMB-Server auf Storage Virtual Machines (SVMs) aktivieren und konfigurieren, um SMB-Clients den Zugriff auf Dateien in Ihrem Cluster zu ermöglichen.

* Jede Daten-SVM im Cluster kann genau einer Active Directory Domäne zugeordnet werden.
* Daten-SVMs müssen nicht an dieselbe Domäne gebunden sein.
* Mehrere SVMs können an denselben Bereich gebunden werden.


Sie müssen die SVMs und LIFs, die Sie zur Datenbereitstellung verwenden, konfigurieren, bevor Sie einen SMB-Server erstellen können. Wenn Ihr Datennetzwerk nicht flach ist, müssen möglicherweise auch IPspaces, Broadcast-Domänen und Subnetze konfiguriert werden.
