= SMB-Sicherheitsstile auf ONTAP FlexVol Volumes konfigurieren
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Der Sicherheitsstil des FlexVol volume wird konfiguriert, um die Art der Berechtigungen festzulegen, die für Daten auf FlexVol volumes der Storage Virtual Machine (SVM) verwendet werden.

.Schritte
. Führen Sie eine der folgenden Aktionen aus:
+
|===


| Wenn das FlexVol Volume... | Verwenden Sie den Befehl... 


 a| 
Existiert noch nicht
 a| 
`volume create` und den  `-security-style` Parameter zur Angabe des Sicherheitsstils einschließen.



 a| 
Existiert bereits
 a| 
`volume modify` und den  `-security-style` Parameter zur Angabe des Sicherheitsstils einschließen.

|===
+
Die möglichen Optionen für den FlexVol volume-Sicherheitsstil sind `unix`, `ntfs`, oder `mixed`.

+
Wenn beim Erstellen eines FlexVol volume kein Sicherheitsstil angegeben wird, übernimmt das Volume den Sicherheitsstil des Stammvolumes.

+
Weitere Informationen zu den  `volume create` oder  `volume modify` Befehlen finden sich unter link:../volumes/index.html["Logische Speicherverwaltung"].

. Um die Konfiguration einschließlich des Sicherheitsstils des von Ihnen erstellten FlexVol Volumes anzuzeigen, ist folgender Befehl einzugeben:
+
`volume show -volume volume_name -instance`
