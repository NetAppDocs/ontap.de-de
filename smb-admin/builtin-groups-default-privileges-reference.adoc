= Vordefinierte BUILTIN Gruppen und standardmäßige ONTAP SMB-Berechtigungen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können einem lokalen Benutzer oder Domänenbenutzer die Mitgliedschaft in einer vordefinierten Gruppe von BUILTIN-Gruppen zuweisen, die von ONTAP bereitgestellt werden. Vordefinierte Gruppen haben vordefinierte Berechtigungen zugewiesen.

Die folgende Tabelle beschreibt die vordefinierten Gruppen:

|===
| Vordefinierte BUILTIN Gruppe | Standardberechtigungen 


 a| 
``BUILTIN\Administrators``RID 544

Wenn das lokale `Administrator` Konto mit einer RID von 500 erstmals erstellt wird, wird es automatisch Mitglied dieser Gruppe. Wenn die Storage Virtual Machine (SVM) einer Domäne beitritt, wird die `domain\Domain Admins` Gruppe zur Gruppe hinzugefügt. Wenn die SVM die Domäne verlässt, wird die `domain\Domain Admins` Gruppe aus der Gruppe entfernt.
 a| 
* `SeBackupPrivilege`
* `SeRestorePrivilege`
* `SeSecurityPrivilege`
* `SeTakeOwnershipPrivilege`
* `SeChangeNotifyPrivilege`




 a| 
``BUILTIN\Power Users``RID 547

Diese Gruppe hat bei ihrer Erstellung keine Mitglieder. Mitglieder dieser Gruppe weisen die folgenden Merkmale auf:

* Kann lokale Benutzer und Gruppen erstellen und verwalten.
* Kann sich selbst oder andere Objekte nicht zur `BUILTIN\Administrators` Gruppe hinzufügen.

 a| 
`SeChangeNotifyPrivilege`



 a| 
``BUILTIN\Backup Operators``RID 551

Diese Gruppe hat bei ihrer Erstellung keine Mitglieder. Mitglieder dieser Gruppe können die Lese- und Schreibberechtigungen für Dateien oder Ordner überschreiben, wenn sie mit Backup-Absicht geöffnet werden.
 a| 
* `SeBackupPrivilege`
* `SeRestorePrivilege`
* `SeChangeNotifyPrivilege`




 a| 
``BUILTIN\Users``RID 545

Diese Gruppe hat nach ihrer Erstellung keine Mitglieder (abgesehen von der impliziten `Authenticated Users` Sondergruppe). Wenn die SVM einer Domäne beitritt, wird die `domain\Domain Users` Gruppe zu dieser Gruppe hinzugefügt. Wenn die SVM die Domäne verlässt, wird die `domain\Domain Users` Gruppe aus dieser Gruppe entfernt.
 a| 
`SeChangeNotifyPrivilege`



 a| 
``Everyone``SID S-1-1-0

Diese Gruppe umfasst alle Benutzer, einschließlich Gäste (jedoch keine anonymen Benutzer). Es handelt sich um eine implizite Gruppe mit impliziter Mitgliedschaft.
 a| 
`SeChangeNotifyPrivilege`

|===
