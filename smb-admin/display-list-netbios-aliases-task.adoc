= NetBIOS-Aliasliste für ONTAP SMB-Server
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können die Liste der NetBIOS-Aliase anzeigen. Dies ist hilfreich, wenn die Liste der Namen ermittelt werden soll, über die SMB-Clients Verbindungen zum CIFS-Server herstellen können.

.Schritt
. Führen Sie eine der folgenden Aktionen aus:
+
|===
| Wenn Informationen über ... angezeigt werden sollen | Eingeben... 


 a| 
NetBIOS-Aliase eines CIFS-Servers
 a| 
`vserver cifs show -display-netbios-aliases`



 a| 
Die Liste der NetBIOS-Aliase als Teil der detaillierten CIFS-Server-Informationen
 a| 
`vserver cifs show -instance`

|===
+
Das folgende Beispiel zeigt Informationen über die NetBIOS-Aliase eines CIFS-Servers:

+
`vserver cifs show -display-netbios-aliases`

+
[listing]
----
Vserver: vs1

         Server Name: CIFS_SERVER
         NetBIOS Aliases: ALIAS_1, ALIAS_2, ALIAS_3
----
+
Das folgende Beispiel zeigt die Liste der NetBIOS-Aliase als Teil der detaillierten CIFS-Server-Informationen:

+
`vserver cifs show -instance`

+
[listing]
----

                                          Vserver: vs1
                         CIFS Server NetBIOS Name: CIFS_SERVER
                    NetBIOS Domain/Workgroup Name: EXAMPLE
                      Fully Qualified Domain Name: EXAMPLE.COM
Default Site Used by LIFs Without Site Membership:
                             Authentication Style: domain
                CIFS Server Administrative Status: up
                          CIFS Server Description:
                          List of NetBIOS Aliases: ALIAS_1, ALIAS_2, ALIAS_3
----
+
Weitere Informationen zu  `vserver cifs show` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/vserver-cifs-show.html["ONTAP-Befehlsreferenz"^].
