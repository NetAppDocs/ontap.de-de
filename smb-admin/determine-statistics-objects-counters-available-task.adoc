= Festgestellt werden kann, welche Statistiken, Objekte und Zähler auf ONTAP SMB-Servern verfügbar sind.
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Bevor Sie Informationen über CIFS, SMB, Auditing und BranchCache Hash-Statistiken sowie zur Überwachung der Leistung erhalten können, muss bekannt sein, welche Objekte und Zähler verfügbar sind, aus denen Daten gewonnen werden können.

.Schritte
. Die Berechtigungsstufe wird auf „Erweitert“ gesetzt: `set -privilege advanced`
. Führen Sie eine der folgenden Aktionen aus:
+
|===
| Wenn Sie feststellen möchten... | Eingeben... 


 a| 
Welche Objekte sind verfügbar
 a| 
`statistics catalog object show`



 a| 
Spezifische verfügbare Objekte
 a| 
`statistics catalog object show -object object_name`



 a| 
Welche Zähler stehen zur Verfügung?
 a| 
`statistics catalog counter show -object object_name`

|===
+
Weitere Informationen zu  `statistics catalog object show`, einschließlich der verfügbaren Objekte und Zähler, finden Sie in der link:https://docs.netapp.com/us-en/ontap-cli/statistics-catalog-object-show.html["ONTAP-Befehlsreferenz"^].

. Zurück zur Administratorrechteebene: `set -privilege admin`


.Beispiele
Der folgende Befehl zeigt Beschreibungen ausgewählter Statistikobjekte im Zusammenhang mit dem CIFS- und SMB-Dateizugriff im Cluster auf der Ebene der erweiterten Berechtigungen an:

[listing]
----
cluster1::> set -privilege advanced

Warning: These advanced commands are potentially dangerous; use them only when directed to do so by support personnel.
Do you want to continue? {y|n}: y

cluster1::*> statistics catalog object show -object audit
    audit_ng                    CM object for exporting audit_ng performance counters

cluster1::*> statistics catalog object show -object cifs
    cifs                        The CIFS object reports activity of the
                                Common Internet File System protocol
                                ...

cluster1::*> statistics catalog object show -object nblade_cifs
    nblade_cifs                 The Common Internet File System (CIFS)
                                protocol is an implementation of the Server
                                ...

cluster1::*> statistics catalog object show -object smb1
    smb1                        These counters report activity from the SMB
                                revision of the protocol. For information
                                ...

cluster1::*> statistics catalog object show -object smb2
    smb2                        These counters report activity from the
                                SMB2/SMB3 revision of the protocol. For
                                ...

cluster1::*> statistics catalog object show -object hashd
    hashd                       The hashd object provides counters to measure
                                the performance of the BranchCache hash daemon.
cluster1::*> set -privilege admin
----
