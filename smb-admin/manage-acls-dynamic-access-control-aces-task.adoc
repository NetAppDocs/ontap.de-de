= Verwaltung von ACLs mit DAC-ACEs, wenn DAC auf ONTAP SMB-Servern deaktiviert ist
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Wenn Sie Ressourcen haben, auf die ACLs mit Dynamic Access Control ACEs angewendet wurden, und Sie Dynamic Access Control auf der Storage Virtual Machine (SVM) deaktivieren, müssen Sie die Dynamic Access Control ACEs entfernen, bevor Sie die nicht-Dynamic Access Control ACEs auf dieser Ressource verwalten können.

.Über diese Aufgabe
Nachdem die dynamische Zugriffskontrolle deaktiviert ist, können bestehende nicht-dynamische Zugriffskontroll-ACEs weder entfernt noch neue nicht-dynamische Zugriffskontroll-ACEs hinzugefügt werden, bis die bestehenden dynamischen Zugriffskontroll-ACEs entfernt wurden.

Sie können das Tool, das Sie normalerweise zur Verwaltung von ACLs verwenden, für diese Schritte einsetzen.

.Schritte
. Ermitteln, welche Dynamic Access Control ACEs auf die Ressource angewendet sind.
. Die ACEs der dynamischen Zugriffssteuerung aus der Ressource entfernen.
. Nicht-dynamische Zugriffskontroll-ACEs können nach Bedarf zur Ressource hinzugefügt oder daraus entfernt werden.
