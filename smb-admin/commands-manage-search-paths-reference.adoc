= ONTAP Befehle zur Verwaltung von SMB-Suchpfaden
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Für die Verwaltung von Suchpfaden in SMB Home-Verzeichnis-Konfigurationen stehen spezielle ONTAP Befehle zur Verfügung. Beispielsweise gibt es Befehle zum Hinzufügen, Entfernen und Anzeigen von Informationen zu Suchpfaden. Es gibt auch einen Befehl zum Ändern der Reihenfolge des Suchpfads.

|===
| Wenn Sie möchten... | Verwenden Sie diesen Befehl... 


 a| 
Einen Suchpfad hinzufügen
 a| 
`vserver cifs home-directory search-path add`



 a| 
Suchpfade anzeigen
 a| 
`vserver cifs home-directory search-path show`



 a| 
Die Reihenfolge des Suchpfads ändern
 a| 
`vserver cifs home-directory search-path reorder`



 a| 
Einen Suchpfad entfernen
 a| 
`vserver cifs home-directory search-path remove`

|===
Weitere Informationen zu  `vserver cifs home-directory search-path` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/search.html?q=vserver+cifs+home-directory+search-path["ONTAP-Befehlsreferenz"^].
