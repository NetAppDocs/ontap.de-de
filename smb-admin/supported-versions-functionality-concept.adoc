= Unterstützte ONTAP SMB Versionen und Funktionen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Server Message Block (SMB) ist ein Protokoll zur gemeinsamen Nutzung von Dateien über Microsoft Windows Clients und Server. Alle SMB Versionen werden unterstützt. Es sollte überprüft werden, ob der ONTAP SMB Server die in Ihrer Umgebung benötigten Clients und Funktionen unterstützt.

Die aktuellsten Informationen darüber, welche SMB-Clients und Domänencontroller ONTAP unterstützt, sind im _Interoperability Matrix Tool_ verfügbar.

SMB 2.0 und spätere Versionen sind für ONTAP SMB-Server standardmäßig aktiviert und können bei Bedarf aktiviert oder deaktiviert werden. SMB 1.0 kann bei Bedarf aktiviert oder deaktiviert werden.

[NOTE]
====
Die Standardeinstellungen für SMB 1.0 und 2.0 Verbindungen zu Domänencontrollern hängen ebenfalls von der ONTAP Version ab. Weitere Informationen zu  `vserver cifs security modify` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/vserver-cifs-security-modify.html["ONTAP-Befehlsreferenz"^]. Für Umgebungen mit vorhandenen CIFS-Servern, auf denen SMB 1.0 ausgeführt wird, empfiehlt sich eine Migration auf eine spätere SMB-Version, um auf Verbesserungen bei Sicherheit und Compliance vorbereitet zu sein. Für Details steht Ihr NetApp Ansprechpartner zur Verfügung.

====
