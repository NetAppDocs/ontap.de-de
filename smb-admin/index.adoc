= Informationen zu ONTAP SMB
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ONTAP Dateizugriffsfunktionen sind für das SMB-Protokoll verfügbar. Ein CIFS-Server kann aktiviert, Freigaben können erstellt und Microsoft-Dienste aktiviert werden.

[NOTE]
====
_SMB_ (Server Message Block) bezeichnet moderne Dialekte des Common Internet File System (CIFS)-Protokolls. _CIFS_ erscheint weiterhin in der ONTAP-Befehlszeilenschnittstelle (CLI) und in OnCommand Management-Tools.

====
