= Informationen zur Verwendung von SnapMirror auf ONTAP SMB SVMs, die lokale Gruppen enthalten
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Es sind die Richtlinien zu beachten, wenn SnapMirror auf Volumes konfiguriert wird, die SVMs gehören und lokale Gruppen enthalten.

Lokale Gruppen können nicht in ACEs verwendet werden, die auf Dateien, Verzeichnisse oder Freigaben angewendet werden, die von SnapMirror auf eine andere SVM repliziert werden. Wenn die SnapMirror Funktion verwendet wird, um ein DR-Mirror für ein Volume auf einer anderen SVM zu erstellen und das Volume eine ACE für eine lokale Gruppe enthält, ist die ACE auf dem Mirror ungültig. Wenn Daten auf eine andere SVM repliziert werden, gelangen die Daten effektiv in eine andere lokale Domäne. Die Berechtigungen, die lokalen Benutzern und Gruppen gewährt werden, sind nur innerhalb des Gültigkeitsbereichs der SVM gültig, auf der sie ursprünglich erstellt wurden.
