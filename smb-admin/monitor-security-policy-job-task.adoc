= ONTAP SMB-Sicherheitsrichtlinienaufträge überwachen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Beim Anwenden der Sicherheitsrichtlinie auf Storage Virtual Machines (SVMs) kann der Fortschritt der Aufgabe überwacht werden, indem der Sicherheitsrichtlinienauftrag überwacht wird. Dies ist hilfreich, wenn festgestellt werden soll, dass die Anwendung der Sicherheitsrichtlinie erfolgreich war. Dies ist auch hilfreich, wenn ein lang andauernder Auftrag vorliegt, bei dem eine Sicherheitsrichtlinie auf eine große Anzahl von Dateien und Ordnern angewendet wird.

.Über diese Aufgabe
Um detaillierte Informationen zu einem Sicherheitsrichtlinienauftrag anzuzeigen, sollte der  `-instance` Parameter verwendet werden.

.Schritt
. Der Auftrag zur Sicherheitsrichtlinie wird überwacht: `vserver security file-directory job show -vserver vserver_name`
+
`vserver security file-directory job show -vserver vs1`
