= Verwaltung der ONTAP SMB Domain Controller-Erkennung
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Ab ONTAP 9.3 können Sie den Standardprozess zur Ermittlung von Domänencontrollern (DCs) anpassen. Dadurch kann die Ermittlung auf Ihren Standort oder auf einen Pool bevorzugter DCs beschränkt werden, was je nach Umgebung zu Leistungsverbesserungen führen kann.

.Über diese Aufgabe
Standardmäßig ermittelt der dynamische Discovery-Prozess alle verfügbaren DCs, einschließlich aller bevorzugten DCs, aller DCs am lokalen Standort und aller Remote-DCs. Diese Konfiguration kann in bestimmten Umgebungen zu Latenz bei der Authentifizierung und beim Zugriff auf Freigaben führen. Wenn der Pool der zu verwendenden DCs bereits festgelegt wurde oder die Remote-DCs unzureichend oder nicht erreichbar sind, kann die Ermittlungsmethode geändert werden.

In ONTAP 9.3 und späteren Versionen ermöglicht der  `discovery-mode` Parameter des  `cifs domain discovered-servers` Befehls die Auswahl einer der folgenden Erkennungsoptionen:

* Alle Domänencontroller in der Domäne werden erkannt.
* Es werden nur DCs am lokalen Standort erkannt.
+
Der  `default-site`Parameter für den SMB-Server kann so definiert werden, dass dieser Modus mit LIFs verwendet wird, die keinem Standort in sites-and-services zugeordnet sind.

* Es findet keine Servererkennung statt, die SMB Server-Konfiguration hängt ausschließlich von den bevorzugten DCs ab.
+
Um diesen Modus zu verwenden, müssen zuerst die bevorzugten Domänencontroller für den SMB-Server definiert werden.



.Bevor Sie beginnen
Sie müssen sich auf der erweiterten Berechtigungsstufe befinden.

.Schritt
. Geben Sie die gewünschte Erkennungsoption an: `vserver cifs domain discovered-servers discovery-mode modify -vserver _vserver_name_ -mode {all|site|none}`
+
Optionen für den  `mode` Parameter:

+
** `all`
+
Alle verfügbaren DCs ermitteln (Standard).

** `site`
+
Die DC-Erkennung ist auf den eigenen Standort beschränkt.

** `none`
+
Es werden nur bevorzugte Domänencontroller verwendet, und eine Erkennung findet nicht statt.
