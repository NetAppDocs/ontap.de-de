= Informationen zu ONTAP SMB eingeschränkten Gruppenrichtlinien (GPOs) anzeigen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Detaillierte Informationen über eingeschränkte Gruppen, die als Gruppenrichtlinienobjekte (GPOs) in Active Directory definiert sind und auf den CIFS-Server angewendet werden, können angezeigt werden.

.Über diese Aufgabe
Standardmäßig werden folgende Informationen angezeigt:

* Name der Gruppenrichtlinie
* Gruppenrichtlinienversion
* Link
+
Gibt die Ebene an, auf der die Gruppenrichtlinie konfiguriert ist. Mögliche Ausgabewerte sind die folgenden:

+
** `Local`wenn die Gruppenrichtlinie in ONTAP konfiguriert ist
** `Site`wenn die Gruppenrichtlinie auf Standortebene im Domänencontroller konfiguriert ist
** `Domain`wenn die Gruppenrichtlinie auf Domänenebene im Domänencontroller konfiguriert ist
** `OrganizationalUnit`wenn die Gruppenrichtlinie auf Abteilungsebene (OU) im Domänencontroller konfiguriert ist
** `RSOP` für die resultierende Menge von Richtlinien, die aus allen auf verschiedenen Ebenen definierten Gruppenrichtlinien abgeleitet werden


* Name der eingeschränkten Gruppe
* Die Benutzer und Gruppen, die der eingeschränkten Gruppe angehören und die nicht der eingeschränkten Gruppe angehören
* Die Liste der Gruppen, zu denen die eingeschränkte Gruppe hinzugefügt wird
+
Eine Gruppe kann Mitglied anderer Gruppen als der hier aufgeführten Gruppen sein.



.Schritt
. Informationen zu allen eingeschränkten Gruppenrichtlinienobjekten für Gruppen werden angezeigt, wenn eine der folgenden Aktionen durchgeführt wird:
+
|===
| Wenn Informationen zu allen eingeschränkten Gruppenrichtlinienobjekten (GPOs) angezeigt werden sollen... | Geben Sie den Befehl ein... 


 a| 
In Active Directory definiert
 a| 
`vserver cifs group-policy restricted-group show-defined -vserver vserver_name`



 a| 
Angewendet auf einen CIFS-Server
 a| 
`vserver cifs group-policy restricted-group show-applied -vserver vserver_name`

|===


.Beispiel
Das folgende Beispiel zeigt Informationen über eingeschränkte Gruppenrichtlinienobjekte (GPOs), die in der Active Directory Domäne definiert sind, zu der die CIFS-Server-fähige SVM mit dem Namen vs1 gehört:

[listing]
----
cluster1::> vserver cifs group-policy restricted-group show-defined -vserver vs1

Vserver: vs1
-------------

     Group Policy Name: gpo1
               Version: 16
                  Link: OrganizationalUnit
            Group Name: group1
               Members: user1
              MemberOf: EXAMPLE\group9

     Group Policy Name: Resultant Set of Policy
               Version: 0
                  Link: RSOP
            Group Name: group1
               Members: user1
              MemberOf: EXAMPLE\group9
----
Das folgende Beispiel zeigt Informationen über eingeschränkte Gruppenrichtlinienobjekte (GPOs) für eingeschränkte Gruppen, die auf die CIFS-aktivierte SVM vs1 angewendet wurden:

[listing]
----
cluster1::> vserver cifs group-policy restricted-group show-applied -vserver vs1

Vserver: vs1
-------------

     Group Policy Name: gpo1
               Version: 16
                  Link: OrganizationalUnit
            Group Name: group1
               Members: user1
              MemberOf: EXAMPLE\group9

     Group Policy Name: Resultant Set of Policy
               Version: 0
                  Link: RSOP
            Group Name: group1
               Members: user1
              MemberOf: EXAMPLE\group9
----
