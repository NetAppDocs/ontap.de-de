= Informationen zur Anwendung von Gruppenrichtlinienobjekten auf ONTAP SMB Servern
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Ihr SMB-Server unterstützt Gruppenrichtlinienobjekte (GPOs), eine Reihe von Regeln, die als _Gruppenrichtlinienattribute_ bezeichnet werden und für Computer in einer Active Directory-Umgebung gelten. Mit GPOs lassen sich die Einstellungen für alle Storage Virtual Machines (SVMs) im Cluster, die zur selben Active Directory-Domäne gehören, zentral verwalten.

Wenn Gruppenrichtlinienobjekte (GPOs) auf Ihrem SMB-Server aktiviert sind, sendet ONTAP LDAP-Abfragen an den Active Directory-Server, um GPO-Informationen anzufordern. Wenn GPO-Definitionen vorhanden sind, die für Ihren SMB-Server gelten, gibt der Active Directory-Server die folgenden GPO-Informationen zurück:

* GPO-Name
* Aktuelle GPO Version
* Speicherort der GPO-Definition
* Listen von UUIDs (universell eindeutigen Kennungen) für GPO Richtliniensätze
