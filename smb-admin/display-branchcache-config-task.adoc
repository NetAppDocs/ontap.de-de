= Informationen zu BranchCache Konfigurationen auf ONTAP SMB-Freigaben
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Informationen zu BranchCache Konfigurationen auf Storage Virtual Machines (SVMs) können angezeigt werden, was bei der Überprüfung einer Konfiguration oder bei der Ermittlung der aktuellen Einstellungen vor einer Änderung der Konfiguration hilfreich ist.

.Schritt
. Führen Sie eine der folgenden Aktionen aus:
+
|===
| Wenn Sie anzeigen möchten... | Geben Sie diesen Befehl ein... 


 a| 
Zusammenfassende Informationen zu BranchCache Konfigurationen auf allen SVMs
 a| 
`vserver cifs branchcache show`



 a| 
Detaillierte Informationen zur Konfiguration einer bestimmten SVM
 a| 
`vserver cifs branchcache show -vserver _vserver_name_`

|===


.Beispiel
Das folgende Beispiel zeigt Informationen zur BranchCache Konfiguration auf SVM vs1:

[listing]
----
cluster1::> vserver cifs branchcache show -vserver vs1

                                   Vserver: vs1
            Supported BranchCache Versions: enable_all
                        Path to Hash Store: /hash_data
            Maximum Size of the Hash Store: 20GB
  Encryption Key Used to Secure the Hashes: -
          CIFS BranchCache Operating Modes: per_share
----
