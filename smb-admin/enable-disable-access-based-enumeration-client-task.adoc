= Zugriffsbasierte Aufzählung von einem Windows-Client auf ONTAP SMB-Freigaben aktivieren oder deaktivieren
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Der Zugriff auf die zugriffsbasierte Aufzählung (ABE) kann auf SMB-Freigaben von einem Windows-Client aus aktiviert oder deaktiviert werden, wodurch sich diese Freigabeeinstellung konfigurieren lässt, ohne dass eine Verbindung zum CIFS-Server erforderlich ist.


NOTE: Das  `abecmd` Dienstprogramm ist in neueren Versionen von Windows Server und Windows Clients nicht verfügbar. Es wurde als Teil von Windows Server 2008 veröffentlicht. Der Support für Windows Server 2008 endete am 14. Januar 2020.

.Schritte
. Auf einem Windows-Client, der ABE unterstützt, wird folgender Befehl eingegeben: `abecmd [/enable | /disable] [/server CIFS_server_name] {/all | share_name}`
+
Weitere Informationen zum  `abecmd` Befehl sind in der Dokumentation Ihres Windows Clients zu finden.
