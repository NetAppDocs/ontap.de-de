= Informationen darüber, was mit statischen ONTAP SMB-Homeverzeichnis-Freigabenamen nach einem Upgrade geschieht
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Die Namen von Home-Verzeichnisfreigaben müssen entweder die  `%w` oder die  `%u` dynamische Variable enthalten. Es ist zu beachten, was mit bestehenden statischen Home-Verzeichnisfreigabenamen nach einem Upgrade auf eine Version von ONTAP mit dieser neuen Anforderung geschieht.

Wenn Ihre Home-Verzeichniskonfiguration statische Freigabenamen enthält und Sie auf ONTAP aktualisieren, bleiben die statischen Home-Verzeichnisfreigabenamen unverändert und weiterhin gültig. Es können jedoch keine neuen Home-Verzeichnisfreigaben erstellt werden, die nicht entweder die  `%w` oder  `%u` Variable enthalten.

Die Anforderung, dass eine dieser Variablen im Freigabenamen des Benutzerverzeichnisses enthalten ist, gewährleistet, dass jeder Freigabename innerhalb der Benutzerverzeichniskonfiguration eindeutig ist. Falls gewünscht, können die statischen Freigabenamen des Benutzerverzeichnisses in Namen geändert werden, die entweder die  `%w` oder die  `%u` Variable enthalten.
