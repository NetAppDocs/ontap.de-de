= NetBIOS-Aliase aus der Liste für ONTAP SMB-Server entfernen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Wenn keine bestimmten NetBIOS-Aliase für einen CIFS-Server benötigt werden, können diese NetBIOS-Aliase aus der Liste entfernt werden. Es ist auch möglich, alle NetBIOS-Aliase aus der Liste zu entfernen.

.Über diese Aufgabe
Sie können mehr als einen NetBIOS-Alias mithilfe einer durch Kommas getrennten Liste entfernen. Alle NetBIOS-Aliase auf einem CIFS-Server können entfernt werden, indem  `-` als Wert für den  `-netbios-aliases` Parameter angegeben wird.

.Schritte
. Führen Sie eine der folgenden Aktionen aus:
+
|===
| Wenn Sie entfernen möchten... | Eingeben... 


 a| 
Spezifische NetBIOS Aliase aus der Liste
 a| 
`+vserver cifs remove-netbios-aliases -vserver _vserver_name_ -netbios-aliases _NetBIOS_alias_,...+`



 a| 
Alle NetBIOS Aliase aus der Liste
 a| 
`vserver cifs remove-netbios-aliases -vserver _vserver_name_ -netbios-aliases -`

|===
+
`vserver cifs remove-netbios-aliases -vserver vs1 -netbios-aliases alias_1`

. Es wird überprüft, ob die angegebenen NetBIOS-Aliase entfernt wurden: `vserver cifs show -vserver _vserver_name_ -display-netbios-aliases`
+
`vserver cifs show -vserver vs1 -display-netbios-aliases`

+
[listing]
----
Vserver: vs1

         Server Name: CIFS_SERVER
         NetBIOS Aliases: ALIAS_2, ALIAS_3
----
