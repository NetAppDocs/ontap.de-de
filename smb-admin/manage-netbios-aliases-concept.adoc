= Informationen zur Verwaltung von NetBIOS-Aliasen für ONTAP SMB-Server
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
NetBIOS-Aliase sind alternative Namen für Ihren SMB-Server, die SMB-Clients beim Verbindungsaufbau mit dem SMB-Server verwenden können. Die Konfiguration von NetBIOS-Aliasen für einen SMB-Server kann nützlich sein, wenn Daten von anderen Dateiservern auf den SMB-Server zusammengeführt werden und der SMB-Server auf die Namen der ursprünglichen Dateiserver reagieren soll.

Sie können beim Erstellen des SMB-Servers oder jederzeit danach eine Liste von NetBIOS-Aliasen festlegen. NetBIOS-Aliase können jederzeit zur Liste hinzugefügt oder daraus entfernt werden. Eine Verbindung zum SMB-Server ist mit jedem der Namen in der NetBIOS-Aliasliste möglich.
