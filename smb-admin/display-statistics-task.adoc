= ONTAP SMB Statistiken anzeigen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Es lassen sich verschiedene Statistiken anzeigen, darunter Statistiken zu CIFS und SMB, Auditing und BranchCache-Hashes, um die Leistung zu überwachen und Probleme zu diagnostizieren.

.Bevor Sie beginnen
Sie müssen Datenbeispiele mithilfe der `statistics start` und `statistics stop` Befehle gesammelt haben, bevor Sie Informationen über Objekte anzeigen können.

Weitere Informationen zu  `statistics start` und  `statistics stop` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/search.html?q=statistics["ONTAP-Befehlsreferenz"^].

.Schritte
. Die Berechtigungsstufe wird auf „Erweitert“ gesetzt: `set -privilege advanced`
. Führen Sie eine der folgenden Aktionen aus:
+
|===
| Wenn Sie Statistiken für... | Eingeben... 


 a| 
Alle Versionen von SMB
 a| 
`statistics show -object cifs`



 a| 
SMB 1.0
 a| 
`statistics show -object smb1`



 a| 
SMB 2.x und SMB 3.0
 a| 
`statistics show -object smb2`



 a| 
CIFS-Subsystem des Knotens
 a| 
`statistics show -object nblade_cifs`



 a| 
Multiprotokoll-Audit
 a| 
`statistics show -object audit_ng`



 a| 
BranchCache Hash-Dienst
 a| 
`statistics show -object hashd`



 a| 
dynamisches DNS
 a| 
`statistics show -object ddns_update`

|===
+
Weitere Informationen zu  `statistics show` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/statistics-show.html["ONTAP-Befehlsreferenz"^].

. Zurück zur Administratorrechteebene: `set -privilege admin`
