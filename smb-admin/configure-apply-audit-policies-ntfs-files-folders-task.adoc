= ONTAP Befehle zum Konfigurieren und Anwenden von SMB-Überwachungsrichtlinien auf NTFS-Dateien und Ordner
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Es sind mehrere Schritte erforderlich, um Überwachungsrichtlinien auf NTFS-Dateien und -Ordner mit der ONTAP CLI anzuwenden. Zuerst wird ein NTFS-Sicherheitsdeskriptor erstellt und SACLs zu diesem Sicherheitsdeskriptor hinzugefügt. Anschließend wird eine Sicherheitsrichtlinie erstellt und Richtlinienaufgaben hinzugefügt. Danach wird die Sicherheitsrichtlinie auf eine Storage Virtual Machine (SVM) angewendet.

.Über diese Aufgabe
Nach der Anwendung der Sicherheitsrichtlinie können Sie den Sicherheitsrichtlinienauftrag überwachen und anschließend die Einstellungen für die angewendete Audit-Richtlinie überprüfen.


NOTE: Beim Anwenden einer Überwachungsrichtlinie und zugehöriger SACLs werden alle vorhandenen DACLs überschrieben. Vor dem Erstellen und Anwenden neuer Sicherheitsrichtlinien empfiehlt sich eine Überprüfung der bestehenden Richtlinien.
