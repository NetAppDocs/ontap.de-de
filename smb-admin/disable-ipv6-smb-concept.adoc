= Informationen zum Deaktivieren von IPv6 für ONTAP SMB Server
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Auch wenn IPv6 im Cluster über eine Netzwerkoption aktiviert ist, kann IPv6 für SMB nicht mit demselben Befehl deaktiviert werden. Stattdessen deaktiviert ONTAP IPv6, wenn der Clusteradministrator die letzte IPv6-fähige Schnittstelle im Cluster deaktiviert. Eine Abstimmung mit dem Clusteradministrator zur Verwaltung der IPv6-fähigen Schnittstellen ist empfehlenswert.
