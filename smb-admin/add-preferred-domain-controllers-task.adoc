= Bevorzugte ONTAP SMB-Domänencontroller hinzufügen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ONTAP erkennt Domänencontroller automatisch über DNS. Optional ist es möglich, einen oder mehrere Domänencontroller zur Liste der bevorzugten Domänencontroller für eine bestimmte Domäne hinzuzufügen.

.Über diese Aufgabe
Existiert für die angegebene Domäne bereits eine Liste bevorzugter Domänencontroller, wird die neue Liste mit der bestehenden Liste zusammengeführt.

.Schritt
. Um die Liste der bevorzugten Domänencontroller zu erweitern, wird folgender Befehl eingegeben: +
`vserver cifs domain preferred-dc add -vserver _vserver_name_ -domain _domain_name_ -preferred-dc IP_address, ...+`
+
`-vserver _vserver_name_`Gibt den Namen der Storage Virtual Machine (SVM) an.

+
`-domain _domain_name_`Gibt den vollqualifizierten Active Directory-Namen der Domäne an, zu der die angegebenen Domänencontroller gehören.

+
`-preferred-dc _IP_address_`,... gibt eine oder mehrere IP-Adressen der bevorzugten Domänencontroller als durch Kommas getrennte Liste in der Reihenfolge der Präferenz an.



.Beispiel
Der folgende Befehl fügt die Domänencontroller 172.17.102.25 und 172.17.102.24 der Liste der bevorzugten Domänencontroller hinzu, die der SMB-Server auf SVM vs1 zur Verwaltung des externen Zugriffs auf die Domäne cifs.lab.example.com verwendet.

[listing]
----
cluster1::> vserver cifs domain preferred-dc add -vserver vs1 -domain cifs.lab.example.com -preferred-dc 172.17.102.25,172.17.102.24
----
