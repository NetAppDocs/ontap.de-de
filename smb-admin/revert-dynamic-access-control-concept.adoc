= Überlegungen zum Zurücksetzen von DAC auf ONTAP SMB-Servern
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie sollten sich darüber im Klaren sein, was passiert, wenn auf eine Version von ONTAP zurückgesetzt wird, die Dynamic Access Control (DAC) nicht unterstützt, und was vor und nach dem Zurücksetzen zu tun ist.

Wenn der Cluster auf eine Version von ONTAP zurückgesetzt werden soll, die Dynamic Access Control nicht unterstützt, und Dynamic Access Control auf einer oder mehreren Storage Virtual Machines (SVMs) aktiviert ist, sind vor dem Zurücksetzen die folgenden Schritte erforderlich:

* Die dynamische Zugriffskontrolle muss auf allen SVMs deaktiviert werden, auf denen sie im Cluster aktiviert ist.
* Sie müssen alle Überwachungskonfigurationen auf dem Cluster, die den  `cap-staging` Ereignistyp enthalten, so ändern, dass nur der  `file-op` Ereignistyp verwendet wird.


Einige wichtige Überlegungen zum Zurücksetzen von Dateien und Ordnern mit Dynamic Access Control-ACEs sollten verstanden und berücksichtigt werden:

* Wenn der Cluster zurückgesetzt wird, werden bestehende Dynamic Access Control ACEs nicht entfernt; sie werden jedoch bei Dateizugriff-Prüfungen ignoriert.
* Da Dynamic Access Control-ACEs nach der Rücksetzung ignoriert werden, ändert sich der Zugriff auf Dateien mit Dynamic Access Control-ACEs.
+
Dies könnte dazu führen, dass Benutzer auf Dateien zugreifen können, auf die sie zuvor keinen Zugriff hatten, oder umgekehrt, dass sie auf Dateien nicht mehr zugreifen können, auf die sie zuvor Zugriff hatten.

* Nicht-dynamische Zugriffskontroll-ACEs sollten auf die betroffenen Dateien angewendet werden, um deren vorheriges Sicherheitsniveau wiederherzustellen.
+
Dies kann entweder vor dem Zurücksetzen oder unmittelbar nach Abschluss der Rücksetzung erfolgen.



[NOTE]
====
Da Dynamic Access Control-ACEs nach dem Zurücksetzen ignoriert werden, ist es nicht erforderlich, sie zu entfernen, wenn nicht-Dynamic Access Control-ACEs auf die betroffenen Dateien angewendet werden. Falls gewünscht, können sie manuell entfernt werden.
