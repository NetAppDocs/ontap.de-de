= Informationen zu den Home Directory-Pfaden von ONTAP SMB-Benutzern
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können den Home Directory-Pfad eines SMB-Benutzers auf der Storage Virtual Machine (SVM) anzeigen, was hilfreich ist, wenn mehrere CIFS-Home Directory-Pfade konfiguriert sind und nachvollzogen werden soll, welcher Pfad das Home Directory des Benutzers enthält.

.Schritt
. Der Home Directory-Pfad wird mit dem  `vserver cifs home-directory show-user` Befehl angezeigt.
+
`vserver cifs home-directory show-user -vserver vs1 -username user1`

+
[listing]
----

     Vserver      User                  Home Dir Path
     -----------  -------------------   -----------------------
     vs1          user1                 /home/user1
----
