= LDAP über TLS konfigurieren
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Um LDAP über TLS zu konfigurieren, muss eine Kopie des selbstsignierten Stamm-CA-Zertifikats exportiert, das selbstsignierte Stamm-CA-Zertifikat mithilfe der exportierten Datei installiert und anschließend LDAP über TLS auf der SVM aktiviert werden.
