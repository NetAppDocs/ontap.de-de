= Informationen zu ONTAP SMB-Freigaben mit der MMC anzeigen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Informationen zu SMB-Freigaben auf Ihrer SVM lassen sich anzeigen und einige Verwaltungsaufgaben mithilfe der Microsoft Management Console (MMC) durchführen. Zum Anzeigen der Freigaben ist eine Verbindung der MMC mit der SVM erforderlich.

.Über diese Aufgabe
Mit der MMC lassen sich die folgenden Aufgaben für in SVMs enthaltene Freigaben durchführen:

* Freigaben anzeigen
* Aktive Sitzungen anzeigen
* Offene Dateien anzeigen
* Die Liste der Sitzungen, Dateien und Baumverbindungen im System wird aufgezählt.
* Offene Dateien im System schließen
* Offene Sitzungen schließen
* Freigaben erstellen/verwalten


[NOTE]
====
Die von den vorherigen Funktionen angezeigten Ansichten sind knotenspezifisch und nicht clusterspezifisch. Daher wird bei Verwendung der MMC zur Verbindung mit dem SMB-Server-Hostnamen (das heißt cifs01.domain.local) abhängig von der DNS-Konfiguration zu einer einzelnen LIF innerhalb des Clusters weitergeleitet.

====
Die folgenden Funktionen werden in MMC für ONTAP nicht unterstützt:

* Erstellen neuer lokaler Benutzer/Gruppen
* Verwaltung/Anzeige bestehender lokaler Benutzer/Gruppen
* Anzeigen von Ereignissen oder Leistungsprotokollen
* Storage
* Dienste und Anwendungen


In Fällen, in denen die Operation nicht unterstützt wird, können Sie  `remote procedure call failed`Fehler erleben.

https://kb.netapp.com/Advice_and_Troubleshooting/Data_Storage_Software/ONTAP_OS/FAQ%3A_Using_Windows_MMC_with_ONTAP["Häufig gestellte Fragen: Verwendung der Windows MMC mit ONTAP"]

.Schritte
. Die Computer Management MMC auf einem beliebigen Windows Server wird in der *Systemsteuerung* unter *Verwaltung* > *Computerverwaltung* geöffnet.
. *Aktion* > *Mit einem anderen Computer verbinden* auswählen.
+
Das Dialogfeld „Computer auswählen“ wird angezeigt.

. Den Namen des Speichersystems eingeben oder auf *Durchsuchen* klicken, um das Speichersystem zu finden.
. Klicken Sie auf *OK*.
+
Der MMC stellt eine Verbindung zum SVM her.

. Im Navigationsbereich auf *Freigegebene Ordner* > *Freigaben* klicken.
+
Im rechten Anzeigebereich wird eine Liste der Shares auf der SVM angezeigt.

. Die Freigabeeigenschaften einer Freigabe werden angezeigt, wenn auf die Freigabe doppelt geklickt wird, wodurch das Dialogfeld *Eigenschaften* geöffnet wird.
. Wenn keine Verbindung zum Speichersystem über die MMC möglich ist, kann der Benutzer der Gruppe BUILTIN\Administrators oder BUILTIN\Power Users auf dem Speichersystem mit einem der folgenden Befehle hinzugefügt werden:
+
[listing]
----

cifs users-and-groups local-groups add-members -vserver <vserver_name> -group-name BUILTIN\Administrators -member-names <domainuser>

cifs users-and-groups local-groups add-members -vserver <vserver_name> -group-name "BUILTIN\Power Users" -member-names <domainuser>
----
