= Informationen zur Unterstützung von ONTAP SMB BranchCache Versionen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie sollten wissen, welche BranchCache Versionen ONTAP unterstützt.

ONTAP unterstützt BranchCache 1 und die erweiterte BranchCache 2:

* Wenn BranchCache auf dem SMB-Server für die Storage Virtual Machine (SVM) konfiguriert wird, kann BranchCache 1, BranchCache 2 oder alle Versionen aktiviert werden.
+
Standardmäßig sind alle Versionen aktiviert.

* Wenn Sie nur BranchCache 2 aktivieren, müssen die Windows-Clientrechner im Remote-Büro BranchCache 2 unterstützen.
+
Nur SMB 3.0 oder neuere Clients unterstützen BranchCache 2.



Weitere Informationen zu BranchCache Versionen finden sich in der Microsoft TechNet Library.
