= Die Auswirkungen der Löschung von ONTAP SMB-Servern auf Benutzer und Gruppen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Die Standardmenge an lokalen Benutzern und Gruppen wird bei der Erstellung eines CIFS-Servers angelegt und der Storage Virtual Machine (SVM) zugeordnet, die den CIFS-Server hostet. SVM-Administratoren können lokale Benutzer und Gruppen jederzeit erstellen. Es sollte beachtet werden, was mit den lokalen Benutzern und Gruppen geschieht, wenn der CIFS-Server gelöscht wird.

Lokale Benutzer und Gruppen sind SVMs zugeordnet, daher werden sie aus Sicherheitsgründen beim Löschen von CIFS-Servern nicht gelöscht. Obwohl lokale Benutzer und Gruppen beim Löschen des CIFS-Servers nicht gelöscht werden, sind sie ausgeblendet. Lokale Benutzer und Gruppen können erst wieder angezeigt oder verwaltet werden, nachdem ein CIFS-Server auf der SVM neu erstellt wurde.

[NOTE]
====
Der administrative Status des CIFS-Servers hat keinen Einfluss auf die Sichtbarkeit lokaler Benutzer oder Gruppen.
