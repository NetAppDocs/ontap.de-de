= Konfigurationsworkflow für Storage-Level Access Guard auf ONTAP SMB Servern
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Der Workflow zur Konfiguration von Storage-Level Access Guard (SLAG) verwendet dieselben ONTAP CLI-Befehle wie die Konfiguration von NTFS-Dateiberechtigungen und Überwachungsrichtlinien. Statt den Datei- und Verzeichniszugriff auf einem bestimmten Ziel zu konfigurieren, wird SLAG auf dem entsprechenden Volume der Storage Virtual Machine (SVM) konfiguriert.
