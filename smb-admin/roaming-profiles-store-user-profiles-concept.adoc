= Informationen zur zentralen Speicherung von ONTAP SMB-Benutzerprofilen mit Roaming-Profilen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ONTAP unterstützt die Speicherung von Windows-Roaming-Profilen auf einem CIFS-Server, der der Storage Virtual Machine (SVM) zugeordnet ist. Die Konfiguration von Benutzer-Roaming-Profilen bietet Vorteile für die Benutzer, wie die automatische Verfügbarkeit von Ressourcen unabhängig davon, wo sich die Benutzer anmelden. Roaming-Profile vereinfachen außerdem die Administration und Verwaltung von Benutzerprofilen.

Roaming-Benutzerprofile haben folgende Vorteile:

* Automatische Ressourcenverfügbarkeit
+
Das individuelle Benutzerprofil steht automatisch zur Verfügung, sobald sich der Benutzer an einem beliebigen Computer im Netzwerk anmeldet, auf dem Windows 8, Windows 7, Windows 2000 oder Windows XP ausgeführt wird. Benutzer müssen nicht auf jedem Computer, den sie im Netzwerk verwenden, ein Profil erstellen.

* Vereinfachter Computeraustausch
+
Da alle Benutzerprofilinformationen separat im Netzwerk gespeichert werden, kann ein Benutzerprofil problemlos auf einen neuen Ersatzcomputer heruntergeladen werden. Wenn sich der Benutzer zum ersten Mal am neuen Computer anmeldet, wird die Serverkopie des Benutzerprofils auf den neuen Computer kopiert.
