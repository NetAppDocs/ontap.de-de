= Informationen zu den Voraussetzungen für die Verwendung der ONTAP SMB-Ordnerumleitung
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Bevor Sie die Ordnerumleitung von Microsoft mit Ihrem CIFS-Server verwenden können, müssen Sie wissen, welche Versionen von ONTAP und SMB sowie welche Windows-Clients diese Funktion unterstützen.



[[ontap-version-requirements]]
== ONTAP Versionsanforderungen

ONTAP unterstützt die Umleitung von Microsoft Ordnern.



[[smb-protocol-version-requirements]]
== Anforderungen an die SMB-Protokollversion

Für Storage Virtual Machine (SVM) unterstützt ONTAP die Ordnerumleitung von Microsoft in allen Versionen von SMB.



[[windows-client-requirements]]
== Windows Client Anforderungen

Bevor ein Benutzer die Ordnerumleitung von Microsoft verwenden kann, muss der Windows Client diese Funktion unterstützen.

Die aktuellsten Informationen darüber, welche Windows-Clients die Ordnerumleitung unterstützen, sind in der Interoperabilitätsmatrix zu finden.

http://mysupport.netapp.com/matrix["mysupport.netapp.com/matrix"^]
