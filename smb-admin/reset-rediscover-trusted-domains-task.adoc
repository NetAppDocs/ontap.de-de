= Vertrauenswürdige ONTAP SMB-Domänen zurücksetzen und neu erkennen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können die erneute Erkennung aller vertrauenswürdigen Domänen erzwingen. Dies kann nützlich sein, wenn die Server der vertrauenswürdigen Domänen nicht angemessen reagieren oder sich die Vertrauensstellungen geändert haben. Es werden nur Domänen erkannt, die über eine bidirektionale Vertrauensstellung zur Heimatdomäne verfügen, also der Domäne, die das Computerkonto des CIFS-Servers enthält.

.Schritt
. Vertrauenswürdige Domänen werden mithilfe des  `vserver cifs domain trusts rediscover` Befehls zurückgesetzt und neu erkannt.
+
`vserver cifs domain trusts rediscover -vserver vs1`
