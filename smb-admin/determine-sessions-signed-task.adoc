= Feststellen, ob ONTAP SMB-Sitzungen signiert sind
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Informationen über verbundene SMB-Sitzungen auf dem CIFS-Server können angezeigt werden. Diese Informationen ermöglichen die Feststellung, ob SMB-Sitzungen signiert sind. Dies kann hilfreich sein, um zu bestimmen, ob SMB-Client-Sitzungen mit den gewünschten Sicherheitseinstellungen verbunden sind.

.Schritte
. Führen Sie eine der folgenden Aktionen aus:
+
|===
| Wenn Informationen über ... angezeigt werden sollen | Geben Sie den Befehl ein... 


 a| 
Alle signierten Sitzungen auf einer bestimmten Storage Virtual Machine (SVM)
 a| 
`vserver cifs session show -vserver _vserver_name_ -is-session-signed true`



 a| 
Details zu einer signierten Sitzung mit einer bestimmten Sitzungs-ID auf der SVM
 a| 
`vserver cifs session show -vserver _vserver_name_ -session-id integer -instance`

|===


.Beispiele
Der folgende Befehl zeigt Sitzungsinformationen zu signierten Sitzungen auf SVM vs1 an. Die Standard-Zusammenfassungsausgabe zeigt das Ausgabefeld „`Ist Sitzung signiert`“ nicht an:

[listing]
----
cluster1::> vserver cifs session show -vserver vs1 -is-session-signed true
Node:    node1
Vserver: vs1
Connection Session                                    Open         Idle
ID          ID      Workstation      Windows User    Files         Time
----------  ------- ---------------- ------------- ------- ------------
3151272279  1       10.1.1.1         DOMAIN\joe          2          23s
----
Der folgende Befehl zeigt detaillierte Sitzungsinformationen an, einschließlich der Information, ob die Sitzung signiert ist, für eine SMB-Sitzung mit der Sitzungs-ID 2:

[listing]
----
cluster1::> vserver cifs session show -vserver vs1 -session-id 2 -instance
                        Node: node1
                     Vserver: vs1
                  Session ID: 2
               Connection ID: 3151274158
Incoming Data LIF IP Address: 10.2.1.1
                 Workstation: 10.1.1.2
    Authentication Mechanism: Kerberos
                Windows User: DOMAIN\joe
                   UNIX User: pcuser
                 Open Shares: 1
                  Open Files: 1
                  Open Other: 0
              Connected Time: 10m 43s
                   Idle Time: 1m 19s
            Protocol Version: SMB3
      Continuously Available: No
           Is Session Signed: true
       User Authenticated as: domain-user
                NetBIOS Name: CIFS_ALIAS1
       SMB Encryption Status: Unencrypted
----
