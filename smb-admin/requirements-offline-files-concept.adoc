= Informationen zu den Anforderungen für die Verwendung von Offline ONTAP SMB-Dateien
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Bevor Sie die Microsoft Offline Files Funktion mit Ihrem CIFS-Server verwenden können, müssen Sie wissen, welche Versionen von ONTAP und SMB sowie welche Windows-Clients die Funktion unterstützen.



[[ontap-version-requirements]]
== ONTAP Versionsanforderungen

ONTAP Releases unterstützen Offline-Dateien.



[[smb-protocol-version-requirements]]
== Anforderungen an die SMB-Protokollversion

Für Storage Virtual Machines (SVM) unterstützt ONTAP Offline-Dateien auf allen Versionen von SMB.



[[windows-client-requirements]]
== Windows Client Anforderungen

Der Windows Client muss die Offline-Dateien unterstützen.

Die aktuellsten Informationen darüber, welche Windows Clients die Funktion Offline-Dateien unterstützen, sind in der Interoperabilitätsmatrix zu finden.

http://mysupport.netapp.com/matrix["mysupport.netapp.com/matrix"^]
