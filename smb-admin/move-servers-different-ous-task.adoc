= ONTAP SMB-Server in andere Organisationseinheiten verschieben
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Der CIFS-Server-Erstellungsprozess verwendet während der Einrichtung die Standard-Organisations­einheit (OU) CN=Computers, sofern keine andere OU angegeben wird. CIFS-Server können nach der Einrichtung in andere OUs verschoben werden.

.Schritte
. Auf dem Windows-Server wird die Struktur *Active Directory Users and Computers* geöffnet.
. Das Active Directory-Objekt für die Storage Virtual Machine (SVM) ermitteln.
. Mit der rechten Maustaste auf das Objekt klicken und *Verschieben* auswählen.
. Die Organisationseinheit, die mit der SVM verknüpft werden soll, auswählen.


.Ergebnisse
Das SVM-Objekt befindet sich in der ausgewählten Organisationseinheit.
