= ONTAP SMB-Serveroptionen konfigurieren
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Die SMB-Serveroptionen lassen sich jederzeit konfigurieren, nachdem ein SMB-Server auf einer Storage Virtual Machine (SVM) erstellt wurde.

.Schritt
. Die gewünschte Aktion ausführen:
+
|===
| Wenn Sie Optionen für den SMB-Server konfigurieren möchten... | Geben Sie den Befehl ein... 


 a| 
Auf Administratorebene
 a| 
`vserver cifs options modify -vserver _vserver_name options_`



 a| 
Auf der Stufe „Erweiterte Berechtigungen“
 a| 
.. `set -privilege advanced`
.. `vserver cifs options modify -vserver _vserver_name options_`
.. `set -privilege admin`


|===
+
Weitere Informationen zu  `vserver cifs options modify` und zur Konfiguration von SMB-Serveroptionen finden Sie in der link:https://docs.netapp.com/us-en/ontap-cli/vserver-cifs-options-modify.html["ONTAP-Befehlsreferenz"^].
