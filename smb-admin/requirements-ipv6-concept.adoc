= Informationen zu den ONTAP SMB-Anforderungen für IPv6
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Bevor Sie IPv6 auf Ihrem SMB-Server verwenden können, müssen Sie wissen, welche Versionen von ONTAP und SMB dies unterstützen und welche Lizenzanforderungen gelten.



[[ontap-license-requirements]]
== ONTAP Lizenzanforderungen

Für IPv6 ist keine spezielle Lizenz erforderlich, wenn SMB lizenziert ist. Die SMB-Lizenz ist mit link:../system-admin/manage-licenses-concept.html#licenses-included-with-ontap-one["ONTAP One"] enthalten. Wenn ONTAP One nicht vorhanden ist und die Lizenz nicht installiert ist, sollte der Vertriebsmitarbeiter kontaktiert werden.



[[smb-protocol-version-requirements]]
== Anforderungen an die SMB-Protokollversion

* Für SVMs unterstützt ONTAP IPv6 auf allen Versionen des SMB-Protokolls.


[NOTE]
====
Der NetBIOS Name Service (NBNS) über IPv6 wird nicht unterstützt.

====
