= Feststellen, ob ONTAP SMB-Clients über NetBIOS-Aliase verbunden sind
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können feststellen, ob SMB-Clients über NetBIOS-Aliase verbunden sind und, falls ja, welcher NetBIOS-Alias für die Verbindung verwendet wird. Dies kann bei der Behebung von Verbindungsproblemen hilfreich sein.

.Über diese Aufgabe
Sie müssen den  `-instance` Parameter verwenden, um den NetBIOS-Alias (falls vorhanden) anzuzeigen, der mit einer SMB-Verbindung verknüpft ist. Wenn für die SMB-Verbindung der CIFS-Servername oder eine IP-Adresse verwendet wird, ist die Ausgabe für das  `NetBIOS Name` Feld  `-` (Bindestrich).

.Schritt
. Die gewünschte Aktion ausführen:
+
|===
| Wenn Sie NetBIOS-Informationen für ... anzeigen möchten | Eingeben... 


 a| 
SMB-Verbindungen
 a| 
`vserver cifs session show -instance`



 a| 
Verbindungen mit einem angegebenen NetBIOS-Alias:
 a| 
`vserver cifs session show -instance -netbios-name _netbios_name_`

|===
+
Das folgende Beispiel zeigt Informationen über den NetBIOS-Alias, der für die SMB-Verbindung mit der Sitzungs-ID 1 verwendet wird:

+
`vserver cifs session show -session-id 1 -instance`
