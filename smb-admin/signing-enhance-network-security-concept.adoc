= Informationen zur Verwendung von ONTAP SMB-Signaturen zur Verbesserung der Netzwerksicherheit
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Die SMB-Signierung trägt dazu bei, sicherzustellen, dass der Netzwerkverkehr zwischen dem SMB-Server und dem Client nicht kompromittiert wird; dies geschieht durch die Verhinderung von Replay-Angriffen. Standardmäßig unterstützt ONTAP die SMB-Signierung, wenn sie vom Client angefordert wird. Optional kann der Speicheradministrator den SMB-Server so konfigurieren, dass die SMB-Signierung erforderlich ist.
