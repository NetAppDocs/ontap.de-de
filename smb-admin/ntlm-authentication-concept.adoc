= NTLM-Authentifizierung
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Die NTLM Client-Authentifizierung erfolgt mittels eines Challenge-Response-Protokolls, das auf dem gemeinsamen Wissen über ein benutzerspezifisches Geheimnis basiert, welches wiederum auf einem Passwort beruht.

Wenn ein Benutzer eine SMB-Verbindung mit einem lokalen Windows-Benutzerkonto herstellt, erfolgt die Authentifizierung lokal durch den CIFS-Server mittels NTLMv2.
