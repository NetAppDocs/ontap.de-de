= Informationen zum Erstellen und Verwalten von ONTAP SMB-Datenvolumes in NAS-Namespaces
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Um den Dateizugriff in einer NAS-Umgebung zu verwalten, müssen Sie Datenvolumes und Verbindungspunkte auf Ihrer Storage Virtual Machine (SVM) verwalten. Dies umfasst die Planung Ihrer Namespace-Architektur, das Erstellen von Volumes mit oder ohne Verbindungspunkte, das Einbinden oder Aushängen von Volumes sowie das Anzeigen von Informationen über Datenvolumes und NFS-Server- oder CIFS-Server-Namespaces.
