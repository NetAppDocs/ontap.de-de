= Informationen zu ONTAP SMB GPO-Konfigurationen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Informationen über Gruppenrichtlinienobjekt (GPO)-Konfigurationen, die in Active Directory definiert sind, sowie über GPO-Konfigurationen, die auf den CIFS-Server angewendet werden, können angezeigt werden.

.Über diese Aufgabe
Es ist möglich, Informationen zu allen GPO-Konfigurationen anzuzeigen, die im Active Directory der Domäne definiert sind, zu der der CIFS-Server gehört, oder nur Informationen zu den GPO-Konfigurationen, die auf einen CIFS-Server angewendet wurden.

.Schritte
. Informationen zu GPO-Konfigurationen werden durch eine der folgenden Aktionen angezeigt:
+
|===
| Wenn Informationen zu allen Gruppenrichtlinienkonfigurationen angezeigt werden sollen... | Geben Sie den Befehl ein... 


 a| 
In Active Directory definiert
 a| 
`vserver cifs group-policy show-defined -vserver vserver_name`



 a| 
Angewendet auf eine CIFS-aktivierte Storage Virtual Machine (SVM)
 a| 
`vserver cifs group-policy show-applied -vserver vserver_name`

|===
