= Vorkalkulation von BranchCache-Hashes auf angegebenen ONTAP SMB-Pfaden
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können den BranchCache Service so konfigurieren, dass Hashwerte für eine einzelne Datei, ein Verzeichnis oder alle Dateien in einer Verzeichnisstruktur vorab berechnet werden. Dies kann hilfreich sein, wenn Hashwerte für Daten in einer BranchCache-fähigen Freigabe außerhalb der Spitzenzeiten berechnet werden sollen.

.Über diese Aufgabe
Wenn Sie vor der Anzeige der Hash-Statistiken eine Datenprobe erfassen möchten, müssen Sie die  `statistics start` und optionalen  `statistics stop` Befehle verwenden.

* Es ist erforderlich, die Storage Virtual Machine (SVM) und den Pfad anzugeben, auf dem Hashes vorab berechnet werden sollen.
* Es muss außerdem angegeben werden, ob die Hashwerte rekursiv berechnet werden sollen.
* Wenn Sie Hashes rekursiv berechnen möchten, durchläuft der BranchCache-Dienst den gesamten Verzeichnisbaum unter dem angegebenen Pfad und berechnet Hashes für jedes geeignete Objekt.
+
Weitere Informationen zu  `statistics start` und  `statistics stop` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/search.html?q=statistics["ONTAP-Befehlsreferenz"^].



.Schritte
. Hashwerte nach Bedarf vorab berechnen:
+
|===
| Wenn Sie Hashwerte vorab berechnen möchten... | Geben Sie den Befehl ein... 


 a| 
Eine einzelne Datei oder ein Verzeichnis
 a| 
`vserver cifs branchcache hash-create -vserver vserver_name -path path -recurse false`



 a| 
Rekursiv auf alle Dateien in einer Verzeichnisstruktur
 a| 
`vserver cifs branchcache hash-create -vserver vserver_name -path absolute_path -recurse true`

|===
. Es kann mit dem  `statistics` Befehl überprüft werden, ob Hashwerte berechnet werden:
+
.. Statistiken für das  `hashd` Objekt auf der gewünschten SVM-Instanz anzeigen: `statistics show -object hashd -instance vserver_name`
.. Es kann überprüft werden, ob die Anzahl der erzeugten Hashes durch Wiederholung des Befehls zunimmt.


+
Weitere Informationen zu  `statistics show` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/statistics-show.html["ONTAP-Befehlsreferenz"^].
