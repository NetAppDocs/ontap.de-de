= Informationen zu ONTAP SMB NetBIOS über TCP-Verbindungen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können Informationen zu NetBIOS over TCP (NBT)-Verbindungen anzeigen. Dies kann bei der Fehlerbehebung von NetBIOS-bezogenen Problemen hilfreich sein.

.Schritt
. Mit dem  `vserver cifs nbtstat` Befehl werden Informationen zu NetBIOS over TCP-Verbindungen angezeigt.
+
[NOTE]
====
Der NetBIOS Name Service (NBNS) über IPv6 wird nicht unterstützt.

====


.Beispiel
Das folgende Beispiel zeigt die NetBIOS-Namensdienstinformationen, die für "`cluster1`" angezeigt werden:

[listing]
----
cluster1::> vserver cifs nbtstat

          Vserver: vs1
          Node:    cluster1-01
          Interfaces:
                   10.10.10.32
                   10.10.10.33
         Servers:
                   17.17.1.2  (active  )
         NBT Scope:
                  [ ]
         NBT Mode:
                  [h]
         NBT Name    NetBIOS Suffix   State   Time Left   Type
         ----------- ---------------  ------- ---------   -----
         CLUSTER_1   00               wins    57
         CLUSTER_1   20               wins    57

         Vserver: vs1
         Node:    cluster1-02
         Interfaces:
                 10.10.10.35
         Servers:
                 17.17.1.2  (active  )
         CLUSTER_1          00               wins          58
         CLUSTER_1          20               wins          58
         4 entries were displayed.
----
