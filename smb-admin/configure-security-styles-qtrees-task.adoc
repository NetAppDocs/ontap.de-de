= SMB-Sicherheitsstile auf ONTAP Qtrees konfigurieren
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Der Sicherheitsstil des Qtree-Volumes wird konfiguriert, um die Art der Berechtigungen festzulegen, die für Daten auf Qtrees verwendet werden.

.Schritte
. Führen Sie eine der folgenden Aktionen aus:
+
|===
| Wenn der Qtree... | Verwenden Sie den Befehl... 


 a| 
Existiert noch nicht
 a| 
`volume qtree create` und den  `-security-style` Parameter zur Angabe des Sicherheitsstils einschließen.



 a| 
Existiert bereits
 a| 
`volume qtree modify` und den  `-security-style` Parameter zur Angabe des Sicherheitsstils einschließen.

|===
+
Die möglichen Optionen für den Qtree-Sicherheitsstil sind `unix`, `ntfs` oder `mixed`.

+
Wenn beim Erstellen eines Qtree kein Sicherheitsstil angegeben wird, ist der Standard-Sicherheitsstil  `mixed`.

+
Weitere Informationen zu den  `volume qtree create` oder  `volume qtree modify` Befehlen finden sich unter link:../volumes/index.html["Logische Speicherverwaltung"].

. Um die Konfiguration, einschließlich des Sicherheitsstils des von Ihnen erstellten Qtrees, anzuzeigen, ist folgender Befehl einzugeben: `volume qtree show -qtree qtree_name -instance`
