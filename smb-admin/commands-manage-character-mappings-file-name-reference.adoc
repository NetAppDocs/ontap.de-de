= ONTAP Befehle zur Verwaltung von Zeichenzuordnungen für die SMB-Dateinamenübersetzung
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können die Zeichenzuordnung verwalten, indem Sie Dateizeichenzuordnungen erstellen, ändern, Informationen darüber anzeigen oder löschen, die für die SMB-Dateinamenübersetzung auf FlexVol Volumes verwendet werden.

|===
| Wenn Sie möchten... | Verwenden Sie diesen Befehl... 


 a| 
Neue Dateizeichenzuordnungen erstellen
 a| 
`vserver cifs character-mapping create`



 a| 
Informationen zu Dateizeichenzuordnungen
 a| 
`vserver cifs character-mapping show`



 a| 
Vorhandene Dateizeichenzuordnungen ändern
 a| 
`vserver cifs character-mapping modify`



 a| 
Dateizeichenzuordnungen löschen
 a| 
`vserver cifs character-mapping delete`

|===
Weitere Informationen zu  `vserver cifs character-mapping` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/search.html?q=vserver+cifs+character-mapping["ONTAP-Befehlsreferenz"^].
