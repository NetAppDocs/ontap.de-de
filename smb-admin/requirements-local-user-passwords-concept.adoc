= Anforderungen an lokale ONTAP SMB-Benutzerpasswörter
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Standardmäßig müssen Passwörter lokaler Benutzer die Komplexitätsanforderungen erfüllen. Die Anforderungen an die Passwortkomplexität ähneln den in der Microsoft Windows _Lokalen Sicherheitsrichtlinie_ definierten Anforderungen.

Das Passwort muss folgende Kriterien erfüllen:

* Muss mindestens sechs Zeichen lang sein
* Darf nicht den Benutzernamen enthalten
* Muss Zeichen aus mindestens drei der folgenden vier Kategorien enthalten:
+
** Englische Großbuchstaben (A bis Z)
** Englische Kleinbuchstaben (a bis z)
** Ziffern im Dezimalsystem (0 bis 9)
** Sonderzeichen:
+
[listing]
----
~ ! @ # $ % {caret} & * _ - + = ` \ | ( ) [ ] : ; " ' < > , . ? /
----
