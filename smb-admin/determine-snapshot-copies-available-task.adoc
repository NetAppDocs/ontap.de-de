= Es kann festgestellt werden, ob ONTAP SMB Snapshots für die Verwendung mit Previous Versions verfügbar sind.
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Snapshots können auf der Registerkarte „Vorherige Versionen“ nur angezeigt werden, wenn für das Volume, das die Freigabe enthält, eine aktivierte Snapshot-Richtlinie gilt und die Volume-Konfiguration den Zugriff auf Snapshots zulässt. Die Ermittlung der Snapshot-Verfügbarkeit ist hilfreich, wenn einem Benutzer beim Zugriff auf frühere Versionen Unterstützung geboten wird.

.Schritte
. Es wird geprüft, ob für das Volume, auf dem sich die Freigabedaten befinden, automatische Snapshots aktiviert sind und ob Clients Zugriff auf die Snapshot-Verzeichnisse haben: `volume show -vserver vserver-name -volume volume-name -fields vserver,volume,snapdir-access,snapshot-policy,snapshot-count`
+
Die Ausgabe zeigt an, welche Snapshot-Richtlinie dem Volume zugeordnet ist, ob der Zugriff auf das Client-Snapshot-Verzeichnis aktiviert ist und wie viele Snapshots verfügbar sind.

. Es wird geprüft, ob die zugehörige Snapshot-Richtlinie aktiviert ist: `volume snapshot policy show -policy policy-name`
. Liste der verfügbaren Snapshots: `volume snapshot show -volume volume_name`
+
Weitere Informationen zum Konfigurieren und Verwalten von Snapshot-Richtlinien und Snapshot-Zeitplänen sind unter link:../data-protection/index.html["Datensicherung"] zu finden.
