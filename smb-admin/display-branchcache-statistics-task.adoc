= ONTAP SMB BranchCache Statistiken anzeigen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können BranchCache Statistiken anzeigen, um unter anderem festzustellen, wie gut das Caching funktioniert, ob Ihre Konfiguration zwischengespeicherte Inhalte für Clients bereitstellt und ob Hash-Dateien gelöscht wurden, um Platz für aktuellere Hash-Daten zu schaffen.

.Über diese Aufgabe
Das `hashd` Statistikobjekt enthält Zähler, die statistische Informationen über BranchCache-Hashwerte liefern. Das `cifs` Statistikobjekt enthält Zähler, die statistische Informationen über BranchCache-bezogene Aktivitäten liefern. Informationen über diese Objekte können auf der erweiterten Berechtigungsebene gesammelt und angezeigt werden.

.Schritte
. Die Berechtigungsstufe wird auf „Erweitert“ gesetzt: `set -privilege advanced`
+
[listing]
----
cluster1::> set -privilege advanced

Warning: These advanced commands are potentially dangerous; use them only when directed to do so by support personnel.
Do you want to continue? {y|n}: y
----
. Die BranchCache-bezogenen Zähler werden mit dem  `statistics catalog counter show` Befehl angezeigt.
+
[listing]
----
cluster1::*> statistics catalog counter show -object hashd

    branchcache_hash_created    Number of times a request to generate
                                BranchCache hash for a file succeeded.
    branchcache_hash_files_replaced
                                Number of times a BranchCache hash file was
                                deleted to make room for more recent hash
                                data. This happens if the hash store size is

    branchcache_hash_rejected   Number of times a request to generate
                                BranchCache hash data failed.
    branchcache_hash_store_bytes
                                Total number of bytes used to store hash data.
    branchcache_hash_store_size Total space used to store BranchCache hash

----
+
Weitere Informationen zu  `statistics catalog counter show` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/statistics-catalog-counter-show.html["ONTAP-Befehlsreferenz"^].

. Mit den  `statistics start`- und  `statistics stop`-Befehlen lassen sich BranchCache-bezogene Statistiken erfassen.
+
[listing]
----
cluster1::*> statistics start -object cifs -vserver vs1 -sample-id 11
Statistics collection is being started for Sample-id: 11

cluster1::*> statistics stop -sample-id 11
Statistics collection is being stopped for Sample-id: 11
----
+
Weitere Informationen zu  `statistics start` und  `statistics stop` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/search.html?q=statistics["ONTAP-Befehlsreferenz"^].

Weitere Informationen zu  `statistics show` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/statistics-show.html["ONTAP-Befehlsreferenz"^].

. Zurück zur Administratorrechteebene: `set -privilege admin`
+
[listing]
----
cluster1::*> set -privilege admin
----
