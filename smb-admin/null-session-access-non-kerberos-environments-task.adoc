= ONTAP SMB-Nullsitzungen ermöglichen den Zugriff auf Speicher in Nicht-Kerberos-Umgebungen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Der Zugriff über eine Null-Sitzung ermöglicht Berechtigungen für Netzwerkressourcen wie Speichersystemdaten sowie für Client-seitige Dienste, die unter dem lokalen System ausgeführt werden. Eine Null-Sitzung tritt auf, wenn ein Clientprozess das Konto „system“ verwendet, um auf eine Netzwerkressource zuzugreifen. Die Konfiguration einer Null-Sitzung ist spezifisch für die Nicht-Kerberos Authentifizierung.
