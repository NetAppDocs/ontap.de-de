= Informationen zur Wiederherstellung von ONTAP SMB-Dateien und -Ordnern mit früheren Versionen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Die Möglichkeit, Microsoft Previous Versions zu nutzen, ist für Dateisysteme verfügbar, die Snapshots in irgendeiner Form unterstützen und diese aktiviert haben. Snapshot Technologie ist ein integraler Bestandteil von ONTAP. Benutzer können Dateien und Ordner aus Snapshots von ihrem Windows-Client mithilfe der Funktion Microsoft Previous Versions wiederherstellen.

Die Funktion „Vorherige Versionen“ bietet eine Methode für Benutzer, Snapshots zu durchsuchen oder Daten aus einem Snapshot wiederherzustellen, ohne dass ein Speicheradministrator eingreifen muss. „Vorherige Versionen“ ist nicht konfigurierbar. Sie ist immer aktiviert. Wenn der Speicheradministrator Snapshots auf einer Freigabe bereitgestellt hat, kann der Benutzer mit „Vorherige Versionen“ die folgenden Aufgaben ausführen:

* Wiederherstellung versehentlich gelöschter Dateien.
* Wiederherstellung nach versehentlichem Überschreiben einer Datei.
* Dateiversionen während der Bearbeitung vergleichen.


Die in Snapshots gespeicherten Daten sind schreibgeschützt. Benutzer müssen eine Kopie einer Datei an einem anderen Speicherort speichern, um Änderungen an der Datei vorzunehmen. Snapshots werden regelmäßig gelöscht, daher müssen Benutzer Kopien der in „Vorherige Versionen“ enthaltenen Dateien erstellen, wenn sie eine frühere Version einer Datei dauerhaft aufbewahren möchten.
