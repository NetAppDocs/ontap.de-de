= Manuelle Aktualisierung der GPO-Einstellungen auf ONTAP SMB Servern
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Wenn Sie die Gruppenrichtlinienobjekt (GPO)-Einstellungen auf Ihrem CIFS-Server sofort aktualisieren möchten, können Sie die Einstellungen manuell aktualisieren. Es ist möglich, nur geänderte Einstellungen zu aktualisieren oder eine Aktualisierung für alle Einstellungen zu erzwingen, einschließlich der Einstellungen, die zuvor angewendet wurden, sich aber nicht geändert haben.

.Schritt
. Die entsprechende Maßnahme ausführen:
+
|===
| Wenn Sie ein Update durchführen möchten... | Geben Sie den Befehl ein... 


 a| 
Geänderte Gruppenrichtlinieneinstellungen
 a| 
`vserver cifs group-policy update -vserver _vserver_name_`



 a| 
Alle GPO-Einstellungen
 a| 
`vserver cifs group-policy update -vserver _vserver_name_ -force-reapply-all-settings true`

|===
