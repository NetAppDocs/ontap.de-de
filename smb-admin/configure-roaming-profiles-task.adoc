= Roaming-ONTAP SMB-Profile über die MMC „Active Directory-Benutzer und -Computer“ konfigurieren
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Wenn das Benutzerprofil automatisch verfügbar sein soll, sobald sich der Benutzer an einem beliebigen Computer im Netzwerk anmeldet, kann die Konfiguration von Roaming-Profilen über das MMC-Snap-In Active Directory Users and Computers erfolgen. Bei der Konfiguration von Roaming-Profilen auf Windows Server kann das Active Directory Administration Center verwendet werden.

.Schritte
. Auf dem Windows-Server die MMC „Active Directory Users and Computers“ (oder das Active Directory Administration Center auf Windows-Servern) öffnen.
. Den Benutzer ermitteln, für den ein Roaming-Profil konfiguriert werden soll.
. Mit der rechten Maustaste auf den Benutzer klicken und auf *Eigenschaften* klicken.
. Auf der Registerkarte *Profil* den Profilpfad zu der Freigabe eingeben, in der das Roaming-Profil des Benutzers gespeichert werden soll, gefolgt von  `%username%`.
+
Ein Profilpfad könnte beispielsweise wie folgt aussehen: `\\vs1.example.com\profiles\%username%`. Beim ersten Anmelden eines Benutzers wird  `%username%` durch den Benutzernamen ersetzt.

+
[NOTE]
====
Im Pfad  `\\vs1.example.com\profiles\%username%`, `profiles` ist der Freigabename einer Freigabe auf der Storage Virtual Machine (SVM) vs1, die Vollzugriffsrechte für Everyone hat.

====
. Klicken Sie auf *OK*.
