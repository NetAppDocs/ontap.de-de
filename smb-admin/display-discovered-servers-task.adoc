= Informationen zu ONTAP SMB erkannten Servern
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Informationen zu erkannten LDAP-Servern und Domänencontrollern auf Ihrem CIFS-Server können angezeigt werden.

.Schritt
. Um Informationen zu den gefundenen Servern anzuzeigen, den folgenden Befehl eingeben: `vserver cifs domain discovered-servers show`


.Beispiel
Das folgende Beispiel zeigt die erkannten Server für SVM vs1:

[listing]
----
cluster1::> vserver cifs domain discovered-servers show

Node: node1
Vserver: vs1

Domain Name     Type     Preference DC-Name     DC-Address    Status
--------------- -------- ---------- ----------- ------------- -------
example.com     MS-LDAP  adequate   DC-1        1.1.3.4       OK
example.com     MS-LDAP  adequate   DC-2        1.1.3.5       OK
example.com     MS-DC    adequate   DC-1        1.1.3.4       OK
example.com     MS-DC    adequate   DC-2        1.1.3.5       OK
----
