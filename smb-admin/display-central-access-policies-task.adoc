= Informationen zu den zentralen Zugriffsrichtlinien von ONTAP SMB anzeigen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Es lassen sich detaillierte Informationen zu den in Active Directory definierten zentralen Zugriffsrichtlinien anzeigen. Es lassen sich auch Informationen zu den zentralen Zugriffsrichtlinien anzeigen, die über Gruppenrichtlinienobjekte (GPOs) auf den CIFS-Server angewendet werden.

.Über diese Aufgabe
Standardmäßig werden folgende Informationen angezeigt:

* SVM-Name
* Name der zentralen Zugriffsrichtlinie
* SID
* Beschreibung
* Erstellungszeit
* Änderungszeit
* Mitgliedsregeln


[NOTE]
====
CIFS-Server im Arbeitsgruppenmodus werden nicht angezeigt, da sie keine Gruppenrichtlinien (GPOs) unterstützen.

====
.Schritt
. Informationen zu zentralen Zugriffsrichtlinien werden durch eine der folgenden Aktionen angezeigt:
+
|===
| Wenn Informationen zu allen zentralen Zugriffsrichtlinien angezeigt werden sollen... | Geben Sie den Befehl ein... 


 a| 
In Active Directory definiert
 a| 
`vserver cifs group-policy central-access-policy show-defined -vserver _vserver_name_`



 a| 
Angewendet auf einen CIFS-Server
 a| 
`vserver cifs group-policy central-access-policy show-applied -vserver _vserver_name_`

|===


.Beispiel
Das folgende Beispiel zeigt Informationen zu allen zentralen Zugriffsrichtlinien, die in Active Directory definiert sind:

[listing]
----
cluster1::> vserver cifs group-policy central-access-policy show-defined

       Description: policy #1
     Creation Time: Tue Oct 22 09:34:13 2013
 Modification Time: Wed Oct 23 08:59:15 2013
      Member Rules: r1

vs1      p2                   S-1-17-1885229282-1100162114-134354072-822349040
       Description: policy #2
     Creation Time: Tue Oct 22 10:28:20 2013
 Modification Time: Thu Oct 31 10:25:32 2013

                    r2
----
