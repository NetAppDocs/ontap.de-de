= Informationen zu den Anforderungen für die Verwendung von Roaming ONTAP SMB-Profilen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Bevor Sie Microsofts Roaming-Profile mit Ihrem CIFS-Server verwenden können, müssen Sie wissen, welche Versionen von ONTAP und SMB sowie welche Windows-Clients diese Funktion unterstützen.



[[ontap-version-requirements]]
== ONTAP Versionsanforderungen

ONTAP unterstützt Roaming-Profile.



[[smb-protocol-version-requirements]]
== Anforderungen an die SMB-Protokollversion

Für Storage Virtual Machines (SVM) unterstützt ONTAP Roaming-Profile auf allen Versionen von SMB.



[[windows-client-requirements]]
== Windows Client Anforderungen

Bevor ein Benutzer die Roaming-Profile verwenden kann, muss der Windows Client diese Funktion unterstützen.

Die aktuellsten Informationen darüber, welche Windows-Clients Roaming-Profile unterstützen, sind in der Interoperabilitätsmatrix zu finden.

https://mysupport.netapp.com/matrix["NetApp Interoperabilitätsmatrix-Tool"^]
