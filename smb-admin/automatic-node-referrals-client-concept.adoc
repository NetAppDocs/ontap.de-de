= Client-seitige ONTAP SMB-Informationen zur automatischen Knotenweiterleitung mit einem Windows-Client überwachen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Um festzustellen, welche Empfehlungen aus Sicht des Clients erfolgen, kann das Windows `dfsutil.exe` Dienstprogramm verwendet werden.

Das in Windows 7 und späteren Versionen enthaltene Remoteserver-Verwaltungstool (RSAT) enthält das  `dfsutil.exe` Dienstprogramm. Mit diesem Dienstprogramm lassen sich Informationen zum Inhalt des Verweis-Caches sowie Informationen zu jedem Verweis anzeigen, den der Client aktuell verwendet. Das Dienstprogramm kann auch zum Leeren des Verweis-Caches des Clients verwendet werden. Weitere Informationen bietet die Microsoft TechNet Library.
