= ONTAP SMB-Sicherheitsrichtlinien anwenden
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Das Anwenden einer Überwachungsrichtlinie auf SVMs ist der letzte Schritt beim Erstellen und Anwenden von NTFS-ACLs auf Dateien oder Ordner.

.Über diese Aufgabe
Sie können die in der Sicherheitsrichtlinie definierten Sicherheitseinstellungen auf NTFS-Dateien und -Ordner anwenden, die sich innerhalb von FlexVol Volumes befinden (NTFS oder gemischter Sicherheitsstil).


NOTE: Beim Anwenden einer Überwachungsrichtlinie und zugehöriger SACLs werden alle vorhandenen DACLs überschrieben. Beim Anwenden einer Sicherheitsrichtlinie und der zugehörigen DACLs werden alle vorhandenen DACLs überschrieben. Vor dem Erstellen und Anwenden neuer Sicherheitsrichtlinien empfiehlt sich eine Überprüfung der bestehenden Richtlinien.

.Schritt
. Eine Sicherheitsrichtlinie anwenden: `vserver security file-directory apply -vserver vserver_name -policy-name policy_name`
+
`vserver security file-directory apply -vserver vs1 -policy-name policy1`

+
Der Auftrag zur Anwendung der Richtlinie wird eingeplant und die Job-ID wird zurückgegeben.
