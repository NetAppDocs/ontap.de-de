= Die Verwendung der Microsoft Management Console mit lokalen ONTAP SMB-Benutzern und -Gruppen wird erläutert.
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Informationen zu lokalen Benutzern und Gruppen können in der Microsoft Management Console angezeigt werden. Mit dieser Version von ONTAP ist es nicht möglich, andere Verwaltungsaufgaben für lokale Benutzer und Gruppen über die Microsoft Management Console durchzuführen.
