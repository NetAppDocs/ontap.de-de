= ONTAP Befehle zur Verwaltung von SMB-Freigaben
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie verwenden die  `vserver cifs share`- und  `vserver cifs share properties`-Befehle, um SMB-Freigaben zu verwalten.

|===
| Wenn Sie möchten... | Verwenden Sie diesen Befehl... 


 a| 
Erstellen Sie eine SMB-Freigabe
 a| 
`vserver cifs share create`



 a| 
SMB-Freigaben anzeigen
 a| 
`vserver cifs share show`



 a| 
Ändern einer SMB-Freigabe
 a| 
`vserver cifs share modify`



 a| 
Eine SMB-Freigabe löschen
 a| 
`vserver cifs share delete`



 a| 
Freigabeeigenschaften zu einer bestehenden Freigabe hinzufügen
 a| 
`vserver cifs share properties add`



 a| 
Freigabeeigenschaften von einer bestehenden Freigabe entfernen
 a| 
`vserver cifs share properties remove`



 a| 
Informationen zu Aktieneigenschaften
 a| 
`vserver cifs share properties show`

|===
Weitere Informationen zu  `vserver cifs` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/search.html?q=vserver+cifs["ONTAP-Befehlsreferenz"^].
