= Zugriffe mithilfe von Exportrichtlinien mit ONTAP System Manager verwalten

[role="lead"]
Linux-Clients erhalten Zugriff auf NFS-Server mithilfe von Exportrichtlinien.

Dieses Verfahren erstellt oder ändert Exportrichtlinien für ein link:task_nas_enable_linux_nfs.html["vorhandene NFS-fähige Storage-VM"].

.Schritte
. Im System Manager auf *Speicher* > *Volumes* klicken.
. Klicken Sie auf ein NFS-fähiges Volume und wählen Sie *Mehr*.
. Auf *Exportrichtlinie bearbeiten* klicken und dann *Vorhandene Richtlinie auswählen* oder *Neue Richtlinie hinzufügen* wählen.
