= ONTAP VLAN-Informationen
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
Sie können Informationen über ein bestimmtes VLAN oder über alle VLANs im Cluster anzeigen.

.Über diese Aufgabe
Detaillierte Informationen zu jedem VLAN werden durch Angabe des  `-instance` Parameters angezeigt. Spezifische Informationen werden durch Angabe von Feldnamen mit dem  `-fields` Parameter angezeigt.

.Schritt
Informationen zu VLANs werden mit dem  `network port vlan show` Befehl angezeigt. Der folgende Befehl zeigt Informationen zu allen VLANs im Cluster an:

....
network port vlan show
                 Network Network
Node   VLAN Name Port    VLAN ID  MAC Address
------ --------- ------- -------- -----------------
cluster-1-01
       a0a-10    a0a     10       02:a0:98:06:10:b2
       a0a-20    a0a     20       02:a0:98:06:10:b2
       a0a-30    a0a     30       02:a0:98:06:10:b2
       a0a-40    a0a     40       02:a0:98:06:10:b2
       a0a-50    a0a     50       02:a0:98:06:10:b2
cluster-1-02
       a0a-10    a0a     10       02:a0:98:06:10:ca
       a0a-20    a0a     20       02:a0:98:06:10:ca
       a0a-30    a0a     30       02:a0:98:06:10:ca
       a0a-40    a0a     40       02:a0:98:06:10:ca
       a0a-50    a0a     50       02:a0:98:06:10:ca
....
Weitere Informationen zu  `network port vlan show` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/network-port-vlan-show.html["ONTAP-Befehlsreferenz"^].
