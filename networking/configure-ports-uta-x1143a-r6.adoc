= UTA X1143A-R6 Ports für das ONTAP Netzwerk konfigurieren
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Standardmäßig ist der X1143A-R6 Unified-Zieladapter im FC-Zielmodus konfiguriert, Sie können seine Ports jedoch entweder als 10-Gb-Ethernet- und FCoE- (CNA-) Ports oder als 16-Gb-FC-Initiator- oder -Zielports konfigurieren. Dies erfordert unterschiedliche SFP+-Adapter.

Bei Konfiguration für Ethernet und FCoE unterstützen X1143A-R6 Adapter gleichzeitigen NIC- und FCoE-Zieldatenverkehr über denselben 10-GBE-Port. Bei FC-Konfiguration kann jedes Zwei-Port-Paar, das denselben ASIC nutzt, individuell für den FC-Ziel- oder FC-Initiator-Modus konfiguriert werden. Das bedeutet, dass ein einzelner X1143A-R6 Adapter den FC-Zielmodus auf einem Zwei-Port-Paar und den FC-Initiator-Modus auf einem anderen Zwei-Port-Paar unterstützen kann. Portpaare, die mit demselben ASIC verbunden sind, müssen im selben Modus konfiguriert werden.

Im FC-Modus verhält sich der X1143A-R6 Adapter wie jedes andere FC-Gerät mit Geschwindigkeiten von bis zu 16 Gbit/s. Im CNA-Modus kann der X1143A-R6 Adapter für gleichzeitigen NIC- und FCoE-Datenverkehr über denselben 10 GbE-Port verwendet werden. Der CNA-Modus unterstützt für die FCoE-Funktion ausschließlich den FC-Zielmodus.

Um den einheitlichen Zieladapter (X1143A-R6) zu konfigurieren, müssen die beiden benachbarten Ports auf demselben Chip im gleichen Personality-Modus konfiguriert werden.

.Schritte
. Portkonfiguration anzeigen:
+
[source, cli]
----
system hardware unified-connect show
----
. Die Ports können nach Bedarf für Fibre Channel (FC) oder Converged Network Adapter (CNA) konfiguriert werden:
+
[source, cli]
----
system node hardware unified-connect modify -node <node_name> -adapter <adapter_name> -mode {fcp|cna}
----
. Die entsprechenden Kabel für FC oder 10 Gbit/s Ethernet anschließen.
. Es sollte überprüft werden, ob das richtige SFP+ installiert ist:
+
[source, cli]
----
network fcp adapter show -instance -node -adapter
----
+
Für CNA sollte ein 10Gb Ethernet SFP verwendet werden. Für FC sollte abhängig vom angeschlossenen FC-Fabric entweder ein 8-Gb-SFP oder ein 16-Gb-SFP verwendet werden.
