= ONTAP Befehle zur Verwaltung des Firewall-Dienstes und der Richtlinien
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
Die  `system services firewall` Befehle dienen zur Verwaltung des Firewall-Dienstes, die  `system services firewall policy` Befehle zur Verwaltung von Firewall-Richtlinien und der  `network interface modify` Befehl zur Verwaltung der Firewall-Einstellungen für LIFs.

Ab ONTAP 9.10.1:

* Firewall-Richtlinien sind veraltet und werden durch LIF-Dienstrichtlinien ersetzt. Zuvor wurde die integrierte Firewall mithilfe von Firewall-Richtlinien verwaltet. Diese Funktionalität wird jetzt mit einer LIF-Dienstrichtlinie umgesetzt.
* Alle Firewall-Richtlinien sind leer und öffnen keine Ports in der zugrunde liegenden Firewall. Stattdessen müssen alle Ports über eine LIF-Dienstrichtlinie geöffnet werden.
* Nach einem Upgrade auf Version 9.10.1 oder höher ist kein weiteres Vorgehen erforderlich, um von Firewall-Richtlinien auf LIF-Dienstrichtlinien umzustellen. Das System erstellt automatisch LIF-Dienstrichtlinien, die mit den in der vorherigen ONTAP Version verwendeten Firewall-Richtlinien konsistent sind. Wenn Skripte oder andere Tools verwendet werden, die benutzerdefinierte Firewall-Richtlinien erstellen und verwalten, kann es erforderlich sein, diese Skripte so anzupassen, dass stattdessen benutzerdefinierte Dienstrichtlinien erstellt werden.


Weitere Informationen finden sich unter link:lifs_and_service_policies96.html["LIFs und Servicerichtlinien in ONTAP 9.6 und später"].

|===


| Wenn Sie möchten... | Verwenden Sie diesen Befehl... 


 a| 
Firewall-Service aktivieren oder deaktivieren
 a| 
`system services firewall modify`



 a| 
Aktuelle Konfiguration des Firewall-Dienstes anzeigen
 a| 
`system services firewall show`



 a| 
Eine Firewall-Richtlinie erstellen oder einen Dienst zu einer vorhandenen Firewall-Richtlinie hinzufügen
 a| 
`system services firewall policy create`



 a| 
Eine Firewall-Richtlinie auf eine LIF anwenden
 a| 
`network interface modify -lif lifname -firewall-policy`



 a| 
Ändern der mit einer Firewall-Richtlinie verknüpften IP-Adressen und Netzmasken
 a| 
`system services firewall policy modify`



 a| 
Informationen zu Firewall-Richtlinien
 a| 
`system services firewall policy show`



 a| 
Eine neue Firewall-Richtlinie erstellen, die eine exakte Kopie einer vorhandenen Richtlinie ist
 a| 
`system services firewall policy clone`



 a| 
Eine Firewall-Richtlinie löschen, die von einem LIF nicht verwendet wird
 a| 
`system services firewall policy delete`
