= IPspaces aus dem ONTAP Netzwerk löschen
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
Wenn ein IPspace nicht mehr benötigt wird, kann er gelöscht werden.

.Bevor Sie beginnen
Dem IPspace, den Sie löschen möchten, dürfen keine Broadcast-Domänen, Netzwerkschnittstellen oder SVMs zugeordnet sein.

Die systemdefinierten „Default“ und „Cluster“ IPspaces können nicht gelöscht werden.

.Schritt
Löschen eines IPspace:

....
network ipspace delete -ipspace ipspace_name
....
Der folgende Befehl löscht den IPspace ipspace1 aus dem Cluster:

....
network ipspace delete -ipspace ipspace1
....
Weitere Informationen zu  `network ipspace delete` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/network-ipspace-delete.html["ONTAP-Befehlsreferenz"^].
