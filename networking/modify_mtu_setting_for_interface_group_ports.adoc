= Die MTU-Einstellung für ONTAP Schnittstellengruppen-Ports anpassen
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
Um die MTU-Einstellung für Schnittstellengruppen zu ändern, muss die MTU der Broadcast-Domäne geändert werden.

Die VLAN-MTU-Größe sollte der Broadcast-Domänen-MTU der zugrunde liegenden Schnittstellengruppen und physischen Ports entsprechen. Falls für ein VLAN eine andere VLAN-Einstellung erforderlich ist, darf diese die von der zugrunde liegenden Broadcast-Domäne festgelegte Größe nicht überschreiten.

.Schritte
. Die Einstellungen der Broadcast-Domäne können angepasst werden:
+
`broadcast-domain modify -broadcast-domain _broadcast_domain_name_ -mtu _mtu_setting_`

+
Es wird folgende Warnmeldung angezeigt:

+
....
Warning: Changing broadcast domain settings will cause a momentary data-serving interruption.
Do you want to continue? {y|n}: y
....
. Geben Sie y ein, um fortzufahren.
. Es sollte überprüft werden, ob die MTU-Einstellungen korrekt geändert wurden:
+
`network port show`
