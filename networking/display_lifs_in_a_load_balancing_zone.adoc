= ONTAP LIFs in einer Load Balancing Zone anzeigen
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
Sie können überprüfen, ob eine Load-Balancing-Zone korrekt konfiguriert ist, indem Sie alle LIFs anzeigen, die zu ihr gehören. Sie können außerdem die Load-Balancing-Zone einer bestimmten LIF oder die Load-Balancing-Zonen aller LIFs anzeigen.

.Schritt
Die gewünschten LIFs und Load-Balancing-Details lassen sich mit einem der folgenden Befehle anzeigen.

[cols="30,70"]
|===


| Zum Anzeigen... | Eingeben... 


 a| 
LIFs in einer bestimmten Lastverteilungszone
 a| 
`network interface show -dns-zone zone_name`

`zone_name`Gibt den Namen der Lastausgleichszone an.



 a| 
Die Lastausgleichszone eines bestimmten LIF
 a| 
`network interface show -lif lif_name -fields dns-zone`



 a| 
Die Lastausgleichszonen aller LIFs
 a| 
`network interface show -fields dns-zone`

|===


[[examples-of-displaying-load-balancing-zones-for-lifs]]
== Beispiele für die Anzeige von Lastausgleichszonen für LIFs

Der folgende Befehl zeigt die Details aller LIFs in der Lastausgleichszone storage.company.com für SVM vs0 an:

....
net int show -vserver vs0 -dns-zone storage.company.com

         Logical    Status     Network           Current   Current Is
Vserver  Interface  Admin/Oper Address/Mask      Node      Port    Home
-------- ---------- ---------- ----------------- --------- ------- ----
vs0
         lif3         up/up    10.98.226.225/20  ndeux-11  e0c     true
         lif4         up/up    10.98.224.23/20   ndeux-21  e0c     true
         lif5         up/up    10.98.239.65/20   ndeux-11  e0c     true
         lif6         up/up    10.98.239.66/20   ndeux-11  e0c     true
         lif7         up/up    10.98.239.63/20   ndeux-21  e0c     true
         lif8         up/up    10.98.239.64/20   ndeux-21  e0c     true
....
Der folgende Befehl zeigt die DNS-Zonendetails der LIF data3 an:

....
network interface show -lif data3 -fields dns-zone
Vserver  lif    dns-zone
-------  -----  --------
vs0      data3  storage.company.com
....
Der folgende Befehl zeigt die Liste aller LIFs im Cluster und ihrer entsprechenden DNS-Zonen an:

....
network interface show -fields dns-zone
Vserver   lif          dns-zone
-------   ------------ --------
cluster   cluster_mgmt none
ndeux-21  clus1        none
ndeux-21  clus2        none
ndeux-21  mgmt1        none
vs0       data1        storage.company.com
vs0       data2        storage.company.com
....
Weitere Informationen zu  `network interface show` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/network-interface-show.html["ONTAP-Befehlsreferenz"^].
