= Routinginformationen für das ONTAP Netzwerk
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
Informationen über Routen innerhalb einer SVM können angezeigt werden.

.Schritt
Je nachdem, welche Art von Routing-Informationen angezeigt werden sollen, ist der entsprechende Befehl einzugeben:

[cols="40,60"]
|===


| Informationen über ... anzeigen | Eingeben... 


 a| 
Statische Routen, pro SVM
 a| 
`network route show`



 a| 
LIFs auf jeder Route, pro SVM
 a| 
`network route show-lifs`

|===
Detaillierte Informationen zu jeder Route werden durch Angabe des  `-instance` Parameters angezeigt. Der folgende Befehl zeigt die statischen Routen innerhalb der SVMs in cluster- 1 an:

....
network route show
Vserver             Destination     Gateway         Metric
------------------- --------------- --------------- ------
Cluster
                    0.0.0.0/0       10.63.0.1       10
cluster-1
                    0.0.0.0/0       198.51.9.1      10
vs1
                    0.0.0.0/0       192.0.2.1       20
vs3
                    0.0.0.0/0       192.0.2.1       20
....
Der folgende Befehl zeigt die Zuordnung von statischen Routen und logischen Schnittstellen (LIFs) innerhalb aller SVMs in Cluster-1 an:

....
network route show-lifs
Vserver: Cluster
Destination             Gateway                 Logical Interfaces
----------------------  ----------------------  -------------------
0.0.0.0/0               10.63.0.1               -

Vserver: cluster-1
Destination             Gateway                 Logical Interfaces
----------------------  ----------------------  -------------------
0.0.0.0/0               198.51.9.1              cluster_mgmt,
                                                cluster-1_mgmt1,

Vserver: vs1
Destination             Gateway                 Logical Interfaces
----------------------  ----------------------  -------------------
0.0.0.0/0               192.0.2.1               data1_1, data1_2

Vserver: vs3
Destination             Gateway                 Logical Interfaces
----------------------  ----------------------  -------------------
0.0.0.0/0               192.0.2.1               data2_1, data2_2
....
Weitere Informationen zu  `network route show` und  `network route show-lifs` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/search.html?q=network+route+show["ONTAP-Befehlsreferenz"^].
