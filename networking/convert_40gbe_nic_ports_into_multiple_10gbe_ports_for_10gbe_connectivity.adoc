= 10GbE-Ports für ONTAP Netzwerke durch Konvertierung von 40GbE-NIC-Ports erstellen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können die X1144A-R6 und die X91440A-R6 40GbE Netzwerkschnittstellenkarten (NICs) so umrüsten, dass sie vier 10GbE Ports unterstützen.

Wenn eine Hardwareplattform, die eine dieser Netzwerkkarten unterstützt, mit einem Cluster verbunden wird, der 10GbE Cluster Interconnect und Kundendatenverbindungen unterstützt, muss die Netzwerkkarte so umgerüstet werden, dass die erforderlichen 10GbE-Verbindungen bereitgestellt werden.

.Bevor Sie beginnen
Es muss ein unterstütztes Breakout-Kabel verwendet werden.

.Über diese Aufgabe
Eine vollständige Liste der Plattformen, die NICs unterstützen, ist unter  https://hwu.netapp.com/["Hardware Universe"^] zu finden.


NOTE: Bei der Netzwerkkarte X1144A-R6 kann nur Port A für die vier 10GbE-Verbindungen umgerüstet werden. Nach der Umrüstung von Port A ist Port e nicht mehr verfügbar.

.Schritte
. In den Wartungsmodus wechseln.
. Die Netzwerkkarte von 40GbE-Unterstützung auf 10GbE-Unterstützung konvertieren.
+
[listing]
----
nicadmin convert -m [40G | 10G] [port-name]
----
. Nach der Ausführung des Konvertierungsbefehls den Knoten anhalten.
. Das Kabel installieren oder austauschen.
. Je nach Hardwaremodell wird der SP (Service Processor) oder BMC (Baseboard Management Controller) verwendet, um den Knoten neu zu starten, damit die Umstellung wirksam wird.
