= Statische Routen aus dem ONTAP Netzwerk löschen
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
Eine nicht benötigte statische Route kann aus einer Storage Virtual Machine (SVM) gelöscht werden.

.Schritt
Mit dem  `network route delete`Befehl lässt sich eine statische Route löschen.

Das folgende Beispiel löscht eine statische Route, die mit SVM vs0, dem Gateway 10.63.0.1 und der Ziel-IP-Adresse 0.0.0.0/0 verknüpft ist:

....
network route delete -vserver vs0 -gateway 10.63.0.1 -destination 0.0.0.0/0
....
Weitere Informationen zu  `network route delete` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/network-route-delete.html["ONTAP-Befehlsreferenz"^].
