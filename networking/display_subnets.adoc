= Subnetze für das ONTAP Netzwerk anzeigen
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
Sie können die Liste der IP-Adressen anzeigen, die jedem Subnetz innerhalb eines IPspace zugewiesen sind. Die Ausgabe zeigt außerdem die Gesamtzahl der in jedem Subnetz verfügbaren IP-Adressen sowie die Anzahl der aktuell verwendeten Adressen.

Die Vorgehensweise hängt von der verwendeten Schnittstelle ab, System Manager oder CLI:

[role="tabbed-block"]
====
.System Manager
--
*Ab ONTAP 9.12.0 kann mit System Manager die Anzeige von Subnetzen erfolgen.*

.Schritte
. *Netzwerk > Übersicht > Subnetze* auswählen.
. Die Liste der Subnetze wird angezeigt.


--
.CLI
--
*Mit der CLI Subnetze anzeigen*

.Schritt
Die Liste der Subnetze und der zugehörigen IP-Adressbereiche, die in diesen Subnetzen verwendet werden, wird angezeigt:

....
network subnet show
....
Der folgende Befehl zeigt die Subnetze und die Eigenschaften der Subnetze an:

....
network subnet show

IPspace: Default
Subnet                 Broadcast               Avail/
Name   Subnet          Domain    Gateway       Total   Ranges
-----  --------------- --------- ------------ -------- -----------------------
sub1   192.0.2.0/24     bcast1   192.0.2.1     5/9     192.0.2.92-192.0.2.100
sub3   198.51.100.0/24  bcast3   198.51.100.1   3/3     198.51.100.7,198.51.100.9
....
Weitere Informationen zu  `network subnet show` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/network-subnet-show.html["ONTAP-Befehlsreferenz"^].
