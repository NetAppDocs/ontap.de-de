= Das ONTAP Netzwerk lässt sich mit System Manager visualisieren.
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
Ab ONTAP 9.8 ist es möglich, mit dem System Manager eine Grafik anzuzeigen, die die Komponenten und die Konfiguration Ihres Netzwerks darstellt, sodass die Netzwerkverbindungspfade über Hosts, Ports, SVMs, Volumes und mehr sichtbar werden. Ab ONTAP 9.12.1 kann die LIF- und Subnetzzuordnung im Raster „Netzwerkschnittstellen“ angezeigt werden.

Die Grafik wird angezeigt, wenn *Netzwerk > Übersicht* ausgewählt wird oder wenn image:icon_arrow.gif["Pfeilsymbol"] im Bereich *Netzwerk* des Dashboards ausgewählt wird.

Die folgenden Komponentenkategorien sind in der Grafik dargestellt:

* Hosts
* Speicheranschlüsse
* Netzwerkschnittstellen
* Storage VMs
* Datenzugriffskomponenten


Jeder Abschnitt zeigt zusätzliche Details an, über die Sie mit der Maus fahren oder die Sie auswählen können, um Aufgaben zur Netzwerkverwaltung und -konfiguration auszuführen.

Wenn Sie den klassischen System Manager verwenden (nur in ONTAP 9.7 und früher verfügbar), siehe link:https://docs.netapp.com/us-en/ontap-system-manager-classic/online-help-96-97/concept_managing_network.html["Netzwerkverwaltung"^].



[[examples]]
== Beispiele

Nachfolgend einige Beispiele für die vielfältigen Möglichkeiten, wie mit der Grafik interagiert werden kann, um Details zu den einzelnen Komponenten anzuzeigen oder Aktionen zur Verwaltung Ihres Netzwerks zu initiieren:

* Durch Auswahl eines Hosts werden dessen Konfiguration, die zugehörigen Ports, Netzwerkschnittstellen, Storage-VMs und Datenzugriffskomponenten angezeigt.
* Fahren Sie mit der Maus über die Anzahl der Volumes in einer Storage-VM, um ein Volume auszuwählen und dessen Details anzuzeigen.
* Eine iSCSI-Schnittstelle kann ausgewählt werden, um deren Leistung in der vergangenen Woche anzuzeigen.
* Klicken Sie auf image:icon_kabob.gif["Symbol für Menüoptionen"] neben einer Komponente, um Aktionen zur Änderung dieser Komponente zu initiieren.
* Schnell lässt sich feststellen, wo in Ihrem Netzwerk Probleme auftreten könnten, angezeigt durch ein „X“ neben fehlerhaften Komponenten.
