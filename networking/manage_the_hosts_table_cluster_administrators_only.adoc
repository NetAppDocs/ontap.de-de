= ONTAP Befehle zur Verwaltung der ONTAP Hosts-Tabelle
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
Ein Clusteradministrator kann die Hostnameneinträge in der Hosttabelle der administrativen Storage Virtual Machine (SVM) hinzufügen, ändern, löschen und anzeigen. Ein SVM-Administrator kann die Hostnameneinträge nur für die zugewiesene SVM konfigurieren.



[[commands-for-managing-local-host-name-entries]]
== Befehle zum Verwalten lokaler Hostname-Einträge

Der  `vserver services name-service dns hosts`Befehl kann verwendet werden, um DNS-Hosttabelleneinträge zu erstellen, zu ändern oder zu löschen.

Beim Erstellen oder Ändern der DNS-Host-Name-Einträge können Sie mehrere Alias-Adressen, getrennt durch Kommas, angeben.

[cols="30,70"]
|===


| Wenn Sie möchten... | Verwenden Sie diesen Befehl... 


 a| 
Einen DNS-Hostnamen-Eintrag erstellen
 a| 
`vserver services name-service dns hosts create`



 a| 
Einen DNS-Hostnamen-Eintrag ändern
 a| 
`vserver services name-service dns hosts modify`



 a| 
Einen DNS-Host-Name-Eintrag löschen
 a| 
`vserver services name-service dns hosts delete`

|===
Weitere Informationen zu den  `vserver services name-service dns hosts`Befehlen finden sich in der  https://docs.netapp.com/us-en/ontap-cli["ONTAP-Befehlsreferenz"^].
