= Informationen zu Subnetzen für das ONTAP Netzwerk
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
Subnetze ermöglichen die Zuweisung spezifischer IP-Adressblöcke oder -pools für Ihre ONTAP Netzwerkkonfiguration. Dadurch ist das Erstellen von LIFs einfacher, da ein Subnetznamen angegeben werden kann, anstatt IP-Adresse und Netzwerkmaske einzeln angeben zu müssen.

Innerhalb einer Broadcast-Domäne wird ein Subnetz erstellt, das einen Pool von IP-Adressen desselben Layer-3-Subnetzes enthält. IP-Adressen eines Subnetzes werden Ports in der Broadcast-Domäne zugewiesen, wenn LIFs erstellt werden. Nach dem Entfernen von LIFs werden die IP-Adressen dem Subnetzpool zurückgegeben und stehen für zukünftige LIFs zur Verfügung.

Es wird empfohlen, Subnetze zu verwenden, da diese die Verwaltung von IP-Adressen erheblich vereinfachen und die Erstellung von LIFs zu einem einfacheren Prozess machen. Wenn beim Definieren eines Subnetzes ein Gateway angegeben wird, wird beim Erstellen eines LIFs mit diesem Subnetz automatisch eine Standardroute zu diesem Gateway zur SVM hinzugefügt.
