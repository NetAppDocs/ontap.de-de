= ONTAP LIF-Informationen
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
Detaillierte Informationen über ein LIF geben Aufschluss über dessen Konfiguration.

Diese Informationen können auch zur Diagnose grundlegender LIF-Probleme herangezogen werden, wie etwa zur Überprüfung auf doppelte IP-Adressen oder zur Verifizierung, ob der Netzwerkport dem richtigen Subnetz zugeordnet ist. Administratoren von storage virtual machines (SVM) können nur die Informationen zu den mit der SVM verknüpften LIFs anzeigen.

.Über diese Aufgabe
Folgende Informationen werden angezeigt:

* Mit dem LIF verknüpfte IP-Adresse
* Verwaltungsstatus des LIF
* Betriebszustand des LIF
+
Der Betriebsstatus der Daten-LIFs wird durch den Status der SVM bestimmt, mit der die Daten-LIFs verknüpft sind. Wenn die SVM angehalten wird, ändert sich der Betriebsstatus des LIF auf „down“. Wenn die SVM wieder gestartet wird, ändert sich der Betriebsstatus auf „up“.

* Knoten und Port, auf dem sich die LIF befindet


Wenn Daten für ein Feld nicht verfügbar sind (z. B. wenn keine erweiterten Statusinformationen vorliegen), wird der Feldwert als  `-` aufgeführt.

.Schritt
LIF-Informationen werden mithilfe des  `network interface show` Befehls angezeigt.

Detaillierte Informationen zu jedem LIF werden durch Angabe des Parameters -instance angezeigt, oder spezifische Informationen durch Angabe von Feldnamen mit dem Parameter -fields.
