= Statische Routen für das ONTAP Netzwerk erstellen
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
Sie können innerhalb einer Storage Virtual Machine (SVM) statische Routen erstellen, um zu steuern, wie LIFs das Netzwerk für ausgehenden Datenverkehr nutzen.

Wenn ein Routeneintrag erstellt wird, der einem SVM zugeordnet ist, wird die Route von allen LIFs verwendet, die dem angegebenen SVM gehören und sich im selben Subnetz wie das Gateway befinden.

.Schritt
Mit dem  `network route create`Befehl wird eine Route erstellt.

....
network route create -vserver vs0 -destination 0.0.0.0/0 -gateway 10.61.208.1
....
Weitere Informationen zu  `network route create` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/network-route-create.html["ONTAP-Befehlsreferenz"^].
