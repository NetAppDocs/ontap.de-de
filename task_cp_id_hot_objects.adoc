= Häufig genutzte Volumes und andere Objekte im ONTAP System Manager erkennen
:hardbreaks:
:toclevels: 1
:allow-uri-read: 
:toclevels: 1
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
Die Cluster-Performance lässt sich steigern, wenn die häufig genutzten Volumes (Hot Volumes) und Daten (Hot Objects) identifiziert werden.


NOTE: Ab ONTAP 9.10.1 ist die Aktivitätsverfolgungsfunktion in File System Analytics verfügbar, mit der häufig genutzte Objekte in einem Volume überwacht werden können.

.Schritte
. *Speicher > Volumes* auswählen.
. Die Spalten für IOPS, Latenz und Durchsatz können gefiltert werden, um die häufig aufgerufenen Volumes und Daten anzuzeigen.
