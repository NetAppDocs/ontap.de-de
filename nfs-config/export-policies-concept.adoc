= Informationen zur Sicherung des ONTAP NFS-Zugriffs mithilfe von Exportrichtlinien
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können Exportrichtlinien verwenden, um den NFS-Zugriff auf Volumes oder Qtrees auf Clients zu beschränken, die bestimmten Parametern entsprechen. Beim Bereitstellen neuen Speichers ist es möglich, eine vorhandene Richtlinie und Regeln zu verwenden, einer vorhandenen Richtlinie Regeln hinzuzufügen oder eine neue Richtlinie und Regeln zu erstellen. Auch die Konfiguration der Exportrichtlinien kann überprüft werden.

[NOTE]
====
Ab ONTAP 9.3 kann die Überprüfung der Exportrichtlinienkonfiguration als Hintergrundprozess aktiviert werden, der alle Regelverstöße in einer Fehlerregelliste protokolliert. Die  `vserver export-policy config-checker` Befehle rufen die Überprüfung auf und zeigen die Ergebnisse an, die zur Überprüfung der Konfiguration und zum Löschen fehlerhafter Regeln aus der Richtlinie verwendet werden können. Die Befehle validieren die Exportkonfiguration nur für Hostnamen, Netzgruppen und anonyme Benutzer.
