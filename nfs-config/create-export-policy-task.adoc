= Erstellen einer ONTAP NFS-Exportrichtlinie
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Bevor Exportregeln erstellt werden, ist eine Exportrichtlinie zu erstellen, die diese enthält. Mit dem  `vserver export-policy create` Befehl kann eine Exportrichtlinie erstellt werden.

.Schritte
. Eine Exportrichtlinie erstellen:
+
`vserver export-policy create -vserver _vserver_name_ -policyname _policy_name_`

+
Der Name der Richtlinie kann bis zu 256 Zeichen lang sein.

. Es wird überprüft, ob die Exportrichtlinie erstellt wurde:
+
`vserver export-policy show -policyname _policy_name_`



.Beispiel
Die folgenden Befehle erstellen und überprüfen die Erstellung einer Exportrichtlinie mit dem Namen exp1 auf der SVM mit dem Namen vs1:

[listing]
----
vs1::> vserver export-policy create -vserver vs1 -policyname exp1

vs1::> vserver export-policy show -policyname exp1
Vserver          Policy Name
---------------  -------------------
vs1              exp1
----
