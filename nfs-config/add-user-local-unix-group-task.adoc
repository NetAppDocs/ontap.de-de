= Benutzer zur lokalen UNIX-Gruppe auf ONTAP NFS SVMs hinzufügen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Mit dem  `vserver services name-service unix-group adduser` Befehl kann ein Benutzer zu einer zusätzlichen UNIX-Gruppe hinzugefügt werden, die lokal zur SVM gehört.

.Schritt
. Einen Benutzer zu einer lokalen UNIX-Gruppe hinzufügen:


`vserver services name-service unix-group adduser -vserver _vserver_name_ -name _group_name_ -username _user_name_`

`-name` `_group_name_`Gibt den Namen der UNIX-Gruppe an, zu der der Benutzer zusätzlich zu seiner primären Gruppe hinzugefügt wird.

.Beispiel
Der folgende Befehl fügt einen Benutzer namens max zu einer lokalen UNIX-Gruppe namens eng auf der SVM namens vs1 hinzu:

[listing]
----
vs1::> vserver services name-service unix-group adduser -vserver vs1 -name eng
-username max
----
