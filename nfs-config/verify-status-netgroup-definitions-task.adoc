= ONTAP NFS SVM-Netzgruppendefinitionen überprüfen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Nach dem Laden der Netzgruppen in die SVM kann mit dem  `vserver services name-service netgroup status` Befehl der Status der Netzgruppendefinitionen überprüft werden. So lässt sich feststellen, ob die Netzgruppendefinitionen auf allen Knoten, die die SVM unterstützen, konsistent sind.

.Schritte
. Stellen Sie die Berechtigungsstufe auf „Erweitert“ ein:
+
`set -privilege advanced`

. Status der Netzgruppendefinitionen überprüfen:
+
`vserver services name-service netgroup status`

+
Zusätzliche Informationen werden in einer detaillierteren Ansicht angezeigt.

. Zurück zur Administrator-Privilegstufe:
+
`set -privilege admin`



.Beispiel
Nachdem die Berechtigungsstufe festgelegt wurde, zeigt der folgende Befehl den Netgroup-Status für alle SVMs an:

[listing]
----
vs1::> set -privilege advanced

         directed to do so by technical support.
