= Informationen zum ONTAP NFS Konfigurationsworkflow
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Die Konfiguration von NFS beinhaltet die Beurteilung der physischen Speicher- und Netzwerkanforderungen sowie die Auswahl eines Workflows, der auf Ihr Ziel zugeschnitten ist: die Konfiguration des NFS-Zugriffs auf eine neue oder bestehende SVM oder das Hinzufügen eines Volumes oder qtree zu einer bestehenden SVM, die bereits vollständig für den NFS-Zugriff konfiguriert ist.

image:nfs-config-pg-workflow_ieops-1616.png["Bietet einen Überblick über den NFS-Einrichtungsprozess, einschließlich der Schritte, die vor Beginn der NFS-Einrichtung erfolgen, und der Schritte, die optional im Anschluss durchgeführt werden können."]
