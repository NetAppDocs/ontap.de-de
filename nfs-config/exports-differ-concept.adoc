= Wie sich ONTAP Exporte von 7-Mode Exporten unterscheiden
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Falls Sie mit der Implementierung von NFS-Exports in ONTAP nicht vertraut sind, besteht die Möglichkeit, die Exportkonfigurationstools von 7-Mode und ONTAP sowie Beispieldateien von 7-Mode `/etc/exports` mit Cluster-Richtlinien und -Regeln zu vergleichen.

In ONTAP gibt es keine `/etc/exports` Datei und keinen `exportfs` Befehl. Stattdessen ist eine Exportrichtlinie zu definieren. Exportrichtlinien ermöglichen eine Steuerung des Clientzugriffs ähnlich wie im 7-Mode, bieten jedoch zusätzliche Funktionen wie die Möglichkeit, dieselbe Exportrichtlinie für mehrere Volumes wiederzuverwenden.
