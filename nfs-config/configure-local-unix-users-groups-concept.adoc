= Informationen zu lokalen UNIX-Benutzern und -Gruppen für ONTAP NFS SVMs
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können lokale UNIX-Benutzer und -Gruppen auf der SVM für die Authentifizierung und Namenszuordnungen verwenden. Sie können UNIX-Benutzer und -Gruppen manuell erstellen oder eine Datei mit UNIX-Benutzern oder -Gruppen von einem Uniform Resource Identifier (URI) laden.

Im Cluster gilt standardmäßig eine maximale Anzahl von 32.768 lokalen UNIX-Benutzergruppen und Gruppenmitgliedern zusammen. Der Clusteradministrator kann dieses Limit anpassen.
