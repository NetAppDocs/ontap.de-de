= Lokale UNIX-Gruppen auf ONTAP NFS SVMs erstellen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Der  `vserver services name-service unix-group create` Befehl kann verwendet werden, um UNIX-Gruppen zu erstellen, die lokal für die SVM sind. Lokale UNIX-Gruppen werden mit lokalen UNIX-Benutzern verwendet.

.Schritt
. Eine lokale UNIX-Gruppe erstellen:
+
`vserver services name-service unix-group create -vserver _vserver_name_ -name _group_name_ -id _integer_`

+
`-name _group_name_` Gibt den Gruppennamen an. Die Länge des Gruppennamens darf 64 Zeichen nicht überschreiten.

+
`-id _integer_`gibt die zugewiesene Gruppen-ID an.



.Beispiel
Der folgende Befehl erstellt eine lokale Gruppe namens eng auf der SVM namens vs1. Die Gruppe hat die ID 101.

[listing]
----
vs1::> vserver services name-service unix-group create -vserver vs1 -name eng -id 101
----
