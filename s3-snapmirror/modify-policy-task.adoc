= Eine ONTAP SnapMirror S3-Richtlinie ändern
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Möglicherweise wird eine S3 SnapMirror Richtlinie geändert, wenn eine Anpassung der RPO- und Drosselungswerte erforderlich ist.

[role="tabbed-block"]
====
.System Manager
--
. Klicken Sie auf *Schutz > Beziehungen* und wählen Sie dann die Datensicherungsstrategie für die Beziehung aus, die Sie ändern möchten.
. Klicken Sie image:icon_kabob.gif["Symbol für Menüoptionen"] neben dem Namen der Policy und dann auf *Bearbeiten*.


--
.CLI
--
Eine SnapMirror S3-Richtlinie ändern:
`snapmirror policy modify -vserver <svm_name> -policy <policy_name> [-rpo <integer>] [-throttle <throttle_type>] [-comment <text>]`

Parameter:

* `-rpo`: Gibt die Zeit für das Recovery Point Objective in Sekunden an.
* `-throttle`: Gibt die Obergrenze für Durchsatz/Bandbreite in Kilobyte/Sekunde an.


....
clusterA::> snapmirror policy modify -vserver vs0 -policy test-policy -rpo 60
....
--
