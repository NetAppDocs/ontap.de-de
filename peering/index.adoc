= Informationen zu ONTAP Cluster und SVM Peering
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können Peer-Beziehungen zwischen Quell- und Ziel-Cluster sowie zwischen Quell- und Ziel-Storage-VMs (Storage Virtual Machines) erstellen. Peer-Beziehungen zwischen diesen Entitäten müssen erstellt werden, bevor Snapshots mit SnapMirror repliziert werden können.

ONTAP 9.3 bietet Verbesserungen, die die Konfiguration von Peer-Beziehungen zwischen Clustern und SVMs vereinfachen. Die Verfahren zum Peering von Clustern und SVMs sind für alle ONTAP 9 Versionen verfügbar. Das jeweils passende Verfahren für Ihre ONTAP Version sollte verwendet werden.

Die Prozeduren werden über die Befehlszeilenschnittstelle (CLI) durchgeführt, nicht über System Manager oder ein automatisiertes Scripting-Tool.
