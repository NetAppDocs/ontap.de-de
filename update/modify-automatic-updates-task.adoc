= Automatische Updates ändern
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Wenn die automatische Update-Funktion aktiviert und die Standardeinstellungen für Systemkomponenten konfiguriert sind, kann ONTAP alle empfohlenen Updates automatisch erkennen, herunterladen und installieren. Wenn empfohlene Updates vor der Installation angezeigt oder Empfehlungen automatisch ausgeblendet werden sollen, lässt sich das Standardverhalten entsprechend anpassen.

.Schritte
[role="tabbed-block"]
====
.ONTAP 9.16.1 und höher
--
. Im System Manager unter *Cluster > Einstellungen* navigieren.
. Im Abschnitt *Software updates* image:icon_arrow.gif["Pfeilsymbol"] auswählen.
. Die Registerkarte *Alle anderen Updates* auswählen und auf *Automatische Update-Einstellungen bearbeiten* klicken.
. Die Standardaktionen, die für jeden Aktualisierungstyp durchgeführt werden, können festgelegt werden.
+
Sie können für jeden Aktualisierungstyp festlegen, ob automatische Aktualisierungen, Benachrichtigungen angezeigt oder Aktualisierungen automatisch abgelehnt werden.

+

NOTE: Die ONTAP Zeitzonendatenbank wird durch den Aktualisierungstyp *Systemdateien* gesteuert.

. Die Allgemeinen Geschäftsbedingungen akzeptieren und *Speichern* auswählen.


--
.ONTAP 9.15.1 und früher
--
. Im System Manager auf *Cluster > Einstellungen* klicken.
. Im Abschnitt *Automatische Aktualisierung* auf image:icon_kabob.gif["Symbol für Menüoptionen"] klicken, um eine Liste der Aktionen anzuzeigen.
. *Einstellungen für automatische Updates bearbeiten* auswählen.
. Die Standardaktionen, die für jeden Aktualisierungstyp durchgeführt werden, können festgelegt werden.
+
Für jeden Aktualisierungstyp kann festgelegt werden, ob automatisch aktualisiert, Benachrichtigungen angezeigt oder Aktualisierungen automatisch verworfen werden.

+

NOTE: Die ONTAP Zeitzonen Datenbank wird durch den Aktualisierungstyp SYSTEM FILES gesteuert.



--
