= Firmware manuell aktualisieren
:hardbreaks:
:toclevels: 1
:allow-uri-read: 
:toclevels: 1
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
Ab ONTAP 9.9.1, wenn Sie bei link:https://docs.netapp.com/us-en/active-iq/["Digital Advisor"^] registriert sind, können Sie im System Manager Benachrichtigungen erhalten, die Sie darüber informieren, wenn Firmware-Updates für unterstützte Geräte wie Festplatten, Festplattengehäuse, den Serviceprozessor (SP) oder den Baseboard Management Controller (BMC) im Cluster ausstehen.

Wenn Sie ONTAP 9.8 verwenden oder nicht bei Digital Advisor registriert sind, navigieren Sie zur NetApp Support-Website, um Firmware-Updates herunterzuladen.

.Bevor Sie beginnen
Für eine reibungslose Firmware-Aktualisierung empfiehlt sich ein Neustart von SP oder BMC vor Beginn des Updates. Der Neustart erfolgt mit dem  `system service-processor reboot-sp -node node_name` Befehl. Weitere Informationen zu  `system service-processor reboot-sp` finden sich in der link:https://docs.netapp.com/us-en/ontap-cli/system-service-processor-reboot-sp.html["ONTAP-Befehlsreferenz"^].

.Schritte
Befolgen Sie das entsprechende Verfahren basierend auf Ihrer Version von ONTAP und falls Sie bei Digital Advisor registriert sind.

[role="tabbed-block"]
====
.ONTAP 9.16.1 und höher mit Digital Advisor
--
.Schritte
. Im System Manager zur *Dashboard* wechseln.
+
Im Abschnitt *Health* wird eine Meldung angezeigt, wenn empfohlene Firmware-Updates für das Cluster verfügbar sind.

. Klicken Sie auf die Warnmeldung.
. Neben den Sicherheitsupdates in der Liste der empfohlenen Updates die Option *Aktionen* auswählen.
. *Aktualisieren* installiert das Update sofort, *Planen* legt die Installation für einen späteren Zeitpunkt fest.
+
Wenn das Update bereits geplant ist, besteht die Möglichkeit, es *zu bearbeiten* oder *zu stornieren*.



--
.ONTAP 9.9.1 bis 9.15.1 mit Digital Advisor
--
. Im System Manager zur *Dashboard* wechseln.
+
Im Abschnitt *Health* wird eine Meldung angezeigt, wenn empfohlene Firmware-Updates für das Cluster verfügbar sind.

. Klicken Sie auf die Warnmeldung.
+
Auf der Seite *Update* wird die Registerkarte *Firmware Update* angezeigt.

. Auf *Download von der NetApp Support Site* klicken, um das gewünschte Firmware-Update durchzuführen.
+
Die NetApp Support Site wird angezeigt.

. Melden Sie sich auf der NetApp Support Site an und laden Sie das für das Update benötigte Firmware-Image-Paket herunter.
. Die Dateien werden auf einen HTTP- oder FTP-Server im Netzwerk oder in einen lokalen Ordner kopiert.
. Im System Manager auf *Cluster > Übersicht* klicken.
. In der rechten Ecke des *Übersicht*-Bereichs auf *Mehr* image:icon_kabob.gif["Symbol für Menüoptionen"] klicken und *ONTAP Update* auswählen.
. *Firmware Update* auswählen.
. Je nach Ihrer Version von ONTAP ist Folgendes zu tun:
+
[cols="2"]
|===
| ONTAP 9.9.1 und 9.10.0 | ONTAP 9.10.1 und höher 


 a| 
.. *Vom Server* oder *Lokaler Client* auswählen
.. Geben Sie die Server-URL oder den Dateispeicherort an.

 a| 
.. In der Liste der empfohlenen Updates die Option *Aktionen* auswählen.
.. *Aktualisieren* installiert das Update sofort, *Planen* legt die Installation für einen späteren Zeitpunkt fest.
+
Wenn das Update bereits geplant ist, besteht die Möglichkeit, es *zu bearbeiten* oder *zu stornieren*.

.. Die Schaltfläche *Firmware aktualisieren* auswählen.


|===


--
--
.ONTAP 9.8 und höher ohne Digital Advisor
. Navigieren Sie zu link:https://mysupport.netapp.com/site/downloads["NetApp Support Site"^] und melden Sie sich an.
. Wählen Sie das Firmware-Paket aus, das Sie zum Aktualisieren der Cluster-Firmware verwenden möchten.
. Die Dateien werden auf einen HTTP- oder FTP-Server im Netzwerk oder in einen lokalen Ordner kopiert.
. Im System Manager auf *Cluster > Übersicht* klicken.
. In der rechten Ecke des *Übersicht*-Bereichs auf *Mehr* image:icon_kabob.gif["Symbol für Menüoptionen"] klicken und *ONTAP Update* oder *Software updates* auswählen (abhängig von Ihrer Version).
. Je nach Ihrer Version von ONTAP ist Folgendes zu tun:
+
** Für ONTAP 9.15.1 und frühere Versionen auf *Firmware Update* klicken.
** Für ONTAP 9.16.1 und höher auf *Alle anderen Updates* klicken.


. Je nach Ihrer Version von ONTAP ist Folgendes zu tun:


[cols="2"]
|===
| ONTAP 9.8, 9.9.1 und 9.10.0 | ONTAP 9.10.1 und höher 


 a| 
. *Vom Server* oder *Lokaler Client* auswählen
. Geben Sie die Server-URL oder den Dateispeicherort an.

 a| 
. In der Liste der empfohlenen Updates die Option *Aktionen* auswählen.
. *Aktualisieren* installiert das Update sofort, *Planen* legt die Installation für einen späteren Zeitpunkt fest.
+
Wenn das Update bereits geplant ist, besteht die Möglichkeit, es *zu bearbeiten* oder *zu stornieren*.

. Die Schaltfläche *Firmware aktualisieren* auswählen.


|===
--
====
.Nachdem Sie fertig sind
Sie können Updates unter *Firmware Update Summary* überwachen oder überprüfen. Um Updates anzuzeigen, die verworfen wurden oder deren Installation fehlgeschlagen ist, gibt es je nach Ihrer Version von ONTAP folgende Möglichkeiten:

* Für ONTAP 9.15.1 und frühere Versionen auf *Cluster > Einstellungen > Automatisches Update > Alle automatischen Updates anzeigen* klicken
* Für ONTAP 9.16.1 und höher unter *Cluster > Einstellungen > Softwareupdates* klicken. In der rechten Ecke des Bereichs *Alle anderen Updates* auf *Mehr* image:icon_kabob.gif["Symbol für Menüoptionen"] klicken und *Alle automatischen Updates anzeigen* auswählen.
