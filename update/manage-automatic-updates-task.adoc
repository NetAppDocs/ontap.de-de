= Empfohlene automatische Updates verwalten
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Das automatische Update-Protokoll zeigt eine Liste mit Update-Empfehlungen und Details zu jeder einzelnen an, darunter Beschreibung, Kategorie, geplanter Installationszeitpunkt, Status und eventuelle Fehler. Das Protokoll kann eingesehen werden, anschließend lässt sich festlegen, welche Aktion für jede Empfehlung ausgeführt werden soll.

.Schritte
. Liste der Empfehlungen anzeigen:
+
[cols="2"]
|===
| Ansicht aus den Cluster Einstellungen | Ansicht aus dem Aktualisieren-Tab 


 a| 
.. *Cluster > Einstellungen* auswählen.
.. Je nach Ihrer ONTAP-Version ist eine der folgenden Optionen zu wählen:
+
*** Für ONTAP 9.15.1 und früher im Abschnitt *Automatische Aktualisierung* auf image:../media/icon_kabob.gif["Symbol für Menüoptionen"] klicken, dann die Option auswählen, um alle Aktualisierungen anzuzeigen.
*** Für ONTAP 9.16.1 und höher im Abschnitt *Software updates* image:icon_arrow.gif["Pfeilsymbol"] auswählen. In der rechten Ecke des Bereichs *All other updates* auf *More* image:icon_kabob.gif["Symbol für Menüoptionen"] klicken und dann die Option zum Anzeigen aller Updates wählen.



 a| 
.. *Cluster > Übersicht* auswählen.
.. Im Abschnitt *Übersicht* auf *Mehr* image:../media/icon_kabob.gif["Symbol für Menüoptionen"] klicken, dann auf *ONTAP Update*.
.. Je nach Ihrer Version von ONTAP ist Folgendes zu tun:
+
*** Für ONTAP 9.15.1 und frühere Versionen auf *Firmware Update* klicken.
*** Für ONTAP 9.16.1 und höher auf *Alle anderen Updates* klicken.


.. Auf der Update-Seite auf *Mehr* image:../media/icon_kabob.gif["Symbol für Menüoptionen"]klicken, dann die Option auswählen, alle Updates anzuzeigen.


|===
. Klicken Sie image:../media/icon_kabob.gif["Symbol für Menüoptionen"] neben der Beschreibung, um eine Liste von Aktionen anzuzeigen, die Sie für die Empfehlung ausführen können.
+
Je nach Status der Empfehlung ist eine der folgenden Aktionen möglich:

+
[cols="35,65"]
|===


| Wenn sich das Update in diesem Zustand befindet... | Sie können... 


 a| 
Ist noch nicht terminiert
 a| 
*Update*: Startet den Aktualisierungsprozess.

*Zeitplan*: Ermöglicht das Festlegen eines Datums für den Beginn des Aktualisierungsprozesses.

*Verwerfen*: Entfernt die Empfehlung aus der Liste.



 a| 
Wurde geplant
 a| 
*Update*: Startet den Aktualisierungsprozess.

*Zeitplan bearbeiten*: Ermöglicht die Änderung des geplanten Datums für den Beginn des Aktualisierungsprozesses.

*Termin stornieren*: Hebt das geplante Datum auf.



 a| 
Wurde entlassen
 a| 
*Wiederherstellen*: Die Empfehlung wird wieder in die Liste aufgenommen.



 a| 
Wird angewendet oder wird heruntergeladen
 a| 
*Abbrechen*: Das Update wird abgebrochen.

|===
