= Erfahren Sie mehr über das ONTAP System Manager SVM Administrator-Dashboard
:hardbreaks:
:toclevels: 1
:allow-uri-read: 
:toclevels: 1
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
Ab ONTAP 9.19.1 können SVM-Administratoren ihre SVMs über ein dediziertes System Manager-Dashboard und die Seite „Storage > Volumes“ überwachen und verwalten. Dies ist besonders in einer mandantenfähigen ONTAP Umgebung nützlich, da der Cluster-Administrator für jeden Mandanten ein eigenes SVM-Dashboard bereitstellen kann.

Der Clusteradministrator stellt den SVM-Administratoren die Anmeldeinformationen bereit. SVM-Administratoren können Details und Leistungsdaten im SVM-Dashboard und auf der Seite *Storage > Volumes* einsehen. Das Dashboard zeigt SVM-Protokolle, Netzwerkschnittstellen, Details und Leistungskennzahlen an. Die Seite *Volumes* enthält Informationen zu den SVM-Volumes und ermöglicht es dem SVM-Administrator, – sofern vom Clusteradministrator freigegeben – neue Volumes hinzuzufügen.
