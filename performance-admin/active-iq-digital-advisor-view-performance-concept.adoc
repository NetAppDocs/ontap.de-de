= Mit ONTAP Active IQ Digital Advisor lässt sich die Systemleistung anzeigen.
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Für jedes ONTAP System, das AutoSupport-Telemetriedaten an NetApp sendet, sind umfassende Leistungs- und Kapazitätsdaten einsehbar. Digital Advisor zeigt die Systemleistung über einen längeren Zeitraum an als im System Manager sichtbar ist.

Sie können Diagramme zur CPU-Auslastung, Latenz, IOPS, IOPS nach Protokoll und Netzwerkdurchsatz anzeigen. Sie können diese Daten auch im .csv-Format herunterladen, um sie in anderen Tools zu analysieren.

Zusätzlich zu diesen Leistungsdaten kann Digital Advisor die Speichereffizienz nach Arbeitslast anzeigen und diese Effizienz mit der erwarteten Effizienz für diese Art von Arbeitslast vergleichen. Kapazitätstrends lassen sich anzeigen und eine Schätzung, wie viel zusätzlicher Speicher in einem bestimmten Zeitraum möglicherweise benötigt wird, ist ersichtlich.

[NOTE]
====
* Die Speichereffizienz ist auf Kunden-, Cluster- und Knotenebene auf der linken Seite des Hauptdashboards verfügbar.
* Die Performance ist auf Cluster- und Knotenebene auf der linken Seite des Hauptdashboards verfügbar.
