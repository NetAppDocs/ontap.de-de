= ONTAP Active IQ Unified Manager Leistungsschwellenwerte festlegen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können Leistungsschwellenwerte festlegen, um kritische Leistungsprobleme zu überwachen. Benutzerdefinierte Schwellenwerte lösen eine Warnung oder eine Benachrichtigung über ein kritisches Ereignis aus, wenn das System sich dem festgelegten Schwellenwert nähert oder diesen überschreitet.

.Schritte
. Die Schwellenwerte für Warnungen und kritische Ereignisse festlegen:
+
.. *Konfiguration* > *Leistungsschwellenwerte* auswählen.
.. Klicken Sie auf *Create*.
.. Wählen Sie den Objekttyp aus und geben Sie einen Namen und eine Beschreibung der Richtlinie an.
.. Wählen Sie die Objektzählerbedingung aus und geben Sie die Grenzwerte an, die Warnung und kritische Ereignisse definieren.
.. Wählen Sie die Zeitspanne aus, in der die Grenzwerte überschritten sein müssen, damit ein Ereignis gesendet wird, und klicken Sie dann auf *Speichern*.


. Die Schwellenwertrichtlinie wird dem Speicherobjekt zugewiesen.
+
.. Zur Inventarseite für denselben Clusterobjekttyp, der zuvor ausgewählt wurde, wechseln und unter der Option *Ansicht* die Option *Leistung* auswählen.
.. Wählen Sie das Objekt aus, dem Sie die Schwellenwertrichtlinie zuweisen möchten, und klicken Sie dann auf *Schwellenwertrichtlinie zuweisen*.
.. Wählen Sie die zuvor erstellte Richtlinie aus und klicken Sie dann auf *Richtlinie zuweisen*.




.Beispiel
Sie können benutzerdefinierte Schwellenwerte festlegen, um kritische Leistungsprobleme zu erkennen. Wenn Sie beispielsweise einen Microsoft Exchange Server verwenden und wissen, dass dieser abstürzt, wenn die Volumenlatenz 20 Millisekunden überschreitet, kann ein Warnschwellenwert von 12 Millisekunden und ein kritischer Schwellenwert von 15 Millisekunden festgelegt werden. Mit dieser Schwellenwerteinstellung können Benachrichtigungen empfangen werden, wenn die Volumenlatenz den Grenzwert überschreitet.
