= Tägliche ONTAP Active IQ Unified Manager Überwachung durchführen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Eine tägliche Überwachung stellt sicher, dass keine unmittelbaren Leistungsprobleme bestehen, die Aufmerksamkeit erfordern.

.Schritte
. In der Benutzeroberfläche von Active IQ Unified Manager zur Seite *Ereignisinventar* wechseln, um alle aktuellen und veralteten Ereignisse anzuzeigen.
. Unter der Option *Ansicht*  `Active Performance Events`wird ausgewählt und bestimmt, welche Aktion erforderlich ist.
