= VMware-Umgebungsanforderungen für ONTAP Active IQ Unified Manager überprüfen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Für eine erfolgreiche Installation von Active IQ Unified Manager ist zu überprüfen, ob Ihre VMware-Umgebung die erforderlichen Voraussetzungen erfüllt.

.Schritte
. Es sollte sichergestellt sein, dass Ihre VMware-Infrastruktur die Größenanforderungen für die Installation von Unified Manager erfüllt.
. Wechseln Sie zu  https://mysupport.netapp.com/matrix["Interoperabilitätsmatrix"^], um zu überprüfen, ob eine unterstützte Kombination der folgenden Komponenten vorliegt:
+
** ONTAP Version
** ESXi Betriebssystemversion
** VMware vCenter Server-Version
** VMware Tools Version
** Browsertyp und Version


+
[NOTE]
====
Die Interoperabilitätsmatrix listet die unterstützten Konfigurationen für Unified Manager auf.

====
. Auf den Namen der ausgewählten Konfiguration klicken.
+
Details zu dieser Konfiguration werden im Fenster „Konfigurationsdetails“ angezeigt.

. Die Informationen in den folgenden Registerkarten werden angezeigt:
+
** Hinweise
+
Listet wichtige Warnungen und Informationen auf, die speziell für Ihre Konfiguration gelten.

** Richtlinien und Leitlinien
+
Bietet allgemeine Richtlinien für alle Konfigurationen.
