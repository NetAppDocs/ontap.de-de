= Konfiguration der Alarmeinstellungen von ONTAP Active IQ Unified Manager
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Es kann festgelegt werden, welche Ereignisse von Active IQ Unified Manager Warnmeldungen auslösen, wer die E-Mail-Empfänger dieser Warnmeldungen sind und wie häufig die Warnmeldungen erfolgen.

.Bevor Sie beginnen
Sie benötigen die Rolle des Anwendungsadministrators.

.Über diese Aufgabe
Sie können individuelle Alarmeinstellungen für die folgenden Arten von Leistungsereignissen konfigurieren:

* Kritische Ereignisse, die durch Überschreitung benutzerdefinierter Schwellenwerte ausgelöst werden
* Warnereignisse, die durch Überschreitungen von benutzerdefinierten Schwellenwerten, systemdefinierten Schwellenwerten oder dynamischen Schwellenwerten ausgelöst werden


Standardmäßig werden E-Mail-Benachrichtigungen für alle neuen Ereignisse an die Unified Manager Administratoren gesendet. E-Mail-Benachrichtigungen können auch an andere Benutzer gesendet werden, indem deren E-Mail-Adressen hinzugefügt werden.

[NOTE]
====
Um zu verhindern, dass Benachrichtigungen für bestimmte Ereignistypen gesendet werden, müssen alle Kontrollkästchen in einer Ereigniskategorie deaktiviert werden. Dadurch wird das Erscheinen von Ereignissen in der Benutzeroberfläche nicht verhindert.

====
.Schritte
. Im linken Navigationsbereich *Speicherverwaltung* > *Benachrichtigungseinrichtung* auswählen.
+
Die Seite „Benachrichtigungseinstellungen“ wird angezeigt.

. *Hinzufügen* auswählen und die entsprechenden Einstellungen für jeden Ereignistyp konfigurieren.
+
Um E-Mail-Benachrichtigungen an mehrere Benutzer zu senden, ist zwischen den einzelnen E-Mail-Adressen ein Komma einzugeben.

. Auf *Speichern* klicken.
