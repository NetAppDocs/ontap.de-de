= Die ONTAP CIFS/SMB Multiplex-Einstellungen prüfen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Wenn eine langsame CIFS-Netzwerkleistung zu einem Leistungsproblem führt, können die Multiplex-Einstellungen geändert werden, um die Leistung zu verbessern und das Problem zu beheben.

.Schritte
. Die CIFS-Multiplex-Einstellung prüfen:
+
`vserver cifs options show -vserver _-vserver_name_ -instance`

. Die CIFS-Multiplex-Einstellung ändern:
+
`vserver cifs options modify –vserver _-vserver_name_ –max-mpx _integer_`



.Beispiel
Das folgende Beispiel ändert die maximale Multiplexanzahl auf `SVM1` 255:

[listing]
----
cluster1::> vserver cifs options modify -vserver SVM1 -max-mpx 255
----
