= ONTAP Storage QoS Workflow
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Wenn Sie die Leistungsanforderungen für die Workloads, die Sie mit QoS verwalten möchten, bereits kennen, kann die Durchsatzbegrenzung beim Erstellen der Richtliniengruppe festgelegt werden. Andernfalls kann mit der Festlegung des Limits gewartet werden, bis die Workloads überwacht wurden.

image:qos-workflow.gif["Diese Abbildung zeigt die folgenden Schritte: 1. Speicherobjekte identifizieren, die Richtliniengruppen zugewiesen werden sollen. 2. Zur Steuerung und Überwachung Richtliniengruppen mit Durchsatzbegrenzung erstellen, zur reinen Überwachung Richtliniengruppen ohne Durchsatzbegrenzung erstellen. 3. Die Speicherobjekte den Richtliniengruppen zuweisen. 4. Leistung durch Ansicht der Statistiken überwachen. 5. Richtlinieneinstellungen bei Bedarf anpassen."]
