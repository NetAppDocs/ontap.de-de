= Die ONTAP Netzwerkeinstellungen auf den Datenswitches prüfen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Obwohl auf Clients, Servern und Speichersystemen (das heißt, Netzwerkendpunkten) die gleichen MTU-Einstellungen beibehalten werden müssen, sollten zwischengeschaltete Netzwerkgeräte wie NICs und Switches auf ihre maximalen MTU-Werte eingestellt sein, damit die Leistung nicht beeinträchtigt wird.

Für optimale Leistung müssen alle Komponenten im Netzwerk Jumbo-Frames (9000 Byte IP, 9022 Byte inklusive Ethernet) weiterleiten können. Data Switches sollten auf mindestens 9022 Byte eingestellt sein, ein typischer Wert von 9216 ist jedoch mit den meisten Switches möglich.

.Schritte
. Für Data Switches prüfen, ob die MTU-Größe auf 9022 oder höher eingestellt ist.


Weitere Informationen sind in der Dokumentation des Switch-Herstellers zu finden.
