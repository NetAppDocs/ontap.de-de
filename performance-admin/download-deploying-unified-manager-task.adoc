= ONTAP Active IQ Unified Manager herunterladen und bereitstellen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Zur Installation der Software muss die Installationsdatei der virtuellen Appliance (VA) heruntergeladen und anschließend mit einem VMware vSphere Client auf einem VMware ESXi Server bereitgestellt werden. Die VA ist als OVA-Datei verfügbar.

.Schritte
. Die Seite *NetApp Support Site Software Download* aufrufen und Active IQ Unified Manager suchen.
+
https://mysupport.netapp.com/products/index.html[]

. Im Dropdown-Menü *Plattform auswählen* die Option *VMware vSphere* auswählen und auf *Los!* klicken.
. Die Datei „`OVA`“ ist an einem lokalen oder Netzwerkspeicherort zu speichern, der für Ihren VMware vSphere Client zugänglich ist.
. Im VMware vSphere Client auf *Datei* > *OVF-Template bereitstellen* klicken.
. Die Datei "`OVA`" kann gefunden und mit dem Assistenten die virtuelle Appliance auf dem ESXi Server bereitgestellt werden.
+
Die Registerkarte *Eigenschaften* im Assistenten ermöglicht die Eingabe Ihrer statischen Konfigurationsinformationen.

. Die VM wird eingeschaltet.
. Auf der Registerkarte *Konsole* lässt sich der anfängliche Startvorgang anzeigen.
. Den Anweisungen zum Installieren von VMware Tools auf der VM folgen.
. Die Zeitzone konfigurieren.
. Einen Benutzernamen und ein Passwort für die Wartung eingeben.
. Zur angezeigten URL in der VM-Konsole wechseln.
