= Grundlegende ONTAP Überwachungsaufgaben einrichten
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können Ihre Systeme auf Leistungsprobleme überwachen, indem Sie die Systeme täglich überprüfen, wöchentliche und monatliche Leistungstrends ermitteln und Schwellenwerte für den Empfang von Benachrichtigungen über potenzielle Leistungsprobleme festlegen.
