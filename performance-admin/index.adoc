= Informationen zur Leistungsüberwachung und zum Management mit ONTAP Active IQ Unified Manager
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können grundlegende Aufgaben zur Leistungsüberwachung und Leistungsverwaltung einrichten sowie häufig auftretende Leistungsprobleme identifizieren und beheben.

Sie können diese Verfahren zur Überwachung und Verwaltung der Cluster-Performance verwenden, wenn die folgenden Annahmen auf Ihre Situation zutreffen:

* Sie möchten bewährte Verfahren anwenden, nicht jede verfügbare Option untersuchen.
* Es besteht der Wunsch, Systemstatus und Warnmeldungen anzuzeigen, die Clusterleistung zu überwachen und eine Ursachenanalyse mithilfe von Active IQ Unified Manager (ehemals OnCommand Unified Manager) zusätzlich zur ONTAP Befehlszeilenschnittstelle durchzuführen.
* Sie verwenden die ONTAP Befehlszeile, um die Storage Quality of Service (QoS) zu konfigurieren. QoS ist auch über Folgendes verfügbar:
+
** System Manager
** ONTAP REST API
** ONTAP Tools for VMware vSphere
** NetApp Service Level Manager (NSLM)
** OnCommand Workflow Automation (WFA)


* Sie möchten Active IQ Unified Manager mithilfe einer virtuellen Appliance installieren, statt einer Installation auf Linux- oder Windows-Basis.
* Sie sind bereit, für die Installation der Software eine statische Konfiguration anstelle von DHCP zu verwenden.
* Sie können auf ONTAP Befehle auf der erweiterten Berechtigungsstufe zugreifen.
* Sie sind ein Cluster-Administrator mit der Rolle "admin".
