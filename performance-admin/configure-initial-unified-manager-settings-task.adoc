= Anfängliche ONTAP Active IQ Unified Manager Einstellungen konfigurieren
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Das Dialogfeld „Active IQ Unified Manager Ersteinrichtung“ erscheint beim ersten Zugriff auf die Web-Benutzeroberfläche, wodurch einige Grundeinstellungen konfiguriert und Cluster hinzugefügt werden können.

.Schritte
. Die standardmäßig aktivierte Einstellung AutoSupport akzeptieren.
. Geben Sie die NTP-Serverdetails, die E-Mail-Adresse des Wartungsbenutzers, den Hostnamen des SMTP-Servers und zusätzliche SMTP-Optionen ein, und wählen Sie anschließend *Speichern*.


.Nachdem Sie fertig sind
Nach Abschluss der Ersteinrichtung wird die Seite „Cluster-Datenquellen“ angezeigt, auf der die Clusterdetails hinzugefügt werden können.
