= ONTAP Festplattendurchsatz und Latenz prüfen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Die Metriken für Festplattendurchsatz und Latenz der Clusterknoten können zur Unterstützung bei der Fehlersuche überprüft werden.

.Über diese Aufgabe
Für diese Aufgabe sind Befehle auf dem erweiterten Berechtigungslevel erforderlich.

.Schritte
. Wechsel zum erweiterten Berechtigungslevel:
+
[source, cli]
----
set -privilege advanced
----
. Die Metriken für Festplattendurchsatz und Latenz prüfen:
+
[source, cli]
----
statistics disk show -sort-key operation_latency
----


.Beispiel
Das folgende Beispiel zeigt die Summen in jedem Benutzer-Lese- oder Schreibvorgang für  `node2` auf  `cluster1`:

----
