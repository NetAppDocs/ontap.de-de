= Workflow für die Virenprüfung mit ONTAP Vscan
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Bevor Sie das Scannen aktivieren können, müssen Sie einen Scannerpool erstellen und eine Scannerrichtlinie anwenden. In der Regel werden sowohl der On-Access- als auch der On-Demand-Scanmodus auf einer SVM aktiviert.


NOTE: Die CIFS Konfiguration muss abgeschlossen sein.

image:avcfg-workflow.gif["Arbeitsablauf zur Virenprüfung"]


NOTE: Um eine Aufgabe bei Bedarf zu erstellen, muss mindestens eine On-Access-Richtlinie aktiviert sein. Dies kann die Standardrichtlinie oder eine vom Benutzer erstellte On-Access-Richtlinie sein.

* xref:create-on-access-policy-task.html[Eine On-Access-Richtlinie erstellen]
