= Informationen zur Antivirenkonfiguration mit ONTAP Vscan
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Vscan ist eine von NetApp entwickelte Antiviren-Scanning-Lösung, die es Kunden ermöglicht, ihre Daten vor Viren oder anderem bösartigen Code zu schützen.

Vscan führt Virenscans durch, wenn Clients über SMB auf Dateien zugreifen. Vscan kann so konfiguriert werden, dass Scans bei Bedarf oder nach einem Zeitplan durchgeführt werden. Die Interaktion mit Vscan ist über die ONTAP Command-Line Interface (CLI) oder ONTAP Application Programming Interfaces (APIs) möglich.
