= Informationen zur Konfiguration der ONTAP Vscan On-Demand-Überprüfung
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können die On-Demand-Scanfunktion verwenden, um Dateien sofort oder nach einem Zeitplan auf Viren zu überprüfen.

Scans können beispielsweise nur außerhalb der Spitzenzeiten durchgeführt werden, oder sehr große Dateien, die von einem On-Access-Scan ausgeschlossen wurden, können gescannt werden. Ein Cron-Schedule kann verwendet werden, um festzulegen, wann die Aufgabe ausgeführt wird.


NOTE: Um eine Aufgabe bei Bedarf zu erstellen, muss mindestens eine On-Access-Richtlinie aktiviert sein. Dies kann die Standardrichtlinie oder eine vom Benutzer erstellte On-Access-Richtlinie sein.

.Zu diesem Thema
* Sie können beim Erstellen einer Aufgabe einen Zeitplan zuweisen.
* Auf einer SVM kann jeweils nur eine Aufgabe eingeplant werden.
* Die On-Demand-Scanfunktion unterstützt das Scannen von symbolischen Links oder Stream-Dateien nicht.
