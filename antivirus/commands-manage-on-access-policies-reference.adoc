= ONTAP Vscan Befehle zur Verwaltung von On-Access-Richtlinien
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können eine On-Access-Richtlinie ändern, deaktivieren oder löschen. Eine Zusammenfassung und Details zur Richtlinie sind einsehbar.

|===


| Wenn Sie möchten... | Führen Sie den folgenden Befehl aus... 


 a| 
Eine On-Access-Richtlinie erstellen
 a| 
`vserver vscan on-access-policy create`



 a| 
Eine On-Access-Richtlinie ändern
 a| 
`vserver vscan on-access-policy modify`



 a| 
Eine On-Access-Richtlinie aktivieren
 a| 
`vserver vscan on-access-policy enable`



 a| 
Eine On-Access-Richtlinie deaktivieren
 a| 
`vserver vscan on-access-policy disable`



 a| 
Eine On-Access-Richtlinie löschen
 a| 
`vserver vscan on-access-policy delete`



 a| 
Zusammenfassung und Details zu einer On-Access-Richtlinie anzeigen
 a| 
`vserver vscan on-access-policy show`



 a| 
Zur Liste der auszuschließenden Pfade hinzufügen
 a| 
`vserver vscan on-access-policy paths-to-exclude add`



 a| 
Aus der Liste der auszuschließenden Pfade löschen
 a| 
`vserver vscan on-access-policy paths-to-exclude remove`



 a| 
Liste der auszuschließenden Pfade anzeigen
 a| 
`vserver vscan on-access-policy paths-to-exclude show`



 a| 
Zur Liste der auszuschließenden Dateierweiterungen hinzufügen
 a| 
`vserver vscan on-access-policy file-ext-to-exclude add`



 a| 
Aus der Liste der auszuschließenden Dateierweiterungen entfernen
 a| 
`vserver vscan on-access-policy file-ext-to-exclude remove`



 a| 
Liste der auszuschließenden Dateierweiterungen anzeigen
 a| 
`vserver vscan on-access-policy file-ext-to-exclude show`



 a| 
Zur Liste der einzuschließenden Dateierweiterungen hinzufügen
 a| 
`vserver vscan on-access-policy file-ext-to-include add`



 a| 
Aus der Liste der einzuschließenden Dateierweiterungen entfernen
 a| 
`vserver vscan on-access-policy file-ext-to-include remove`



 a| 
Liste der einzuschließenden Dateierweiterungen anzeigen
 a| 
`vserver vscan on-access-policy file-ext-to-include show`

|===
Weitere Informationen zu den in diesem Verfahren beschriebenen Befehlen befinden sich in der link:https://docs.netapp.com/us-en/ontap-cli/["ONTAP-Befehlsreferenz"^].
