= Informationen zu NetApp Virenscans mit ONTAP Vscan
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Vscan ist eine von NetApp entwickelte Antiviren-Scanning-Lösung, die es Kunden ermöglicht, ihre Daten vor Viren oder anderem bösartigen Code zu schützen. Sie kombiniert von Partnern bereitgestellte Antivirensoftware mit ONTAP Funktionen, um Kunden die Flexibilität zu geben, die sie für die Verwaltung der Dateiprüfung benötigen.



[[how-virus-scanning-works]]
== Funktionsweise der Virenprüfung

Speichersysteme lagern Scanvorgänge an externe Server aus, auf denen Antivirensoftware von Drittanbietern gehostet wird.

Basierend auf dem aktiven Scanmodus sendet ONTAP Scananforderungen, wenn Clients über SMB auf Dateien zugreifen (On-Access) oder auf Dateien an bestimmten Speicherorten zugreifen, entweder nach einem Zeitplan oder sofort (On-Demand).

* Die _On-Access-Prüfung_ kann verwendet werden, um Dateien beim Öffnen, Lesen, Umbenennen oder Schließen über SMB auf Viren zu überprüfen. Dateivorgänge werden ausgesetzt, bis der externe Server den Scanstatus der Datei meldet. Wurde die Datei bereits geprüft, erlaubt ONTAP die Dateioperation. Andernfalls wird eine Prüfung vom Server angefordert.
+
On-access-Scanning wird für NFS nicht unterstützt.

* Die bedarfsgesteuerte Überprüfung kann verwendet werden, um Dateien sofort oder nach einem Zeitplan auf Viren zu prüfen. Es wird empfohlen, bedarfsgesteuerte Scans nur außerhalb der Spitzenzeiten auszuführen, um eine Überlastung der bestehenden Antiviren-Infrastruktur zu vermeiden, die normalerweise für die Prüfung bei Zugriff ausgelegt ist. Der externe Server aktualisiert den Scanstatus der geprüften Dateien, sodass die Zugriffsverzögerung über SMB reduziert wird. Wenn Dateiänderungen oder Software-Updates vorliegen, wird ein neuer Dateiscan vom externen Server angefordert.
+
Sie können die On-Demand-Scanfunktion für jeden Pfad im SVM-Namensraum verwenden, auch für Volumes, die nur über NFS exportiert werden.



In der Regel werden sowohl der On-Access- als auch der On-Demand-Scanmodus auf einer SVM aktiviert. In beiden Modi ergreift die Antivirensoftware basierend auf Ihren Softwareeinstellungen Maßnahmen zur Behebung infizierter Dateien.

Der ONTAP Antivirus Connector, bereitgestellt von NetApp und auf dem externen Server installiert, übernimmt die Kommunikation zwischen dem Speichersystem und der Antivirensoftware.
