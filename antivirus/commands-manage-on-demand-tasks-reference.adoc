= ONTAP Vscan Befehle zur Verwaltung von On-Demand-Aufgaben
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können eine Aufgabe nach Bedarf bearbeiten, löschen oder deren Ausführung aufheben. Sie können eine Zusammenfassung und Details zur Aufgabe anzeigen und Berichte zu dieser Aufgabe verwalten.

|===


| Wenn Sie möchten... | Führen Sie den folgenden Befehl aus... 


 a| 
Eine Aufgabe auf Abruf erstellen
 a| 
`vserver vscan on-demand-task create`



 a| 
Eine On-Demand-Aufgabe ändern
 a| 
`vserver vscan on-demand-task modify`



 a| 
Eine On-Demand-Aufgabe löschen
 a| 
`vserver vscan on-demand-task delete`



 a| 
Eine Aufgabe bei Bedarf ausführen
 a| 
`vserver vscan on-demand-task run`



 a| 
Eine Aufgabe auf Abruf planen
 a| 
`vserver vscan on-demand-task schedule`



 a| 
Eine On-Demand-Aufgabe aus dem Zeitplan entfernen
 a| 
`vserver vscan on-demand-task unschedule`



 a| 
Zusammenfassung und Details zu einer On-Demand-Aufgabe
 a| 
`vserver vscan on-demand-task show`



 a| 
On-Demand-Berichte anzeigen
 a| 
`vserver vscan on-demand-task report show`



 a| 
Berichte auf Abruf löschen
 a| 
`vserver vscan on-demand-task report delete`

|===
Weitere Informationen zu den in diesem Verfahren beschriebenen Befehlen befinden sich in der link:https://docs.netapp.com/us-en/ontap-cli/["ONTAP-Befehlsreferenz"^].
