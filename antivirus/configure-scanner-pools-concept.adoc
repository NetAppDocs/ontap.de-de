= Informationen zur Konfiguration von ONTAP Vscan Scannerpools
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Ein Scanner Pool definiert die Vscan Server und privilegierten Benutzer, die eine Verbindung zu SVMs herstellen können. Eine Scanner Richtlinie legt fest, ob ein Scannerpool aktiv ist.

[NOTE]
====
Wenn Sie auf einem SMB-Server eine Exportrichtlinie verwenden, müssen Sie jeden Vscan Server zur Exportrichtlinie hinzufügen.

====
