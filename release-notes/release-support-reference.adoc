= ONTAP 9 Release-Unterstützung
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Ab der ONTAP 9.8-Version liefert NetApp ONTAP-Releases zweimal pro Kalenderjahr aus. Auch wenn sich die Pläne ändern können, ist beabsichtigt, neue ONTAP-Releases im zweiten und vierten Quartal jedes Kalenderjahres bereitzustellen. Diese Informationen können zur Planung des Zeitrahmens für ein Upgrade genutzt werden, um von der neuesten ONTAP-Version zu profitieren.
