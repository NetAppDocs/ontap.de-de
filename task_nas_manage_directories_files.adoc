= Verzeichnisse und Dateien mit ONTAP System Manager verwalten
:hardbreaks:
:toclevels: 1
:allow-uri-read: 
:toclevels: 1
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
Die System Manager Volumeanzeige kann erweitert werden, um Verzeichnisse und Dateien anzuzeigen und zu löschen.

Ab ONTAP 9.9.1 werden Verzeichnisse mit niedriger Latenz und asynchroner Verzeichnislöschfunktion gelöscht.

Weitere Informationen zum Anzeigen von Dateisystemen in ONTAP 9.9.1 und späteren Versionen sind unter link:concept_nas_file_system_analytics_overview.html["Übersicht zur Dateisystemanalyse"] zu finden.

.Schritt
. *Speicher > Volumes* auswählen. Ein Volume erweitern, um dessen Inhalt anzuzeigen.
