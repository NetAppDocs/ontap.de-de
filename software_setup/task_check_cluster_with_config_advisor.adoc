= Ihr ONTAP Cluster kann mit Digital Advisor Config Advisor überprüft werden
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Nachdem Sie alle Knoten zu Ihrem neuen Cluster hinzugefügt haben, empfiehlt es sich, Active IQ Config Advisor auszuführen, um Ihre Konfiguration zu validieren und auf häufige Konfigurationsfehler zu überprüfen.

Config Advisor ist eine webbasierte Anwendung, die auf einem Laptop, einer virtuellen Maschine oder einem Server installiert wird und auf Windows-, Linux- und Mac-Plattformen funktioniert.

Config Advisor führt eine Reihe von Befehlen aus, um Ihre Installation zu validieren und den allgemeinen Zustand der Konfiguration, einschließlich der Cluster- und Speicherschalter, zu überprüfen.

. Active IQ Config Advisor herunterladen und installieren.
+
link:https://mysupport.netapp.com/site/tools/tool-eula/activeiq-configadvisor["Active IQ Config Advisor"^]

. Digital Advisor wird gestartet, und bei Aufforderung wird eine Passphrase festgelegt.
. Die Einstellungen überprüfen und auf *Speichern* klicken.
. Auf der Seite *Ziele* *ONTAP Post-Deployment Validation* auswählen.
. Entweder der geführte Modus oder der Expertenmodus kann ausgewählt werden.
+
Im geführten Modus werden angeschlossene Switches automatisch erkannt.

. Geben Sie die Cluster-Anmeldeinformationen ein.
. (Optional) Auf *Formular validieren* klicken.
. Um mit der Datenerfassung zu beginnen, auf *Speichern & Auswerten* klicken.
. Nach Abschluss der Datenerfassung sind unter *Job Monitor > Aktionen* die erfassten Daten über das Symbol *Datenansicht* und die Ergebnisse über das Symbol *Ergebnisse* einsehbar.
. Die vom Config Advisor identifizierten Probleme beheben.
