= Optional kann die ONTAP Management-LIF von IPv4 auf IPv6 konvertiert werden.
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Ab ONTAP 9.13.1 ist es möglich, Management-LIFs auf AFF A800 und FAS 8700 Plattformen während der Ersteinrichtung des Clusters über die ONTAP Befehlszeilenschnittstelle (CLI) IPv6-Adressen zuzuweisen. Für ONTAP Versionen vor 9.13.1 oder für 9.13.1 und höher auf anderen Plattformen müssen zunächst IPv4-Adressen den Management-LIFs zugewiesen werden, und nach Abschluss der Clustereinrichtung erfolgt die Umstellung auf IPv6-Adressen.


NOTE: Wenn der System Manager nach Abschluss der Hardware-Einrichtung mit DHCP und einer automatisch zugewiesenen IP-Adresse sowie mit Windows-Erkennung gestartet wird, kann der System Manager eine IPv6-Managementadresse konfigurieren.

.Schritte
. IPv6 für das Cluster aktivieren:
+
[source, cli]
----
network options ipv6 modify -enable true
----
. Berechtigungsstufe auf „Erweitert“ festlegen:
+
[source, cli]
----
set priv advanced
----
. Die Liste der auf verschiedenen Schnittstellen gelernten RA-Präfixe anzeigen:
+
[source, cli]
----
network ndp prefix show
----
. Eine IPv6-Management-LIF erstellen:
+
Das Format  `prefix::id` im Adressparameter dient dazu, die IPv6-Adresse manuell zu erstellen.

+
[source, cli]
----
network interface create -vserver <svm_name> -lif <LIF> -home-node <home_node> -home-port <home_port> -address <IPv6prefix::id> -netmask-length <netmask_length> -failover-policy <policy> -service-policy <service_policy> -auto-revert true
----
. Es sollte überprüft werden, ob die LIF erstellt wurde:
+
[source, cli]
----
network interface show
----
. Es sollte überprüft werden, ob die konfigurierte IP-Adresse erreichbar ist:
+
[source, cli]
----
network ping6
----
. Der IPv4 LIF wird als administrativ deaktiviert markiert:
+
[source, cli]
----
network interface modify -vserver <svm_name> -lif <lif_name> -status-admin down
----
. Die IPv4-Management-LIF löschen:
+
[source, cli]
----
network interface delete -vserver <svm_name> -lif <lif_name>
----
. Bestätigen, dass die IPv4-Management-LIF gelöscht wurde:
+
[source, cli]
----
network interface show
----
